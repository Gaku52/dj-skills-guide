# DJ Skills Guide

DJテクニックとスキルを体系的にまとめたナレッジベースです。

## 概要

このリポジトリは、DJに必要な技術や知識を明確化し、体系的に学習できるようにすることを目的としています。
将来的にはWebアプリケーションやモバイルアプリとしての展開も視野に入れています。

## コンテンツ

### EQ操作

- [DJ EQ操作 完全ガイド](./docs/DJ_EQ_Complete_Guide.md) - キーの一致・不一致を問わず、あらゆる状況でのEQ管理技術

## 対象読者

- 初心者DJから中級者まで
- EQ操作の基礎から応用まで学びたい方
- キーミックスやハーモニックミキシングを習得したい方
