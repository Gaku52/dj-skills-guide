# DJ Skills Guide

DJ テクニックと楽曲制作（Ableton Live）を体系的に学べる包括的なガイドです。

## 概要

このリポジトリは、**DJ スキル**と **Ableton Live での楽曲制作**の両方を、初心者から上級者まで体系的に学習できるようにすることを目的としています。

**使用ツール:**
- **DJ**: Rekordbox 7.2.7
- **制作**: Ableton Live 12 Suite

**特徴:**
- 「どうやるんだろう？」を解決できる実践的な手順書
- 基礎から応用まで段階的に学べる構成
- DJ と制作を統合した応用テクニック

---

## コンテンツ構成

### [00-fundamentals](./docs/00-fundamentals/) - 音楽の基礎知識

DJ と楽曲制作の両方に共通する、音楽理論とオーディオの基礎。

- [音楽理論基礎](./docs/00-fundamentals/music-theory.md) - スケール、コード、インターバル
- [リズム基礎](./docs/00-fundamentals/rhythm-basics.md) - BPM、拍子、グルーヴ
- [和声基礎](./docs/00-fundamentals/harmony-basics.md) - コード進行、ダイアトニック

- [キーとスケール](./docs/00-fundamentals/key-scales.md) - ハーモニックミキシング入門

### [dj/](./docs/dj/) - DJ スキル（Rekordbox 7.2.7）

#### [02-equipment](./docs/dj/02-equipment/) - 機材ガイド
- コントローラー、CDJ、ミキサー、Rekordbox 設定

#### [03-basic-techniques](./docs/dj/03-basic-techniques/) - 基本テクニック

#### [04-advanced-techniques](./docs/dj/04-advanced-techniques/) - 応用テクニック

#### [05-track-selection](./docs/dj/05-track-selection/) - 選曲とセットリスト
- ジャンル知識、エネルギーカーブ、フロアリーディング

#### [06-performance](./docs/dj/06-performance/) - パフォーマンス
- ステージプレゼンス、セットアップ、ライブストリーミング

#### [07-genres](./docs/dj/07-genres/) - ジャンル別ガイド
- House、Techno、Hip Hop、Dubstep、Drum & Bass など

#### [09-career](./docs/dj/09-career/) - DJ キャリア

### [production/](./docs/production/) - 楽曲制作（Ableton Live 12 Suite）

Ableton Live での楽曲制作を、基礎から完成まで学びます。

#### 01-ableton-basics - Ableton Live 基礎
- インターフェイス、セッション vs アレンジメント、プロジェクト設定

#### 02-audio-midi - Audio & MIDI
- オーディオクリップ、MIDI編集、ワーピング

#### 03-instruments - 音源・楽器
- Wavetable、Operator、Analog、Sampler、Drum Rack

#### 04-sound-design - サウンドデザイン
- シンセシス、サンプリング、レイヤリング、モジュレーション

#### 05-drums-rhythm - ドラム・リズム
- ドラムプログラミング、キック/スネア/ハイハット設計

#### 06-bass-melody - ベース・メロディ
- ベースライン、コード進行、メロディ作成

#### 07-arrangement - アレンジメント
- 曲構造、Intro/Breakdown/Drop、トランジション

#### 08-effects - エフェクト
- EQ、コンプレッサー、リバーブ、ディレイ、フィルター

#### 09-mixing - ミキシング
- ゲインステージング、パンニング、オートメーション

#### 10-mastering - マスタリング
- マスタリングEQ、リミッター、ラウドネス、エクスポート

#### 11-genre-production - ジャンル別制作
- House、Techno、Dubstep、Drum & Bass などの制作手法

#### 12-workflow - ワークフロー
- 制作フロー、クリエイティブプロセス、サンプル管理

### [integration/](./docs/integration/) - DJ × 制作の統合

DJ と楽曲制作を組み合わせた応用テクニック。

- エディット・リミックス

## 対象読者

**DJ 志向:**
- これから DJ を始める初心者
- スキルアップしたい中級者
- プロを目指す上級者

**制作志向:**
- Ableton Live で曲を作りたい初心者
- ジャンル特有のテクニックを学びたい人
- DJ と制作を両方やりたい人

**両方:**
- DJ も制作もマスターしたい人
- オリジナル曲で DJ したい人
- リミックスやマッシュアップを作りたい人

## 学習の進め方

### DJ を始めたい方

```
1. 00-fundamentals で音楽の基礎
2. dj/01-basics で DJ の基礎知識
3. dj/02-equipment で Rekordbox の使い方
4. dj/03-basic-techniques で実践開始
```

### 楽曲制作を始めたい方

```
1. 00-fundamentals で音楽の基礎
2. production/01-ableton-basics で Ableton の基礎
3. production/05-drums-rhythm でドラム作成
4. production/07-arrangement で曲構成
```

### 両方を学びたい方

```
1. 00-fundamentals で共通基礎
2. DJ と制作を並行して学ぶ
3. integration/ で統合テクニック
4. オリジナル曲でDJパフォーマンス
```
