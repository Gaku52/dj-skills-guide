# オーディオ基礎

デジタルオーディオの仕組みと音質の基礎を学びます。DJ・制作の両方で重要な知識です。

## この章で学ぶこと

- デジタルオーディオの仕組み
- サンプルレートとビット深度
- ファイルフォーマットの違い
- 音質の考え方

## なぜオーディオ基礎が重要なのか

**DJの場合:**
- 音質の良いファイルを選べる
- クラブの音響システムに適した設定
- レコーディング時の最適な設定

**制作の場合:**
- プロジェクト設定を正しく行える
- エクスポート時の設定を理解
- マスタリング後の音質を保てる

## 1. デジタルオーディオとは

### アナログからデジタルへ

**アナログ音声:**
- 連続的な波形
- レコード、カセットテープ

**デジタル音声:**
- 数値データに変換
- CD、MP3、WAV

### A/D変換（Analog to Digital）

アナログ信号をデジタルデータに変換するプロセス。

```
1. サンプリング: 一定間隔で音の高さを記録
2. 量子化: 記録した値を数値に変換
3. デジタルデータ: 0と1の数列として保存
```

## 2. サンプルレート（Sample Rate）

### 定義

1秒間に何回音を記録するか。単位は **Hz（ヘルツ）** または **kHz（キロヘルツ）**。

```
44.1 kHz = 1秒間に 44,100回 サンプリング
48 kHz   = 1秒間に 48,000回 サンプリング
96 kHz   = 1秒間に 96,000回 サンプリング
```

### 一般的なサンプルレート

| サンプルレート | 用途 | 特徴 |
|-------------|------|------|
| 44.1 kHz | CD、音楽配信 | 標準、互換性高い |
| 48 kHz | 映像、DJ機器 | CDJ、動画の標準 |
| 88.2 kHz | マスタリング | 44.1の倍数 |
| 96 kHz | ハイレゾ、制作 | 高音質、容量大 |
| 192 kHz | 最高品質 | 非常に大容量 |

### ナイキスト定理

**人間が聞ける周波数:**
- 約 20 Hz ~ 20,000 Hz（20 kHz）

**サンプルレートと再現可能な周波数:**
- サンプルレートの半分まで再現可能
- 44.1 kHz → 22.05 kHz まで再現
- これで人間の可聴域をカバー

### 推奨設定

**DJ（Rekordbox）:**
```
48 kHz
- CDJとの互換性
- クラブ機材の標準
```

**制作（Ableton Live）:**
```
48 kHz（基本）
96 kHz（高音質を求める場合）
- CPU負荷とのバランス
- 最終的に44.1/48にダウンサンプル
```

## 3. ビット深度（Bit Depth）

### 定義

音の大きさをどれだけ細かく記録するか。単位は **bit（ビット）**。

```
16 bit = 65,536段階
24 bit = 16,777,216段階
32 bit float = ほぼ無限の範囲（浮動小数点）
```

### ダイナミックレンジ

ビット深度が高いほど、**小さい音と大きい音の差（ダイナミックレンジ）**を正確に記録できます。

```
16 bit = 約 96 dB
24 bit = 約 144 dB
32 bit float = 約 1,680 dB（実質無限）
```

### 一般的なビット深度

| ビット深度 | 用途 | 特徴 |
|----------|------|------|
| 16 bit | CD、配信 | 十分な音質、容量小 |
| 24 bit | 録音、制作 | プロ標準、余裕あり |
| 32 bit float | DAW内部処理 | クリッピングしない |

### 推奨設定

**DJ（Rekordbox）:**
```
16 bit で十分
- 配信用音源は基本16bit
- クラブで違いは分からない
```

**制作（Ableton Live）:**
```
プロジェクト: 32 bit float（内部処理）
エクスポート: 24 bit（WAV）
           16 bit（MP3、配信）
```

## 4. ファイルフォーマット

### 非圧縮フォーマット

**WAV（Waveform Audio File Format）:**
- Windows標準
- 非圧縮、高音質
- ファイルサイズ大
- DJと制作で推奨

**AIFF（Audio Interchange File Format）:**
- Mac標準
- 非圧縮、高音質
- WAVとほぼ同等
- メタデータの扱いが異なる

### 可逆圧縮フォーマット

**FLAC（Free Lossless Audio Codec）:**
- 圧縮しても音質劣化なし
- WAVの約50-60%のサイズ
- ストリーミングで普及
- Rekordbox対応

**ALAC（Apple Lossless Audio Codec）:**
- Apple版FLAC
- iTunesとの親和性
- 音質はFLACと同等

### 非可逆圧縮フォーマット

**MP3（MPEG Audio Layer-3）:**
- 最も普及
- 非可逆圧縮（音質劣化）
- 320 kbps なら実用レベル
- 128 kbps はクラブで使わない

**AAC（Advanced Audio Coding）:**
- MP3の後継
- 同じビットレートでMP3より高音質
- Apple Musicで使用
- 256 kbps で十分

**OGG（Ogg Vorbis）:**
- オープンソース
- AACと同等の音質
- ゲーム、配信で使用

### ビットレート比較（MP3/AAC）

| ビットレート | 音質 | 用途 |
|------------|------|------|
| 128 kbps | 低音質 | 避けるべき |
| 192 kbps | 中音質 | 練習用 |
| 256 kbps | 高音質 | 配信レベル |
| 320 kbps | 最高音質 | DJ使用可 |

## 5. ファイルサイズの計算

### 計算式

```
ファイルサイズ(MB) = サンプルレート × ビット深度 × チャンネル数 × 秒数 / 8 / 1,000,000
```

### 実例：5分の曲

**WAV 44.1kHz 16bit ステレオ:**
```
44,100 × 16 × 2 × 300 ÷ 8 ÷ 1,000,000 = 52.9 MB
```

**MP3 320kbps:**
```
320 × 300 ÷ 8 ÷ 1,000 = 12 MB
```

**FLAC（約60%圧縮）:**
```
52.9 × 0.6 = 約 32 MB
```

## 6. 音質の選び方

### DJの場合

**推奨フォーマット:**
```
1. WAV / AIFF（最高音質）
2. FLAC（容量節約）
3. MP3 320kbps（互換性重視）
```

**避けるべき:**
```
× MP3 128kbps
× YouTube rip
× 違法ダウンロード
```

### 制作の場合

**プロジェクト内:**
```
- サンプル: WAV 48kHz 24bit
- 録音: 48kHz 24bit 以上
- 内部処理: 32bit float
```

**エクスポート:**
```
- マスター: WAV 48kHz 24bit
- 配信用: WAV 44.1kHz 16bit
- MP3: 320kbps CBR
```

## 7. Rekordbox の設定

### 音声出力設定

```
1. 環境設定 > オーディオ
2. オーディオデバイス: 使用するインターフェイス
3. サンプルレート: 48000 Hz
4. バッファサイズ: 512 samples（レイテンシと安定性のバランス）
```

### インポート設定

```
1. 環境設定 > 詳細
2. ファイル解析時の品質: 高
3. 自動ゲイン: オフ（手動で調整推奨）
```

---

## 8. Ableton Live の設定

### プロジェクト設定

```
1. Preferences > Audio
2. Sample Rate: 48000 Hz
3. Bit Depth: 32 bit（内部処理）
4. Buffer Size: 512 samples（制作時）
             128 samples（パフォーマンス時）
```

### エクスポート設定

```
1. File > Export Audio/Video
2. Sample Rate: 48000 Hz（または 44100 Hz）
3. Bit Depth: 24 bit（WAV）
4. Dither: Triangular（ビット深度を下げる場合）
5. Normalize: オフ（マスタリング済みの場合）
```

## 9. 実践：音質を比較する

### 手順

1. **同じ曲の異なるフォーマットを用意**
   - WAV 16bit
   - MP3 320kbps
   - MP3 128kbps

2. **Rekordboxで波形を比較**
   - 高域の情報量を確認
   - 波形の密度を見る

3. **A/Bテスト**
   - ヘッドフォンで聴き比べ
   - クラブスピーカーで聴き比べ

4. **違いを確認**
   - 高音のクリアさ
   - ベースの圧力
   - 全体の空間感

## 10. 練習方法

### 初級

1. **ファイル形式を確認**: 自分の音源ライブラリを整理
2. **サンプルレートを統一**: 全て48kHzに
3. **低品質ファイルを除外**: 128kbps以下を削除

### 中級

1. **音質比較**: WAV vs MP3 320kbps を聴き比べ
2. **Rekordbox設定最適化**: バッファサイズを調整
3. **FLAC導入**: 容量節約しながら高音質

### 上級

1. **マスタリング音源の作成**: 24bit WAVでエクスポート
2. **ディザリング理解**: ビット深度変換時の設定
3. **高解像度制作**: 96kHzでプロジェクトを試す

## まとめ

- **サンプルレート**: 48kHz が DJ・制作の標準
- **ビット深度**: 制作は24bit、配信は16bit
- **フォーマット**: WAV/FLAC推奨、MP3は320kbps以上
- **設定**: Rekordbox/Abletonで適切に設定
