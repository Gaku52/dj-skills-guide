# 周波数とスペクトラム

音の周波数帯域を理解し、EQ操作やミキシングの基礎となる知識を学びます。

## この章で学ぶこと

- 周波数帯域の理解
- Low / Mid / High の特性
- 楽器ごとの周波数帯域
- EQの基礎概念

## なぜ周波数が重要なのか

**DJの場合:**
- EQでスムーズなトランジション
- 周波数のかぶりを避ける
- クラブの音響に合わせた調整

**制作の場合:**
- 各楽器の住み分け
- ミックスダウンの基礎
- マスタリングの土台

---

## 1. 周波数とは

### 定義

音の高さを表す物理量。単位は **Hz（ヘルツ）**。

```
低い音 = 低い周波数（例: 50 Hz のキック）
高い音 = 高い周波数（例: 10,000 Hz のシンバル）
```

### 人間の可聴域

```
約 20 Hz ~ 20,000 Hz（20 kHz）
```

**年齢による変化:**
- 若い人: ~20 kHz
- 30代: ~16 kHz
- 50代: ~12 kHz

---

## 2. 周波数帯域の分類

音楽制作とDJでは、周波数を以下のように分類します。

### 詳細な分類

| 帯域名 | 周波数範囲 | 特徴 | 主な楽器 |
|-------|----------|------|---------|
| **Sub Bass** | 20-60 Hz | 感じる音、物理的な圧力 | サブベース、キックの最低域 |
| **Bass** | 60-250 Hz | 楽曲の土台、パワー | キック、ベース |
| **Low Mid** | 250-500 Hz | 温かみ、厚み | ベース上部、ギター、男性ボーカル |
| **Mid** | 500 Hz-2 kHz | 楽器の芯、明瞭さ | ボーカル、ギター、スネア |
| **High Mid** | 2-4 kHz | 存在感、アタック | ボーカル子音、スネアアタック |
| **Presence** | 4-6 kHz | 明瞭度、前に出る | ボーカル、リードシンセ |
| **Highs** | 6-12 kHz | 明るさ、エアー感 | ハイハット、シンバル |
| **Air** | 12-20 kHz | 空間、キラキラ感 | シンバル、リバーブ |

### 3分割（DJ向け）

DJ機器のEQは通常3バンド。

```
Low  : ~250 Hz 以下
Mid  : 250 Hz ~ 4 kHz
High : 4 kHz 以上
```

---

## 3. 各帯域の特性と役割

### Sub Bass（20-60 Hz）

**特性:**
- 耳で「聴く」より体で「感じる」
- クラブの大型スピーカーで効果的
- ヘッドフォンでは聴こえにくい

**注意点:**
- 過剰だと音がこもる
- モノラルが基本（ステレオにしない）
- 重なると濁る

**DJでの使い方:**
- ローカットで除去または調整
- 2曲のサブベースは重ねない

### Bass（60-250 Hz）

**特性:**
- 楽曲のパワーと土台
- キックとベースの主要帯域
- 最も重要な周波数帯

**注意点:**
- 重なると最も濁りやすい
- キックとベースの住み分けが重要
- モノラル推奨

**DJでの使い方:**
- トランジション時は片方をカット
- Low EQで調整
- バスブースト注意

### Low Mid（250-500 Hz）

**特性:**
- 温かみと厚み
- ベースの上部、ギターの中心
- 過剰だと「箱鳴り」

**注意点:**
- 最もこもりやすい帯域
- 「Mud（泥）」と呼ばれる濁り
- 引き算EQで整理

**DJでの使い方:**
- Mid EQでやや下げ気味
- クラブの音響でブーストされやすい

### Mid（500 Hz - 2 kHz）

**特性:**
- ボーカルの中心帯域
- 楽器の「芯」
- 人間の耳が最も敏感

**注意点:**
- 過剰だと耳が痛い
- 削りすぎると薄くなる
- ボーカルの明瞭度に直結

**DJでの使い方:**
- ボーカルトラックはMidを残す
- インストは状況に応じて調整

### High Mid（2-4 kHz）

**特性:**
- 楽器の存在感
- アタック音の核心
- スネアのパンチ

**注意点:**
- 過剰だと耳障り
- 削りすぎると抜けない
- 最もデリケートな帯域

**DJでの使い方:**
- High EQの下部
- パーカッションの調整

### Presence（4-6 kHz）

**特性:**
- ボーカルの明瞭度
- リードシンセの前面性
- 「前に出る」感覚

**注意点:**
- 過剰だと攻撃的
- 削ると奥に引っ込む
- 耳の疲労に注意

**DJでの使い方:**
- High EQで調整
- ボーカル曲は慎重に

### Highs（6-12 kHz）

**特性:**
- ハイハット、シンバルの中心
- 明るさとキラキラ感
- トランジションで重要

**注意点:**
- 過剰だと耳が痛い
- 削りすぎると暗い
- シビランス（サ行のきつさ）

**DJでの使い方:**
- High EQで頻繁に調整
- 2曲のハイハットが重なる時カット

### Air（12-20 kHz）

**特性:**
- 空間感、エアー感
- リバーブ、アンビエンス
- 微細な倍音

**注意点:**
- 聴こえにくい人も多い
- クラブでは減衰しやすい
- 過剰でもわかりにくい

**DJでの使い方:**
- ほとんど調整不要
- High EQブーストで少し影響

## 4. 楽器別の周波数マップ

### キック（Kick Drum）

```
Sub (40-60 Hz)    : 圧力、物理的パワー
Body (60-100 Hz)  : 太さ、パンチ
Attack (2-5 kHz)  : ビーターのアタック
Click (5-8 kHz)   : クリック音
```

### ベースライン

```
Sub (40-80 Hz)    : 最低音
Fundamental (80-200 Hz) : 音程の核
Harmonics (200-500 Hz)  : 倍音、厚み
Presence (1-3 kHz)      : アタック
```

### スネア

```
Body (150-250 Hz) : 太さ
Tone (400-800 Hz) : スネアの音色
Crack (2-5 kHz)   : アタック、パンチ
Snap (6-10 kHz)   : スナッピー
```

### ハイハット

```
Body (200-500 Hz) : 厚み（少ない）
Tone (3-8 kHz)    : ハイハットの核
Sizzle (8-15 kHz) : シャリシャリ感
Air (15-20 kHz)   : 空気感
```

### ボーカル

```
Warmth (100-250 Hz)  : 温かみ（男性）
Body (250-500 Hz)    : 厚み
Clarity (1-3 kHz)    : 明瞭度
Presence (4-6 kHz)   : 前面性
Sparkle (8-12 kHz)   : 煌めき
```

### シンセサイザー

```
Sub (30-60 Hz)       : サブオシレーター
Bass (60-250 Hz)     : ベース音
Mids (250 Hz-4 kHz)  : 音色の中心
Highs (4-12 kHz)     : 明るさ
Air (12-20 kHz)      : 倍音
```

## 5. EQの基本操作

### EQの種類

**DJ用EQ（3バンド）:**
```
Low  : シェルビング（~250 Hz）
Mid  : ピーキング（250 Hz ~ 4 kHz）
High : シェルビング（4 kHz~）
```

**制作用EQ（パラメトリック）:**
```
- 周波数を自由に選択
- Qで帯域幅を調整
- Gain で増減
```

### EQ操作の基本原則

**1. 引き算が基本**
```
× ブースト（上げる）→ 濁りやすい
○ カット（下げる）→ すっきり
```

**2. 少しずつ調整**
```
× 一気に大きく変える
○ 1-3 dB ずつ調整
```

**3. ソロで聴かない**
```
× ソロで完璧を求める
○ ミックス全体で判断
```

## 6. DJでの実践的なEQ使用法

### トランジション時のEQ操作

**Step 1: 低域の管理**
```
1. 次の曲をキューで確認
2. Low EQを完全カット（-∞）
3. フェーダーを上げる
4. 徐々にLow EQを戻す
5. 前の曲のLowを下げる
```

**Step 2: 高域の調整**
```
1. ハイハットの重なりを確認
2. High EQで調整（両方の曲）
3. シンバルのクラッシュに注意
```

**Step 3: 中域の整理**
```
1. ボーカルやメロディの重なり確認
2. Mid EQで住み分け
3. どちらかを主役に
```

### ジャンル別のEQ設定

**House:**
```
Low  : しっかり残す
Mid  : やや控えめ
High : 明るく
```

**Techno:**
```
Low  : 圧力重視
Mid  : 削ってダーク
High : 控えめ
```

**Hip Hop:**
```
Low  : 非常に重要
Mid  : ボーカル重視
High : 控えめ
```

## 7. Rekordboxでの周波数確認

### スペクトラムアナライザー

```
1. ブラウザで曲を選択
2. 波形表示を拡大
3. スペクトル表示をオン
4. 周波数の分布を視覚的に確認
```

### EQプリセット活用

```
1. Preferences > Effect
2. EQカーブを保存
3. よく使う設定を登録
```

---

## 8. Ableton Liveでの周波数確認

### Spectrum デバイス

```
1. トラックの最後にSpectrumを配置
2. 再生して周波数分布を確認
3. 他のトラックと比較
```

### EQ Eight

```
1. オーディオエフェクト > EQ Eight
2. 視覚的に周波数を確認
3. 各バンドで調整
   - Low Cut: 不要な低域カット
   - Notch: 特定の周波数を削除
   - Bell: 狭い範囲を調整
   - High Shelf: 高域全体を調整
```

## 9. 練習方法

### 初級

1. **周波数帯域を聴き分ける**: ソロで各帯域を消して聴く
2. **楽器の帯域を確認**: キック、スネア、ハイハットを個別に
3. **3バンドEQで遊ぶ**: Rekordboxで各バンドの影響を確認

### 中級

1. **トランジションEQ**: 2曲のミックスでLowを切り替え
2. **周波数マップ作成**: 自分の好きな曲の帯域を図解
3. **Abletonでスペクトラム確認**: 各楽器の周波数を視覚化

### 上級

1. **パラメトリックEQ使用**: 細かい周波数調整
2. **ジャンル別の特徴理解**: House vs Techno の違い
3. **マスキング対策**: 楽器同士の住み分け
