# 和声基礎

コードとコード進行の基礎を学び、DJ と楽曲制作の両方で活用できる知識を身につけます。

## この章で学ぶこと

- コードとは何か
- コード進行の基本
- ダイアトニックコード
- テンションとカラー
- 実践的な使い方

## なぜ和声が重要なのか

**DJの場合:**
- ハーモニックミキシングでスムーズなトランジション
- キーの相性を理解して選曲
- メロディがぶつからない組み合わせを選べる

**制作の場合:**
- 感情を表現するコード進行を作れる
- ジャンルごとの定番進行を理解
- リハーモナイズでオリジナリティを出せる

---

## 1. コードとは

### 定義

**コード（和音）** = 3つ以上の音を同時に鳴らしたもの

```
C メジャーコード = C + E + G
（ドとミとソを同時に鳴らす）
```

### トライアド（3和音）

最も基本的なコードです。

**メジャーコード（Major）:**
```
構造: ルート + メジャー3度 + 完全5度
例  : C + E + G
響き: 明るい、安定
```

**マイナーコード（Minor）:**
```
構造: ルート + マイナー3度 + 完全5度
例  : C + Eb + G
響き: 暗い、哀愁
```

**ディミニッシュコード（Diminished）:**
```
構造: ルート + マイナー3度 + 減5度
例  : C + Eb + Gb
響き: 不安定、緊張
```

**オーギュメントコード（Augmented）:**
```
構造: ルート + メジャー3度 + 増5度
例  : C + E + G#
響き: 浮遊感、不思議
```

## 2. ダイアトニックコード

### 定義

スケール上の各音をルートにして作られるコードの集合。

### C Major のダイアトニックコード

```
I   : C    (メジャー)  - トニック（安定）
ii  : Dm   (マイナー)  - サブドミナント
iii : Em   (マイナー)  - メディアント
IV  : F    (メジャー)  - サブドミナント
V   : G    (メジャー)  - ドミナント（緊張）
vi  : Am   (マイナー)  - トニック代理
vii°: Bdim (ディミニッシュ) - ドミナント代理
```

### 機能（Function）

コードは以下の3つの機能を持ちます：

**1. トニック（T）- 安定**
- I, vi
- 曲の始まりと終わりに使う
- 「家に帰ってきた」感覚

**2. サブドミナント（SD）- 準備**
- IV, ii
- ドミナントへ向かう準備
- 「出発する」感覚

**3. ドミナント（D）- 緊張**
- V, vii°
- トニックへ戻りたくなる
- 「帰りたい」感覚

## 3. コード進行の基本

### 王道進行

**I - V - vi - IV（1-5-6-4）**
```
C - G - Am - F
```
- ポップス、EDMで最も使われる
- 安定感がありながら感情的
- 例: Let It Be, Don't Stop Believin'

### カノン進行

**I - V - vi - iii - IV - I - IV - V**
```
C - G - Am - Em - F - C - F - G
```
- クラシックから現代まで
- パッヘルベルのカノンが起源
- 例: Canon in D

### 循環コード（ジャズ）

**I - vi - ii - V**
```
C - Am - Dm - G
```
- ジャズの基本進行
- スムーズな流れ
- 何度でも繰り返せる

### ツーファイブワン（ii-V-I）

**ii - V - I**
```
Dm - G - C
```
- ジャズで最重要
- 完璧な解決感
- アレンジで頻繁に使われる

## 4. セブンスコード

### メジャーセブンス（Maj7）

```
Cmaj7 = C + E + G + B
響き: 洗練された、ジャジー
用途: House、R&B、Jazz
```

### ドミナントセブンス（7）

```
C7 = C + E + G + Bb
響き: ブルージー、緊張感
用途: Blues、Funk、Soul
```

### マイナーセブンス（m7）

```
Cm7 = C + Eb + G + Bb
響き: 落ち着いた、都会的
用途: Jazz、Neo Soul、Chill
```

### ハーフディミニッシュ（m7♭5）

```
Cm7♭5 = C + Eb + Gb + Bb
響き: 不安定、ジャジー
用途: Jazz、ii-V-I の ii として
```

## 5. テンションノート

### テンションとは

基本的な3和音や4和音に、さらに音を加えて色付けすること。

**9th（ナインス）:**
```
Cadd9 = C + E + G + D
響き: 明るく広がりのある
```

**11th（イレブンス）:**
```
C11 = C + E + G + Bb + D + F
響き: 浮遊感、アンビエント
```

**13th（サーティーンス）:**
```
C13 = C + E + G + Bb + D + F + A
響き: ジャジー、複雑
```

### 使いどころ

- **House**: maj7、add9 が定番
- **Techno**: sus4、テンション少なめ
- **Jazz/Soul**: 9th、11th、13th を多用
- **Trance**: sus2、sus4 で浮遊感

## 6. コード進行の感情

### 明るい進行

**I - IV - V - I**
```
C - F - G - C
```
- 古典的、元気
- ロック、ポップス

**I - V - vi - IV**
```
C - G - Am - F
```
- 現代的、前向き
- EDM、ポップス

### 暗い進行

**i - VII - VI - VII**
```
Am - G - F - G
```
- 力強い、ダーク
- Techno、Dubstep

**i - iv - VII - III**
```
Am - Dm - G - C
```
- 哀愁、エモい
- アニソン、Trance

### 浮遊感のある進行

**Imaj7 - IVmaj7 - IIIm7 - VIm7**
```
Cmaj7 - Fmaj7 - Em7 - Am7
```
- 洗練された、浮遊
- Jazz、Neo Soul、Chill

## 7. 転調（Modulation）

### 転調とは

曲の途中でキーが変わること。

### 平行調転調

**C Major → A Minor**
```
同じ音を使うキーへの転調
自然で滑らか
```

### 近親調転調

**C Major → G Major（5度上）**
```
共通する音が多い
エネルギーが上がる
```

### 半音転調

**C Major → C# Major（半音上）**
```
ド派手な盛り上がり
J-POPのサビで定番
```

### DJでの転調活用

**スムーズな転調:**
- Camelot Wheel で隣接するキー
- +1 / -1 の移動

**エネルギーを上げる転調:**
- 半音上、全音上へ
- ビルドアップで使用

## 8. 実践：コードを聴き取る

### 手順

1. **ルート音を探す**: ベースラインを聴く
2. **メジャーかマイナーか判断**: 明るいか暗いか
3. **コード進行を確認**: I-IV-V か I-V-vi-IV か
4. **Abletonで確認**: MIDIで音を鳴らして確認

### Abletonでコード進行を作る

```
1. MIDIトラックを作成
2. Wavetable や Analog を読み込み
3. ピアノロールでコードを入力
   - C: C, E, G
   - F: F, A, C
   - G: G, B, D
4. ボイシングを調整（転回形）
5. アルペジエーターで動きをつける
```

---

## 9. 練習方法

### 初級

1. **メジャーとマイナーの聴き分け**: 10曲でコードの明暗を判断
2. **ダイアトニックコードを覚える**: C Major の I-vii° を暗記
3. **簡単な進行を弾く**: I-IV-V-I をピアノで

### 中級

1. **王道進行を耳コピ**: 好きな曲のコード進行を採譜
2. **セブンスコードを使う**: maj7、m7 を曲に追加
3. **ハーモニックミキシング**: 同じコード進行の曲を繋ぐ

### 上級

1. **リハーモナイズ**: 既存の進行を別のコードで置き換え
2. **モーダルインターチェンジ**: 他のモードからコードを借りる
3. **複雑なテンション**: 9th、11th、13th を使いこなす

---

## まとめ

- **コード**: 3つ以上の音の重なり
- **ダイアトニック**: スケールから作られる7つのコード
- **進行**: I-V-vi-IV など定番パターンがある
- **テンション**: 9th、11th、13th で色付け
