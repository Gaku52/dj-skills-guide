# キーとスケール

音楽のキー（調）とスケール、そしてDJに必須のハーモニックミキシングの基礎を学びます。

## この章で学ぶこと

- キーの概念と仕組み
- スケールの種類
- Circle of Fifths（五度圏）
- ハーモニックミキシングの基礎
- Camelot Keyシステム

## なぜキーが重要なのか

**DJの場合:**
- キーの合う曲を繋いでハーモニックミキシング
- 不協和音を避けてスムーズなトランジション
- エネルギーをコントロールした選曲

**制作の場合:**
- メロディとコードを自由に作れる
- サンプルやループの組み合わせ
- リミックス・マッシュアップの制作

---

## 1. キー（調）とは

### 定義

**キー** = 楽曲の中心となる音とスケールの組み合わせ

```
C Major = Cの音を中心としたメジャースケール
A Minor = Aの音を中心としたマイナースケール
```

### メジャーキーとマイナーキー

**メジャーキー（長調）:**
- 明るく、前向き
- 例: C Major, G Major, F Major

**マイナーキー（短調）:**
- 暗く、感情的
- 例: A Minor, E Minor, D Minor

---

## 2. Circle of Fifths（五度圏）

### 五度圏とは

キーの関係性を円形に表した図。DJとハーモニックミキシングの基礎となる重要な概念です。

```
        C
    F       G
  Bb          D
 Eb            A
  Ab          E
    Db      B
       F#/Gb
```

### 完全五度の関係

時計回りに進むと、**完全五度（7半音）上**のキーになります。

```
C → G → D → A → E → B → F#/Gb
```

反時計回りは**完全五度下（完全四度上）**。

```
C → F → Bb → Eb → Ab → Db → Gb/F#
```

### 相対調（Relative Key）

円の内側と外側は**相対調**の関係。

```
外側（メジャー） - 内側（マイナー）
C Major         - A Minor
G Major         - E Minor
D Major         - B Minor
```

同じ音を使うが、中心音が違う。

---

## 3. キーの相性（ハーモニックミキシング）

### 相性の良いキーの組み合わせ

**1. 同じキー**
```
C Major → C Major
最も安全、確実
```

**2. 相対調（Relative Key）**
```
C Major ⇔ A Minor
同じ音を使うので非常に相性が良い
```

**3. 完全五度上下**
```
C Major → G Major（上）
C Major → F Major（下）
共通する音が多く、スムーズ
```

**4. 平行調（Parallel Key）**
```
C Major ⇔ C Minor
同じルート、雰囲気が変わる
```

### 相性の悪いキーの組み合わせ

**半音の関係:**
```
C Major × C# Major
非常に不協和、避けるべき
```

**増四度（トライトーン）:**
```
C Major × F# Major
緊張感が強すぎる
```

## 4. Camelot Keyシステム

### Camelot Wheelとは

キーを数字とアルファベットで表す、DJ向けの表記システム。

### 表記方法

```
数字: 1-12（Circle of Fifthsの位置）
文字: A（マイナー）、B（メジャー）
```

### Camelot Key 対応表

| 音楽キー | Camelot | Open Key |
|---------|---------|----------|
| C Major | 8B | 1d |
| A Minor | 8A | 1m |
| G Major | 9B | 2d |
| E Minor | 9A | 2m |
| D Major | 10B | 3d |
| B Minor | 10A | 3m |
| A Major | 11B | 4d |
| F# Minor | 11A | 4m |
| E Major | 12B | 5d |
| C# Minor | 12A | 5m |
| B Major | 1B | 6d |
| G# Minor | 1A | 6m |
| F# Major | 2B | 7d |
| Eb Minor | 2A | 7m |
| Db Major | 3B | 8d |
| Bb Minor | 3A | 8m |
| Ab Major | 4B | 9d |
| F Minor | 4A | 9m |
| Eb Major | 5B | 10d |
| C Minor | 5A | 10m |
| Bb Major | 6B | 11d |
| G Minor | 6A | 11m |
| F Major | 7B | 12d |
| D Minor | 7A | 12m |

### Camelot でのミキシングルール

**完璧にマッチ（Perfect Match）:**
```
同じキー: 8B → 8B
```

**エネルギーアップ（Energy Boost）:**
```
+1: 8B → 9B（完全五度上）
```

**エネルギーダウン（Energy Drop）:**
```
-1: 8B → 7B（完全五度下）
```

**ムード転換（Mood Change）:**
```
A ⇔ B: 8B ⇔ 8A（メジャー⇔マイナー）
```

**安全な組み合わせ:**
```
- 同じ番号（8B → 8B、8B → 8A）
- ±1の番号（8B → 7B、8B → 9B）
- 同じ文字で±1（8A → 9A、8A → 7A）
```

## 5. Rekordboxでのキー管理

### キー解析

```
1. 曲をライブラリに追加
2. 右クリック > 「解析」
3. BPMとキーが自動検出
4. ブラウザで「Key」列を表示
```

### 表記の切り替え

```
1. 環境設定 > 表示
2. キー表記:
   - 音楽キー（C、Am など）
   - Camelot（8B、8A など）
   - Open Key（1d、1m など）
```

### キーでソート・フィルタ

```
1. ブラウザで「Key」列をクリック
2. 同じキーの曲をグループ化
3. スマートプレイリスト作成
   条件: Key is 8B
```

### キーマッチング機能

```
1. 曲をロード
2. ブラウザ画面「Related Tracks」
3. キーの合う曲が自動表示
4. BPMも考慮した推奨曲
```

## 6. Ableton Liveでのキー確認

### キー解析（Live 12）

```
1. 曲をセッションビューにドロップ
2. クリップビューでワープ
3. キー情報が表示される（一部のファイル）
```

### 手動でキー確認

```
1. MIDIトラックを作成
2. Scale MIDIエフェクトを追加
3. いくつかのスケールを試す
4. 曲に合うスケールを見つける
```

### Tunerデバイス

```
1. オーディオトラックにTunerを追加
2. ベースラインやメロディをソロ再生
3. ルート音を確認
4. スケールタイプを判断（メジャー/マイナー）
```

### MIDI Pitchで転調

```
1. オーディオをMIDI変換（Convert Harmony/Melody to MIDI）
2. MIDI Pitchで半音単位で転調
3. 別のキーの曲と組み合わせる
```

## 7. ハーモニックミキシングの実践

### ステップ1: キーを確認

```
1. 両方の曲のキーをRekordboxで確認
2. Camelot表記で相性をチェック
3. 相性の良い組み合わせか判断
```

### ステップ2: ミックスポイントを決める

```
1. 前の曲のアウトロを確認
2. 次の曲のイントロを確認
3. メロディやコードが少ない部分を選ぶ
```

### ステップ3: EQで調整

```
1. 次の曲のLowをカット
2. フェーダーを上げる
3. Midも調整してメロディの衝突を避ける
4. 徐々に前の曲をフェードアウト
```

### ステップ4: 耳で確認

```
1. ヘッドフォンで両方を聴く
2. 不協和音がないか確認
3. 違和感があればEQやタイミング調整
```

## 8. 転調テクニック

### DJ中に転調する方法

**1. ドラムブレイク利用**
```
メロディがないドラムセクションで切り替え
キーの衝突を避ける
```

**2. エフェクト活用**
```
Echoやリバーブでメロディをぼかす
不協和を目立たなくする
```

**3. ±1セミトーン調整**
```
Rekordboxのキー調整機能
半音ずつ上下して合わせる
音質劣化に注意
```

**4. キーロック（Master Tempo）**
```
BPMを変えてもキーを保持
テンポ調整の自由度向上
```

## 9. ジャンル別のキー傾向

### House

```
よく使われるキー: A Minor, F Major, C Major
明るいメジャーキーが多い
ハーモニックミキシング重視
```

### Techno

```
よく使われるキー: A Minor, D Minor, E Minor
マイナーキーが主流
ダークで力強い
```

### Trance

```
よく使われるキー: C Major, D Major, A Minor
転調を多用
エネルギー変化が激しい
```

### Hip Hop

```
よく使われるキー: 様々（サンプリングベース）
キーよりグルーヴ重視
ハーモニックミキシングは二の次
```

## 10. 練習方法

### 初級

1. **五度圏を覚える**: 紙に書いて暗記
2. **Camelot表を理解**: よく使うキーを覚える
3. **同じキーの曲を集める**: プレイリスト作成

### 中級

1. **ハーモニックミックス実践**: 8B → 9B など
2. **相対調を使う**: 8B → 8A の切り替え
3. **エネルギーカーブを作る**: キーでストーリー

### 上級

1. **複雑な転調**: 8B → 8A → 9A → 9B
2. **キー調整機能活用**: ±1で合わせる
3. **オープンキー**: Camelot外の組み合わせも試す

## 11. トラブルシューティング

### Q: Rekordboxのキー解析が間違っている

```
A: 手動で修正
1. 波形画面で確認
2. ブラウザでキーを手動入力
3. Abletonで確認
```

### Q: キーは合っているのに不協和に聴こえる

```
A: 以下を確認
1. メロディが複雑すぎる
2. テンションノートが衝突
3. ベースラインが合わない
→ ドラムブレイクでミックス
```

### Q: どのキーの組み合わせから始めるべき？

```
A: 安全な組み合わせ
1. 同じキー（8B → 8B）
2. メジャー⇔マイナー（8B → 8A）
3. ±1（8B → 9B）
```
