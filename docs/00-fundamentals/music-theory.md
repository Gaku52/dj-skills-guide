# 音楽理論基礎

DJ と楽曲制作に必要な音楽理論の基礎を、実践的な視点で学びます。

## この章で学ぶこと

- 音階（スケール）とは何か
- 音程（インターバル）の理解
- メジャーとマイナーの違い
- 実際のDJ・制作での使い方

## なぜDJ・制作に音楽理論が必要なのか

**DJの場合:**
- キーの合う曲を選んでハーモニックミキシングができる
- 不協和音を避け、心地よいトランジションを作れる
- フロアに合った曲選びができる

**制作の場合:**
- メロディやコードを自由に作れる
- 音の響きを理解してミックスダウンできる
- ジャンルの特性を理解できる

---

## 1. 音とは何か

### 音の高さ（ピッチ）

音の高さは**周波数（Hz）**で決まります。

- **高い音**: 周波数が高い（例: 女性の声、シンバル）
- **低い音**: 周波数が低い（例: 男性の声、キック）

### 音名

西洋音楽では、音に以下の名前がついています：

```
C - D - E - F - G - A - B (- C)
ド - レ - ミ - ファ - ソ - ラ - シ (- ド)
```

これが**オクターブ**を形成します。

### 半音と全音

- **半音**: 隣り合う鍵盤1つ分の距離（例: C → C#）
- **全音**: 半音2つ分の距離（例: C → D）

**ピアノの鍵盤で確認:**
```
C  C# D  D# E  F  F# G  G# A  A# B  C
│半│全│半│全│全│半│全│半│全│半│全│全│
```

## 2. スケール（音階）

### メジャースケール（長音階）

明るく、ポジティブな響きを持つスケールです。

**構造（全音と半音の並び）:**
```
全-全-半-全-全-全-半
```

**C Major Scale（ハ長調）の例:**
```
C - D - E - F - G - A - B - C
全  全  半  全  全  全  半
```

**実際の使われ方:**
- House、Techno、Tranceなどの明るい曲
- アッパーなメロディ
- ポップス全般

### マイナースケール（短音階）

暗く、感情的な響きを持つスケールです。

**ナチュラルマイナースケールの構造:**
```
全-半-全-全-半-全-全
```

**A Minor Scale（イ短調）の例:**
```
A - B - C - D - E - F - G - A
全  半  全  全  半  全  全
```

**実際の使われ方:**
- Techno、Dubstep、Drum & Bassなどのダークな曲
- エモーショナルなメロディ
- ドラマチックな展開

### メジャーとマイナーの関係

**同じ音を共有する関係（相対調）:**
- C Major と A Minor は同じ音を使う
- キーが「1つ」ズレた関係（Cから数えてAは6番目）

これは**ハーモニックミキシング**で重要な概念です。

## 3. インターバル（音程）

2つの音の距離を**インターバル**と呼びます。

### 主要なインターバル

| インターバル名 | 半音の数 | C からの例 | 響き |
|--------------|---------|-----------|------|
| ユニゾン (Unison) | 0 | C - C | 同じ音 |
| マイナー2度 (m2) | 1 | C - C# | 不協和 |
| メジャー2度 (M2) | 2 | C - D | 安定 |
| マイナー3度 (m3) | 3 | C - Eb | 暗い |
| メジャー3度 (M3) | 4 | C - E | 明るい |
| 完全4度 (P4) | 5 | C - F | 安定 |
| 完全5度 (P5) | 7 | C - G | 非常に安定 |
| オクターブ (Oct) | 12 | C - C | 同じ音（高さ違い）|

### DJでの使い方

**協和音程（良く響く）:**
- 完全4度、完全5度、オクターブ
- これらのキー関係の曲は混ぜやすい

**不協和音程（ぶつかる）:**
- マイナー2度、増4度
- これらのキー関係は避けるべき

## 4. コードの基礎

### トライアド（3和音）

3つの音を重ねたものが**コード**です。

**メジャーコード（明るい）:**
```
C メジャー = C + E + G
（ルート + メジャー3度 + 完全5度）
```

**マイナーコード（暗い）:**
```
C マイナー = C + Eb + G
（ルート + マイナー3度 + 完全5度）
```

### ダイアトニックコード

スケール上の各音から作られるコードです。

**C Major のダイアトニックコード:**
```
I    : C (メジャー)
ii   : Dm (マイナー)
iii  : Em (マイナー)
IV   : F (メジャー)
V    : G (メジャー)
vi   : Am (マイナー)
vii° : Bdim (ディミニッシュ)
```

**実践的な使い方:**
- I - IV - V は「王道進行」（安定感がある）
- I - vi - IV - V は「カノン進行」（ポップス定番）
- ii - V - I は「ジャズ進行」（洗練された響き）

## 5. 実践：曲のキーを聴き取る

### 手順

1. **曲を聴く**: 何度も繰り返し聴く
2. **ルート音を探す**: 一番「落ち着く」音を見つける
3. **明るいか暗いか**: メジャーかマイナーかを判断
4. **Rekordbox/Abletonで確認**: キー解析機能を使う

### Rekordboxでのキー確認

```
1. 曲をロード
2. ブラウザ画面で「Key」列を表示
3. Camelot Key または 音楽キー表記で確認
```

### Abletonでのキー確認

```
1. 曲をセッションビューにドロップ
2. ワープマーカーを設定
3. MIDI トラックでメロディをなぞって確認
```

---

## 6. 練習方法

### 初級

1. **好きな曲のキーを調べる**: Rekordboxで10曲のキーを確認
2. **同じキーの曲を集める**: プレイリストを作成
3. **メジャーとマイナーを聴き比べる**: 違いを耳で覚える

### 中級

1. **ハーモニックミキシングを試す**: キーの合う曲を繋げる
2. **コード進行を聴き取る**: 好きな曲のコードを探す
3. **Abletonで簡単なメロディを作る**: Cメジャースケールで遊ぶ

### 上級

1. **転調を理解する**: 曲中でキーが変わる箇所を見つける
2. **モードを学ぶ**: ドリアン、フリジアンなどのスケール
3. **ジャンル別の理論**: ジャズ、クラシックなど深掘り

---

## まとめ

- **音階**: メジャー（明るい）とマイナー（暗い）
- **インターバル**: 音の距離、協和と不協和
- **コード**: 3つ以上の音の重なり
- **実践**: Rekordbox/Abletonで確認しながら学ぶ
