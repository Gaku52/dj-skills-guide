# リズム基礎

DJ と楽曲制作に必要なリズムの基礎知識を学びます。ビートマッチングやドラム打ち込みの土台となる重要な概念です。

## この章で学ぶこと

- 拍子とテンポの理解
- 小節とフレーズの構造
- リズムパターンの種類
- グルーヴの概念

## なぜリズムが重要なのか

**DJの場合:**
- ビートマッチングの基礎理解
- フレーズの切れ目でミックス
- グルーヴを保ってフロアを盛り上げる

**制作の場合:**
- ドラムパターンを打ち込める
- リズムの「ノリ」を作れる
- ジャンルごとのグルーヴを再現できる

---

## 1. 拍子とは

### 拍（ビート）

音楽の基本単位です。手拍子で「1, 2, 3, 4」と数えるそれぞれが拍です。

```
♩ ♩ ♩ ♩
1  2  3  4
```

### 拍子（Time Signature）

拍のグループ化の仕方を示します。

**4/4拍子（Four on the Floor）:**
```
4   ← 1小節に4拍ある
─
4   ← 4分音符が1拍
```

ほとんどのダンスミュージックは **4/4拍子** です。

**その他の拍子:**
- **3/4拍子**: ワルツ（1-2-3, 1-2-3）
- **6/8拍子**: シャッフルビート（1-2-3-4-5-6）
- **5/4拍子**: 変拍子（Take Five など）

---

## 2. テンポ（BPM）

### BPMとは

**BPM（Beats Per Minute）** = 1分間に何拍あるか

```
BPM 120 = 1分間に120拍
       = 1秒間に2拍
       = 0.5秒ごとに1拍
```

### ジャンル別の典型的なBPM

| ジャンル | 典型的なBPM |
|---------|-----------|
| Hip Hop | 80-100 |
| House | 120-130 |
| Techno | 125-135 |
| Trance | 130-145 |
| Drum & Bass | 170-180 |
| Dubstep | 140 (ハーフタイム 70) |

### テンポの感じ方

**遅いテンポ（80-100 BPM）:**
- ゆったり、重い
- Hip Hop、Lo-Fi、Chill

**中間テンポ（120-130 BPM）:**
- 踊りやすい
- House、Disco、Pop

**速いテンポ（140-180 BPM）:**
- エネルギッシュ、興奮
- Trance、Drum & Bass、Hardcore

## 3. 小節とフレーズ

### 小節（Bar / Measure）

拍子によって区切られたリズムの単位です。

**4/4拍子の1小節:**
```
|1  2  3  4|
```

### フレーズ（Phrase）

複数の小節がまとまった音楽的な単位です。

**典型的なフレーズ構造:**
```
4小節 = 1フレーズ（短い）
8小節 = 1フレーズ（標準）
16小節 = 1セクション
32小節 = 大きなセクション（Intro、Verse など）
```

### 実践例：House楽曲の構造

```
Intro      : 32小節 (8小節×4)
Verse 1    : 16小節 (8小節×2)
Build Up   : 8小節
Drop 1     : 32小節
Break      : 16小節
Build Up 2 : 8小節
Drop 2     : 32小節
Outro      : 32小節
```

### DJでフレーズを意識する理由

**ミックスのタイミング:**
- フレーズの切れ目（8小節、16小節）でミックスすると自然
- 中途半端な場所でミックスすると違和感

**Rekordboxでフレーズを確認:**
```
1. 波形ビューを開く
2. ビートグリッドを表示
3. 8小節ごとにマーカーを付ける
4. Hot Cue で重要な場所を保存
```

## 4. リズムパターンの種類

### キック（Kick / Bass Drum）

**4つ打ち（Four on the Floor）:**
```
|♩  ♩  ♩  ♩|
 1  2  3  4
```
- House、Techno、Tranceの基本
- 全ての拍にキックが入る

**2拍4拍（Backbeat）:**
```
|   ♩     ♩|
    2     4
```
- Hip Hop、R&Bの基本
- 2拍目と4拍目にスネア

**シンコペーション:**
```
|♩  ♩♩   ♩♩|
 1  2& 3 4&
```
- オフビートにキック
- Breakbeat、Drum & Bass

### スネア / クラップ（Snare / Clap）

**基本パターン（House）:**
```
|   ♩     ♩|
    2     4
```
- 2拍目と4拍目にスネア

**ダブル スネア:**
```
|   ♩    ♩♩|
    2    4&
```
- 4拍目の裏にもスネア
- Fill-inでよく使われる

### ハイハット（Hi-Hat）

**8分音符:**
```
|♩♩♩♩♩♩♩♩|
 1&2&3&4&
```
- House、Techno の基本

**16分音符:**
```
|♬♬♬♬♬♬♬♬|
 1e&a2e&a3e&a4e&a
```
- Drum & Bass、Trap
- より細かいグルーヴ

**オープン/クローズ:**
```
Closed: チッ チッ チッ チッ
Open  : ツァー（伸びる音）
```

## 5. グルーヴ（Groove）

### グルーヴとは

リズムの「ノリ」や「揺れ」のこと。機械的な正確さだけではない、人間的な「気持ち良さ」です。

### クオンタイズ（Quantize）

**完全にクオンタイズ（ジャストタイミング）:**
```
|♩   ♩   ♩   ♩  |
 1   2   3   4
正確だが機械的
```

**少しズレている（Human Feel）:**
```
|♩  ♩  ♩  ♩ |
 1  2  3  4
わずかに前後にズレて「ノリ」が生まれる
```

### スウィング（Swing）

8分音符や16分音符を「タッタ、タッタ」とハネさせるリズム。

**ストレート（Swing 0%）:**
```
|♩ ♩ ♩ ♩|
 1 & 2 &
均等
```

**スウィング（Swing 50-70%）:**
```
|♩  ♩♩  ♩|
 1  & 2  &
後ろが少し遅れる = ハネる
```

**ジャンル別のスウィング:**
- **House**: 0-10%（ほぼストレート）
- **Hip Hop**: 50-60%（しっかりハネる）
- **Jazz**: 60-70%（強くハネる）

### Abletonでスウィングを設定

```
1. Groove Pool を開く
2. Swing プリセットを選択（例: MPC 16 Swing）
3. トラックにドラッグ＆ドロップ
4. Groove Amount で強さを調整（0-100%）
```

## 6. ポリリズムとシンコペーション

### ポリリズム

複数の異なるリズムが同時に鳴ること。

**例: 3対4**
```
Track 1: |♩  ♩  ♩  |（3連符）
Track 2: |♩ ♩ ♩ ♩|（4分音符）
```

Technoでパーカッションが複雑に絡むときに使われます。

### シンコペーション

強拍と弱拍を逆転させてアクセントをつける技法。

**通常:**
```
|♩   ♩   ♩   ♩  |
 強  弱  強  弱
```

**シンコペーション:**
```
|  ♩ ♩     ♩ ♩ |
   弱 強   弱 強
```

ダンスミュージックの「グルーヴ」を生む重要な要素です。

## 7. 実践：リズムを聴き取る

### 手順

1. **好きな曲を選ぶ**
2. **BPM を測る**: Rekordboxで自動解析
3. **拍を数える**: 「1, 2, 3, 4」で手拍子
4. **キックのパターンを聴く**: 4つ打ちか、Backbeat か
5. **フレーズを数える**: 8小節ごとに変化を確認

### Rekordboxでリズム解析

```
1. 曲をロード
2. ブラウザで「BPM」列を確認
3. ビートグリッドを表示
4. 正しく拍が検出されているか確認
5. ズレていたら手動で調整
```

### Abletonでリズムを作る

```
1. MIDIトラックを作成
2. Drum Rack を読み込み
3. ピアノロールで打ち込み
   - Kick: 1, 2, 3, 4
   - Snare: 2, 4
   - Hi-Hat: 1&2&3&4&
4. Groove を適用してスウィングを加える
```

## 8. 練習方法

### 初級

1. **BPMを体感する**: 120 BPMで1分間手拍子
2. **フレーズを数える**: 好きな曲で8小節ごとに数える
3. **ドラムパターンを聴き分ける**: キック、スネア、ハイハットを個別に聴く

### 中級

1. **Abletonでリズムパターンを打ち込む**: House、Technoの基本パターン
2. **スウィングを試す**: 0%、50%、70%で聴き比べ
3. **フレーズに合わせてDJミックス**: 8小節、16小節の切れ目で繋ぐ

### 上級

1. **ポリリズムを作る**: 3連符と4分音符を重ねる
2. **ジャンル別グルーヴを再現**: Hip Hop、Drum & Bass、Techno
3. **ライブでリズムを操作**: Abletonのセッションビューでリズムを変化させる

---

## まとめ

- **拍子**: ほとんどのダンスミュージックは4/4拍子
- **BPM**: ジャンルごとに典型的なテンポがある
- **フレーズ**: 8小節、16小節がミックスのタイミング
- **グルーヴ**: スウィングやタイミングのズレで「ノリ」を作る
