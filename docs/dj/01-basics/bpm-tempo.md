**ビートマッチングの基礎:**

## 1. BPMとは

### 定義

**BPM = Beats Per Minute（ビーツ・パー・ミニット）**

```
1分間に何回「拍」があるか
```

### 計算例

**BPM 120 の場合:**
```
1分間に120拍
= 1秒間に2拍（120 ÷ 60）
= 0.5秒ごとに1拍
```

**BPM 140 の場合:**
```
1分間に140拍
= 1秒間に約2.33拍
= 約0.43秒ごとに1拍
```

### 手動で測る方法

```
1. 曲を再生
2. ストップウォッチで15秒測る
3. 15秒間の拍数を数える
4. 拍数 × 4 = BPM

例: 15秒で30拍
    30 × 4 = 120 BPM
```

## 2. テンポの感じ方

### BPM帯域別の感覚

#### 遅いテンポ（60-100 BPM）

```
感覚: ゆったり、重い、落ち着く
用途: Chill、Lo-Fi、Hip Hop
```

**特徴:**
- リラックス
- グルーヴ重視
- ビートの1つ1つをしっかり感じる

**代表ジャンル:**
- Hip Hop: 80-100 BPM
- Trip Hop: 80-95 BPM
- Downtempo: 70-100 BPM
- Lo-Fi Hip Hop: 70-90 BPM

#### 中間テンポ（100-130 BPM）

```
感覚: 歩くペース、踊りやすい、心地よい
用途: House、Disco、Pop
```

**特徴:**
- 最も踊りやすい
- 幅広い層に受け入れられる
- 長時間聴いても疲れにくい

**代表ジャンル:**
- House: 120-130 BPM
- Disco: 110-125 BPM
- Tech House: 125-130 BPM
- Deep House: 118-125 BPM

#### 速いテンポ（130-150 BPM）

```
感覚: エネルギッシュ、興奮、駆け足
用途: Techno、Trance、Dubstep
```

**特徴:**
- 高揚感
- ピークタイムに適している
- 長時間は疲れる可能性

**代表ジャンル:**
- Techno: 125-150 BPM（標準125-135、Industrial 135-150）
- Trance: 130-145 BPM
- Dubstep: 140 BPM（ハーフタイム = 70 BPM感覚）
- Hard Style: 150 BPM

#### 非常に速いテンポ（150-180+ BPM）

```
感覚: 猛スピード、激しい、アグレッシブ
用途: Drum & Bass、Hardcore、Jungle
```

**特徴:**
- 超高速
- 専門的なジャンル
- 高いテクニカルスキル要求

**代表ジャンル:**
- Drum & Bass: 170-180 BPM
- Jungle: 160-170 BPM
- Hardcore: 160-200 BPM
- Footwork: 160 BPM

## 4. Rekordboxでのbpm確認と調整

### BPMの自動解析

```
1. Rekordboxを起動
2. 曲をライブラリに追加（ドラッグ＆ドロップ）
3. 曲を右クリック > 「解析」
4. BPMが自動検出される
```

**精度:**
- ほとんどの曲で正確
- 複雑なリズムは誤認識することも
- 手動で修正可能

### BPM表示列を追加

```
1. ブラウザ画面を開く
2. カラムヘッダーを右クリック
3. 「BPM」にチェック
4. BPM列が表示される
```

### BPMでソート

```
1. 「BPM」列のヘッダーをクリック
2. BPM順に並び替え
3. 近いBPMの曲を見つけやすい
```

### BPMの手動修正

```
1. 曲を選択
2. 情報パネルで「BPM」フィールドを編集
3. 正しいBPMを入力
4. Enter で確定
```

**誤認識の例:**
```
実際のBPM 128 → 64 と検出（半分）
実際のBPM 128 → 256 と検出（倍）
```

**修正方法:**
- 倍 / 半分に手動修正
- または再解析

### ビートグリッドの調整

```
1. 曲をデッキにロード
2. 波形画面を開く
3. ビートグリッドがズレている場合:
   - Shift + グリッド調整ボタン
   - 最初の拍に合わせる
4. BPMを微調整（±0.01単位）
```

## 5. BPMを耳で判断する練習

### レベル1: 大まかな判断

**目標: ±20 BPM の精度**

```
練習方法:
1. 曲を聴く
2. 手拍子でテンポを取る
3. 「遅い・中間・速い」を判断
4. Rekordboxで答え合わせ
```

**基準:**
- 遅い: 100以下
- 中間: 100-140
- 速い: 140以上

### レベル2: ジャンルでの判断

**目標: ±10 BPM の精度**

```
練習方法:
1. ジャンルを聞き分ける
2. そのジャンルの典型的なBPMを想起
   - House なら 125前後
   - Techno なら 130前後
3. Rekordboxで答え合わせ
```

### レベル3: 正確な判断

**目標: ±5 BPM の精度**

```
練習方法:
1. 15秒間拍数を数える
2. 拍数 × 4 = BPM
3. Rekordboxで答え合わせ
4. 繰り返し練習
```

### レベル4: 瞬時の判断

**目標: ±2 BPM の精度**

```
練習方法:
1. 毎日10曲のBPMを当てる
2. 聴いた瞬間に「これは128だ」と判断
3. 体に染み込むまで反復
4. 数ヶ月で身につく
```

## 6. テンポマッチングの基礎

### ピッチフェーダーの使い方

**ピッチフェーダー（テンポフェーダー）:**
```
- 上: BPM 下げる（遅くする）
- 中央: 元のBPM（0%）
- 下: BPM 上げる（速くする）
```

**範囲設定:**
```
Rekordbox設定で選択可能:
- ±6%（細かい調整向け）
- ±10%（標準）
- ±16%（大きな調整向け）
- Wide（±100%）
```

### テンポマッチングの手順

```
1. 曲Aを再生中（例: 128 BPM）
2. 曲BをキューでBPM確認（例: 130 BPM）
3. 曲Bのピッチフェーダーを下げる（130→128）
4. ヘッドフォンで両方聴き比べ
5. ピッタリ合ったら準備完了
```

### Sync（シンク）機能

**自動同期:**
```
Rekordbox の Sync ボタン:
- 自動でBPMを合わせる
- ビートグリッドも同期
- 初心者に便利
```

**使用の是非:**
- 練習時: 使わない（耳を鍛える）
- 本番: 状況次第（複雑なセットでは有効）
- プロの意見: 賛否両論

## 7. BPMの±幅とキーの変化

### ピッチとキーの関係

**Master Tempo OFF の場合:**
```
BPM を上げる → キーが高くなる
BPM を下げる → キーが低くなる

例: C Major の曲
+5% → C# Major に
-5% → B Major に
```

### Master Tempo（キーロック）

**機能:**
```
BPM を変えてもキーを保持
```

**使用場面:**
- BPMが大きく違う曲を繋ぐ
- キーを合わせたいとき
- ハーモニックミキシング重視

**音質:**
- わずかに劣化する可能性
- 最新のRekordboxは高品質
- ±6%以内なら問題なし

## 8. 実践：BPM活用テクニック

### BPMでセットリストを組む

**エネルギーカーブ:**
```
オープニング: 120 BPM（落ち着き）
  ↓
ウォームアップ: 125 BPM
  ↓
ピークタイム: 130 BPM（最高潮）
  ↓
クールダウン: 125 BPM
  ↓
エンディング: 120 BPM（着地）
```

### BPMの変化幅

**推奨:**
```
- 隣接する曲: ±5 BPM以内
- 徐々に変化: 2-3 BPM ずつ
- 急激な変化: ブレイク・ドラムロールで
```

**例:**
```
曲1: 125 BPM
曲2: 128 BPM（+3）
曲3: 130 BPM（+2）
曲4: 128 BPM（-2、落ち着かせる）
```

## 9. 練習方法

### 初級（0-1ヶ月）

```
1. 毎日5曲のBPMを当てる練習
2. Rekordboxで BPM を確認する癖をつける
3. ジャンル別のBPM表を暗記
4. Syncを使わずにピッチ合わせ練習
```

### 中級（1-3ヶ月）

```
1. 耳だけで±5 BPM の精度
2. 2曲のBPMを合わせて1曲再生
3. BPM変化でエネルギーカーブを作る
4. Master Tempo の使い分け
```

### 上級（3ヶ月以降）

```
1. 瞬時にBPMを判断（±2 BPM）
2. ピッチ調整なしでも近いBPMの曲を選曲
3. BPMの微調整でグルーヴを作る
4. 異なるBPMをクリエイティブに繋ぐ
```
