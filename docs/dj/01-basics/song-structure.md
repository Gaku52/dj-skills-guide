# 曲構造の理解

Intro、Verse、Drop、Outro などのセクションを見分け、DJ ミックスに活用します。

## この章で学ぶこと

- 曲の典型的な構造
- 各セクションの特徴と役割
- ジャンル別の構造の違い
- Rekordboxでの構造分析
- 構造を活かしたミックス戦略
- セクションの聴き取り方

## なぜ曲構造が重要なのか

**ミックスの戦略:**
- どこで繋ぐか判断できる
- 展開を予測できる
- スムーズなトランジション

**エネルギー管理:**
- セクションごとのエネルギーレベル
- フロアの盛り上げ方
- 観客の期待をコントロール

---

## 1. 基本的な曲構造

### ダンスミュージックの典型的構造

```
Intro（導入）
  ↓
Verse / Build（展開）
  ↓
Drop（クライマックス）
  ↓
Breakdown（休憩）
  ↓
Build Up（再構築）
  ↓
Drop 2（最高潮）
  ↓
Outro（エンディング）
```

### 各セクションの長さ（目安）

| セクション | 小節数 | 秒数（128BPM） | 用途 |
|----------|-------|--------------|------|
| **Intro** | 16-32 | 30-60秒 | DJミックス用 |
| **Verse** | 16-32 | 30-60秒 | 展開・ボーカル |
| **Build** | 8-16 | 15-30秒 | 緊張感構築 |
| **Drop** | 16-32 | 30-60秒 | クライマックス |
| **Breakdown** | 16-32 | 30-60秒 | 休憩・再構築 |
| **Build 2** | 8-16 | 15-30秒 | 再び緊張 |
| **Drop 2** | 16-32 | 30-60秒 | 最高潮 |
| **Outro** | 16-32 | 30-60秒 | DJミックス用 |

## 2. 各セクションの詳細

### Intro（イントロ）

**目的:**
- 曲の導入
- DJがミックスしやすいように
- 徐々にエネルギーを上げる

**特徴:**
```
- ドラムのみ（キック、ハイハット）
- ベースなし / 最小限
- メロディなし / 最小限
- 徐々に楽器が追加される
```

**典型的な展開（32小節）:**
```
小節1-8  : キックのみ
小節9-16 : ハイハット追加
小節17-24: パーカッション追加
小節25-32: ベース/シンセのヒント
```

**DJ的活用:**
- **ミックスイン**: 前の曲と重ねやすい
- **EQ調整**: Lowを徐々に入れる
- **長さ**: 32小節あれば十分な時間

### Verse（ヴァース / Aメロ）

**目的:**
- 曲のメインパート
- ストーリーを展開
- ボーカルやメロディの紹介

**特徴:**
```
- ボーカルが入る（ボーカル曲の場合）
- メロディが明確
- 比較的落ち着いたエネルギー
- コード進行が確立
```

**DJ的活用:**
- **ボーカルミックス注意**: 2曲のボーカルが重ならないように
- **Mid EQ調整**: メロディの衝突を避ける
- **タイミング**: Verse途中でのミックスは避ける

### Build Up / Build（ビルドアップ）

**目的:**
- Dropへの準備
- 緊張感を高める
- 観客の期待を煽る

**特徴:**
```
- 音が徐々に増える
- ピッチ上昇（ライザー）
- ドラムロール
- フィルター（ローパス→オープン）
- リバーブ・ディレイが増える
```

**典型的な手法:**
```
- ホワイトノイズのスウェル
- スネアロール（ダダダダダ）
- シンセの音程上昇
- "Here we go!"などのボーカルサンプル
```

**DJ的活用:**
- **ミックスアウト**: Buildの途中で次の曲へ
- **エフェクト**: Echo Out で前の曲を終わらせる
- **タイミング**: Build最後の8小節が最適

### Drop（ドロップ）

**目的:**
- 曲のクライマックス
- エネルギーの解放
- フロアを盛り上げる

**特徴:**
```
- 全ての楽器が鳴る
- 最大のエネルギー
- 覚えやすいフック（リフ）
- ベースラインが強烈
```

**House/Technoの場合:**
```
小節1: ドン！（全楽器一斉）
小節1-16: ピークエネルギー維持
小節17-32: 若干の変化・発展
```

**Dubstep/Bass Musicの場合:**
```
小節1: ドロップ！
小節1-8: ワブルベース
小節9-16: パターン変化
```

**DJ的活用:**
- **Drop to Drop**: 前の曲Dropから次の曲Dropへ
- **エネルギーマッチ**: 同じくらいのエネルギーレベル
- **クイックミックス**: 8-16小節で切り替え

### Breakdown（ブレイクダウン / ブレイク）

**目的:**
- 一旦エネルギーを下げる
- 息継ぎ
- 次のDropへの準備

**特徴:**
```
- ドラムが抜ける / 最小限
- メロディやパッドが前面
- アンビエンス、雰囲気重視
- 静かで美しい
```

**典型的な展開:**
```
小節1-8 : ドラム抜ける、メロディのみ
小節9-16: 徐々に要素追加
小節17-24: パーカッション戻る
小節25-32: Build Upへ
```

**DJ的活用:**
- **休憩タイム**: フロアを一息つかせる
- **ハーモニックミックス**: メロディが目立つのでキー重要
- **ミックスポイント**: Breakdownの途中は避ける

### Outro（アウトロ）

**目的:**
- 曲の終わり
- DJが次の曲と繋ぎやすく
- 徐々にエネルギーを下げる

**特徴:**
```
- Introの逆
- 楽器が徐々に減る
- ドラムのみに戻る
- メロディ・ベースが抜ける
```

**典型的な展開（32小節）:**
```
小節1-8  : メロディ・ベース抜ける
小節9-16 : パーカッション減少
小節17-24: ハイハットのみ
小節25-32: キックのみ / フェードアウト
```

**DJ的活用:**
- **ミックスアウト**: 次の曲のIntroと重ねる
- **32小節重ねる**: 十分な時間でスムーズに
- **EQ**: Low徐々にカット、High残す

## 3. ジャンル別の構造

### House

```
Intro (32)
 → Verse (32)
 → Build (8-16)
 → Drop (32)
 → Breakdown (16-32)
 → Build (8)
 → Drop 2 (32)
 → Outro (32)

合計: 約4-5分
```

**特徴:**
- 対称的な構造
- 32小節単位
- 明確なIntro/Outro

### Techno

```
Intro (32-64)
 → Build (16)
 → Drop (64)
 → Breakdown (32)
 → Build (16)
 → Drop 2 (64)
 → Outro (32-64)

合計: 約6-8分
```

**特徴:**
- 長い
- ミニマルな変化
- 長いIntro/Outro（DJ向け）

### Trance

```
Intro (32)
 → Verse (32)
 → Build (32-64) ← 長い！
 → Drop (32)
 → Break (32)
 → Build 2 (32)
 → Drop 2 (32-64)
 → Outro (32)

合計: 約6-8分
```

**特徴:**
- 非常に長いBuild
- ドラマチック
- メロディック

### Dubstep / Bass Music

```
Intro (16)
 → Build (16)
 → Drop (16)
 → Build (8)
 → Drop 2 (16)
 → Outro (16)

合計: 約3-4分
```

**特徴:**
- 短い
- Dropが主役
- 140 BPM だがハーフタイム感覚

### Hip Hop

```
Intro (8)
 → Verse 1 (16)
 → Hook (8)
 → Verse 2 (16)
 → Hook (8)
 → Bridge (8)
 → Outro (8)

合計: 約3-4分
```

**特徴:**
- Verse-Hook構造
- ボーカル中心
- 不規則な構成も多い

## 4. Rekordboxでの構造分析

### 波形で視覚的に確認

```
1. 曲をデッキにロード
2. 波形ビューを開く（拡大）
3. 色の変化で構造を読む
   - 暗い部分: 音が少ない（Intro, Breakdown）
   - 明るい部分: 音が多い（Drop）
```

### ビートグリッドとフレーズ

```
1. 8小節ごとに色が変わる
2. 32小節（セクション）を数える
3. 大きな変化点を見つける
```

### Hot Cueでマーキング

**推奨マーキング:**
```
Hot Cue 1（赤）: Intro開始
Hot Cue 2（黄）: Verse開始
Hot Cue 3（緑）: Drop 1開始
Hot Cue 4（青）: Breakdown開始
Hot Cue 5（緑）: Drop 2開始
Hot Cue 6（赤）: Outro開始
```

### メモリーポイント

```
1. 重要なセクションにメモリーポイント
2. 名前を付ける
   - "Drop 1"
   - "Breakdown"
   - "Outro Start"
3. 色で識別
```

## 5. ミックス戦略：構造の活用

### パターン1: Intro × Outro

```
前の曲: Drop → Outro (32小節)
次の曲: Intro (32小節) → Verse
```

**最も基本的で安全**

### パターン2: Drop × Intro

```
前の曲: Drop → 終わり
次の曲: Intro → Drop
```

**エネルギーを維持**

### パターン3: Breakdown × Intro

```
前の曲: Breakdown
次の曲: Intro → Drop
```

**雰囲気を変える**

### パターン4: Drop × Drop

```
前の曲: Drop終わり
次の曲: Drop開始
```

**ピークタイムの畳み掛け**

### パターン5: Build × Build

```
前の曲: Build Up
次の曲: Build Up
```

**緊張感を倍増**
**注意**: やりすぎると疲れる

## 6. セクション聴き取りの練習

### ステップ1: Dropを見つける

```
1. 曲を通して聴く
2. 最もエネルギーが高い部分を見つける
3. そこがDrop
4. Rekordboxの波形で確認
```

### ステップ2: Intro/Outroを見つける

```
1. 曲の最初32-64秒 = Intro
2. 曲の最後32-64秒 = Outro
3. ドラムのみの部分を探す
```

### ステップ3: Build Upを見つける

```
1. Dropの直前8-16秒
2. 音が増えていく部分
3. ドラムロール、ライザー
```

### ステップ4: Breakdownを見つける

```
1. Dropとdrop 2の間
2. 静かになる部分
3. ドラムが抜ける
```

## 7. 実践例：具体的な曲で分析

### 例1: House楽曲（128 BPM）

```
00:00-00:30 Intro (32小節)
  - キック導入
  - ハイハット追加

00:30-01:00 Verse (32小節)
  - ベース追加
  - ボーカル

01:00-01:15 Build (16小節)
  - エネルギー↑
  - フィルター開く

01:15-01:45 Drop (32小節)
  - 全楽器
  - メインフック

01:45-02:15 Breakdown (32小節)
  - ドラム抜ける
  - メロディのみ

02:15-02:30 Build 2 (16小節)
  - 再び緊張

02:30-03:00 Drop 2 (32小節)
  - クライマックス

03:00-03:30 Outro (32小節)
  - 徐々に減少
  - ドラムのみに

合計: 約3分30秒
```

## 8. よくある構造のバリエーション

### ロングイントロ（Techno）

```
Intro: 64小節（2分以上）
- DJフレンドリー
- ミックスしやすい
- クラブ向け
```

### ショートイントロ（Radio Edit）

```
Intro: 8小節（15秒）
- すぐにサビ
- ラジオ向け
- DJには不向き
```

### ダブルドロップ（Trance）

```
Drop 1 → Break → Build → Drop 2 → Drop 3
- 3回目のDropがある
- 非常に長い曲
```

### No Intro/Outro（Pop曲）

```
いきなりVerse開始
突然終わる
- DJ向けではない
- Edit/Remix必要
```

## 9. 練習方法

### 初級（0-1ヶ月）

```
1. 好きな曲のセクションを紙に書き出す
2. Dropを見つける練習
3. Intro/Outroの長さを測る
4. Rekordboxで答え合わせ
```

### 中級（1-3ヶ月）

```
1. 聴くだけでセクション判別
2. Hot Cueでマーキング
3. 構造に合わせたミックス練習
4. ジャンル別の違いを理解
```

### 上級（3ヶ月以降）

```
1. 初見の曲で構造予測
2. 構造を活かしたクリエイティブミックス
3. 複雑な構造の曲も理解
4. 自分で曲をEdit/Remix
```

## まとめ

- **基本構造**: Intro → Verse → Build → Drop → Breakdown → Build → Drop → Outro
- **セクションの役割**: それぞれ明確な目的がある
- **ジャンル別**: House、Techno、Tranceで構造が違う
- **DJ活用**: Intro×Outro、Drop×Drop など戦略的に
- **分析**: Rekordboxの波形とHot Cueで視覚化
