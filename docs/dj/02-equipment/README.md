# 02. DJ Equipment - DJ機材ガイド

Rekordbox 7.2.7 と各種 DJ 機材の使い方を学びます。

## このセクションで学ぶこと

- DJ機材の種類と選び方
- Rekordbox の基本設定
- コントローラー、CDJ、ターンテーブルの使い方
- DJミキサーの操作
- ヘッドフォン・モニター環境の構築
- オーディオインターフェイスの接続

## なぜ機材知識が重要なのか

**適切な選択:**
- 自分のスタイルに合った機材
- 予算内で最適な構成
- 将来を見据えた投資

**トラブル対応:**
- 機材トラブルに対処できる
- クラブの機材を使いこなせる
- セットアップを素早く行える

## 学習の順序

### 初心者（0-3ヶ月）

```
1. DJコントローラー
   - 基本操作を学ぶ
   - Rekordbox との接続

2. ヘッドフォン
   - DJ用ヘッドフォンを選ぶ
   - キューイング練習

3. モニタースピーカー（予算があれば）
   - 自宅練習環境の構築
```

### 中級者（3-12ヶ月）

```
1. CDJ セットアップ
   - クラブ機材に慣れる
   - USBメモリでのプレイ

2. DJミキサー
   - ミキサーの詳細機能
   - エフェクトの活用

3. オーディオインターフェイス
   - より高音質な環境
```

### 上級者（12ヶ月以降）

```
1. ターンテーブル（興味があれば）
   - アナログの魅力
   - スクラッチ技術

2. プロ機材の理解
   - CDJ-3000、DJM-A9
   - 最新技術の習得

3. ソフトウェアの使い分け
   - 状況に応じた選択
```

## セットアップの基本手順

### 1. PC/Mac の準備

```
1. OS を最新に更新
2. 不要なソフトを停止
3. Rekordbox 7.2.7 をインストール
4. オーディオドライバのインストール
```

### 2. コントローラー接続

```
1. USB ケーブルで PC と接続
2. Rekordbox が自動認識
3. 環境設定 > オーディオでデバイス選択
4. テスト音声で確認
```

### 3. ヘッドフォン接続

```
1. コントローラーのヘッドフォン端子に接続
2. ヘッドフォン音量を調整
3. Cue ボタンで片方の曲を確認
```

### 4. スピーカー接続

```
1. コントローラーの Master Out → スピーカー
2. または PC → スピーカー
3. Master 音量を調整
```

## トラブルシューティング

### 音が出ない

```
□ ケーブルの接続確認
□ 音量が 0 になっていないか
□ Rekordbox のオーディオ設定確認
□ PC の音量設定確認
□ ドライバの再インストール
```

### レイテンシ（遅延）が大きい

```
□ バッファサイズを小さく（128 samples）
□ 不要なソフトを終了
□ USB 3.0 ポートを使用
□ ASIOドライバを使用（Windows）
```

### Rekordbox が認識しない

```
□ USB ケーブルの交換
□ 別の USB ポートを試す
□ Rekordbox を再起動
□ ファームウェアの更新
```

## よくある質問

### Q: 最初に買うべき機材は？

```
A: DJコントローラー + ヘッドフォン
理由:
- オールインワンで簡単
- 予算5万円以内
- すぐに練習開始できる
```

### Q: CDJ は必要？

```
A: 将来的には
理由:
- クラブの標準機材
- 慣れておくと有利
- 初心者はコントローラーで OK
```

### Q: Mac vs Windows どっち？

```
A: どちらでも OK
Mac: 安定性、トラブル少ない
Windows: コスパ、選択肢多い
```

### Q: ターンテーブルから始めるべき？

```
A: 特にこだわりがなければ NO
理由:
- 学習曲線が急
- レコード購入コスト
- 現代は CDJ/コントローラーが主流
ただし、アナログの魅力は格別
```
