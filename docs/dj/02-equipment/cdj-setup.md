# CDJセットアップ

クラブ標準機材CDJの使い方を学びます。

## この章で学ぶこと

- CDJとは何か
- CDJ-3000の操作方法
- CDJ-2000NXS2の操作方法
- USBメモリの準備
- Link機能の活用
- クラブでの使い方

## なぜCDJが重要なのか

**クラブの標準:**
- 世界中のクラブで使用
- プロDJの必須スキル
- レジデントDJには必須

**コントローラーとの違い:**
- スタンドアローン（PC不要）
- USBメモリで再生
- 高品質オーディオ
- プロフェッショナル仕様

## 1. CDJとは

### 定義

```
CDJ = CD + DJ
Pioneer DJが開発したデジタルDJ機材
```

**進化:**
```
1994年: CDJ-500（初代）
2001年: CDJ-1000（革命）
2009年: CDJ-2000（USB対応）
2016年: CDJ-2000NXS2（現行標準）
2020年: CDJ-3000（最新）
```

### 主要モデル

**CDJ-3000（最新）**
```
- 9インチタッチスクリーン
- Key Sync機能
- USB-C接続
- 高解像度波形
- 価格: 約40万円
```

**CDJ-2000NXS2（標準）**
```
- 7インチタッチスクリーン
- 4デッキ同期
- Rekordbox Link
- Beat Jump
- 価格: 約30万円
```

**XDJ-1000MK2（コスパ）**
```
- CD非対応（USB/Link）
- タッチスクリーン
- コンパクト
- 価格: 約20万円
```

## 2. USBメモリの準備

### Rekordboxでのエクスポート

**手順:**
```
1. Rekordboxを起動
2. プレイリストを作成
3. USBメモリを接続
4. プレイリストを右クリック
5. 「デバイスへエクスポート」
6. 解析データも含める
7. エクスポート完了を待つ
```

### 推奨USBメモリ

**スペック:**
```
- 容量: 64GB以上
- 規格: USB 3.0以上
- 速度: 読み込み速度重視
- フォーマット: FAT32またはexFAT
```

**おすすめモデル:**
```
- SanDisk Extreme Pro
- Samsung BAR Plus
- Kingston DataTraveler
```

### バックアップ

**必須:**
```
USBメモリ × 3本用意
- メイン
- バックアップ1
- バックアップ2

理由: USBトラブルは頻繁
```

## 3. CDJ-3000の操作

### 基本レイアウト

```
┌─────────────────┐
│  タッチスクリーン  │
├─────────────────┤
│ Performance Pads │
├─────────────────┤
│   ジョグホイール   │
├─────────────────┤
│  Tempo Fader    │
└─────────────────┘
```

### タッチスクリーン操作

**波形表示:**
```
- スワイプで波形スクロール
- ピンチで拡大/縮小
- タップでキューポイント設定
- Key表示（新機能）
```

**ブラウザ:**
```
- タッチで曲選択
- Related Tracks表示
- キー互換曲の表示
- BPM範囲で絞り込み
```

### 新機能（CDJ-3000）

**Key Sync:**
```
自動でキーを合わせる
- Master Tempo不要
- 音質劣化なし
- ハーモニックミキシングが簡単
```

**高解像度波形:**
```
- より詳細な波形表示
- セクション判別が容易
- Beat Jump精度向上
```

## 4. CDJ-2000NXS2の操作

### 基本操作

**曲のロード:**
```
1. USBメモリを挿入
2. ブラウザボタンを押す
3. ロータリーで曲選択
4. ロータリーを押してロード
```

**再生:**
```
1. Play/Pause ボタン
2. Cueボタンで頭出し
3. ジョグホイールでピッチベンド
4. Tempo Faderで微調整
```

### Hot Cue

**設定:**
```
1. 曲の任意の位置でCall/Delete押す
2. パッドABCのいずれかを押す
3. Hot Cue設定完了
```

**使用:**
```
1. パッドを押すとジャンプ
2. Shift+パッドで削除
3. 色分けで識別
```

### Loop

**オートループ:**
```
1. Loop Inボタン
2. 4/8/16/32 Beat選択
3. 自動でループ
4. Reloop/Exitで解除
```

**マニュアルループ:**
```
1. Loop Inでスタート地点
2. Loop Outでエンド地点
3. ループ再生開始
```

### Beat Jump

**機能:**
```
指定したビート数だけジャンプ
- 1, 2, 4, 8, 16, 32 Beat
- 前方/後方ジャンプ可能
- フレーズ頭への移動が簡単
```

## 5. Pro DJ Link（ネットワーク機能）

### Link接続

**接続方法:**
```
CDJ × 2 + DJミキサー
→ LANケーブルで接続
→ 自動でネットワーク構築
```

**メリット:**
```
✓ 1本のUSBを共有
✓ BPM同期
✓ Beat Sync
✓ 曲情報共有
```

### Beat Sync

**自動BPM同期:**
```
1. Masterデッキを決定
2. もう1台でSyncボタン
3. 自動でBPM一致
4. ビートグリッド同期
```

**使用の是非:**
```
練習時: 使わない（耳を鍛える）
本番: 状況次第
  - 複雑なセットでは有効
  - 基本は手動推奨
```

## 6. クラブでの使い方

### 到着時の確認

**チェックリスト:**
```
□ CDJ/ミキサーの動作確認
□ USBメモリ認識
□ ヘッドフォン接続
□ Master Out音量確認
□ モニタースピーカー位置確認
□ 照明の明るさ確認
```

### サウンドチェック

**手順:**
```
1. 自分の曲をロード
2. ゲインを12時位置
3. Master Outを適切な音量に
4. EQフラット確認
5. エフェクトOFF確認
6. クロスフェーダー中央確認
```

### セット中の注意

**トラブル回避:**
```
✓ USB抜き差しは慎重に
✓ 飲み物を機材に近づけない
✓ ケーブルを引っかけない
✓ 次のDJのためリセット
```

### 終了時のマナー

**次のDJへの配慮:**
```
□ 曲を停止
□ EQフラット
□ エフェクトOFF
□ クロスフェーダー中央
□ Tempo Fader 0%
□ ゲイン12時
□ USBメモリを抜く
```

## 7. CDJ-3000 vs CDJ-2000NXS2

### 比較表

| 機能 | CDJ-3000 | CDJ-2000NXS2 |
|------|---------|-------------|
| **画面** | 9インチタッチ | 7インチタッチ |
| **Key Sync** | ○ | × |
| **波形解像度** | 高 | 標準 |
| **USB-C** | ○ | × |
| **価格** | 約40万円 | 約30万円 |


## 9. トラブルシューティング

### USBが認識しない

**対処法:**
```
1. USB抜き差し
2. 別のUSBポートを試す
3. CDJ再起動
4. 別のUSBメモリを試す
5. Rekordboxで再エクスポート
```

### 曲がロードできない

**原因と対処:**
```
原因1: ファイル破損
→ Rekordboxで再エクスポート

原因2: フォーマット非対応
→ FAT32/exFATを使用

原因3: 解析データなし
→ Rekordboxで解析
```

### 音が出ない

**チェック:**
```
□ Cueボタンが押されている
□ チャンネルフェーダーが下
□ ミキサーのゲインが0
□ Master Outケーブル確認
□ ミキサーの入力選択確認
```

## 10. 上級テクニック

### Beat Jump活用

**フレーズジャンプ:**
```
32 Beatジャンプで次のフレーズへ
- Intro→Verse
- Breakdown→Drop
- 瞬時にセクション移動
```

### Slip Mode

**機能:**
```
ループやHot Cueを使っても
バックグラウンドで曲は進行

用途:
- ループでアレンジ
- ワンショットサンプル
- クリエイティブな表現
```

### Quantize

**ビートに合わせる:**
```
Hot Cue、Loop、Beat Jump
自動でビートに合わせる

ON: ズレない（安全）
OFF: 自由度高い（上級者）
```
