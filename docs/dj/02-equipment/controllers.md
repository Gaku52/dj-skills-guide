# DJコントローラー

初心者に最適なオールインワンDJ機材の選び方と使い方を学びます。

## この章で学ぶこと

- DJコントローラーとは
- 初心者向けおすすめモデル
- DDJシリーズの比較
- セットアップ方法
- Rekordboxとの接続
- 基本操作

## なぜコントローラーから始めるべきか

**初心者に最適:**
- オールインワン（ミキサー内蔵）
- Rekordbox完全対応
- 比較的安価（3-15万円）
- 自宅練習に最適

**CDJへの移行も容易:**
- 操作感が似ている
- 基礎が身につく
- クラブ機材の理解

## 1. DJコントローラーとは

### 定義

```
PC/Mac + Rekordbox + コントローラー
= 完全なDJシステム
```

**構成要素:**
- ジョグホイール × 2
- チャンネルフェーダー × 2
- クロスフェーダー × 1
- EQノブ × 6
- エフェクトボタン
- ブラウザコントロール
- オーディオインターフェイス内蔵

## 2. おすすめモデル比較

### DDJ-400（入門）

**価格:** 約3万円
**対象:** 完全初心者

**特徴:**
```
✓ 最も人気の入門機
✓ チュートリアル機能
✓ 2チャンネル
✓ コンパクト
✓ Rekordbox無料版でも使用可（制限あり）
```

### DDJ-FLX4（入門+）

**価格:** 約2.5万円
**対象:** 予算重視の初心者

**特徴:**
```
✓ Rekordbox + Serato対応
✓ Smart CFX機能
✓ 2チャンネル
✓ 最もコンパクト
```

### DDJ-FLX6（中級入門）

**価格:** 約5-6万円
**対象:** 長く使いたい初心者

**特徴:**
```
✓ 4デッキ対応
✓ Merge FX
✓ Jog Cue
✓ Rekordbox + Serato対応
```

### DDJ-800（中級）

**価格:** 約8-9万円
**対象:** 本格的に始めたい人

**特徴:**
```
✓ コンパクトでクラブ級
✓ CDJ風レイアウト
✓ Beat FX搭載
✓ 16パッドでHot Cue
```

### DDJ-1000（上級）

**価格:** 約12-13万円
**対象:** プロ志向

**特徴:**
```
✓ CDJに最も近い操作感
✓ フルサイズジョグ
✓ 4チャンネル
✓ 高品質オーディオ
```

## 3. セットアップ方法

### 必要なもの

```
□ PC/Mac
□ DJコントローラー
□ USBケーブル（付属）
□ Rekordbox 7.2.7
□ ヘッドフォン
□ スピーカー
```

### 手順

```
1. Rekordboxインストール
2. USBでコントローラー接続
3. 自動認識
4. オーディオ設定
5. 動作確認
```

---

## 4. 基本操作

### ジョグホイール

**機能:**
- ピッチベンド（テンポ微調整）
- スクラッチ
- 曲の頭出し

### フェーダー

**チャンネルフェーダー:**
- 各チャンネルの音量

**クロスフェーダー:**
- AデッキとBデッキの切り替え

### EQ

**3バンドEQ:**
- High（高音）
- Mid（中音）
- Low（低音）
