## 1. DJミキサーとは

### 定義

```
DJミキサー = 複数の音源をミックスする機材

通常構成:
- CDJ/ターンテーブル × 2-4台
+ DJミキサー × 1台
= DJセットアップ
```

**主な機能:**
- 音源の切り替え
- EQによる音質調整
- エフェクト
- モニタリング
- マスターアウト

## 3. ミキサーの各部名称

### 基本レイアウト（DJM-900NXS2）

```
┌─────────────────────────┐
│  Master Level   Booth    │
├─────────────────────────┤
│ ┌───┐ ┌───┐ ┌───┐ ┌───┐ │
│ │CH1│ │CH2│ │CH3│ │CH4│ │
│ │Trm│ │Trm│ │Trm│ │Trm│ │
│ │Hi │ │Hi │ │Hi │ │Hi │ │
│ │Mid│ │Mid│ │Mid│ │Mid│ │
│ │Low│ │Low│ │Low│ │Low│ │
│ │Flt│ │Flt│ │Flt│ │Flt│ │
│ │Fdr│ │Fdr│ │Fdr│ │Fdr│ │
│ └───┘ └───┘ └───┘ └───┘ │
├─────────────────────────┤
│    Crossfader           │
├─────────────────────────┤
│    Beat FX              │
└─────────────────────────┘
```

### チャンネルストリップ

**各チャンネルの構成:**
```
Trim（Gain）: 入力レベル調整
↓
Hi EQ: 高域（13kHz）
Mid EQ: 中域（1kHz）
Low EQ: 低域（70Hz）
↓
Color FX: Space, Dub Echo, Sweep等
Filter: HPF/LPF
↓
Channel Fader: チャンネル音量
```

## 4. チャンネル操作

### Trim（Gain）調整

**目的:**
```
全チャンネルの音量を揃える
→ ミックス時の音圧を一定に保つ
```

**手順:**
```
1. 曲を再生
2. Channel Faderを最大
3. Level Meterを見る
4. Trimで0dBに調整（赤ランプの直前）
5. Channel Faderを下げる
```

**重要:**
```
✓ 赤ランプ（クリップ）は避ける
✓ 0dB付近が目標
✓ 曲ごとに調整必須
```

### EQ（イコライザー）操作

**基本:**
```
12時位置: フラット（原音）
右回し: ブースト（+6dB）
左回し: カット（-26dB）
```

**ミックス時の基本テクニック:**
```
Aデッキ: 高域・中域メイン
Bデッキ: 低域・中域メイン

ミックス開始:
1. BデッキLow完全カット
2. Bデッキをフェードイン
3. 徐々にBデッキLow上げる
4. 同時にAデッキLow下げる
5. Aデッキをフェードアウト
```

### Filter（フィルター）

**HPF（High Pass Filter）:**
```
低域をカット
→ 高域だけ残る
→ 軽い音に

用途: イントロ/ブレイク
```

**LPF（Low Pass Filter）:**
```
高域をカット
→ 低域だけ残る
→ こもった音に

用途: ビルドアップ/トランジション
```

## 5. クロスフェーダー

### 基本操作

```
左（A）: Aデッキのみ
中央: A + B ミックス
右（B）: Bデッキのみ
```

### クロスフェーダーカーブ

**設定（DJM-900NXS2）:**
```
┌─────────────┐
│ •  •  •  •  │ カーブ選択
└─────────────┘
 1  2  3  4

1: Smooth（ミックス用）
2: Medium
3: Fast（スクラッチ用）
4: Sharp（カット）
```

### 使い方の違い

**ミックスDJ:**
- チャンネルフェーダー主体
- クロスフェーダーは中央固定
- EQで切り替え

**スクラッチDJ:**
- クロスフェーダー主体
- 細かいカット
- Fast/Sharp設定

## 6. Master Out & Booth Out

### Master Out

**設定:**
```
用途: フロアのメインスピーカー
接続: XLR/RCA
レベル: Master Level ノブで調整
目標: PAシステムの最適入力
```

**重要:**
```
✓ クラブ到着時に音響と確認
✓ 基本は12時位置スタート
✓ セット中は触らない
✓ 個別チャンネルで調整
```

### Booth Out

**設定:**
```
用途: DJブース内モニター
接続: XLR/TRS
レベル: Booth Level ノブで調整
独立: Master Outと独立
```

### ヘッドフォンキュー

**操作:**
```
Cueボタン: 各チャンネルのCue
Mixing: Master/Cue のミックス
Level: ヘッドフォン音量

実践:
1. 次の曲をCueで確認
2. ビートマッチング
3. MixingでMasterと比較
4. ミックスイン
```

## 7. エフェクト

### Beat FX（DJM-900NXS2）

**主要エフェクト:**
```
Echo: ディレイ（1/4, 1/2, 3/4拍）
Reverb: 残響
Spiral: 上昇/下降
Trans: ゲートリバース
Filter: HPF/LPF
Flanger: うねり
Phaser: 位相変化
Slip Roll: スリップループ
```

**操作:**
```
1. Beat FX Select でエフェクト選択
2. Beat で長さ選択（1/8, 1/4, 1/2, 1拍等）
3. Channel Assign で対象選択
4. Level/Depth で深さ調整
5. On/Off で起動
```

**実践例:**
```
ビルドアップ:
- Echo 1/2拍
- Level徐々に上げる
- ドロップ直前でOff

トランジション:
- Filter HPF
- 徐々に開く/閉じる
```

### Sound Color FX

**6種類:**
```
Space: 空間系
Dub Echo: ダブディレイ
Sweep: フィルタースイープ
Noise: ホワイトノイズ
Crush: ビットクラッシャー
Filter: HPF/LPF
```

**操作:**
```
Color FX Select: エフェクト選択
Color FX Knob: 深さ調整（各チャンネル）

実践:
- Sweep: ブレイク時
- Dub Echo: トランジション
- Noise: ビルドアップ
```

## 8. ゲインステージング

### 重要性

```
目的:
✓ 音割れ防止
✓ 音圧の一貫性
✓ クリアなミックス
✓ PA負荷軽減
```

### 正しい手順

**セットアップ時:**
```
1. 全Fader、Trimを0
2. Master Levelを12時
3. 1曲目をロード
4. Channel Fader最大
5. Trimで0dB付近に調整
6. Channel Faderを適切な位置に
7. 全チャンネル同様に
```

**ミックス時:**
```
✓ 各チャンネルのTrimで音量揃え
✓ Channel Faderで操作
✓ Master Levelは固定
✗ Master Levelで音量調整しない
```

## 10. クラブでの使い方

### 到着時チェックリスト

```
□ Master Level位置確認
□ Booth Level調整
□ ヘッドフォン接続確認
□ 全EQフラット確認
□ 全Fader下確認
□ Beat FX OFF確認
□ サウンドチェック
```

### サウンドチェック手順

```
1. 自分の曲をロード
2. Channel Fader最大
3. Trimで0dB調整
4. Channel Faderを70-80%に
5. EQフラット確認
6. Master音量確認
7. 問題あれば音響に相談
```

### セット中の注意

```
✓ Master Levelは触らない
✓ 次のDJのためリセット
✓ 飲み物を近づけない
✓ ケーブル引っかけない
```

### 終了時のマナー

```
□ 全チャンネルFader下げる
□ EQフラット（12時）
□ Beat FX OFF
□ Color FX OFF
□ Crossfader中央
□ Trimを12時に戻す
```
