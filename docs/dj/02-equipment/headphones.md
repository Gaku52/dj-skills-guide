# DJ用ヘッドフォン

DJ用ヘッドフォンの選び方と使い方を学びます。

## この章で学ぶこと

- DJ用ヘッドフォンの条件
- おすすめモデル
- 密閉型の重要性
- 片耳モニタリング
- メンテナンス方法

## なぜDJ用ヘッドフォンが必要か

**一般用との違い:**
- 密閉型（音漏れ防止）
- 高耐久性
- 片耳モニター可能
- 大音量に耐える

## 選び方のポイント

### 必須条件

```
1. 密閉型
2. 片耳モニター可能
3. 交換ケーブル
4. 頑丈な作り
5. 快適な装着感
```

### 予算別

```
1万円: Audio-Technica ATH-M40x
1.5万円: Audio-Technica ATH-M50x
2.5万円: Sennheiser HD25
3万円: Pioneer HDJ-X5
4万円: Pioneer HDJ-X10
```

---

## 使い方

### 片耳モニタリング

```
1. 右耳でフロア音
2. 左耳（ヘッドフォン）で次の曲
3. ビートマッチング
4. 両耳でミックス確認
```

### メンテナンス

```
□ イヤーパッド定期交換（1-2年）
□ ケーブル確認
□ 汚れを拭く
□ 乾燥保管
```
