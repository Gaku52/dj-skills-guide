## 2. Rekordbox 7（本ガイド使用）

### 概要

**バージョン:** 7.2.7（本ガイド対象）
**開発:** Pioneer DJ

**特徴:**
```
✓ CDJ/XDJとの完全連携
✓ クラブ標準
✓ プロフェッショナル仕様
✓ Cloud機能
✓ ビートグリッド精度高い
```

### プラン比較

**Free Plan（無料）**
```
✓ 基本的な楽曲管理
✓ 解析機能
✓ エクスポート（USB）
✗ Performance Mode制限
✗ DVS使用不可
✗ Cloud Unlimited不可
✗ Lyric機能なし
```

**Core Plan（¥1,738/月）**
```
✓ Performance Mode無制限
✓ DVS対応
✓ Cloud Unlimited
✓ Lyric機能
✓ 全エフェクト
✓ 動画対応
✓ 商用利用可
```

### 主要機能

**ライブラリ管理:**
```
- Smart Playlist（自動プレイリスト）
- Related Tracks（関連曲提案）
- Tag編集（ID3タグ）
- カラー分類
- Rating（5段階評価）
- プレイカウント
```

**解析機能:**
```
- BPM検出（高精度）
- Key検出（Camelot表示）
- ビートグリッド自動生成
- 波形解析
- ダイナミックレンジ表示
```

**Performance Mode:**
```
- 2/4デッキ
- エフェクト（Beat FX、Color FX）
- サンプラー
- HOT CUE
- Beat Jump
- Loop
```

### Rekordbox 7の新機能

**Ableton Link:**
```
Abletonと同期
→ ライブセットとDJミックス融合
```

**Stem Separation:**
```
楽曲を4トラックに分離
- Vocal
- Drum
- Bass
- Melody
```

**Cloud Unlimited:**
```
無制限のクラウドストレージ
→ どこでもライブラリアクセス
```

## 7. どれを選ぶべきか

### Pioneer DJ機器を使う場合

**Rekordbox一択:**
```
理由:
✓ CDJ/XDJとの完璧な連携
✓ クラブ標準
✓ Pro DJ Link活用
✓ USBエクスポート必須
```

### スクラッチDJ志望

**Serato DJ Pro推奨:**
```
理由:
✓ スクラッチDJ業界標準
✓ DVS追従精度最高
✓ 波形見やすい
✓ Rane機器と完璧連携
```

### 音質・エフェクト重視

**Traktor Pro推奨:**
```
理由:
✓ 高音質エンジン
✓ 最多エフェクト
✓ Stems対応
✓ 買い切り（コスパ）
```

### モバイル/ウェディングDJ

**Virtual DJ推奨:**
```
理由:
✓ ビデオDJ機能
✓ 自動ミックス
✓ 対応機器最多
✓ 安価（$99/年）
```

### 初心者（まだ機材未定）

**Rekordbox推奨:**
```
理由:
✓ 無料で始められる
✓ クラブ移行が楽
✓ 将来性高い
✓ DDJ-400等の入門機と相性◎
```

## 8. Rekordbox 7 詳細設定

### 初期設定（必須）

**Preferences → Audio**
```
Audio Device: 使用機器選択
  - DDJ-400
  - DDJ-800
  - INTERFACE 2（DVS）

Sample Rate: 44100Hz推奨
Buffer Size: 512 samples（低レイテンシー）
```

**Preferences → Analysis**
```
BPM範囲: 60-180（House/Techno）
Key Detection: ON
ビートグリッド: 自動調整ON
```

**Preferences → View**
```
波形表示: 3 Band
波形カラー: RGB
グリッド線: ON
Key表示: Camelot
```

### ライブラリ管理

**インポート:**
```
1. [File] → [Import]
2. フォルダ選択
3. 自動解析開始
4. 完了待ち（数分-数時間）
```

**プレイリスト作成:**
```
右クリック → [Create New Playlist]
→ 名前入力（例: House 128BPM）
→ 曲をドラッグ&ドロップ
```

**Smart Playlist:**
```
例: House 128BPM
条件:
- Genre: House
- BPM: 126-130
- Rating: ★★★以上
→ 自動更新
```

### USBエクスポート

**手順:**
```
1. プレイリスト作成
2. USBメモリ接続（FAT32/exFAT）
3. プレイリスト右クリック
4. [Export to Device]
5. 解析データ含める
6. エクスポート開始
7. 完了確認
```

## 10. 複数ソフトの併用

### ライブラリ共有

**Rekordbox → Serato:**
```
Rekordbox/Settingsフォルダ
→ Seratoがライブラリ読み込み可能
→ BPM、Cue等は再解析必要
```

**iTunes連携:**
```
全ソフトでiTunesライブラリ読み込み可能
→ プレイリスト同期
→ ただし解析は各ソフトで必要
```

## まとめ

- **Rekordbox**: Pioneer DJ機器必須、クラブ標準
- **Serato**: スクラッチDJ、DVS最強
- **Traktor**: 音質・エフェクト重視、買い切り
- **Virtual DJ**: モバイルDJ、ビデオDJ、コスパ◎
- **初心者**: Rekordbox無料版で始める
- **ライブラリ**: 丁寧な管理が上達の鍵
- **設定**: BPM範囲、Key検出は必須ON
