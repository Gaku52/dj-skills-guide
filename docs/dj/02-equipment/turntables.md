## なぜターンテーブルが重要か

**DJの原点:**
- Hip Hop、House、Technoの原点
- 最も純粋なDJスタイル
- スクラッチの必須機材

**現代でも重要:**
- バイナルの温かみのある音質
- DVSでデジタルとアナログの融合
- クラブでの存在感
- コレクターとしての楽しみ

## 1. ターンテーブルとは

### 定義

```
ターンテーブル = レコードを回転させる機材
DJ用 = ピッチコントロール付き
```

**構成要素:**
- プラッター（回転盤）
- トーンアーム
- カートリッジ（針）
- ピッチフェーダー
- モーター
- ターゲットライト

## 3. ターンテーブルの各部名称

### 基本構造

```
┌────────────────┐
│  ターゲットライト │
├────────────────┤
│   トーンアーム    │
│ ┌──────────┐  │
│ │ プラッター  │  │
│ └──────────┘  │
├────────────────┤
│  Pitch Fader   │
│  33/45 切替     │
│  Start/Stop   │
└────────────────┘
```

### 各部の役割

**プラッター:**
- レコードを載せる回転盤
- 重量: 約1.6kg（安定性）
- スリップマット装着

**トーンアーム:**
- 針をレコードに接触させる
- カウンターウェイトで圧力調整
- アンチスケート機能

**カートリッジ:**
- レコード溝を読み取る
- 針圧: 2-4g（DJ用）
- MM型が一般的

**ピッチフェーダー:**
- 回転速度調整
- ±8%が標準
- ビートマッチングに必須

## 4. セットアップ方法

### 基本セットアップ

**手順:**
```
1. ターンテーブルを水平に設置
2. スリップマットを装着
3. カートリッジを取り付け
4. トーンアームバランス調整
5. 針圧設定（3g推奨）
6. アンチスケート設定
7. RCAケーブルをミキサーに接続
8. アース線を接続
```

### 針圧調整

**重要:**
```
DJ用カートリッジ: 3-4g
- 軽すぎる: スキップする
- 重すぎる: レコードを傷める
- 推奨: 3g（Shure M44Gの場合）
```

**調整方法:**
```
1. カウンターウェイトを回す
2. トーンアームが水平になる位置
3. 目盛りを0に合わせる
4. カウンターウェイトを3gに回す
5. 動作確認
```

## 6. バイナルの扱い方

### 基本ルール

**厳守:**
```
✓ 必ず両手で持つ
✓ 溝に触れない（縁とレーベル）
✓ スリーブに保管
✓ 縦置き保管
✗ 平積み禁止
✗ 直射日光禁止
```

### クリーニング

**道具:**
```
- カーボンブラシ（毎回）
- クリーニング液（月1回）
- マイクロファイバークロス
```

**手順:**
```
1. カーボンブラシで埃を取る
2. クリーニング液を塗布
3. 回転させながら拭く
4. 完全に乾かす
5. スリーブに戻す
```

## 7. DVS（Digital Vinyl System）

### DVSとは

```
DVS = タイムコードバイナルを使用
→ ターンテーブルでデジタル音源を操作

メリット:
✓ バイナルの操作感
✓ デジタルの利便性
✓ 無限の曲数
✓ エフェクト使用可能
```

### タイムコードバイナル

**仕組み:**
```
特殊な信号が録音されたバイナル
→ Rekordboxが信号を読み取る
→ デジタル音源を操作

必要:
- タイムコードバイナル × 2
- DVS対応オーディオIF
- Rekordbox DVSライセンス
```

## 8. Rekordbox DVSの設定

### 必要機材

**最小構成:**
```
□ ターンテーブル × 2
□ DJミキサー（Phono入力）
□ DVS対応オーディオIF
  - Pioneer DJ DJM-250MK2
  - Pioneer DJ INTERFACE 2
  - Rane SL3
□ タイムコードバイナル × 2
□ Rekordbox DVSライセンス
```

### Rekordbox設定手順

**Step 1: ライセンス認証**
```
1. Rekordbox起動
2. [Preferences] → [License]
3. DVSライセンス有効化
```

**Step 2: オーディオ設定**
```
1. [Preferences] → [Audio]
2. Audio Device: DVS対応IFを選択
3. Input Channels:
   - CH1: Turntable 1
   - CH2: Turntable 2
4. Output Channels設定
```

**Step 3: DVS設定**
```
1. [Preferences] → [DVS]
2. Input: Phono/Line選択
3. Tracking: 推奨設定のまま
4. Position: RELATIVE推奨
5. Test Toneで動作確認
```

### キャリブレーション

**重要:**
```
初回必須の設定
目的: 最適な追従性

手順:
1. タイムコードバイナルをかける
2. [DVS] → [Calibration]
3. バイナルを回す
4. 自動調整完了
```

## 9. スクラッチ基礎

### Baby Scratch

**最も基本:**
```
1. クロスフェーダーを開ける
2. バイナルを前後に動かす
3. リズムに合わせる

ポイント:
- フェーダーは触らない
- 一定のリズムで
```

### Forward Scratch

**フェーダー使用:**
```
1. バイナルを前に押す（フェーダーON）
2. バイナルを戻す（フェーダーOFF）
3. 繰り返す

ポイント:
- 押す時だけ音が出る
- タイミング重要
```

### Transform

**フェーダーを細かく:**
```
バイナルを一定速度で動かす
+ フェーダーを細かくON/OFF

結果: 音が刻まれる
```

## 10. メンテナンス

### 定期メンテナンス

**毎回:**
```
□ 針の埃を取る
□ プラッターを拭く
□ スリップマットの埃を払う
```

**月1回:**
```
□ 針の状態確認
□ トーンアームのバランス確認
□ RCAケーブルの接触確認
```

**年1回:**
```
□ 針交換（使用頻度による）
□ スリップマット交換
□ 内部クリーニング（プロ推奨）
```

### トラブルシューティング

**音が出ない:**
```
□ RCAケーブル確認
□ アース線確認
□ ミキサーのPhono/Line設定
□ 針の接続確認
```

**スキップする:**
```
□ 針圧確認（増やす）
□ 針の汚れ確認
□ レコードの汚れ確認
□ ターンテーブルの水平確認
```

**ノイズが入る:**
```
□ アース線確認
□ RCAケーブルの劣化確認
□ 針の摩耗確認
□ レコードのクリーニング
```

## 12. 練習方法

### 初級（1-3ヶ月）

**目標: 基本操作をマスター**
```
Week 1-2:
- セットアップ練習
- 針の扱いに慣れる
- バイナルの持ち方

Week 3-4:
- ピッチコントロール
- 曲の頭出し
- Baby Scratch

Month 2-3:
- ビートマッチング（アナログ）
- Forward Scratch
- DVSセットアップ
```

### 中級（3-6ヶ月）

**目標: ミックスとスクラッチ**
```
- EQを使ったミックス
- Transform Scratch

```

### 上級（6ヶ月-）

**目標: 独自のスタイル確立**

```
- Crab Scratch
```

## 13. バイナル vs DVS

### アナログバイナル

**メリット:**
```
✓ 温かみのある音質
✓ 物理メディアの楽しさ
✓ コレクションの価値
✓ 純粋なDJスタイル
```

**デメリット:**
```
✗ 重い（持ち運び大変）
✗ 高価（1枚2,000-3,000円）
✗ 曲数に限界
✗ 入手困難な曲も
```

### DVS

**メリット:**
```
✓ 無限の曲数
✓ 持ち運び楽
✓ エフェクト使用可能
✓ Hot Cue使える
```

**デメリット:**
```
✗ PCが必要
✗ セットアップ複雑
✗ レイテンシーに注意
✗ タイムコードバイナル必要
```
