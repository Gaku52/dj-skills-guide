# DJ基本テクニック

DJの核となる実践的なテクニックを学びます。

## このセクションで学ぶこと

- ビートマッチング（最重要）
- ミキシング基礎
- EQ操作
- クロスフェーダーテクニック
- ピッチコントロール
- キューイング技術
- ゲインステージング

## なぜ基本テクニックが重要か

**DJの核心:**
```
基本テクニック = DJスキルの80%
✓ ビートマッチングができれば「DJができる」
✓ EQ操作でミックスの質が決まる
✓ ゲインステージングで音質が変わる
```

**プロとアマチュアの差:**
```
アマチュア: 曲をかけるだけ
プロ: 基本に忠実、完璧な実行
→ 基本の反復練習が全て
```

## 学習の順序

### Step 1: ビートマッチング（最優先）

```
期間: 1-2ヶ月
目標: 耳だけでビートを合わせる

なぜ最初か:
- DJ の最も基本的なスキル
- これができないと何も始まらない
- 耳を鍛える（最重要）
```

### Step 2: ミキシング基礎

```
期間: 2週間
目標: スムーズな曲の切り替え

前提: ビートマッチングができる
内容:
- フェーダー操作
- タイミングの取り方
- トランジションの種類
```

### Step 3: EQ操作

```
期間: 1ヶ月
目標: EQを使った音質調整

これで一人前:
- Low カット/ブーストの理解
- 音が濁らないミックス
- クリアなサウンド
```

### Step 4: その他のテクニック

```
期間: 1-2ヶ月
内容:
- ピッチコントロール（微調整）
- キューイング（次の曲の準備）
- ゲインステージング（音量管理）
- クロスフェーダー（スタイルによる）
```

## 練習プレイリスト推奨

### ビートマッチング練習用

**条件:**
```
- BPM範囲: 120-130（狭く）
- ジャンル: 統一（House推奨）
- 曲構造: シンプル（4つ打ち）
- 曲数: 10-20曲
```

**理由:**
```
BPM範囲が狭い
→ ピッチ調整が小さい
→ 合わせやすい
→ 成功体験が増える
→ モチベーション維持
```

### ミックス練習用

**条件:**
```
- BPM範囲: 広げる（120-140）
- ジャンル: 2-3種類
- キー: 互換性を考慮
- 曲構造: 明確なセクション
```

## 練習スケジュール（推奨）

### Week 1-4: ビートマッチング集中

```
毎日30分-1時間
- 同じ2曲を繰り返し
- Sync OFF必須
- ジョグホイールに慣れる
- 耳だけでズレを聞き取る
```

### Week 5-6: ミキシング基礎

```
毎日30分
- ビートマッチング（15分）
- フェーダー操作（15分）
- スムーズなトランジション
```

### Week 7-8: EQ操作

```
毎日30分
- ビートマッチング + ミキシング（15分）
- EQを使ったミックス（15分）
- Low重複回避の練習
```

### Week 9-12: 統合練習

```
毎日30分-1時間
- 30分ミックス録音
- 聞き返して改善点発見
- 全テクニックの統合
```

## よくある質問

### Q1: Syncボタンは使っていい？

**A: 練習時は使わない、本番は状況次第**

```
練習時:
✗ Sync OFF推奨
→ 耳を鍛える（最重要）
→ CDJでも対応できる

本番:
○ 状況次第
→ 3デッキ以上の複雑なミックス
→ エフェクト多用時
→ ただし基礎ができている前提
```

### Q2: どのくらいでビートマッチングできる？

**A: 個人差あり、1-3ヶ月**

```
早い人: 2週間-1ヶ月
平均: 1-2ヶ月
じっくり型: 2-3ヶ月

重要: 焦らない
→ 毎日少しずつ
→ 必ずできるようになる
```

### Q3: DDJ-FLX4で十分？

**A: 基礎習得には完璧**

```
✓ プロと同じ基礎が学べる
✓ CDJへの移行が容易
✓ コンパクトで練習しやすい

クラブでは:
- CDJが標準
- ただし操作は似ている
- 月1回レンタルスタジオ推奨
```

### Q4: ヘッドフォンは必須？

**A: 絶対必須**

```
理由:
- 次の曲をCueで確認
- ビートマッチング必須
- フロア音と分離

推奨: 密閉型（2万円前後）
→ [DJ用ヘッドフォン](../02-equipment/headphones.md)
```

## チェックリスト

### ビートマッチングができる基準

```
□ Sync OFFでビートが合わせられる
□ 耳だけでズレを聞き取れる
□ ジョグホイールで微調整できる
□ 30秒以上ビートがズレない
```

### ミキシングができる基準

```
□ スムーズに曲を切り替えられる
□ 違和感のないトランジション
□ フェーダー操作がスムーズ
□ EQで音が濁らない
```

### 基本テクニック完成の基準

```
□ 30分のミックスが録音できる
□ 聞き返して違和感がない
□ フロアで流せるレベル
□ 友人に「いいね」と言われる
```
