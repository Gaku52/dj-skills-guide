# ビートマッチング

DJの最も重要な基礎スキル、ビートマッチングを完全マスターします。

## この章で学ぶこと

- ビートマッチングとは何か
- なぜ最も重要なのか
- 耳でビートを聞き取る方法
- ジョグホイールの使い方
- Tempo Faderの調整
- DDJ-FLX4での実践
- 段階的な練習方法
- よくある失敗と対処法

## なぜビートマッチングが最重要か

**DJの定義そのもの:**
```
ビートマッチングができない = DJではない

これができれば:
✓ 曲をつなげられる
✓ ダンスフロアを止めない
✓ プロと呼べる
```

**Syncボタンがあるのになぜ？:**
```
理由1: 耳が鍛えられる
→ 音楽的感覚の向上
→ ミックスの質が上がる

理由2: CDJでも対応できる
→ クラブのCDJにはSync機能なし（多くの場合）
→ 機材が変わっても対応可能

理由3: 自信につながる
→ 手動でできる = 真のスキル
→ どんな状況でも対応可能
```

## 1. ビートマッチングとは

### 定義

```
ビートマッチング = 2曲のBPMとビート位相を一致させる技術

BPM一致: 曲の速度を合わせる
位相一致: ビートのタイミングを合わせる

結果: 2曲が同時に流れても違和感なし
```

### 視覚的イメージ

**ズレている状態:**
```
曲A: ドン・ドン・ドン・ドン
曲B: ・ドン・ドン・ドン・ドン
        ↑ ズレている
```

**合っている状態:**
```
曲A: ドン・ドン・ドン・ドン
曲B: ドン・ドン・ドン・ドン
     ↑ 完全に重なる
```

---

## 2. ビートマッチングの2段階

### Step 1: BPM（速度）を合わせる

**使用:**
- Tempo Fader（ピッチフェーダー）
- 粗調整

**目標:**
```
曲A: 128.0 BPM
曲B: 128.0 BPM
→ 同じ速度
```

### Step 2: 位相（タイミング）を合わせる

**使用:**
- ジョグホイール
- 微調整

**目標:**
```
曲Aのキック: ドン
曲Bのキック: ドン
→ 同時に鳴る
```

## 3. DDJ-FLX4でのビートマッチング

### コントローラーレイアウト

```
┌─────────────────────┐
│     Deck A  Deck B  │
├─────────────────────┤
│   Jog      Jog      │
│  Wheel    Wheel     │
├─────────────────────┤
│  Tempo    Tempo     │
│  Fader    Fader     │
└─────────────────────┘
```

### 各部の役割（DDJ-FLX4）

**ジョグホイール:**
```
上面タッチ: 一時停止+スクラッチ
上面回す（再生中）: ピッチベンド
側面回す: 曲送り/戻し

使用: 位相合わせ（細かい調整）
```

**Tempo Fader:**
```
中央: ±0%（原曲BPM）
上: マイナス（遅く）
下: プラス（速く）

DDJ-FLX4: ±8%調整可能
使用: BPM合わせ（速度調整）
```

## 4. ビートマッチング手順（完全版）

### 準備

**Rekordbox設定:**
```
1. [Preferences] → [Performance]
2. Beat Sync: OFF（必須）
3. Quantize: OFF（必須）
4. Metronome: ON（練習時のみ）
```

**曲の選択:**
```
最初の練習:
- 同じBPM（128同士など）
- 同じジャンル（House推奨）
- 4つ打ちキック明確
- 曲数: 2曲だけ
```

### Step 1: 曲Aを再生（マスター）

```
1. Deck Aに曲をロード
2. Play ボタンを押す
3. チャンネルフェーダーを上げる
4. 音が出ることを確認
```

### Step 2: 曲BのBPMを確認

```
Rekordbox画面を見る:
- 曲A: 128.0 BPM
- 曲B: 126.5 BPM
→ 差分: -1.5 BPM
```

### Step 3: Tempo Faderで曲BのBPMを調整

**計算:**
```
曲B: 126.5 BPM（現在）
曲A: 128.0 BPM（目標）

差分: +1.5 BPM
割合: 1.5 / 126.5 = 約1.2%

→ Deck B の Tempo Faderを +1.2% に
```

**DDJ-FLX4での操作:**
```
1. Deck B の Tempo Fader を下に下げる
2. Rekordbox画面でBPM確認
3. 128.0 BPM になるまで調整
4. 目盛りは目安、最終的には耳で
```

### Step 4: 曲Bをヘッドフォンで再生

```
1. Deck B に曲をロード
2. Cue ボタンを押す（ヘッドフォンで確認）
3. Play ボタンを押す
4. ヘッドフォンで曲Bが聞こえる
5. チャンネルフェーダーは下げたまま
```

### Step 5: ビートを聞き比べる

**ヘッドフォンMixing:**
```
DDJ-FLX4: Headphone Mixing ノブ

左: Master（曲A）のみ
中央: Master + Cue（両方）
右: Cue（曲B）のみ

中央位置で両方を聞く
```

**聞き取り:**
```
ヘッドフォンで:
- 左耳: 曲A（スピーカーからも）
- 右耳: 曲B（ヘッドフォンのみ）

両方のキックを聞く:
ドドドド（A）
ドドドド（B）

合っている？ズレている？
```

### Step 6: ジョグホイールで位相を合わせる

**ズレている場合:**

```
曲Bが速い（先行している）:
→ ジョグホイール上面を時計回り
→ 曲Bが遅くなる
→ 曲Aに追いつくまで待つ

曲Bが遅い（遅れている）:
→ ジョグホイール上面を反時計回り
→ 曲Bが速くなる
→ 曲Aに追いつく
```

**DDJ-FLX4の感覚:**
```
軽くタッチ: わずかに加速/減速
強くタッチ: 大きく加速/減速

最初は軽く
慣れたら強弱を使い分ける
```

### Step 7: 微調整

**完璧に合わせる:**
```
1. キックが重なるタイミングを探す
2. ジョグホイールで微調整
3. 数秒間ズレないか確認
4. ズレたら再調整
```

**Tempo Faderの微調整:**
```
数秒でズレる場合:
→ BPMがまだ完全に合っていない

曲Bが徐々に先行:
→ Tempo Faderをわずかに上げる（遅く）

曲Bが徐々に遅れる:
→ Tempo Faderをわずかに下げる（速く）
```

### Step 8: チャンネルフェーダーを上げる

```
1. 位相が完璧に合っている状態
2. Deck B のチャンネルフェーダーを徐々に上げる
3. 両方の曲がスピーカーから鳴る
4. ズレていないか確認
5. ズレたらジョグホイールで即修正
```

## 5. 耳でビートを聞き取る方法

### キックの聞き取り

**4つ打ちキック:**
```
ドン・ドン・ドン・ドン
1   2   3   4

これが明確な曲を選ぶ（最初は）
```

**聞き取り練習:**
```
1. 曲Aだけを流す
2. キックに合わせて数える
3. 「1, 2, 3, 4, 1, 2, 3, 4」
4. 体を揺らす、タップする
```

### ズレの聞き取り

**ドドドド（完璧）:**
```
2つの音が完全に重なる
→ 1つの音に聞こえる
```

**ド・ド・ド・ド（ズレている）:**
```
2つの音が別々に聞こえる
→ エコーのような感じ
→ これを聞き取る力が最重要
```

**練習方法:**
```
1. 意図的にズラす
2. ズレを聞く
3. 合わせる
4. 繰り返し（100回以上）
```

## 6. 段階的な練習方法

### Week 1: 同じBPM、同じ曲

```
練習:
- 全く同じ曲 × 2
- BPMも同じ
- 位相合わせだけに集中

目標:
- ジョグホイールの感覚を掴む
- 10秒間ズレない
```

### Week 2: 同じBPM、違う曲

```
練習:
- 同じBPM（128同士）
- 違う曲
- ジャンルは同じ

目標:
- 30秒間ズレない
- 耳でズレを聞き取れる
```

### Week 3-4: BPM差1-2

```
練習:
- BPM差: 1-2（例: 128と129）
- Tempo Faderで調整
- 位相合わせ

目標:
- 1分間ズレない
- Tempo Faderの感覚を掴む
```

### Week 5-8: BPM差3-5

```
練習:
- BPM差: 3-5（例: 126と128）
- より大きな調整が必要
- 実践的な練習

目標:
- 2分以上ズレない
- すぐに合わせられる
```

## 7. DDJ-FLX4特有のコツ

### ジョグホイールの感度

**DDJ-FLX4の特徴:**
```
感度: 中程度
サイズ: コンパクト
反応: 良好

コツ:
- 軽く触る（最初は）
- 指の腹を使う
- 爪は使わない
```

### Tempo Faderの範囲

```
DDJ-FLX4: ±8%

十分な範囲:
- House: 120-130 BPM → 対応可能
- Techno: 125-135 BPM → 対応可能
- Bass Music: 140-150 BPM → 対応可能

±8%あれば:
128 BPM × 1.08 = 138.24 BPM
128 BPM × 0.92 = 117.76 BPM
→ 十分
```

### Master Tempo機能

**DDJ-FLX4:**
```
Master Tempo ボタン: あり

ON: キーが変わらない（ピッチ変更時）
OFF: キーが変わる

推奨:
- 練習時: OFF（キーの変化を学ぶ）
- 本番: ON（曲によって）
```

## 8. よくある失敗と対処法

### 失敗1: すぐにズレる

**原因:**
```
BPMが完全に合っていない
```

**対処:**
```
Tempo Faderを0.1-0.2%単位で調整
→ 数秒観察
→ まだズレる？ さらに調整
→ 完璧になるまで繰り返し
```

### 失敗2: 合わせたつもりが合っていない

**原因:**
```
耳がまだ鍛えられていない
```

**対処:**
```
1. ヘッドフォンの音量を上げる
2. スピーカーの音量を下げる
3. Cueだけを聞く
4. キックに集中
5. 完璧に合うまで調整
```

### 失敗3: ジョグホイールの調整が大きすぎる

**原因:**
```
力み過ぎ、タッチが強すぎる
```

**対処:**
```
1. 指1本で軽く触る
2. 指の腹を使う
3. 回すのではなく「なでる」
4. 微調整を意識
```

### 失敗4: どちらがズレているかわからない

**原因:**
```
両方の曲を同時に聞きすぎ
```

**対処:**
```
1. Headphone Mixing を Cue側に
2. 曲Bだけを聞く
3. スピーカーから曲Aを感じる
4. 曲Bのキックが先か後か判断
```

## 9. 上達のためのヒント

### 毎日30分

```
一番効果的:
- 毎日30分 × 2ヶ月
> 週1回 3時間 × 2ヶ月

理由: 筋肉記憶、耳の訓練
```

### 録音して聞き返す

```
1. 練習ミックスを録音（Rekordbox機能）
2. 聞き返す
3. ズレを客観的に発見
4. 次回改善
```

### 同じ曲を繰り返す

```
変化を求めない:
- 最初の1ヶ月は同じ10曲
- 完璧にマスター
- その後新しい曲

理由:
曲を覚える
→ ビートのタイミングがわかる
→ 合わせやすい
```

### プレッシャーをかける

```
友人に聞いてもらう:
- 緊張する
- でも上達が早い
- フィードバックをもらえる
```

## 10. Sync機能との向き合い方

### 練習時: 絶対に使わない

```
理由:
✓ 耳が鍛えられない
✓ CDJで困る
✓ 本当のスキルが身につかない
```

### 本番: 状況次第で使う

**使っても良い場合:**
```
○ 3デッキ以上の複雑なミックス
○ エフェクト多用時
○ ライブ楽器との同期
○ 時間がない緊急時
```

**使わない方が良い場合:**
```
× 2デッキの通常ミックス
× 練習中
× ビートマッチングができない場合
```

### プロの意見

```
Carl Cox（レジェンドDJ）:
「Syncは便利だが、手動でできることが前提」

Richie Hawtin（Technoの神）:
「技術は使うべき。ただし基礎があってこそ」

結論: 基礎をマスターしてから
```

## 11. ビートマッチング完成の基準

### 初級レベル（1ヶ月）

```
□ 同じBPM同士で合わせられる
□ 30秒間ズレない
□ ヘッドフォンでズレを聞き取れる
□ ジョグホイールの基本操作ができる
```

### 中級レベル（2ヶ月）

```
□ BPM差3-5でも合わせられる
□ 2分間ズレない
□ すぐに（10秒以内）合わせられる
□ Tempo Faderの微調整ができる
```

### 上級レベル（3ヶ月-）

```
□ どんなBPM差でも合わせられる
□ 5分以上ズレない
□ 即座に（5秒以内）合わせられる
□ 目をつぶっても合わせられる
□ 違うジャンルでも対応可能
```

## まとめ

- **定義**: 2曲のBPMと位相を一致させる技術
- **最重要**: これができないとDJではない
- **2段階**: BPM合わせ → 位相合わせ
- **DDJ-FLX4**: ジョグホイール + Tempo Fader で実践
- **練習**: Sync OFF、毎日30分、同じ曲を繰り返す
- **目標**: 1ヶ月で30秒、2ヶ月で2分ズレない
- **耳**: 最も重要、ズレを聞き取る力を鍛える

**次のステップ:** [ミキシング基礎](./mixing-basics.md) でスムーズな曲つなぎを学ぶ
