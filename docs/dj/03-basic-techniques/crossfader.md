# クロスフェーダー

クロスフェーダーの使い方とSmart CFX機能を学びます。

## この章で学ぶこと

- クロスフェーダーとは
- チャンネルフェーダーとの違い
- 基本的な使い方
- カーブ設定
- ミックスDJ vs スクラッチDJ
- DDJ-FLX4のSmart CFX
- 使い分け

## なぜクロスフェーダーを学ぶか

**スタイルによって重要度が異なる:**
```
ミックスDJ:
- ほとんど使わない
- チャンネルフェーダー中心
- 中央固定が一般的

スクラッチDJ:
- 必須
- 高速カット
- カーブ設定重要
```

---

## 1. クロスフェーダーとは

### 定義

```
クロスフェーダー = 2デッキを切り替えるフェーダー

位置:
左（A）: Deck A のみ
中央: A + B 両方
右（B）: Deck B のみ
```

### DDJ-FLX4の配置

```
┌─────────────────┐
│ Deck A  Deck B  │
├─────────────────┤
│                 │
│  Crossfader     │
│  A [===] B      │
└─────────────────┘
```

---

## 2. チャンネルフェーダーとの違い

### チャンネルフェーダー

```
機能: 各デッキの音量を個別に調整

特徴:
✓ 独立している
✓ 同時に両方上げられる
✓ 細かい調整が可能

用途: ミックスDJ（主流）
```

### クロスフェーダー

```
機能: 2デッキを切り替える

特徴:
✓ 連動している
✓ 片方が上がると片方が下がる
✓ 高速操作が可能

用途: スクラッチDJ、カットミックス
```

---

## 3. 基本的な使い方

### ミックスDJの使用法

**中央固定:**
```
多くのミックスDJ:
- クロスフェーダーを中央に
- 触らない
- チャンネルフェーダーのみ使用

理由:
- より細かい調整が可能
- 両手が使える
- EQ操作と同時にできる
```

### クロスフェーダーミックス

**手順:**
```
1. 両方のチャンネルフェーダーを上げる
2. クロスフェーダーを左（Deck A）
3. ビートマッチング
4. フレーズの頭でクロスフェーダーを中央へ
5. ゆっくり右（Deck B）へ移動
6. 16-32 bar かけて

結果: Deck A → Deck B
```

## 4. カーブ設定

### カーブとは

```
カーブ = フェーダー移動に対する音量変化の割合

種類（DDJ-FLX4）:
- Smooth（なだらか）
- Fast（急峻）
```

### Smooth カーブ（ミックスDJ向け）

```
フェーダー位置 → 音量
0% → 0%
25% → 15%
50% → 50%
75% → 85%
100% → 100%

特徴:
✓ ゆっくりとした変化
✓ ミックスに最適
✓ 細かい調整が可能
```

### Fast カーブ（スクラッチDJ向け）

```
フェーダー位置 → 音量
0% → 0%
25% → 5%
50% → 50%
75% → 95%
100% → 100%

特徴:
✓ 急激な変化
✓ カットに最適
✓ 少しの動きで大きく変化
```

### DDJ-FLX4での設定

**Rekordbox設定:**
```
[Preferences] → [Controller]
→ DDJ-FLX4 → Crossfader Curve

選択:
- Smooth（ミックス）
- Fast（スクラッチ）

推奨: Smooth（初心者）
```

## 5. ミックスDJ vs スクラッチDJ

### ミックスDJ

**スタイル:**
```
- ロングミックス
- EQ使用
- スムーズなトランジション
```

**クロスフェーダー使用:**
```
✗ ほとんど使わない
→ チャンネルフェーダー中心

理由:
- 両手で EQ 操作
- より細かい調整
- 業界標準
```

### スクラッチDJ

**スタイル:**
```
- カットミックス
- スクラッチ
- 高速切り替え
```

**クロスフェーダー使用:**
```
✓ 必須
→ 片手でクロスフェーダー
→ もう片手でジョグホイール

理由:
- 高速カット必要
- スクラッチと連動
- Hip Hop、Breaks標準
```

## 6. DDJ-FLX4のSmart CFX

### Smart CFX とは

**機能:**
```
クロスフェーダーに自動エフェクト

種類:
- Echo（ディレイ）
- Filter（フィルター）
- Wash Out（フェードアウト）
- その他

使用:
Smart CFX ボタン ON
→ クロスフェーダー操作でエフェクト発動
```

### 使い方

**基本操作:**
```
1. Smart CFX ボタンを押す
2. エフェクト種類を選択
3. クロスフェーダーを操作
4. 自動でエフェクト付き切り替え
```

**例: Echo**
```
クロスフェーダーを A → B へ移動:
→ Deck A にエコーがかかる
→ エコーの余韻が残る
→ Deck B にスムーズに移行
```

### 使うべきか？

**練習時:**
```
✗ 使わない
理由:
- 基礎が身につかない
- 手動の感覚が掴めない
```

**本番:**
```
○ クリエイティブに使用
使用場面:
- トランジション時
- 実験的なミックス
- 観客を驚かせる

ただし基礎ができている前提
```

## 7. 実践テクニック

### Technique 1: クロスフェーダーカット

**用途:** Hip Hop、Breaks
```
1. 両チャンネルフェーダー上げる
2. クロスフェーダーを高速で左右
3. リズムに合わせてカット
4. Deck A と Deck B を交互に
```

### Technique 2: Smart CFX トランジション

**用途:** House、Techno
```
1. Smart CFX ON（Echo選択）
2. ビートマッチング
3. クロスフェーダーをゆっくり移動
4. 自動でエコーエフェクト
5. スムーズなトランジション
```

### Technique 3: ハーフウェイミックス

**用途:** 2曲を同時に
```
1. 両チャンネルフェーダー上げる
2. クロスフェーダーを中央
3. 両方の曲が鳴る
4. EQ で Low 管理
5. 長時間維持可能
```

## 8. 練習方法

### 初級: チャンネルフェーダー中心

**推奨（ミックスDJ志望）:**
```
- クロスフェーダーは触らない
- 中央固定
- チャンネルフェーダーのみ使用
- 両手で EQ 操作

理由: 業界標準、細かい調整
```

### 中級: クロスフェーダーも試す

**練習:**
```
1. クロスフェーダーミックス 10回
2. チャンネルフェーダーミックス 10回
3. 比較して違いを理解
```

### 上級: Smart CFX 活用

**練習:**
```
1. 基礎ができている前提
2. Smart CFX を ON
3. 様々なエフェクトを試す
4. クリエイティブなミックス
```
