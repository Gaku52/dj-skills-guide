# キューイング

ヘッドフォンでの次の曲の準備、Cueの使い方を学びます。

## この章で学ぶこと

- キューイングとは
- Cueボタンの使い方
- ヘッドフォンモニタリング
- 曲の頭出し
- DDJ-FLX4での実践
- ミックスの準備

## なぜキューイングが重要か

**ダンスフロアを止めない:**
```
良いDJ:
✓ 次の曲を完璧に準備
✓ スムーズなミックス
✓ ダンスフロアが止まらない

悪いDJ:
✗ 次の曲を探している
✗ 準備不足
✗ ダンスフロアが止まる
```

---

## 1. キューイングとは

### 定義

```
キューイング（Cueing）= 次の曲の準備

含まれること:
1. 曲の選択
2. ロード
3. 頭出し
4. ビートマッチング準備
5. タイミング計算
```

---

## 2. Cueボタン

### DDJ-FLX4のCueボタン

```
各デッキに1つ:
Deck A: Cue ボタン
Deck B: Cue ボタン

機能:
- ヘッドフォンで確認
- 曲の頭出し
- 一時停止
```

### 基本操作

**ヘッドフォンで聞く:**
```
1. Cueボタンを押す
2. ヘッドフォンでそのデッキが聞こえる
3. スピーカーからは聞こえない
4. チャンネルフェーダーが下でもOK
```

**曲の頭出し:**
```
再生中:
1. Playボタンで一時停止
2. Cueボタンを押す
3. その位置がCueポイントに設定

次回:
Cueボタンを押す → Cueポイントから再生
```

---

## 3. ヘッドフォンモニタリング

### Headphone Mixingノブ

**DDJ-FLX4:**
```
┌──────────────┐
│ Master  Cue  │
│   ←  ○  →   │
└──────────────┘

左: Master のみ
中央: Master + Cue 両方
右: Cue のみ
```

### 使い方

**ビートマッチング時:**
```
位置: 中央〜Cue寄り

聞こえる:
- 左耳: Master（曲A）
- 右耳: Cue（曲B）

両方聞いてビートを合わせる
```

**曲選択時:**
```
位置: Cue側

聞こえる:
- Cue（次の曲候補）のみ

スピーカーから流れている曲の邪魔にならない
```

## 4. 曲の頭出し

### Step 1: Cueポイント設定

**自動設定（Rekordbox）:**
```
曲をロード時:
→ 自動で最初の音が出る位置にCue
→ ほとんどの場合問題なし
```

**手動設定:**
```
1. Playで再生
2. 希望の位置で一時停止
3. Cueボタンを押す
4. 新しいCueポイントに設定
```

### Step 2: Cueから再生

**方法1: Play + Cue**
```
1. Cueボタンを押す（Cueポイントへ移動）
2. Playボタンを押す
3. そこから再生開始
```

**方法2: Cueボタン長押し**
```
Cueボタンを押し続ける:
→ 押している間だけ再生
→ 離すとCueポイントに戻る

用途: ビートの確認
```

---

## 5. ビートマッチング準備

### Step 1: 次の曲をロード

```
タイミング:
- 現在の曲が1-2分残っている時
- 早すぎない、遅すぎない
```

### Step 2: Cueで確認

```
1. Cueボタンを押す
2. ヘッドフォンで聞く
3. BPM確認
4. キー確認（Rekordbox画面）
5. 曲構造の把握
```

### Step 3: ビートの確認

```
Cueボタン長押し:
- キックのタイミング確認
- ビートの強さ確認
- フレーズの頭を探す
```

### Step 4: ビートマッチング開始

```
1. Tempo Faderで BPM 調整
2. ジョグホイールで位相合わせ
3. Cueボタン ON のまま
4. 完璧に合ったら Cueボタン OFF
5. ミックス開始
```

---

## 6. タイミング管理

### ミックス開始のタイミング

**理想:**
```
曲Aの残り:
- 1分: 次の曲を探す
- 45秒: 次の曲をロード
- 30秒: ビートマッチング開始
- 15秒: ミックス開始準備完了
- 0秒: ミックス開始（フレーズの頭で）
```

### フレーズカウント

**Rekordbox波形:**
```
現在位置から次のフレーズまで:
- 4 bar = 約8秒（128 BPM）
- 8 bar = 約16秒
- 16 bar = 約32秒
- 32 bar = 約64秒

これを見て逆算:
「あと32秒で次のフレーズの頭
→ 今ミックス準備開始」
```

## 7. DDJ-FLX4での実践

### 練習1: Cue操作

```
目標: Cue操作に慣れる

1. 曲をロード
2. Cueボタンで確認
3. Playで再生
4. 一時停止
5. Cueで頭出し
6. 繰り返し（10回）
```

### 練習2: ビートマッチング with Cue

```
目標: Cue使用でビートマッチング

1. 曲Aを再生
2. 曲BをCueで確認
3. ビートマッチング
4. Cueボタン ON のまま
5. 完璧に合ったら OFF
6. ミックス
```

### 練習3: タイミング管理

```
目標: 時間内に準備完了

1. 曲Aを再生
2. 1分後に次の曲をロード（タイマー使用）
3. 45秒でビートマッチング開始
4. 30秒で準備完了
5. フレーズの頭でミックス
```

## 8. よくある失敗と対処法

### 失敗1: Cueポイントがズレている

**原因:**
```
自動Cueポイントが曲の途中
```

**対処:**
```
1. 手動で設定し直す
2. Rekordboxで事前に設定
3. Hot Cue使用（上級）
```

### 失敗2: ビートマッチング中にCue OFF忘れ

**原因:**
```
集中しすぎ
```

**対処:**
```
習慣化:
ビートマッチング完了
→ すぐCue OFF
→ ミックス開始
```

### 失敗3: 準備が間に合わない

**原因:**
```
タイミング管理の失敗
```

**対処:**
```
1. 早めに準備開始（1分前）
2. 曲を覚える（構造を把握）
3. プレイリスト整理
```

## 9. 上級テクニック

### Hot Cue使用

**DDJ-FLX4:**
```
Performance Pads:
- 複数のCueポイント設定可能
- 瞬時にジャンプ

使用:
曲の重要なポイントに設定
→ ビルドアップ
→ ドロップ
→ ブレイクダウン
```

### Slip Cue

**機能:**
```
Cue中もバックグラウンドで曲は進行

用途:
- Cueで確認
- 離すと本来の位置に戻る
```
