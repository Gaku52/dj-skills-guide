# EQ操作

ミックスの質を決める、EQ（イコライザー）操作を完全マスターします。

## この章で学ぶこと

- EQとは何か
- 3バンドEQの理解
- 周波数帯域の役割
- ミックス時のEQ調整
- Low重複の回避
- DDJ-FLX4でのEQ操作
- 実践的なテクニック
- よくある失敗

## なぜEQ操作が重要か

**ミックスの質を決める:**
```
ビートマッチング = 曲を合わせる
EQ操作 = 音をクリアにする

良いEQ操作:
✓ 濁らないミックス
✓ クリアなサウンド
✓ プロフェッショナル

悪いEQ操作:
✗ 音が濁る
✗ Low が重複
✗ アマチュア感
```

**プロとアマチュアの差:**
```
アマチュア: EQ フラット（12時固定）
プロ: EQ を積極的に使う

プロの特徴:
- Low を常に調整
- Mid、Hi も適宜カット
- 音の空間を作る
```

---

## 1. EQとは

### 定義

```
EQ (Equalizer) = 音質を調整するツール

機能:
- 特定の周波数帯域をブースト（上げる）
- 特定の周波数帯域をカット（下げる）
```

### 3バンドEQ（DDJ-FLX4）

```
┌────────┐
│  Hi    │ 高域（High）
├────────┤
│  Mid   │ 中域（Middle）
├────────┤
│  Low   │ 低域（Low）
└────────┘

各ノブ:
- 12時: フラット（原音）
- 右回し: ブースト（+6dB）
- 左回し: カット（-26dB）
- 完全左: 完全カット（-∞dB）
```

---

## 2. 周波数帯域の理解

### Low（低域）: 20-250Hz

**音の特徴:**
```
- キック（バスドラム）
- ベースライン
- サブベース
- 重低音
```

**印象:**
```
✓ パワフル
✓ 体に響く
✓ ダンスフロアの基礎
✗ 多すぎると濁る
```

**DDJ-FLX4:**
```
Low ノブ: 約 70Hz を中心に調整
```

### Mid（中域）: 250Hz-4kHz

**音の特徴:**
```
- ボーカル
- シンセメロディ
- スネア
- 多くの楽器
```

**印象:**
```
✓ 明瞭さ
✓ メロディ
✓ 存在感
✗ 多すぎると耳が痛い
```

**DDJ-FLX4:**
```
Mid ノブ: 約 1kHz を中心に調整
```

### Hi（高域）: 4kHz-20kHz

**音の特徴:**
```
- ハイハット
- シンバル
- エアー感
- 煌びやか さ
```

**印象:**
```
✓ 明るさ
✓ 抜け
✓ クリアさ
✗ 多すぎると刺さる
```

**DDJ-FLX4:**
```
Hi ノブ: 約 13kHz を中心に調整
```

## 3. 基本的なEQ操作

### フラット（原音）

```
Hi: 12時
Mid: 12時
Low: 12時

使用:
- 曲が単独で再生されている時
- ミックス外
```

### ブースト（+6dB）

```
ノブを右に回す

効果:
- その帯域が強調される
- 音量が上がる

注意:
✗ 多用しない
✗ クリッピングの原因
```

### カット（-26dB）

```
ノブを左に回す

効果:
- その帯域が減少
- 音量が下がる
- 空間ができる

推奨:
✓ 積極的に使う
✓ ミックスの基本
```

### 完全カット（-∞dB）

```
ノブを完全に左（DDJ-FLX4）

効果:
- その帯域が完全に消える
- 他の帯域は残る

使用:
- Low カット（最も一般的）
- トランジション時
```

## 4. ミックス時のEQ調整（最重要）

### Low重複の回避（絶対ルール）

**問題:**
```
曲A: Low フラット（キック鳴っている）
曲B: Low フラット（キック鳴っている）
↓
Low が重複
↓
音が濁る、パワーが失われる
```

**解決:**
```
ミックス時:
曲A（マスター）: Low フラット
曲B（新規）: Low 完全カット

徐々に:
曲B の Low を上げる
同時に曲A の Low を下げる

結果:
Low は常に1曲だけ（または適度に分散）
```

### 基本的なEQミックス手順

**Step 1: 曲Bをミックスイン（Low カット）**
```
1. 曲B をビートマッチング
2. 曲B の Low を完全カット（左いっぱい）
3. 曲B のフェーダーを上げる
4. 高域・中域のみが聞こえる
```

**Step 2: Low の入れ替え**
```
タイミング: 8-16 bar 後

同時に:
- 曲B の Low を徐々に上げる（左→12時）
- 曲A の Low を徐々に下げる（12時→左）

4-8 bar かけてゆっくり
```

**Step 3: 曲Aをフェードアウト**
```
曲B の Low が完全に戻ったら:
- 曲A の全EQ を下げる
- 曲A のフェーダーを下げる
- 曲B のみになる
```

## 5. DDJ-FLX4でのEQ操作

### EQノブの配置

```
Deck A:
┌────┐
│ Hi │
│Mid │
│Low │
└────┘

Deck B:
┌────┐
│ Hi │
│Mid │
│Low │
└────┘
```

### 操作のコツ

**両手を使う:**
```
左手: Deck A の EQ
右手: Deck B の EQ

同時操作:
- Deck A の Low を下げながら
- Deck B の Low を上げる
```

**滑らかに:**
```
✓ ゆっくり回す
✓ 4-8 bar かけて
✗ 急激に変化させない
```

## 6. 実践的なEQテクニック

### Technique 1: Low カットミックス

**最も基本:**
```
用途: House, Techno

手順:
1. 曲B Low 完全カット
2. フェードイン
3. Low 入れ替え
4. 完成
```

### Technique 2: Hi/Mid カット（フィルター効果）

**用途:** ブレイク、ビルドアップ
```
曲が盛り上がる直前:

1. Hi を徐々にカット
2. Mid もカット
3. Low のみになる
4. Drop 直前で全て戻す
5. 爆発的な盛り上がり
```

### Technique 3: Mid カット（ボーカル回避）

**用途:** ボーカル曲のミックス
```
問題: 2曲のボーカルが重なる

解決:
- 出ていく曲: Mid をカット
→ ボーカルが消える
- 入ってくる曲: Mid フラット
→ ボーカルが際立つ
```

### Technique 4: Hi ブースト（明るさ追加）

**用途:** Breakdown、静かなセクション
```
エネルギーを保ちたい時:
- Hi をわずかにブースト（1-2時方向）
→ 明るさ、抜けが良くなる
→ ダンスフロアのエネルギー維持
```

### Technique 5: Low スワップ

**用途:** 2曲のベースを交互に
```
8 bar ごとに:
- 曲A の Low: フラット → カット → フラット
- 曲B の Low: カット → フラット → カット
→ ダイナミックなミックス
```

## 7. ジャンル別EQ戦略

### House / Tech House

**特徴:**
```
- 4つ打ちキック重要
- ベースライン明確
```

**EQ戦略:**
```
✓ Low の管理を最優先
✓ Mid は比較的フラット
✓ Hi で明るさ調整
```

### Techno

**特徴:**
```
- ミニマル
- キックとハイハット主体
```

**EQ戦略:**
```
✓ Low 完璧に管理
✓ Mid は積極的にカット
✓ Hi は残す（ハイハット）
```

### Drum & Bass

**特徴:**
```
- 高速ビート
- サブベース重要
```

**EQ戦略:**
```
✓ Low 非常に重要
✓ 短いミックス
✓ カットミックス多用
```

### Hip Hop

**特徴:**
```
- ボーカル中心
- ビート重要
```

**EQ戦略:**
```
✓ Mid を注意（ボーカル）
✓ Low はキープ
✓ カットミックス推奨
```

## 8. よくある失敗と対処法

### 失敗1: Low が濁る

**原因:**
```
2曲の Low が同時に鳴っている
```

**対処:**
```
✓ 必ず1曲の Low をカット
✓ 入れ替えはゆっくり
✓ 完全に戻ってから次へ
```

### 失敗2: 音が薄い

**原因:**
```
全体的にカットしすぎ
```

**対処:**
```
✓ 1曲はフラット維持
✓ カットは最小限
✓ ブーストは基本使わない
```

### 失敗3: 急激な変化

**原因:**
```
EQ を急に回しすぎ
```

**対処:**
```
✓ 4-8 bar かけてゆっくり
✓ 滑らかに回す
✓ フレーズの頭で開始
```

### 失敗4: どのEQを触るか迷う

**原因:**
```
経験不足、音の聞き取り不足
```

**対処:**
```
✓ まずは Low だけに集中
✓ 曲を聴き込んで覚える
✓ 練習を繰り返す
```

## 9. 練習方法

### Week 1: Low カットのみ

**練習:**
```
1. 2曲を用意
2. ビートマッチング
3. 曲B Low 完全カット
4. フェードイン
5. Low 入れ替え（8 bar）
6. 完成

100回繰り返す
```

**目標:**
```
□ 滑らかな Low 入れ替え
□ 濁らない
```

### Week 2: Hi/Mid も使う

**練習:**
```
1. Low カットミックス（基本）
2. Hi をわずかにカット/ブースト
3. Mid をカット（ボーカル時）
4. 様々なパターン
```

**目標:**
```
□ 3バンド全て使える
□ 状況に応じて判断
```

### Week 3-4: 実践

**練習:**
```
30分ミックス録音:
- 全てEQを使う
- Low は必ず管理
- Mid、Hi で表現

聞き返して改善
```

## 10. EQ完成の基準

### 初級レベル（1ヶ月）

```
□ Low カットミックスができる
□ Low 入れ替えが滑らかにできる
□ 濁らない
```

### 中級レベル（2ヶ月）

```
□ Hi、Mid も使える
□ ジャンルに応じたEQ戦略
□ 30分ミックスで適切なEQ使用
```

### 上級レベル（3ヶ月-）

```
□ 無意識にEQ操作
□ クリエイティブなEQ使用
□ プロレベルのクリアなサウンド
```
