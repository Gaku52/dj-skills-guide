# ゲインステージング

音量管理とクリッピング回避、ゲインステージングを学びます。

## この章で学ぶこと

- ゲインステージングとは
- なぜ重要なのか
- Gain/Trimの使い方
- Level Meterの見方
- クリッピングの回避
- DDJ-FLX4での実践
- 音圧の一貫性

## なぜゲインステージングが重要か

**音質を決める:**
```
良いゲインステージング:
✓ クリアな音質
✓ 一貫した音圧
✓ クリッピングなし
✓ プロフェッショナル

悪いゲインステージング:
✗ 音割れ（クリッピング）
✗ 音量バラバラ
✗ PA負荷大
✗ アマチュア感
```

**プロとアマチュアの差:**
```
アマチュア: Master Volume で音量調整
プロ: 各チャンネルの Gain で調整

結果:
アマチュア → 音割れ、音量不安定
プロ → クリアな音、一貫した音圧
```

---

## 1. ゲインステージングとは

### 定義

```
ゲインステージング = 信号の流れ全体で音量を最適化

信号の流れ:
曲 → Gain/Trim → EQ → Fader → Master → PA

各段階で適切な音量:
→ クリッピング回避
→ 音質維持
→ 一貫した音圧
```

---

## 2. DDJ-FLX4のGain/Trim

### 配置

```
各デッキの上部:
Deck A: Trim ノブ
Deck B: Trim ノブ

機能: 入力レベル調整
```

### 役割

```
目的:
全ての曲の音量を揃える

理由:
曲ごとに録音レベルが異なる
→ Trimで調整
→ 同じ音量に統一
```

---

## 3. Level Meter

### Rekordbox Level Meter

```
表示:
Green: -18dB 〜 -6dB（適正）
Yellow: -6dB 〜 0dB（注意）
Red: 0dB 以上（クリッピング）

目標: Green の上部
```

### DDJ-FLX4 Level Indicator

```
LEDインジケーター:
Green: 適正
Red: クリッピング

目標: Green 点灯、Red 点灯しない
```

---

## 4. 正しいゲインステージング手順

### セットアップ時

**Step 1: 全てリセット**
```
1. 全チャンネルフェーダー: 下
2. 全Trim: 12時
3. Master Volume: 12時（または指定位置）
4. 全EQ: 12時（フラット）
```

**Step 2: 1曲目のGain調整**
```
1. Deck A に曲をロード
2. Playで再生
3. チャンネルフェーダーを最大に
4. Level Meter を見る
5. Trimで調整:
   - 目標: Green の上部（約-6dB〜0dB）
   - Red は絶対NG
6. チャンネルフェーダーを下げる（70-80%）
```

**Step 3: 2曲目も同様に**
```
Deck B も同じ手順:
→ Deck A と同じ音量になるように
```

### ミックス時

**曲ごとに調整:**
```
新しい曲をロード時:
1. Cueで確認
2. Trimで音量調整
3. Level Meterを見る
4. 前の曲と同じ音量に
```

---

## 5. クリッピングとは

### 定義

```
クリッピング = 音量が大きすぎて波形が歪む

視覚的:
正常: 滑らかな波形
クリッピング: 波形の頭が切れる

聴覚的:
正常: クリアな音
クリッピング: 歪んだ音、ビリビリ音
```

### 影響

```
音質:
✗ 歪む
✗ 不快な音
✗ 周波数成分が崩れる

機材:
✗ スピーカー破損の可能性
✗ PA負荷大
✗ アンプ保護回路が働く
```

---

## 6. 適切な音量レベル

### 各段階の目標

**Trim後（Channel）:**
```
目標: -6dB 〜 0dB（Green の上部）
- Red は絶対NG
- Yellow上部までOK
- 余裕を持たせる
```

**Fader後:**
```
Fader位置: 70-80%
→ 調整の余地を残す
→ 緊急時に上げられる
```

**Master:**
```
目標: -6dB 〜 0dB
- セット中は触らない
- サウンドチェック時に設定
- PA担当と相談
```

---

## 7. DDJ-FLX4での実践

### 基本設定

```
1. Rekordbox起動
2. [Preferences] → [Audio]
3. Output Level: 確認（通常デフォルトでOK）
```

### 曲ごとのGain調整

**手順:**
```
新しい曲をロード:
1. Cueボタン ON
2. Playで再生（ヘッドフォンのみ）
3. Rekordbox Level Meter を見る
4. Trimで調整
5. チャンネルフェーダー最大時に Green 上部
6. Cue OFF、フェーダー下げる
```

### ミックス中の確認

```
両方の曲が鳴っている時:
- Master Level Meter を確認
- Red になっていないか
- なっていたら:
  → 両方のTrimをわずかに下げる
  または
  → どちらかのチャンネルフェーダーを下げる
```

## 8. よくある失敗と対処法

### 失敗1: 音が割れる（クリッピング）

**原因:**
```
Trimが大きすぎる
```

**対処:**
```
1. すぐにTrimを下げる
2. Level Meterを確認
3. Green上部を目標に
4. Red絶対NG
```

### 失敗2: 曲ごとに音量が違う

**原因:**
```
Gain調整をしていない
```

**対処:**
```
全ての曲で:
1. ロード時にCueで確認
2. Trimで調整
3. 前の曲と同じ音量に
4. 習慣化
```

### 失敗3: 音が小さすぎる

**原因:**
```
Trimが小さすぎる
または
Master Volumeが小さい
```

**対処:**
```
Trim調整:
- Green上部まで上げる
- ダイナミックレンジ確保

Master Volume:
- サウンドチェック時に設定
- セット中は触らない
```

### 失敗4: Master Volumeで音量調整してしまう

**原因:**
```
ゲインステージングの理解不足
```

**対処:**
```
正しい方法:
✓ Trimで各曲の音量を揃える
✓ チャンネルフェーダーで調整
✗ Master Volumeは固定
```

## 9. クラブでのゲインステージング

### サウンドチェック

```
到着時:
1. PA担当に挨拶
2. Master Volume位置を確認
3. テスト曲を流す
4. Trimで調整
5. チャンネルフェーダー70-80%
6. Master Level を確認
7. PA担当に確認
8. 問題なければOK
```

### セット中

```
原則:
✓ Master Volumeは触らない
✓ Trimで各曲を調整
✓ チャンネルフェーダーで微調整
✓ Level Meterを常に確認
```

## 10. 練習方法

### Week 1: Gain調整の習慣化

```
練習:
全ての曲をロード時:
1. Cueで確認
2. Trimで調整
3. Level Meter確認
4. Green上部を目標

目標: 習慣化、自動的にできる
```

### Week 2: ミックス中のLevel管理

```
練習:
30分ミックス:
- 全曲でGain調整
- Master Level Meter 確認
- Redにならない
- 録音して聞き返す

目標: 一貫した音圧
```

## まとめ

- **定義**: 信号の流れ全体で音量を最適化
- **Trim**: 各曲の音量を揃える
- **目標**: Green上部（-6dB〜0dB）、Red は NG
- **原則**: Master固定、Trimで調整
- **クリッピング**: 絶対回避、音質劣化
- **DDJ-FLX4**: Level Meter常に確認
- **習慣**: 全曲ロード時にGain調整
