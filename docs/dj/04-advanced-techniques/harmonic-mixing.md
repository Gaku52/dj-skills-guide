# ハーモニックミキシング

キーを合わせた音楽的に美しいミックス、ハーモニックミキシングを完全マスターします。

## この章で学ぶこと

- ハーモニックミキシングとは
- Camelot Wheel完全理解
- Rekordboxでのキー確認
- 互換キーの選び方
- キーシフトの使い方
- セット全体の構築
- 実践的なテクニック

## なぜハーモニックミキシングが重要か

**音楽的に美しい:**
```
キー無視のミックス:
- 不協和音
- 違和感
- アマチュア感

ハーモニックミキシング:
- 和音が美しい
- 自然な流れ
- プロフェッショナル
```

**House/Technoでは必須:**
```
✓ ロングミックス（32 bar以上）
✓ 両方の曲が長時間鳴る
✓ キーが合わないと目立つ
✓ プロは必ず意識している
```

---

## 1. ハーモニックミキシングとは

### 定義

```
ハーモニックミキシング = キー（調）を合わせたミックス

目的:
- 2曲が同時に鳴っても和音が美しい
- 不協和音を避ける
- 音楽的に自然な流れ
```

### 効果

**良い例（キーが合っている）:**
```
曲A: 8A（Am）
曲B: 8A（Am）または 8B（C）

結果: 和音が美しい、違和感なし
```

**悪い例（キーが合っていない）:**
```
曲A: 8A（Am）
曲B: 1A（Abm）

結果: 不協和音、違和感
```

## 2. Camelot Wheel完全理解

### Camelot Wheelとは

```
Camelot Wheel = キーの互換性を示す円形図

12の番号 × 2タイプ（A/B）= 24キー

A: マイナーキー（暗い）
B: メジャーキー（明るい）
```

### Camelot Wheel図

```
      12B
   11B   1B
  10B     2B
   9B    3B
     8B
   7B    4B
  6B      5B

      12A
   11A   1A
  10A     2A
   9A    3A
     8A
   7A    4A
  6A      5A

中央: どのキーとも互換性あり
```

### 互換キーのルール

**ルール1: 同じ番号は完璧に合う**
```
8A + 8A = 完璧
8B + 8B = 完璧

理由: 同じキー
```

**ルール2: ±1番号は合う**
```
8A + 7A = 良い
8A + 9A = 良い

理由: 隣接キー、音楽理論的に近い
```

**ルール3: A ↔ B は合う（同じ番号）**
```
8A + 8B = 良い

理由:
8A = Am（Aマイナー）
8B = C（Cメジャー）
→ 関係調（Relative Key）
```

**ルール4: それ以外は避ける**
```
8A + 3A = NG
8A + 12B = NG

理由: 音楽理論的に遠い
```

## 3. Rekordboxでのキー確認

### キー解析

**自動解析:**
```
1. 曲をRekordboxにインポート
2. 右クリック → 解析
3. キーが自動検出
4. Camelot表記で表示（設定次第）
```

### 表示設定

**Camelot表記に変更:**
```
1. [Preferences] → [View]
2. Key Display: Camelot
3. Apply

表示例:
- 8A（Am）
- 8B（C）
- 9A（Em）
```

### ブラウザでの確認

**カラム表示:**
```
Rekordbox Browser:
Title | Artist | BPM | Key | ...

Key列:
8A、8B、9A等が表示

ソート可能: Keyでソート→同じキーの曲を探せる
```

## 4. 互換キーの選び方

### 基本戦略

**Step 1: 現在の曲のキーを確認**
```
再生中: 8A（Am）
```

**Step 2: 次の曲の候補を探す**
```
互換キー:
- 8A（同じ）
- 7A（隣接）
- 9A（隣接）
- 8B（関係調）

Rekordboxで検索:
Key列でフィルター
```

**Step 3: BPMも考慮**
```
キー: 8A
BPM: 126-130（±3%以内）

両方の条件を満たす曲を選択
```

### Rekordboxの Related Tracks 機能

**便利な機能:**
```
再生中の曲を右クリック
→ Related Tracks

表示:
- 同じキーの曲
- 似たBPMの曲
- AIによる推薦

非常に便利
```

## 5. キーシフト（Keylock）

### Master Tempo復習

**DDJ-FLX4:**
```
Master Tempo ON:
- BPMを変えてもキーは変わらない
- ハーモニックミキシングに便利

例:
曲B: 132 BPM, 8A
曲A: 128 BPM, 8A

Master Tempo ON で曲Bを128に
→ キーは8Aのまま
→ 完璧なハーモニックミキシング
```

### キーシフト（上級）

**Rekordboxの機能:**
```
曲を半音上げ/下げ

例:
曲B: 9A（Em）
→ -1半音
→ 8A（Am）

結果: 曲Aとキーが合う
```

**注意:**
```
音質がわずかに劣化
→ 多用は避ける
→ どうしても使いたい曲の場合のみ
```

## 6. セット全体の構築

### エネルギーカーブとキー

**基本戦略:**
```
エネルギー: 上げる/維持/下げる
キー: 互換性を保つ

両方を同時に考える
```

### 60分セット例

**構成:**
```
0-15分: ウォームアップ
- キー: 8A中心
- BPM: 124-126
- エネルギー: 低→中

15-30分: 徐々に盛り上げる
- キー: 8A → 9A → 10A（+1ずつ）
- BPM: 126-128
- エネルギー: 中→高

30-45分: ピーク維持
- キー: 10A → 10B → 11A（A/B切り替え、±1）
- BPM: 128-130
- エネルギー: 高

45-60分: クールダウン
- キー: 11A → 10A → 9A（-1ずつ）
- BPM: 128-126
- エネルギー: 高→中
```

### キーの流れ

**スムーズな流れ:**
```
8A → 8A → 7A → 7A → 8A → 8B → 8B → 9B → 9A → 9A

ルール:
- 同じキーを2-3曲続ける
- 変えるときは±1またはA/B
- 急激な変化は避ける
```

## 7. 実践的なテクニック

### Technique 1: 同じキーで変化をつける

**問題:**
```
同じキーばかりで飽きる
```

**解決:**
```
8A → 8B → 8A → 8B

A/Bを交互に:
- キーは互換
- マイナー/メジャーで変化
- 飽きない
```

### Technique 2: 円を描くように移動

**例:**
```
8A → 9A → 10A → 10B → 9B → 8B → 8A

Camelot Wheel上で円を描く:
- スムーズな流れ
- 音楽的に自然
- プロの定番
```

### Technique 3: エネルギー変化とキー変化

**上げる:**
```
キーを+1ずつ上げる
+ BPMを上げる
= エネルギーアップ

8A（124 BPM）→ 9A（126 BPM）→ 10A（128 BPM）
```

**下げる:**
```
キーを-1ずつ下げる
+ BPMを下げる
= クールダウン

10A（128 BPM）→ 9A（126 BPM）→ 8A（124 BPM）
```

## 8. よくある質問

### Q1: 必ず完璧にキーを合わせる必要がある？

**A: ミックスの長さによる**

```
短いミックス（8-16 bar）:
→ キーが合わなくても大丈夫
→ すぐに切り替わるため

長いミックス（32 bar以上）:
→ キーを合わせるべき
→ 両方が長時間鳴るため
```

### Q2: Hip HopやOpen Formatでも必要？

**A: ジャンルによる**

```
House/Techno:
→ 必須（ロングミックス）

Hip Hop/Breaks:
→ 不要（短いカットミックス）

Open Format:
→ ケースバイケース
→ ロングミックスなら考慮
```

### Q3: Rekordboxのキー検出は正確？

**A: ほぼ正確、たまに間違いも**

```
対処:
1. 耳で確認（最終的には耳）
2. 違和感があれば手動修正
3. 同じキーの曲と比較
```

### Q4: キーが合わない場合は？

**A: 選曲を変える、または短いミックス**

```
選択肢:
1. 別の曲を選ぶ（推奨）
2. キーシフト使用（音質劣化）
3. 短いミックスにする（8 bar）
4. EQでLow完全カット（高域だけミックス）
```

## 9. 練習方法

### Week 1: Camelot Wheel暗記

```
練習:
1. Camelot Wheelを紙に書く
2. 互換キーのルールを覚える
3. ランダムなキーから互換キーを言う
4. 完璧に暗記するまで

目標: 見なくても互換キーがわかる
```

### Week 2: Rekordboxでの曲選び

```
練習:
1. 30曲のプレイリスト作成
2. 全てキーでソート
3. 互換キーの流れを作る
4. 実際にミックス

目標: スムーズなキーの流れ
```

### Week 3: セット構築

```
練習:
60分セット:
- 全てハーモニックミキシング
- エネルギーカーブも考慮
- 録音して聞き返す

目標: 音楽的に美しいセット
```
