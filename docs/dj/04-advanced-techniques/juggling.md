# ジャグリング（2枚使い）

同じレコードを2枚使った高度なテクニック、ジャグリングをマスターします。

## この章で学ぶこと

- ジャグリングとは
- Beat Juggling（ビートジャグリング）
- Phrase Juggling（フレーズジャグリング）
- DDJ-FLX4での実践
- Hot Cue活用
- タイミングとリズム感
- ターンテーブリストへの道

## なぜジャグリングが重要か

**ターンテーブリストの芸術:**
```
ジャグリング = 同じ曲2枚でオリジナルビートを作る

歴史:
- DJ Jazzy Jeff

→ ジャグリングで伝説に

目的: 楽器として使う
```

## 1. ジャグリングとは

### 定義

```
ジャグリング = 同じレコードを2枚使い交互に再生

特徴:
- 同じビート/フレーズを繰り返す
- 独自のリズムパターンを作る
- 観客を驚かせる

起源: 1980年代のHip Hop
```

### 基本原理

**2枚のレコード:**
```
デッキA: ビートの最初
デッキB: 同じビートの最初

交互に再生:
A: キック → B: キック → A: キック → B: キック

結果: 連続したキックのループ
```

**必要なスキル:**
```
前提条件:
□ ビートマッチング完璧
□ スクラッチ基礎完璧
□ Hot Cue設定完璧
□ リズム感が優れている
□ 両手が独立して動く

全て完璧でない場合は先に習得
```

## 2. Beat Juggling（ビートジャグリング）

### 概要

```
Beat Juggling = ドラムのビートを2枚で交互に

用途:
- ブレイク延長
- オリジナルビート作成
- ライブでのパーカッション

最もクラシックなジャグリング
```

### 基本パターン: 2ビートループ

**準備（DDJ-FLX4）:**
```
1. 同じ曲を両デッキにロード
2. ドラムのキックにHot Cue 1設定
3. 両方とも同じ位置
4. Master Tempo ON（両方）
5. BPM完璧に合わせる
```

**Step 1: 基本動作**
```
デッキA:
1. Hot Cue 1を押す
2. 2拍再生（キック - スネア）
3. 2拍目の終わりで止める

デッキB:
1. デッキAが終わる瞬間にHot Cue 1
2. 2拍再生（キック - スネア）
3. 2拍目の終わりで止める

デッキA:
1. またHot Cue 1

繰り返す
```

**タイミング図:**
```
時間軸: |--1--|--2--|--3--|--4--|
デッキA: [キック-スネア]  待機  [キック-スネア]  待機
デッキB:   待機  [キック-スネア]  待機  [キック-スネア]

結果: 連続したキック-スネアパターン
```

### DDJ-FLX4での操作

**両手の動き:**
```
左手: デッキAのHot Cue 1パッド
右手: デッキBのHot Cue 1パッド

リズム:
左 - - - 右 - - - 左 - - - 右
|       |       |       |
1   2   3   4   1   2   3   4

2拍ごとに押す
```

**Channel Fader:**
```
オプション1: 両方上げたまま
- シンプル
- 音が重なる瞬間あり

オプション2: 交互に上げ下げ
- クリーンな音
- 難易度高い
```

## 3. Phrase Juggling（フレーズジャグリング）

### 概要

```
Phrase Juggling = ボーカルやメロディを2枚で交互に

効果:
- 「Yeah - Yeah - Yeah - Yeah」連続
- フレーズの繰り返し
- スタッター効果

使用: Hip Hopのブレイク中
```

### 基本パターン

**準備:**
```
1. ボーカルフレーズのある曲を両デッキに
2. フレーズ開始にHot Cue設定
   例: "Check it out"の"Check"
3. フレーズの長さを把握（2拍 or 4拍）
```

**実行:**
```
デッキA:
"Check it out" 再生

デッキB:
デッキAが終わる瞬間
"Check it out" 再生

デッキA:
またフレーズ開始

結果: "Check it out - Check it out - Check it out"
```

### 応用: スタッター

**短いフレーズを繰り返す:**
```
フレーズ: "Yeah"（1拍）

デッキA: "Yeah" → すぐHot Cue戻る
デッキB: "Yeah" → すぐHot Cue戻る

高速で交互:
"Yeah-Yeah-Yeah-Yeah-Yeah-Yeah"

効果: スタッター、マシンガン効果
```

## 4. DDJ-FLX4での実践

### Performance Pads活用

**Hot Cueモード:**
```
両デッキのPad 1-4:
Pad 1: キック位置
Pad 2: スネア位置
Pad 3: ボーカル開始
Pad 4: ブレイク開始

ジャグリング中:
両手でパッドを交互に押す
```

### Slip Modeとの組み合わせ

**Slip Mode ON:**
```
メリット:
- ジャグリング中も曲は進行
- 解除すると元の位置
- ミックスがスムーズ

使用例:
1. 曲Aが再生中（Bar 64）
2. Slip Mode ON
3. 8 barジャグリング
4. Slip Mode OFF
5. Bar 72に戻る（本来の位置）
```

## 5. 高度なテクニック

### Technique 1: トランスフォームジャグリング

**組み合わせ:**
```
ジャグリング + Transform Scratch

1. デッキAでビートジャグリング開始
2. デッキBのChannel FaderでTransform
3. 「タ・タ・タ・タ」と断続的に
4. デッキ切り替え

効果: リズミカルで複雑
```

### Technique 2: 3デッキジャグリング（概念）

**DDJ-FLX4では不可能だが概念を理解:**
```
デッキA: キック
デッキB: スネア
デッキC: ハイハット

3つを組み合わせて:
オリジナルドラムパターン

DDJ-FLX4: 2デッキまで
上位機材: 3-4デッキ可能
```

### Technique 3: ライブリミックス

**曲の構造を変える:**
```
元の曲:
イントロ - Verse - コーラス - Verse - アウトロ

ジャグリング後:
イントロ - コーラス - コーラス - コーラス - アウトロ

コーラスを何度も繰り返す
→ 盛り上がる
```

## 6. タイミングとリズム感

### 完璧なタイミングの重要性

**1msのズレも許されない:**
```
ビートマッチング: ±5ms許容範囲
ジャグリング: ±1ms許容範囲

理由:
- 同じビートが2回聞こえる
- わずかなズレが目立つ
- プロレベルの精度必要
```

### 練習方法

**メトロノーム使用:**
```
Week 1-4:
1. メトロノーム60 BPMで開始
2. 両手で交互にパッドを叩く
3. 完璧にリズムを刻む
4. 80, 100, 120 BPMへ

Week 5-8:
1. 実際の曲で練習
2. Hot Cueで2ビートループ
3. 100回繰り返す
4. 録音して聞き返す
```

## 9. よくある失敗と対処法

### 失敗1: タイミングがズレる

**原因:**
```
リズム感が不十分
```

**対処:**
```
✓ メトロノーム1000回
✓ ゆっくりから始める（60 BPM）
✓ 焦らない
```

### 失敗2: Hot Cue位置がズレる

**原因:**
```
Hot Cue設定が正確でない
```

**対処:**
```
✓ Rekordboxで波形を拡大
✓ 完璧に1拍目に設定
✓ ビートグリッド確認
```

### 失敗3: 両手がバラバラ

**原因:**
```
両手の独立性が不足
```

**対処:**
```
✓ 片手ずつ練習
✓ ゆっくり両手
✓ 数週間かけて習得
```

### 失敗4: DDJ-FLX4でできないことを試す

**原因:**
```
機材の限界を理解していない
```

**対処:**
```
✓ DDJ-FLX4: 基礎練習用
✓ 本格的ならターンテーブル
✓ 目的を明確に
```
