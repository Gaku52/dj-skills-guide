# ループとHot Cue

Performance Padsを使った創造的なアレンジテクニックを学びます。

## この章で学ぶこと

- Hot Cueとは
- Loopの基礎
- Loop Roll
- Slip Mode
- DDJ-FLX4 Performance Pads
- 即興アレンジ
- クリエイティブな使用例

## なぜループとHot Cueが重要か

**ライブ感と個性:**
```
普通のミックス:
- 曲をそのまま再生
- 予測可能
- 個性が出にくい

ループ/Hot Cue活用:
- その場でアレンジ
- ライブ感
- 個性が爆発
```

**プロデューサーDJスタイル:**
```
✓ 曲を再構築
✓ オリジナリティ
✓ 観客を驚かせる
✓ 記憶に残るセット
```

## 1. Hot Cueとは

### 定義

```
Hot Cue = 曲内の任意の位置に設定するマーカー

機能:
- 瞬時にその位置にジャンプ
- 複数設定可能（DDJ-FLX4: 8個/デッキ）
- 色分けで識別

用途: 即座のセクションジャンプ
```

### 設定方法（DDJ-FLX4）

**基本手順:**
```
1. 曲を再生
2. Hot Cue設定したい位置で
3. Performance Pad Mode → Hot Cue
4. Padを押す（1-8のいずれか）
5. Hot Cue設定完了
6. Padが光る
```

**推奨Hot Cue配置:**
```
Pad 1: イントロ（最初）
Pad 2: Verse開始
Pad 3: ビルドアップ開始
Pad 4: ドロップ
Pad 5: ブレイクダウン
Pad 6: 2nd ドロップ
Pad 7: アウトロ
Pad 8: 特別なポイント
```

---

## 2. Loopの基礎

### Loopとは

```
Loop = 特定の区間を繰り返し再生

種類:
- Auto Loop: 自動で設定（4/8/16/32 beat）
- Manual Loop: 手動で設定（In/Out点）
- Loop Roll: 一時的なループ

用途: セクション延長、アレンジ
```

### Auto Loop（DDJ-FLX4）

**操作:**
```
Performance Pad Mode → Beat Loop

Pad配置:
1/4 beat | 1/2 beat | 1 beat | 2 beat
4 beat  | 8 beat  | 16 beat | 32 beat

Padを押すとループ開始
もう一度押すと解除
```

**推奨:**
```
4 beat または 8 beat が最も使いやすい
→ 1フレーズ = 4 bar = 16 beat
→ 8 beat = 半フレーズ
```

---

## 3. Loop Roll

### 概要

```
Loop Roll = 押している間だけループ

特徴:
- Padを押している間: ループ
- 離すと: 元の位置に戻る
- バックグラウンドで曲は進行

用途: 一時的なアレンジ、フィルイン
```

### DDJ-FLX4での操作

**基本手順:**
```
1. Performance Pad Mode → Beat Loop
2. Shift + Pad（Loop Rollモード）
3. Padを押し続ける
4. 離すと元の位置
```

### 使用シーン

**シーン1: ビルドアップ**
```
ドロップ直前:
1. 1/2 beat Loop Roll
2. Pad押し続ける（4-8拍）
3. 緊張感が高まる
4. ドロップで離す
5. 爆発的な盛り上がり
```

**シーン2: フィルイン**
```
フレーズの終わり:
1. 1/4 beat Loop Roll
2. 2拍だけPad押す
3. リズミカルな変化
4. 次のフレーズへ
```

---

## 4. Slip Mode

### 概要

```
Slip Mode = バックグラウンドで曲は進行し続ける

機能:
- Hot Cue、Loop使用中も曲は進行
- 解除すると本来の位置に戻る
- シームレスなアレンジ

DDJ-FLX4: Slipボタン ON/OFF
```

### 使用例

**例1: Hot Cue + Slip Mode**
```
Slip Mode ON:
1. 曲が再生中（Bar 64）
2. Hot Cue 4（ドロップ）をPad押し
3. ドロップが再生される
4. 4 bar後に離す
5. Bar 68（本来の位置）に戻る

効果: ドロップを繰り返しても曲は進む
```

**例2: Loop + Slip Mode**
```
Slip Mode ON:
1. 8 beat Loopを開始
2. 8 bar間ループ
3. Loop解除
4. 本来の位置（8 bar後）に戻る

効果: ループ使用でも曲の流れが止まらない
```

## 5. 創造的な使用例

### Technique 1: ドロップリピート

**やり方:**
```
1. ドロップにHot Cue設定
2. ドロップが終わる直前に
3. Hot Cue Padを押す
4. もう一度ドロップ
5. 2-3回繰り返す
6. 次の曲へ

効果: ピークのエネルギー維持
```

### Technique 2: ビルドアップ延長

**やり方:**
```
1. ビルドアップで8 beat Loop
2. ループを2-3回繰り返す
3. 緊張感が最大に
4. Loop解除
5. ドロップで爆発

効果: ドロップのインパクトが増す
```

### Technique 3: ブレイク即興

**やり方:**
```
1. ブレイクダウンで4 beat Loop
2. エフェクト追加（Echo）
3. ループを変化させる
4. Loop Roll追加
5. オリジナルアレンジ

効果: 完全にオリジナルな表現
```

### Technique 4: セクションジャンプ

**やり方:**
```
1. イントロで再生開始
2. Hot Cue 4（ドロップ）にジャンプ
3. 即座にエネルギーアップ
4. 次の曲をミックスイン

効果: サプライズ、時間短縮
```

### Technique 5: Hot Cueジャグリング

**やり方:**
```
1. Hot Cue 3と4を交互に
2. リズミカルに押す
3. 2つのセクションが交互に再生
4. 新しいリズムパターン

効果: 即興的なリミックス
```

## 6. 実践的なセットアップ

### 事前準備（Rekordbox）

**Hot Cue設定:**
```
セット前に全ての曲で:
1. Rekordbox で曲を開く
2. 重要なポイントにHot Cue設定
3. 色分け:
   - 赤: ドロップ
   - 黄: ビルドアップ
   - 青: ブレイク
   - 緑: アウトロ
4. コメント追加（任意）
```

**Memory Cue:**
```
Rekordbox機能:
- Hot Cue設定を保存
- 次回も同じ位置
- 全ての曲で準備

本番で即座に使える
```

## 7. DDJ-FLX4 Performance Pads詳細

### モード切り替え

**4つのモード:**
```
1. Hot Cue: キューポイントジャンプ
2. Beat Loop: ループ作成
3. Beat Jump: フレーズジャンプ
4. Sampler: サンプル再生

モード切り替えボタンで変更
```

### Pad配置（Beat Loop）

```
DDJ-FLX4 Beat Loop Mode:

┌────────────────┐
│ 1/4 │ 1/2 │ 1  │ 2  │
├────────────────┤
│ 4   │ 8   │ 16 │ 32 │
└────────────────┘

数字 = beat数
```

## 8. よくある失敗と対処法

### 失敗1: ループがズレる

**原因:**
```
ビートグリッドが正確でない
```

**対処:**
```
1. Rekordboxでビートグリッド確認
2. 必要なら手動調整
3. 1拍目が正確な位置か確認
```

### 失敗2: Hot Cueのタイミングが悪い

**原因:**
```
フレーズの途中にHot Cue設定
```

**対処:**
```
✓ 必ずフレーズの頭（1拍目）に設定
✓ ビートグリッドを参考に
✓ 事前にRekordboxで設定
```

### 失敗3: やりすぎ

**原因:**
```
全ての曲でループ/Hot Cue使用
```

**対処:**
```
✓ 特別な瞬間のみ
✓ 1セットに5-10回程度
✓ 基本ミックスを大切に
```

### 失敗4: Slip Mode理解不足

**原因:**
```
Slip Mode ON/OFF を忘れる
```

**対処:**
```
習慣化:
- 使用前にSlip Mode確認
- 使用後はOFF
- 視覚的に確認（LED）
```

## まとめ

- **Hot Cue**: 瞬時のセクションジャンプ、事前設定必須
- **Loop**: セクション延長、4/8 beat が基本
- **Loop Roll**: 一時的なアレンジ、フィルイン
- **Slip Mode**: バックグラウンドで曲は進行
- **DDJ-FLX4**: Performance Padsで全て可能
- **創造性**: 即興アレンジ、オリジナリティ
- **バランス**: やりすぎない、特別な瞬間のみ
