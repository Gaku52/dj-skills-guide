# マッシュアップ

2曲以上を組み合わせた創造的なミックス、マッシュアップをマスターします。

## この章で学ぶこと

- マッシュアップとは
- アカペラ + インストゥルメンタル
- キーとBPMの調整
- Rekordboxでの準備
- DDJ-FLX4でのライブマッシュアップ
- 事前準備 vs ライブ
- 音楽的に美しいマッシュアップ
- 著作権の注意点

## なぜマッシュアップが重要か

**Open Format DJの武器:**
```
マッシュアップ = 異なる曲を組み合わせる

効果:
- 意外性
- 創造性
- 観客を驚かせる
- 記憶に残る

プロの定番テクニック
```

**ジャンルを超える:**
```
例:
Hip Hopアカペラ + House instrumental
→ 全く新しい曲

観客: 両方のジャンルファンが楽しめる
```

## 1. マッシュアップとは

### 定義

```
マッシュアップ = 2曲以上の要素を組み合わせる

典型的な組み合わせ:
- アカペラ（ボーカルのみ） + インストゥルメンタル（伴奏のみ）
- 曲Aのメロディ + 曲Bのビート
- 異なるジャンルの融合

目的: 新しい音楽体験
```

### 種類

**Type 1: アカペラ + インスト**
```
最も一般的

例:
Notorious B.I.G. - Juicy（アカペラ）
+
Daft Punk - One More Time（インスト）

結果: Hip Hop × House fusion
```

**Type 2: 2つのインスト**
```
メロディとビートの組み合わせ

例:
曲A: 美しいピアノメロディ
+
曲B: 強力なドラムビート

結果: 両方の良いとこ取り
```

**Type 3: 3曲以上**
```
上級者向け

曲A: ボーカル
曲B: ベースライン
曲C: ドラム

結果: 完全にオリジナルな曲
```

## 2. キーとBPMの調整

### キーの重要性

**必ず合わせる:**
```
マッシュアップ = 2曲が同時に鳴る
→ キーが合わないと不協和音

Camelot Wheel復習:
- 同じキー: 8A + 8A
- 関係調: 8A + 8B
- 隣接: 8A + 7A または 9A

必須条件
```

### BPMの調整

**Master Tempo活用:**
```
曲A: 100 BPM（Hip Hop）
曲B: 128 BPM（House）

どちらかを調整:
1. 曲Aを128 BPMに（Master Tempo ON）
2. または曲Bを100 BPMに

推奨: ±6%以内（音質維持）
```

### Rekordboxでの事前確認

**準備手順:**
```
1. 両方の曲をRekordboxで開く
2. キー確認:
   曲A: 8A（Am）
   曲B: 8B（C） → 互換性あり

3. BPM確認:
   曲A: 95 BPM
   曲B: 128 BPM
   → 差: 33 BPM（大きい）

4. 調整可能か計算:
   95 BPM → 128 BPM = +35%
   → 音質劣化大（避けるべき）

5. 別の曲を探す
```

   - 自分で制作
   - 著作権クリア

## 4. DDJ-FLX4でのライブマッシュアップ

### 基本セットアップ

**デッキ配置:**
```
デッキA: アカペラ（ボーカル）
デッキB: インストゥルメンタル（伴奏）

両方:
- Master Tempo ON
- 同じBPMに調整
- キーが互換
```

### 手順

**Step 1: インストゥルメンタル開始**
```
1. デッキBでインストを再生
2. ビートが安定
3. 8-16 bar再生
```

**Step 2: アカペラをビートマッチ**
```
1. デッキAでアカペラを同期
2. ヘッドホンで確認
3. BPM完璧に合わせる
4. フレーズの頭を確認
```

**Step 3: ミックスイン**
```
1. インストのフレーズの頭で
2. アカペラのChannel Faderを上げる
3. ボーカルが入る
4. EQ調整（アカペラのLowカット推奨）
```

**Step 4: バランス調整**
```
EQ設定:

アカペラ（デッキA）:
- Low: -6dB〜-12dB（カット）
- Mid: 0dB（フラット）
- Hi: +3dB（ボーカル強調）

インスト（デッキB）:
- Low: 0dB（ビート維持）
- Mid: -3dB（空間確保）
- Hi: 0dB

ゲイン: 両方同じレベル
```

## 5. 事前準備 vs ライブ

### 事前準備マッシュアップ

**Rekordboxで作成:**
```
メリット:
✓ 完璧な同期
✓ エフェクト追加済み
✓ Hot Cue設定済み
✓ 失敗なし

デメリット:
✗ 柔軟性なし
✗ 観客の反応に対応できない
✗ 既製品感

使用シーン: 重要な瞬間、確実に成功させたい時
```

**作成方法:**
```
1. Ableton Liveまたはrekordbox で編集
2. アカペラとインストをインポート
3. BPM調整
4. キー調整
5. エフェクト追加
6. エクスポート
7. Rekordboxライブラリに追加

事前に完璧に仕上げる
```

### ライブマッシュアップ

**DDJ-FLX4でその場で:**
```
メリット:
✓ 柔軟性
✓ 観客の反応に対応
✓ ライブ感
✓ 即興的

デメリット:
✗ 失敗のリスク
✗ 高度なスキル必要
✗ 練習必須

使用シーン: 通常のセット、実験的な試み
```

## 6. 音楽的に美しいマッシュアップ

### 相性の良い組み合わせ

**ジャンル的相性:**
```
良い組み合わせ:
✓ Hip Hopアカペラ + House/Technoインスト
✓ Soulアカペラ + Drum & Bassインスト
✓ Popアカペラ + Technoインスト
✓ R&Bアカペラ + Dubstepインスト

悪い組み合わせ:
✗ Rockアカペラ + Technoインスト（リズム感が違う）
✗ Jazzアカペラ + Hardstyleインスト（テンポが合わない）
```

### メロディとハーモニー

**音楽理論的考慮:**
```
アカペラのメロディ:
- Aマイナースケール

インストのコード進行:
- Aマイナーキー

→ 互換性あり、美しい

キーが違う:
→ 不協和音、避けるべき
```

## 7. 実践例

### Example 1: Hip Hop × House

**組み合わせ:**
```
アカペラ: Notorious B.I.G. - Big Poppa
- キー: 8A（Am）
- BPM: 94

インスト: CamelPhat - Cola（Inst）
- キー: 8A（Am）
- BPM: 123

調整:
1. Big Poppaを123 BPMに（Master Tempo）
2. キーは同じ（8A）→ 完璧
3. ライブでミックス
```

**DDJ-FLX4操作:**
```
1. Colaのインストを再生（デッキB）
2. Big Poppaのアカペラを123 BPMに調整（デッキA）
3. ビートマッチング
4. フレーズの頭でミックスイン
5. EQ調整（アカペラのLowカット）
6. 32 bar維持
7. 次の曲へ
```

### Example 2: Pop × Techno

**組み合わせ:**
```
アカペラ: Dua Lipa - Levitating（Acapella）
- キー: 6B（F#）
- BPM: 103

インスト: Charlotte de Witte - Technoトラック
- キー: 6A（D#m）
- BPM: 130

調整:
1. Levitatingを130 BPMに
2. キーシフト: 6B → 6A（-1半音）
3. ライブでミックス
```

## 8. よくある失敗と対処法

### 失敗1: キーが合っていない

**原因:**
```
Rekordboxのキー検出ミス、または確認不足
```

**対処:**
```
✓ 必ず事前に耳で確認
✓ 両方を同時に再生して確認
✓ 違和感があれば別の曲を選ぶ
```

### 失敗2: BPM調整しすぎ

**原因:**
```
±10%以上の調整
```

**対処:**
```
✓ 音質劣化を避ける（±6%以内）
✓ 大きく違う曲は諦める
✓ 別の組み合わせを探す
```

### 失敗3: EQバランスが悪い

**原因:**
```
アカペラのLowが残っている
```

**対処:**
```
✓ アカペラのLowは必ずカット
✓ インストのMidは少しカット（空間確保）
✓ ゲインレベルを合わせる
```

### 失敗4: タイミングがズレる

**原因:**
```
フレーズの頭を間違える
```

**対処:**
```
✓ Hot Cueで正確な位置を設定
✓ ヘッドホンで事前確認
✓ 100%確信してからミックスイン
```

## 9. 著作権の注意点

### 重要な警告

```
マッシュアップ = 2曲の著作権が関係

法的問題:
✗ 無断でマッシュアップを配信
✗ 無断でYouTubeにアップロード
✗ 無断でSoundCloudにアップロード

許可されること:
✓ ライブDJセット内での使用（グレーゾーン）
✓ 個人的な練習
✓ 非営利の使用（限定的）

重要: 配信は避ける、ライブのみ
```

### クリアな使用

**安全な方法:**
```
1. 公式リミックス/マッシュアップパックを購入
2. ライセンスを取得
3. オリジナルのアカペラ/インストを使用（自分の曲）
4. ライブDJセットのみで使用（配信しない）

疑問があれば: 使わない
```

## 10. 練習方法

### Week 1-2: キーとBPMの理解

```
練習:
1. 10組のアカペラ/インストを選ぶ
2. Rekordboxでキー確認
3. 互換性をチェック
4. BPM調整を試す
5. 最適な組み合わせを見つける

目標: 相性の良い組み合わせを判断できる
```

### Week 3-4: ライブミックス

```
練習:
1. 5組のマッシュアップを準備
2. DDJ-FLX4でライブミックス練習
3. EQバランスを研究
4. 録音して聞き返す

目標: スムーズなライブマッシュアップ
```

### Week 5-6: セット統合

```
練習:
60分セット:
- 通常のミックス: 80%
- マッシュアップ: 20%（3-4回）
- 特別な瞬間に使用

目標: セット内で自然に使える
```

## まとめ

- **定義**: 2曲以上の要素を組み合わせる、新しい音楽体験
- **基本**: アカペラ + インストゥルメンタル
- **キー**: 必ず互換性確認、Camelot Wheel使用
- **BPM**: Master Tempoで調整、±6%以内
- **EQ**: アカペラのLowカット、バランス調整
- **事前準備**: 完璧だが柔軟性なし
- **ライブ**: 柔軟だがスキル必要
- **著作権**: 配信は避ける、ライブのみ
- **使用頻度**: 1セットに3-4回、特別な瞬間
