Hip Hop DJの伝統的なテクニック、スクラッチを基礎から上級まで学びます。

## なぜスクラッチを学ぶか

**Hip Hop DJの魂:**
```
Hip Hop DJ = スクラッチ

歴史:
- Grandmaster Flash
- DJ Qbert
- DJ Craze
→ スクラッチで名を馳せた

必須スキル: Hip Hop志望なら必修
```

**重要な注意:**
```
全てのDJがスクラッチを学ぶ必要はない

必要な人:
✓ Hip Hop DJ志望
✓ ターンテーブリスト志望
✓ Open Format DJでHip Hopもプレイ

不要な人:
✗ House/Techno専門
✗ その他の電子音楽専門
```

## 1. スクラッチとは

### 定義

```
スクラッチ = レコード/ジョグホイールを前後に動かす

効果:
- リズミカルな音
- パーカッシブな要素
- 即興的な表現

起源: 1970年代のHip Hop、ブロンクス
```

### 2つの要素

**要素1: 手の動き（スクラッチ）**
```
ジョグホイールを前後に動かす

前: 曲が進む
後: 曲が戻る

リズミカルに繰り返す
```

**要素2: フェーダー操作**
```
Channel FaderまたはCrossfader

開く: 音が聞こえる
閉じる: 音が聞こえない

組み合わせて: 様々なスクラッチパターン
```

## 2. DDJ-FLX4でのスクラッチ

### ジョグホイールの特性

**ターンテーブルとの違い:**
```
ターンテーブル:
- ビニールレコード
- 重い、慣性がある
- 直感的

DDJ-FLX4:
- ジョグホイール（プラスチック）
- 軽い、電子的
- 練習には十分
```

### セットアップ

**推奨設定:**
```
Rekordbox設定:

1. [Preferences] → [Controller]
2. Jog Mode: Vinyl
3. Jog Sensitivity: 調整（お好みで）
4. Slip Mode: OFF（スクラッチ時）

DDJ-FLX4:
- Vinyl Mode ON（ジョグホイールの赤いライト）
```

### スクラッチ用の音源

**推奨:**
```
スクラッチ用サンプル:
- "Fresh"（Doug E. Fresh）
- "Ahhh"（声）
- "Fresh"（Kool & The Gang）

または:
Rekordbox Sampler:
- 短い音源（1-2秒）
- はっきりした音
- ボーカルサンプル
```

## 3. Baby Scratch（基礎）

### 概要

```
Baby Scratch = 最も基本的なスクラッチ

特徴:
- フェーダー操作なし
- 手だけで前後
- リズムを作る

全てのスクラッチの基礎
```

### DDJ-FLX4での操作

**Step 1: 準備**
```
1. スクラッチ用サンプルをロード
2. Hot Cueで開始位置設定
3. Channel Faderを上げておく
4. Vinyl Mode ON
```

**Step 2: 基本動作**
```
1. ジョグホイール上面に手を置く
2. 時計回りに回す（前）
   → 音が「Fresh」と聞こえる
3. 反時計回りに回す（後）
   → 音が逆再生
4. リズミカルに繰り返す
```

**Step 3: リズムパターン**
```
基本リズム（4拍子）:
前 - 後 - 前 - 後 - 前 - 後 - 前 - 後
|   |   |   |   |   |   |   |
1   &   2   &   3   &   4   &

8分音符のリズム
```

### 練習方法

```
初級（Week 1-2）:
1. ゆっくり前後 × 100回
2. テンポを徐々に上げる
3. 4拍子を意識
4. メトロノーム使用（60 BPM → 120 BPM）

目標: リズミカルに前後できる
```

## 4. Forward/Backward Scratch

### 概要

```
Forward Scratch = 前方向のみ音を出す
Backward Scratch = 後方向のみ音を出す

フェーダー操作を追加
→ より音楽的
```

### DDJ-FLX4での操作

**Forward Scratch:**
```
1. ジョグホイールを前に回す
   → 同時にChannel Faderを上げる
   → 「Fresh」と聞こえる

2. ジョグホイールを後に戻す
   → 同時にChannel Faderを下げる
   → 音が聞こえない

3. 繰り返す

効果: クリーンな前方向の音のみ
```

**Backward Scratch:**
```
1. ジョグホイールを前に回す
   → Channel Faderは下げたまま
   → 音が聞こえない

2. ジョグホイールを後に戻す
   → Channel Faderを上げる
   → 逆再生の音

3. 繰り返す

効果: 逆再生の音のみ
```

### 練習方法

```
中級（Week 3-4）:
1. Forward Scratch × 100回
2. Backward Scratch × 100回
3. 両手のタイミングを合わせる
4. メトロノームに合わせる（80-100 BPM）

目標: 両手が無意識に連動
```

## 5. Transform Scratch

### 概要

```
Transform = 音を断続的にカット

特徴:
- 「タ・タ・タ・タ」という音
- フェーダーを高速で開閉
- ロボティックな効果

使用頻度: 高い（Hip Hopで定番）
```

### DDJ-FLX4での操作

**基本動作:**
```
1. ジョグホイールを前にゆっくり回す
2. 同時にChannel Faderを高速で開閉
   開 - 閉 - 開 - 閉 - 開 - 閉
3. 音が断続的に聞こえる

コツ:
- ジョグホイール: ゆっくり
- フェーダー: 高速
- リズムを意識
```

**パターン例:**
```
3クリック Transform:
前に回しながらフェーダーを3回開閉
「タ・タ・タ」

4クリック Transform:
前に回しながらフェーダーを4回開閉
「タ・タ・タ・タ」
```

### 練習方法

```
中級（Week 5-6）:
1. 3クリック × 50回
2. 4クリック × 50回
3. テンポを徐々に上げる
4. 曲に合わせて使用

目標: クリーンなTransform
```

## 6. Chirp Scratch

### 概要

```
Chirp = 「チュクチュク」という鳥のような音

特徴:
- Transformより速い
- フェーダーを中途半端に開く
- 独特の音色

使用頻度: 中程度
```

### DDJ-FLX4での操作

```
1. ジョグホイールを前に回す
   → Channel Faderを半分だけ開く（50%）
   → すぐに閉じる
   → 「チュク」という音

2. ジョグホイールを後に戻す
   → Channel Faderを半分だけ開く
   → すぐに閉じる
   → 「チュク」という音

3. 繰り返す

重要: フェーダーを完全に開かない
```

## 7. Crab Scratch（上級）

### 概要

```
Crab Scratch = 最も高速なスクラッチ

特徴:
- 4本指を使用
- 超高速フェーダー操作
- カニの動きに似ている

難易度: 非常に高い
```

### 指の使い方

**ターンテーブルの場合:**
```
Crossfader上に4本指:
1. 小指でフェーダーを押し込む
2. 薬指を離す（1クリック）
3. 中指を離す（2クリック）
4. 人差し指を離す（3クリック）
5. 指を戻す（4クリック）

1回の動作で4クリック
```

**DDJ-FLX4での操作:**
```
Channel Faderで代用:

1. ジョグホイールを前にゆっくり回す
2. 4本指でChannel Faderを高速で叩く
   タンタンタンタン
3. 音が超高速で断続的に

注意: DDJ-FLX4では難しい（練習は可能）
```

### 練習方法

```
上級（Week 10-12）:
1. 指の動きを練習（フェーダーなしで）× 1000回
2. ゆっくりCrab × 50回
3. 徐々に速度を上げる
4. 完璧には数ヶ月かかる

目標: 滑らかな4クリック
```

## 8. スクラッチの組み合わせ

### Beat Juggleへの応用

```
Baby Scratch + Hot Cue:
1. Hot Cue 1: ドラムのキック
2. Baby Scratchでリズムパターン作成
3. 即興的なビート

効果: オリジナルビートを作る
```

### ミックス中の使用

```
Hip Hopミックス:
1. 曲Aと曲Bをミックス中
2. 曲Aの特定部分にHot Cue
3. Transform Scratchで装飾
4. 曲Bへスムーズに移行

使用頻度: 控えめに（2-3回/曲）
```

## 9. ターンテーブルとDDJ-FLX4の違い

### メリット・デメリット

**ターンテーブル（Technics 1200等）:**
```
メリット:
✓ 本物の感触
✓ 重い慣性が使いやすい
✓ プロの標準

デメリット:
✗ 高価（2台で20万円以上）
✗ 重い
✗ メンテナンス必要
```

**DDJ-FLX4:**
```
メリット:
✓ 安価（3万円程度）
✓ コンパクト
✓ 練習には十分
✓ Rekordbox統合

デメリット:
✗ 軽い（慣性がない）
✗ プラスチック製
✗ 本番には不向き（Hip Hopクラブ）
```

### 推奨学習パス

**初心者:**
```
1. DDJ-FLX4で基礎を学ぶ（6ヶ月）
2. Baby、Forward、Transform習得
3. Hip Hop DJ続けるか判断
4. 続けるならターンテーブル購入検討
```

## 10. 練習スケジュール

### Week 1-2: Baby Scratch

```
毎日30分:
- ウォームアップ: 10分

- クールダウン: 5分

練習内容:
1. メトロノーム60 BPMで前後
2. 徐々に80, 100, 120 BPMへ
3. リズムを正確に

目標: リズミカルな前後
```

### Week 3-4: Forward/Backward

```
毎日45分:
- Baby Scratch復習: 10分
- Forward Scratch: 15分
- Backward Scratch: 15分
- 組み合わせ: 5分

目標: 両手の連動

### Week 5-8: Transform & Chirp

毎日60分:

目標: クリーンなTransform

```
毎日60分:

- 録音して聞き返す: 5分

目標: ミックス中に自然に使える

## 11. よくある失敗と対処法

### 失敗1: 音が汚い

**原因:**
```
ジョグホイールの動きが不安定
```

**対処:**
```
✓ ゆっくり正確に
✓ メトロノーム使用
✓ 100回練習してから速度を上げる
```

### 失敗2: フェーダーのタイミングがズレる

**原因:**
```
両手の連動不足
```

**対処:**
```
✓ ゆっくり練習
✓ 鏡で両手を見る
✓ 1000回反復
```

### 失敗3: すぐにCrabをやりたがる

**原因:**
```
基礎を飛ばす
```

**対処:**
```
✓ Baby Scratchを完璧に（最低1ヶ月）
✓ Transform完璧に（最低2ヶ月）
✓ 焦らない
```

### 失敗4: DDJ-FLX4でCrabが難しい

**原因:**
```
機材の限界
```

**対処:**
```
✓ 基礎的なスクラッチに集中
✓ Crabはターンテーブルで
✓ DDJ-FLX4では練習程度
```

- **定義**: ジョグホイールを前後、フェーダー操作と組み合わせ
