# トランジション技法

基本的なフェードイン/アウトを超えた、プロフェッショナルなトランジション技法を学びます。

## この章で学ぶこと

- トランジションとは何か
- Echo Out（エコーアウト）
- Filter Sweep（フィルタースイープ）
- Spin Back（スピンバック）
- Volume Cut（ボリュームカット）
- Beat Jump Cut
- DDJ-FLX4での実践
- 使い分けとタイミング

## なぜトランジション技法が重要か

**基本ミックスとの違い:**
```
基本ミックス:
- フェードイン/アウト
- シンプル
- 問題ないが、印象が薄い

トランジション技法:
- 変化がある
- 記憶に残る
- プロフェッショナル
```

**プロは必ず使う:**
```
✓ セットに変化をつける
✓ 観客を飽きさせない
✓ 次の曲への期待感
✓ 個性を出せる
```

## 1. トランジションとは

### 定義

```
トランジション = 曲から曲への移行方法

基本: フェードイン/アウト
上級: エフェクト、テクニックを使った移行

目的: 自然かつ印象的な切り替え
```

### 種類

**スムーズ系:**
- Echo Out
- Filter Sweep
- 基本フェード

**インパクト系:**
- Spin Back
- Volume Cut
- Beat Jump Cut

**使い分け:**
```
スムーズ系: House、Technoの通常ミックス
インパクト系: ブレイク明け、ドロップ前、特別な瞬間
```

## 2. Echo Out（エコーアウト）

### 概要

```
Echo Out = エコーエフェクトで曲を消す

効果:
- 曲が徐々に遠ざかる
- 残響が残る
- 自然で美しい

使用頻度: 非常に高い（プロの定番）
```

### DDJ-FLX4での操作

**Step 1: 基本セットアップ**
```
1. 曲Aが再生中
2. 曲Bをビートマッチング
3. 曲Bをミックスイン（通常通り）
4. 両方が鳴っている状態
```

**Step 2: Echo Outの実行**
```
タイミング: 曲Aを消す直前（16-8 bar前）

1. Beat FX ボタンを押す
2. Rekordbox画面でEchoを選択
3. Beat設定: 1/4 または 1/2
4. Channel Assign: 曲A（消す方）に
5. Level/Depth: 50%くらい
6. FX ON
```

**Step 3: フェードアウト**
```
1. エコーがかかった状態で
2. 曲Aのチャンネルフェーダーをゆっくり下げる
3. 8-16 bar かけて
4. エコーの余韻が残る
5. 曲Bだけになる
6. FX OFF
```

### タイミング

**最適なタイミング:**
```
- 曲Aのアウトロ
- ブレイクダウン
- フレーズの終わり（32 bar）

避けるべき:
- ドロップ中
- ボーカル中
- ビルドアップ中
```

## 3. Filter Sweep（フィルタースイープ）

### 概要

```
Filter Sweep = フィルターで徐々に高域/低域をカット

効果:
- こもった音→クリアな音
- クリアな音→こもった音
- ビルドアップ感

使用: House、Technoで頻繁
```

### 種類

**HPF Sweep（High Pass Filter）:**
```
低域をカット→全域に戻す

使用:
曲Bのイントロ時
- 最初はこもった音
- 徐々にクリアに
- ドロップで全開
```

**LPF Sweep（Low Pass Filter）:**
```
高域をカット→全域に戻す

使用:
曲Aのアウトロ時
- 徐々にこもった音に
- 低域だけ残る
- フェードアウト
```

### DDJ-FLX4での操作

**方法1: EQのHiノブ使用（簡易）**
```
曲Bをミックスイン時:
1. 曲BのHi EQを完全カット（左いっぱい）
2. フェーダーを上げる
3. 4-8 bar かけて
4. Hi EQを徐々に12時に戻す
5. 明るい音に変化
```

**方法2: Beat FX Filter使用（上級）**
```
1. Beat FX → Filter選択
2. Channel Assign: 曲B
3. FX ON
4. Filterノブを回す
   - 左: LPF（こもる）
   - 右: HPF（軽い）
5. 中央に戻してFX OFF
```

## 4. Spin Back（スピンバック）

### 概要

```
Spin Back = レコードを止める効果

効果:
- 「ギュイーン」という音
- 急激な停止
- インパクト大

使用: ブレイク、ドロップ前
```

### DDJ-FLX4での操作

**方法1: ジョグホイールで手動**
```
1. 曲が再生中
2. ジョグホイール上面を強くタッチ
3. 時計回りに速く回す
4. 徐々に遅く
5. 停止

コツ:
- 最初は速く
- 徐々に遅く
- 自然な減速カーブ
```

**方法2: Beat FX Spiral使用**
```
1. Beat FX → Spiral選択
2. Direction: Down
3. Beat: 1 bar
4. FX ON
5. 自動でスピンバック効果
```

### タイミング

**効果的なタイミング:**
```
✓ ブレイク明け（次の曲へ）
✓ ビルドアップの頂点
✓ セクションの変わり目

✗ 多用しすぎない（1セットに2-3回）
✗ 予測可能なタイミングは避ける
```

## 5. Volume Cut（ボリュームカット）

### 概要

```
Volume Cut = 急激な音量カット

効果:
- 即座の切り替え
- 緊張感
- インパクト

使用: Hip Hop、Breaks、Open Format
```

### DDJ-FLX4での操作

**基本:**
```
フレーズの頭（1拍目）で:
1. 曲AのChannel Faderを即座に下げる
2. 同時に曲BのChannel Faderを上げる
3. 瞬間的な切り替え

重要: ビート位相が完璧に合っていること
```

**DDJ-FLX4 Crossfader使用:**
```
1. 両方のChannel Faderを上げておく
2. Crossfaderを曲A側（左）
3. フレーズの頭で
4. Crossfaderを一気に曲B側（右）へ
5. クリーンなカット
```

---

## 6. Beat Jump Cut

### 概要

```
Beat Jump Cut = Beat Jumpで瞬間移動

効果:
- セクションスキップ
- 意外性
- テクニカル

使用: Techno、実験的なセット
```

### DDJ-FLX4での操作

**Performance Padsを使用:**
```
1. 曲が再生中
2. Beat Jump Modeに切り替え
3. 16 beat または 32 beat選択
4. フレーズの頭でパッドを押す
5. 次のセクションにジャンプ
6. 即座に曲Bをミックスイン

効果: 予想外のタイミングで切り替え
```

## 7. 使い分けとタイミング

### ジャンル別の使用頻度

**House / Tech House:**
```
Echo Out: ★★★★★（多用）
Filter Sweep: ★★★★★（多用）
Spin Back: ★★☆☆☆（たまに）
Volume Cut: ★☆☆☆☆（稀）
Beat Jump Cut: ★★☆☆☆（たまに）
```

**Techno:**
```
Echo Out: ★★★★★（多用）
Filter Sweep: ★★★★☆（よく使う）
Spin Back: ★★★☆☆（時々）
Volume Cut: ★☆☆☆☆（稀）
Beat Jump Cut: ★★★☆☆（時々）
```

**Hip Hop / Breaks:**
```
Echo Out: ★★☆☆☆（たまに）
Filter Sweep: ★☆☆☆☆（稀）
Spin Back: ★★★★☆（よく使う）
Volume Cut: ★★★★★（多用）
Beat Jump Cut: ★★☆☆☆（たまに）
```

### セット内での配分

**60分セット例:**
```
Echo Out: 10-15回（通常のミックス）
Filter Sweep: 5-8回（変化をつける）
Spin Back: 2-3回（特別な瞬間）
Volume Cut: 1-2回（Hip Hopの場合多め）
Beat Jump Cut: 1-2回（実験的に）

基本フェード: 残りのミックス

バランスが重要
```

## 8. 練習方法

```
練習:
1. 同じ2曲で100回

3. Beat設定を変える（1/4, 1/2, 1）
4. 録音して聞き返す

目標: 自然に使える

```
練習:

## 9. よくある失敗と対処法

### 失敗1: エフェクトが強すぎる

**原因:**
```
Level/Depthを上げすぎ
```

**対処:**
```
✓ 最初は50%から
✓ 徐々に調整
✓ 控えめが美しい
```

### 失敗2: タイミングが悪い

**原因:**
```
フレーズの途中で実行
```

**対処:**
```
✓ 必ずフレーズの頭で
✓ 16 bar、32 barの区切り
✓ 曲の構造を理解
```

### 失敗3: 多用しすぎ

**原因:**
```
技を見せたい
```

**対処:**
```
✓ Less is more
✓ 特別な瞬間のみ
✓ 基本ミックスも重要
```

### 失敗4: Spin Backがぎこちない

**原因:**
```
減速カーブが不自然
```

**対処:**
```
✓ 最初は速く
✓ 徐々に遅く
✓ 自然なカーブを研究
✓ 100回練習
```
