# 選曲とセットリスト

技術を超えた、DJの真髄である選曲とセットリスト構築を完全マスターします。

## このセクションで学ぶこと

- ジャンル知識（House、Techno、Hip Hop等）
- セットリスト構築の基本と応用
- エネルギーカーブ理論
- フロアリーディング（観客の反応を読む）
- 曲分析（構造、キー、BPM）
- 音楽ライブラリ管理（Rekordbox整理術）
- レコードディギング（良い曲の探し方）
- プロの選曲センスの身につけ方

## なぜ選曲がDJで最も重要なのか

**技術 vs 選曲:**
```
完璧な技術 + 悪い選曲 = 失敗
普通の技術 + 素晴らしい選曲 = 成功

真実:
技術は1ヶ月で習得できる
選曲センスは一生磨き続ける

プロDJの差別化要因 = 選曲
```

**観客は技術を見に来ていない:**
```
観客が求めるもの:
✓ 良い音楽
✓ 盛り上がり
✓ 感動
✓ 楽しい時間

観客が求めないもの:
✗ 完璧なビートマッチング
✗ 高度なスクラッチ
✗ 複雑なエフェクト

技術は前提、選曲が全て
```

## 選曲の3つの柱

### 1. ジャンル知識

**深い理解:**
```
ジャンルの歴史:
- 起源
- 代表的なアーティスト
- サブジャンル
- トレンド

例: House
→ Deep House, Tech House, Progressive House
→ それぞれの特徴を理解
→ 適切に組み合わせる
```

### 2. 観客理解

**誰のために選ぶか:**
```
クラブ: その日の観客層
- 年齢層
- 音楽嗜好
- エネルギーレベル
- 時間帯

選曲はコミュニケーション
```

### 3. ストーリーテリング

**セットは物語:**
```
起：ウォームアップ（静かに始める）
承：徐々に盛り上げる
転：ピークを作る
結：クールダウン

60分 = 1つの映画
```

## このセクションの学習順序

### Step 1: ジャンル知識（2-3週間）

**目標:** 主要ジャンルの理解

```
学ぶこと:
- House、Techno、Hip Hopの歴史
- サブジャンルの違い
- 代表的なアーティスト100名
- 各ジャンルの名曲50曲

方法:
1. [ジャンル知識](./genre-knowledge.md) を読む
2. プレイリスト100曲作成
3. 毎日2時間聴き込む
```

### Step 2: セットリスト構築（1-2週間）

**目標:** 60分セットの設計

```
学ぶこと:
- セットリストの基本構造
- BPMとキーの流れ
- エネルギーカーブ
- フレーズミックスのタイミング

方法:
1. [セットリスト構築](./setlist-building.md) を読む
2. 紙に60分セットを書く
3. 実際にミックスして検証
```

### Step 3: エネルギーカーブ（1週間）

**目標:** 観客のエネルギーをコントロール

```
学ぶこと:
- エネルギーレベルの定義
- 時間帯別のエネルギー管理
- ピークの作り方
- クールダウンの技術

方法:
1. [エネルギーカーブ](./energy-curve.md) を読む
2. プロのセットを分析
3. エネルギーグラフを描く
```

### Step 4: フロアリーディング（実践）

**目標:** 観客の反応に対応

```
学ぶこと:
- 観客の反応の読み方
- リアルタイムでの選曲変更
- 失敗の対処法
- 予備曲の準備

方法:
1. [フロアリーディング](./floor-reading.md) を読む
2. 実際のギグで練習
3. 反応をメモして改善
```

### Step 5: ライブラリ管理（継続的）

**目標:** 効率的な曲管理

```
学ぶこと:
- Rekordboxでの整理術
- プレイリスト戦略
- タグ付けシステム
- 新曲の発見と追加

方法:
1. [音楽ライブラリ管理](./music-library.md) を読む
2. Rekordboxを完全整理
3. 週に10曲追加する習慣
```

## 選曲のよくある失敗

### 失敗1: 自分の好みだけで選ぶ

**問題:**
```
DJ = 自分が好きな曲を流す
観客 = 盛り上がらない、帰る
```

**対処:**
```
✓ 観客のために選ぶ
✓ 自分の好み: 70%
✓ 観客の好み: 30%
✓ バランスが重要
```

### 失敗2: トレンドだけ追う

**問題:**
```
Beatport Top 10を全て使う
→ 他のDJと同じ
→ 個性がない
```

**対処:**
```
✓ トレンド: 40%
✓ クラシック: 30%
✓ 掘り出し物: 30%
✓ 独自のカラー
```

### 失敗3: エネルギー管理の失敗

**問題:**
```
最初から全開
→ ピークがない
→ 観客が疲れる
```

**対処:**
```
✓ 静かに始める
✓ 徐々に上げる
✓ ピークは1-2回
✓ 波を作る
```

### 失敗4: 準備不足

**問題:**
```
曲を聴いていない
→ 構造を知らない
→ ミックスポイントがわからない
→ 失敗
```

**対処:**
```
✓ 全ての曲を最低3回聴く
✓ ミックスポイントを記憶
✓ Hot Cue設定済み
✓ 完璧な準備
```

## よくある質問

### Q1: どのくらいの曲数が必要？

**A: 60分セットで60-80曲**

```
理由:
- 平均ミックス時間: 3-5分
- 60分 = 12-20曲
- 予備: 40-60曲

合計: 60-80曲あれば安心
```

### Q2: 新曲と定番曲のバランスは？

**A: 新曲60% + 定番40%**

```
新曲:
- トレンド
- 新鮮さ
- プロらしさ

定番曲:
- 確実な盛り上がり
- 観客が知っている
- 安心感

バランスが重要
```

### Q3: 選曲センスはどう磨く？

**A: 毎日2時間、音楽を聴く**

```
方法:
1. 毎日新しい曲を10曲聴く
2. プロのミックスを分析（週5本）
3. クラブに通う（月2-4回）
4. 他DJの選曲を研究
5. 音楽ブログを読む

継続が全て
```

### Q4: ジャンルを混ぜてもいい？

**A: YES、Open Formatなら必須**

```
ジャンル別:
House/Techno: 1ジャンル推奨
Open Format: 複数ジャンル必須
Hip Hop: サブジャンル混在OK

重要: スムーズな移行
```

- **セットリスト**: 事前準備、エネルギーカーブ設計

- **ライブラリ管理**: Rekordbox整理、週に10曲追加
- **継続学習**: 毎日2時間音楽を聴く、プロを研究
