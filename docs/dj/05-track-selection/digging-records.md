# レコードディギング

良い曲を発掘し、他のDJと差別化するためのディギング術を完全マスターします。

## この章で学ぶこと

- ディギングとは
- Beatport、Traxsourceの使い方
- SoundCloudでの発掘方法
- Bandcampの探し方
- レコードショップ巡り
- プロモーションメールリスト
- 週次ディギングルーティン
- プロの発掘術

## なぜディギングが重要か

**Beatport Top 10 vs 掘り出し物:**
```
Beatport Top 10だけ:
- 他のDJと同じ曲
- 個性がない
- 予測可能
- つまらない

掘り出し物:
- 他のDJが使わない
- 個性が爆発
- サプライズ
- 記憶に残る

ディギング = DJの真髄
```

---

## 1. ディギングとは

### 定義

```
ディギング（Digging）= 曲を掘る、発掘する

元々:
- レコードショップで掘る
- クレート（箱）を漁る
- 隠れた名曲を見つける

現代:
- オンラインで掘る
- デジタルプラットフォーム
- 同じ精神

目的: 他人が知らない良い曲を見つける
```

---

## 2. Beatport（メインストリーム）

### Beatportとは

```
Beatport = 世界最大のDJ音楽ストア

特徴:
- 電子音楽専門
- 全ジャンル網羅
- 高品質（WAV, FLAC）
- プロ御用達

URL: https://beatport.com/
```

### 使い方

**Top 100チャート:**
```
各ジャンルのTop 100:
- Tech House Top 100
- Techno Top 100
- Deep House Top 100

メリット: トレンドがわかる
デメリット: 他のDJも使う

使用頻度: 30%程度（全て使わない）
```

**Dig Deeper（重要）:**
```
チャートの外を掘る:

1. ジャンルを選択
2. "New Releases"（新リリース）
3. 評価でソート
4. 試聴
5. 良い曲を購入

これがディギング
```

**Label（レーベル）で探す:**
```
好きなアーティストのレーベル:

例:
- Jamie Jones → Hot Creations
- Carl Cox → Intec

レーベルページ:
→ 同じ方向性の曲が見つかる
→ 質が高い

レーベル買いもあり
```

---

## 3. Traxsource（House専門）

### Traxsourceとは

```
Traxsource = House音楽専門ストア

特徴:
- House、Disco、Soul
- Beatportより深い
- DJ Harvey、Black Coffee等
- アンダーグラウンド

URL: https://traxsource.com/
```

### 使い方

**Essentialチャート:**
```
Traxsource Essential:
- プロDJが選ぶTop 100
- 質が非常に高い
- Beatportより玄人向け

House DJは必ずチェック
```

**Hype Machine:**
```
注目の新曲:
- AIが推薦
- まだ知られていない
- 先取りできる

早めに購入 = 他DJより先
```

---

## 4. SoundCloud（無料発掘）

### SoundCloudとは

```
SoundCloud = 音楽共有プラットフォーム

特徴:
- 無料で聴ける
- アーティスト直接アップロード
- リミックス、ブートレグ
- プロモーション

URL: https://soundcloud.com/
```

### 発掘方法

**Method 1: アーティストをフォロー:**
```
好きなアーティストをフォロー:
→ 新曲通知
→ Repost（リポスト）で関連アーティスト発見
→ 芋づる式

100人フォローすれば毎日新曲
```

**Method 2: プレイリストを漁る:**
```
"Tech House 2025"等で検索:
→ プレイリスト表示
→ 何時間分も聴ける
→ 良い曲をメモ

時間はかかるが宝の山
```

**Method 3: Liked Tracksを辿る:**
```
プロDJのアカウント:
→ Liked Tracks
→ プロが気に入った曲
→ 質が高い

例:
- Carl Cox
- Jamie Jones
- Fisher

彼らがLikeした曲を聴く
```

### ダウンロード

**Free Download:**
```
"Free Download"タグ:
- アーティストがプロモーションで無料配布
- 合法
- DJ用

注意: 著作権確認必須
```

---

## 5. Bandcamp（インディー、実験的）

### Bandcampとは

```
Bandcamp = インディーアーティストのプラットフォーム

特徴:
- アーティストに直接支払い
- 実験的な音楽
- アンダーグラウンド
- コミュニティ

URL: https://bandcamp.com/
```

### 発掘方法

**Discover機能:**
```
1. ジャンルを選択（Electronic）
2. サブジャンル選択（House、Techno）
3. 試聴
4. "Pay What You Want"（価格自由）も多い

掘り出し物だらけ
```

**Tag検索:**
```
"deep house techno minimal"等:
→ 複数タグで検索
→ ニッチな曲が見つかる

他のDJが絶対使わない曲
```

## 6. レコードショップ（伝統的）

### なぜレコードショップか

```
デジタル時代でも価値:

理由:
✓ 店員のおすすめ（プロの目利き）
✓ 試聴できる
✓ 偶然の出会い
✓ コミュニティ

月に1回は訪問
```

### 日本の名店

**東京:**
```
- Disk Union（渋谷）: 全ジャンル
- DMR（下北沢）: House、Techno
- Technique（渋谷）: Hip Hop
- Manhattan Records（渋谷）: House、Hip Hop
```

**大阪:**
```
- Technique（アメ村）: Hip Hop
- Sunshine Records（心斎橋）: House、Techno
```

### 店員との関係

**重要:**
```
店員と仲良くなる:
→ おすすめを聞ける
→ 新入荷を教えてもらえる
→ レア盤を取り置き

人間関係 = 情報源
```

## 7. プロモーションメールリスト

### レーベルメールリスト

**購読方法:**
```
好きなレーベルのサイト:
1. "Newsletter"または"Mailing List"
2. メールアドレス登録
3. 新リリース情報が届く

例:
- Toolroom Records
- Hot Creations
- Drumcode

週に数通、質の高い情報
```

### Promopool

**DJ専用プロモサービス:**
```
Promopool、DMS等:
- レーベルから直接プロモ曲
- DJ専用
- リリース前の曲
- 無料または格安

条件: プロDJである証明必要
（ギグ実績、SoundCloud等）
```

## 8. 週次ディギングルーティン

### 月曜日（2時間）: メインストリーム

**Beatport、Traxsource:**
```
作業:
1. 各ジャンルのTop 10確認
2. New Releases を100曲試聴
3. 良い曲10曲を購入

時間: 2時間
予算: 2,000-3,000円
```

### 水曜日（1時間）: SoundCloud

**無料発掘:**
```
作業:
1. フォローアーティストの新曲チェック
2. プレイリスト2-3個聴く
3. Free Download 5曲ダウンロード

時間: 1時間
予算: 無料
```

### 金曜日（1時間）: Bandcamp、その他

**アンダーグラウンド:**
```
作業:
1. Bandcamp Discover
2. YouTubeミックスから曲特定
3. 掘り出し物5曲購入

時間: 1時間
予算: 1,000-2,000円
```

### 日曜日（30分）: 整理

**ライブラリに統合:**
```
作業:
1. 全ての新曲をRekordboxに
2. 解析、Hot Cue設定
3. プレイリストに追加

習慣化
```

## 9. プロの発掘術

### Shazam活用

**他DJのセットから:**
```
クラブで:
1. 良い曲が流れる
2. Shazamで曲名特定
3. 後で購入

YouTubeのミックス動画:
1. 再生しながらShazam
2. 全ての曲を特定
3. プロの選曲を学ぶ

合法的なスパイ
```

### 1001tracklists

**プロDJのセットリスト:**
```
1001tracklists.com:
- プロDJのセットリスト
- 曲IDが全て載っている
- フェスティバル、クラブセット

例:
Carl Coxの最新セット:
→ 使用曲リスト
→ 全て購入可能

プロの選曲を研究
```

## 10. 予算管理

### 月間予算

**推奨:**
```
初心者: 5,000円/月（50曲）
中級者: 10,000円/月（100曲）
プロ: 20,000円/月（200曲）

年間:
初心者: 60,000円（600曲）
中級者: 120,000円（1,200曲）
プロ: 240,000円（2,400曲）

投資 = DJ活動の必要経費
```

### コスト削減方法

**賢く購入:**
```
✓ SoundCloud Free Download活用
✓ Bandcamp（安い）
✓ レーベルセール時に購入
✓ Beatport Link（月額制、試せる）
✓ プロモプール（条件あり）

無駄遣いしない
```
