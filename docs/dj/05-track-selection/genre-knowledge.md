# ジャンル知識

主要DJ音楽ジャンルの歴史、特徴、代表的アーティストを完全網羅します。

## この章で学ぶこと

- House音楽（Deep、Tech、Progressive）
- Techno音楽（Minimal、Industrial、Acid）
- Hip Hop（Boom Bap、Trap）
- その他の主要ジャンル
- サブジャンルの違いと特徴
- 代表的アーティスト100名
- 必聴曲リスト
- ジャンルの選び方

## なぜジャンル知識が重要か

**選曲の基礎:**
```
ジャンルを知らない = 選曲できない

例:
Deep HouseとTech Houseの違いがわからない
→ セットがバラバラ
→ フロアが混乱

ジャンル知識 = DJの教養
```

## 1. House

### 起源と歴史

```
起源: 1980年代、シカゴ
創始者: Frankie Knuckles（"Godfather of House"）

特徴:
- BPM: 118-130
- 4つ打ちキック
- ディスコの影響
- ソウルフルなボーカル
- ピアノ、ストリングス

Warehouse Club（シカゴ）が名前の由来
```

### サブジャンル

#### Deep House

**特徴:**
```
BPM: 118-125（遅め）
雰囲気: 深い、メロウ、ソウルフル

サウンド:
- 深いベースライン
- ジャズの影響
- 温かいパッド
- ボーカル（時々）

用途: ウォームアップ、ラウンジ
```

**代表的アーティスト:**
```
- Kerri Chandler
- Larry Heard
- Moodymann
- Osunlade
- Black Coffee
```

**必聴曲:**
```
- Kerri Chandler - "Rain"
- Larry Heard - "Can You Feel It"
- Moodymann - "I Can't Kick This Feeling"
```

#### Tech House

**特徴:**
```
BPM: 125-130（速め）
雰囲気: エネルギッシュ、ダンサブル

サウンド:
- ミニマルなTechnoの影響
- グルーヴィーなベース
- パーカッシブ
- ボーカルサンプル

用途: ピークタイム
```

**代表的アーティスト:**
```
- Carl Cox
- Jamie Jones
- Green Velvet
- Fisher
- Chris Lake
```

**必聴曲:**
```
- Green Velvet - "La La Land"
- Fisher - "Losing It"
- Chris Lake - "Turn Off The Lights"
```

#### Progressive House

**特徴:**
```
BPM: 125-130
雰囲気: ビルドアップ、エピック

サウンド:
- 長いビルドアップ
- メロディアス
- レイヤー重ね
- エモーショナル

用途: フェスティバル、大型クラブ
```

**代表的アーティスト:**
```
- Sasha
- John Digweed
- Eric Prydz
- Deadmau5
- Anjunadeep
```

## 2. Techno

### 起源と歴史

```
起源: 1980年代、デトロイト
創始者: The Belleville Three
- Juan Atkins
- Derrick May
- Kevin Saunderson

特徴:
- BPM: 125-145
- 工業的
- 未来的
- ミニマル
- 反復的

ヨーロッパで爆発的人気（特にベルリン）
```

### サブジャンル

#### Minimal Techno

**特徴:**
```
BPM: 125-135
雰囲気: ミニマル、催眠的

サウンド:
- 最小限の要素
- 反復的なループ
- 微細な変化
- ディープなグルーヴ

用途: 長時間セット、アフターアワーズ
```

**代表的アーティスト:**
```
- Richie Hawtin
- Robert Hood
- Ricardo Villalobos
- Dubfire
```

#### Industrial Techno

**特徴:**
```
BPM: 130-145（速い）
雰囲気: ダーク、アグレッシブ

サウンド:
- ハードなキック
- ノイズ
- ディストーション
- 工業的な音

用途: ピークタイム、アンダーグラウンド
```

**代表的アーティスト:**
```
- Amelie Lens
- Charlotte de Witte
- I Hate Models
- Kobosil
```

#### Acid Techno

**特徴:**
```
BPM: 130-145
雰囲気: サイケデリック

サウンド:
- TB-303ベースライン
- スキーキーな音
- アシッド
- グルーヴィー

用途: 実験的なセット
```

**代表的アーティスト:**
```
- Josh Wink
- Phuture
- Hardfloor
```

## 3. Hip Hop

### 起源と歴史

```
起源: 1970年代、ブロンクス（NY）
創始者:
- DJ Kool Herc
- Grandmaster Flash
- Afrika Bambaataa

特徴:
- BPM: 80-110
- ブレイクビート
- サンプリング
- ラップボーカル
- ターンテーブリズム

4大要素: DJ、MC、B-Boy、Graffiti
```

### サブジャンル

#### Boom Bap

**特徴:**
```
BPM: 85-95
雰囲気: クラシック、ノスタルジック

サウンド:
- サンプルベース
- ハードなドラム
- "Boom"（キック）と"Bap"（スネア）
- ジャズサンプル

時代: 90年代ゴールデンエラ
```

**代表的アーティスト:**
```
- DJ Premier
- Pete Rock
- J Dilla
- 9th Wonder
```

**必聴曲:**
```
- Gang Starr - "Mass Appeal"
- Pete Rock & CL Smooth - "They Reminisce Over You"
- A Tribe Called Quest - "Electric Relaxation"
```

#### Trap

**特徴:**
```
BPM: 130-150（ハーフタイムで65-75に聞こえる）
雰囲気: モダン、ダーク

サウンド:
- 808ベース
- ハイハットロール
- スネアロール
- シンセサイザー

時代: 2010年代〜現在
```

**代表的アーティスト:**
```
- RL Grime
- Baauer
- Flosstradamus
- What So Not
```

## 4. その他の主要ジャンル

### Drum & Bass

**特徴:**
```
BPM: 160-180（非常に速い）
起源: イギリス、1990年代

サウンド:
- 高速ブレイクビート
- 重低音
- Jungle の影響

代表的アーティスト:
- Andy C
- Pendulum
- Netsky
```

### Dubstep

**特徴:**
```
BPM: 140（ハーフタイムで70）
起源: イギリス、2000年代

サウンド:
- ウォブルベース
- ハーフタイムドラム
- ダーク

代表的アーティスト:
- Skream
- Benga
- Excision
```

### Trance

**特徴:**
```
BPM: 130-150
起源: ドイツ、1990年代

サウンド:
- 長いビルドアップ
- エモーショナルなメロディ
- エピックなブレイクダウン

代表的アーティスト:
- Armin van Buuren
- Paul van Dyk
- Above & Beyond
```

## 5. ジャンルの聴き分け

### House vs Techno

**違い:**
```
House:
- ソウルフル
- ボーカル多い
- メロディアス
- 温かい

Techno:
- 工業的
- ボーカル少ない
- ミニマル
- クール

両方: 4つ打ち、でも雰囲気が違う
```

### Deep House vs Tech House

**違い:**
```
Deep House:
- 遅い（118-125 BPM）
- ジャジー
- メロウ

Tech House:
- 速い（125-130 BPM）
- パーカッシブ
- エネルギッシュ

ミックス: スムーズに移行可能
```

## 6. ジャンル別のミックス戦略

### House DJ

**セット構成:**
```
60分セット:
0-15分: Deep House（ウォームアップ）
15-40分: Tech House（ピーク）
40-50分: Progressive House（メインピーク）
50-60分: Deep House（クールダウン）

BPM: 118 → 125 → 128 → 125
```

### Techno DJ

**セット構成:**
```
60分セット:
0-15分: Minimal Techno（125 BPM）
15-40分: Techno（130 BPM）
40-50分: Industrial Techno（138 BPM、ピーク）
50-60分: Techno（130 BPM）

長時間セット推奨（2-3時間）
```

### Hip Hop DJ

**セット構成:**
```
60分セット:
- Boom Bap（クラシック）
- 現代Hip Hop
- Trap
- スクラッチで装飾
- ジャグリング

BPM: 85-100（変化が少ない）
```

## 7. ジャンル知識の学習方法

### Week 1-2: House完全理解

```
毎日2時間:
- House プレイリスト50曲作成
- 毎日聴き込む
- Deep、Tech、Progressiveの違いを体感
- アーティスト20名を覚える

目標: House を聴いてサブジャンルが判別できる
```

### Week 3-4: Techno完全理解

```
毎日2時間:
- Techno プレイリスト50曲作成
- Minimal、Industrial、Acidを聴き分ける
- Berghain（ベルリン）のセットを研究
- アーティスト20名を覚える

目標: Techno の雰囲気を理解
```

### Week 5-6: その他ジャンル

```
Hip Hop、Drum & Bass、Dubstep:
各ジャンル20曲
アーティスト10名ずつ

目標: 主要ジャンルを網羅
```

## まとめ

- **House**: ソウルフル、ディスコの影響、118-130 BPM
- **Techno**: 工業的、ミニマル、125-145 BPM
- **Hip Hop**: ブレイクビート、80-110 BPM、ターンテーブリズム
- **サブジャンル**: 各ジャンル内の多様性を理解
- **聴き分け**: 100曲聴けば自然に判別できる
- **学習**: 毎日2時間、6週間で主要ジャンル完全網羅
- **実践**: プレイリスト作成、プロのセット研究
