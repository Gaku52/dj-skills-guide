# セットリスト構築

60分の完璧なセットリストを設計する方法を学びます。

## この章で学ぶこと

- セットリストの基本構造
- BPMとキーの流れ
- 時間配分
- 曲順の決め方
- 予備プレイリストの準備
- セットリストの書き方
- プロのセット分析

## なぜセットリストが重要か

**即興 vs 事前準備:**
```
完全即興:
- 柔軟性あり
- 失敗のリスク高い
- 経験豊富なDJ向け

セットリスト準備:
- 安心感
- 流れが完璧
- 初心者〜中級者必須

プロ: 80%準備 + 20%即興
```

## 1. 60分セットの基本構造

### 4つのフェーズ

**Phase 1: ウォームアップ (0-15分)**
```
目的: 観客を温める

特徴:
- 低〜中エネルギー
- 遅めのBPM
- メロウな曲
- グルーヴ重視

曲数: 3-5曲
```

**Phase 2: エネルギー構築 (15-35分)**
```
目的: 徐々にエネルギーを上げる

特徴:
- 中〜高エネルギー
- BPMを徐々に上げる
- ビルドアップ
- ダンサブル

曲数: 5-7曲
```

**Phase 3: ピーク (35-50分)**
```
目的: 最高潮

特徴:
- 最高エネルギー
- 速いBPM
- ドロップ多用
- 盛り上がり曲

曲数: 4-6曲
```

**Phase 4: クールダウン (50-60分)**
```
目的: 次のDJへの橋渡し

特徴:
- エネルギー下げる
- BPM下げる
- スムーズな終わり方

曲数: 2-4曲
```

## 2. BPMの流れ

### House セット例

**時間別BPM:**
```
0-15分:   120-123 BPM (ウォームアップ)
15-25分:  123-126 BPM (徐々に上げる)
25-40分:  126-128 BPM (ピーク前)
40-50分:  128-130 BPM (ピーク)
50-60分:  126-128 BPM (クールダウン)

ルール: +2-3 BPMずつ上げる
```

### Techno セット例

**時間別BPM:**
```
0-15分:   125-128 BPM
15-35分:  128-132 BPM
35-50分:  132-138 BPM (ピーク)
50-60分:  130-132 BPM

Techno: Houseより速め
```

### Hip Hop セット例

**時間別BPM:**
```
全体: 85-100 BPM (変化が少ない)

理由:
- Hip HopはBPM変化が小さい
- エネルギーは選曲とスクラッチで調整
- グルーヴ重視
```

## 3. キーの流れ

### Camelot Wheel復習

**ハーモニックミキシング:**
```
基本ルール:
- 同じキー: 8A + 8A
- ±1: 8A + 7A または 9A
- A ↔ B: 8A + 8B

60分セットのキーの流れ例:
8A → 8A → 9A → 9B → 10B → 10A → 9A → 8A

円を描くように移動
```

### キーの流れ例

**House セット:**
```
0-15分:   8A (Am) - Deep House
15-25分:  8A → 9A (Em) - 徐々に上げる
25-35分:  9A → 10A (Bm) - エネルギー上げる
35-45分:  10A → 10B (D) - ピーク
45-50分:  10B → 9B (A) - まだピーク
50-60分:  9A → 8A - クールダウン

マイナーキーから始め、ピークでメジャーキー
```

## 5. 時間配分の計算

### 平均ミックス時間

**ジャンル別:**
```
House/Techno:
- ウォームアップ: 5-7分/曲
- ピーク: 3-4分/曲
- クールダウン: 5-7分/曲

Hip Hop:
- 2-3分/曲（短め）
- スクラッチで延長

計算: 60分 ÷ 平均時間 = 曲数
```

### 時間調整の方法

**長すぎる場合:**
```
1. ミックス時間を短くする（8 bar → 16 bar）
2. イントロを短く
3. アウトロを短く
4. 曲数を減らす

目標: 58-62分（±2分許容範囲）
```

**短すぎる場合:**
```
1. ミックス時間を長くする（16 bar → 32 bar）
2. ループで延長
3. 曲数を増やす
```

## 7. よくある失敗と対処法

### 失敗1: BPMの上げすぎ

**問題:**
```
120 → 130 → 140 BPM（急上昇）
→ 観客がついていけない
```

**対処:**
```
✓ +2-3 BPMずつ
✓ 5曲で+10 BPM程度
✓ 段階的に
```

### 失敗2: キーがバラバラ

**問題:**
```
8A → 3B → 11A → 5B
→ 不協和音、違和感
```

**対処:**
```
✓ Camelot Wheelを守る
✓ ±1キーずつ移動
✓ 事前にチェック
```

### 失敗3: ピークが多すぎる

**問題:**
```
全ての曲がピーク曲
→ メリハリがない
→ 観客が疲れる
```

**対処:**
```
✓ ピークは1-2回
✓ 波を作る
✓ クールダウンも重要
```

### 失敗4: 予備曲がない

**問題:**
```
観客の反応が悪い
→ 変更したい
→ 予備がない
→ パニック
```

**対処:**
```
✓ 各フェーズに予備2-3曲
✓ 合計10曲の予備
✓ 柔軟に対応
```

## 8. プロのセット分析

### Analysisの方法

**Step 1: プロのミックスを入手**
```
推奨:
- Mixcloud
- SoundCloud
- Resident Advisor podcasts
- Boiler Room sets

1時間のセットをダウンロード
```

**Step 2: 分析**
```
聴きながらメモ:
0:00 - Deep House, メロウ
5:00 - Tech Houseに移行
10:00 - BPM上がった（126くらい）
...

曲名も可能なら特定（Shazam使用）
```

**Step 3: セットリスト再現**
```
Rekordboxで:
1. 特定した曲を並べる
2. BPMとキーを確認
3. 流れを分析
4. 自分のセットに応用
```
