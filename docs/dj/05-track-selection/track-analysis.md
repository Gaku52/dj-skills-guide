# 曲分析

曲の構造を深く理解し、完璧なミックスポイントを見つける技術を学びます。

## この章で学ぶこと

- 曲構造の完全理解
- Rekordboxでの分析方法
- ミックスポイントの発見
- キーとBPMの確認
- Hot Cue設定戦略
- 相性の良い曲の見つけ方
- プロのアナリストになる方法

## なぜ曲分析が重要か

**知らない vs 知っている:**
```
曲を知らない:
- ミックスポイントがわからない
- ビルドアップが突然来る
- 失敗のリスク高い

曲を完全に理解:
- 完璧なミックスポイント
- ビルドアップを予測
- 失敗しない

プロは全ての曲を3回以上聴く
```

## 1. 曲構造の理解

### House/Technoの基本構造

**典型的な構造:**
```
1. Intro (イントロ) - 32 bar
   - ドラムから始まる
   - 徐々に音が追加
   - キックのみ（最初の16 bar）

2. Verse 1 (第1セクション) - 32 bar
   - メロディ追加
   - ベースライン
   - ボーカル（時々）

3. Buildup 1 (ビルドアップ) - 16 bar
   - 緊張感が高まる
   - フィルター、ロール
   - ドロップへの準備

4. Drop 1 (ドロップ) - 32 bar
   - 最もエネルギッシュ
   - 全ての要素
   - メインセクション

5. Breakdown (ブレイクダウン) - 32 bar
   - キックが抜ける
   - メロディのみ
   - 次のドロップへの準備

6. Buildup 2 - 16 bar
   - 再度ビルドアップ

7. Drop 2 - 32 bar
   - 2回目のドロップ
   - Drop 1と似ている

8. Outro (アウトロ) - 32 bar
   - 徐々に音が減る
   - ドラムのみ
   - 終わり

合計: 約6-7分
```

### Hip Hopの基本構造

**典型的な構造:**
```
1. Intro - 8 bar
   - ビート開始
   - サンプル

2. Verse 1 - 16 bar
   - ラップ
   - メインビート

3. Chorus (Hook) - 8 bar
   - キャッチーな部分
   - 覚えやすい

4. Verse 2 - 16 bar

5. Chorus - 8 bar

6. Bridge - 8 bar
   - 変化

7. Verse 3 - 16 bar

8. Chorus - 8 bar

9. Outro - 8 bar

合計: 約3-4分（House/Technoより短い）
```

## 2. Rekordboxでの分析

### 波形表示

**拡大表示:**
```
Rekordbox:
1. 曲をダブルクリック
2. 波形が表示される
3. +/- で拡大/縮小

観察:
- 青い波形: 音量
- 高い部分: ドロップ、ピーク
- 低い部分: ブレイクダウン
- パターンを読む
```

### セクションマーカー

**視覚的に理解:**
```
波形から判断:

Intro: _____/‾‾‾
  → 徐々に音が増える

Drop: ████████
  → 最も高い波形

Breakdown: ‾‾‾‾‾
  → 波形が低い

Outro: ‾‾‾\_____
  → 徐々に音が減る

これでセクションがわかる
```

## 3. ミックスポイントの発見

### ミックスイン可能なポイント

**House/Techno:**
```
推奨ミックスポイント:

1. Intro (0:00-1:00)
   - 最も簡単
   - ドラムのみ
   - 完璧なミックスイン

2. Breakdown後 (約4:00)
   - 2番目のビルドアップ前
   - エネルギーを引き継げる

3. Drop中 (約2:30)
   - 上級者向け
   - ダブルドロップ
   - 難易度高い

イントロが最も無難
```

### ミックスアウト可能なポイント

**House/Techno:**
```
推奨ミックスアウトポイント:

1. Outro (5:30-6:30)
   - 最も自然
   - ドラムのみに戻る
   - 完璧なミックスアウト

2. Breakdown (約4:00)
   - キックが抜ける瞬間
   - 次の曲へスムーズ

3. Drop終わり (約3:30)
   - ブレイクダウン直前
   - タイミング重要

アウトロが最も無難
```

## 4. Hot Cue設定戦略

### 8つのHot Cue

**推奨配置（House/Techno）:**
```
DDJ-FLX4: 8つのパッド

Pad 1 (赤): Intro開始（曲の最初）
Pad 2 (黄): ドラム完成（Intro終わり、32 bar）
Pad 3 (緑): Verse開始（64 bar）
Pad 4 (青): Drop 1開始
Pad 5 (紫): Breakdown開始
Pad 6 (ピンク): Drop 2開始
Pad 7 (オレンジ): Outro開始
Pad 8 (水色): 特別なポイント（ボーカル等）

全てフレーズの頭（1拍目）に設定
```

**推奨配置（Hip Hop）:**
```
Pad 1: Intro
Pad 2: Verse 1
Pad 3: Chorus
Pad 4: Verse 2
Pad 5: Bridge
Pad 6: Verse 3
Pad 7: Outro
Pad 8: キック位置（ジャグリング用）
```

### Hot Cue設定手順

**Rekordboxで:**
```
1. 曲を再生
2. セクション開始位置で止める
3. Waveform拡大で1拍目を確認
4. Hot Cueボタンをクリック
5. 色を選択
6. コメント追加（"Drop 1"等）
7. 次のセクションへ

全ての曲で設定（時間かかるが必須）
```

## 5. キーとBPMの確認

### Rekordbox自動解析

**解析手順:**
```
1. 曲を右クリック
2. [解析] → [全て]
3. BPM、キーが自動検出
4. 結果を確認

注意: たまに間違いあり
```

### 手動確認

**BPMが怪しい場合:**
```
確認方法:
1. メトロノームアプリ使用
2. 曲に合わせてタップ
3. BPMを確認
4. Rekordboxで手動修正

よくある間違い:
- 倍（130 → 65）
- 半分（65 → 130）
```

**キーが怪しい場合:**
```
確認方法:
1. 同じキーの曲と同時再生
2. 不協和音がないか確認
3. 違和感があれば手動修正

耳が最終ジャッジ
```

## 6. 相性の良い曲の見つけ方

### Related Tracks機能

**Rekordbox:**
```
曲を右クリック → [Related Tracks]

表示:
- 同じキーの曲
- 似たBPMの曲
- AIによる推薦

非常に便利
```

### 手動での探し方

**条件:**
```
曲A: 126 BPM, 8A (Am)

相性の良い曲B:
BPM: 124-128 (±2%)
キー: 8A, 7A, 9A, 8B

Rekordboxでフィルター:
1. キーでソート → 8A
2. BPM範囲設定 → 124-128
3. 候補が表示される
4. 聴いて確認
```

## 7. 50曲完全分析プロジェクト

### Week 1-2: 分析作業

**1曲あたり30分:**
```
1. Rekordboxで曲を開く (2分)
2. 3回通して聴く (18分)
3. 波形を見ながらセクション確認 (5分)
4. Hot Cue設定 (3分)
5. キー、BPM確認 (2分)

50曲 = 25時間（2週間で完了）
```

## 8. よくある失敗と対処法

### 失敗1: Hot Cueがズレている

**原因:**
```
フレーズの頭でなく、途中に設定
```

**対処:**
```
✓ Waveform拡大
✓ ビートグリッド確認
✓ 必ず1拍目に設定
✓ 再生して確認
```

### 失敗2: 曲を聴いていない

**問題:**
```
Hot Cueだけ設定
→ 曲の雰囲気を知らない
→ ミックスが不自然
```

**対処:**
```
✓ 最低3回通して聴く
✓ 雰囲気を覚える
✓ メモを取る
```

### 失敗3: Rekordboxの自動解析を信じすぎる

**問題:**
```
BPMが間違っている（倍/半分）
キーが間違っている
→ ミックスが失敗
```

**対処:**
```
✓ 必ず耳で確認
✓ 疑わしい場合は手動修正
✓ 同じキーの曲と比較
```

## まとめ

- **曲構造**: Intro → Verse → Buildup → Drop → Breakdown → Drop → Outro
- **Rekordbox**: 波形表示、セクション視覚化
- **ミックスポイント**: Introが最も簡単、Outroで自然に終わる
- **Hot Cue**: 8つ全て設定、フレーズの頭（1拍目）
- **キーとBPM**: 自動解析 + 手動確認、耳が最終ジャッジ
- **相性**: Related Tracks機能、同じキー・似たBPM
- **50曲プロジェクト**: 2週間で完全分析、1曲30分
- **練習**: 分析→Hot Cue設定→聴き込む
