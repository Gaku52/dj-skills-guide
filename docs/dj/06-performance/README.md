# パフォーマンス実践

技術と選曲を学んだ後は、実際のパフォーマンス。プロの現場力を完全マスターします。

## このセクションで学ぶこと

- ステージプレゼンス
- 観客との交流（MC、アイコンタクト）
- セットアップとサウンドチェック
- パフォーマンス準備
- セットのレコーディング
- ライブストリーミング
- クラブでのプロトコル、マナー
- Back to Back（他DJとの共演）
- オープンデッキ、トライアウト
- 本番中のトラブル対応

## なぜパフォーマンススキルが重要か

**技術完璧 vs パフォーマンス:**
```
技術完璧、でもステージで緊張:
- 手が震える
- 失敗する
- 観客とのコネクションなし
→ 印象に残らない

技術 + パフォーマンス:
- 堂々としている
- 観客とコネクション
- エネルギーがある
- 記憶に残る

パフォーマンス = プロの証
```

**クラブでのマナー:**
```
技術があっても:
- 遅刻する
- 機材を壊す
- 他DJに失礼
→ 二度と呼ばれない

プロトコルを守る:
- 時間厳守
- 機材を大切に
- 他DJとの協力
→ リピートブッキング
```

## DDJ-FLX4での実践

### 持ち運び

**コンパクトな利点:**
```
DDJ-FLX4:
- 重量: 約2.1kg
- サイズ: コンパクト
- USB給電可能

持ち運び:
✓ バックパックに入る
✓ 電車で移動可能
✓ 海外遠征も楽

プロの機材（CDJ-3000等）より圧倒的に楽
```

### クラブでのセットアップ

**基本手順:**
```
1. DDJ-FLX4をブースに設置
2. USBケーブルでPCに接続
3. Rekordbox起動
4. オーディオ設定確認
5. ヘッドフォン接続
6. サウンドチェック（5分）

合計: 10分で完了

CDJセットアップより簡単
```

## チェックリスト

### 本番前（1週間前）

```
□ ギグの詳細確認（日時、場所、時間）
□ セットリスト作成
□ 全ての曲をRekordboxで確認
□ Hot Cue設定済み
□ 機材動作確認
□ バックアップUSB作成
□ 移動手段確認
```

### 本番前（当日）

```
□ 十分な睡眠（7-8時間）
□ 軽い食事（重すぎない）
□ 持ち物チェック
□ 到着時間確認（1時間前）
□ セットリスト最終確認
□ メンタル準備
```

### 本番後

```
□ 機材撤収

□ 観客とのコミュニケーション
□ SNSで感謝投稿
□ セットを振り返り
□ 改善点をメモ
```

## よくある質問

### Q1: 緊張で手が震える、どうすれば？

**A: 練習と経験、深呼吸**

```
対処法:
1. 深呼吸（4秒吸う、4秒止める、8秒吐く）
2. 「失敗しても死なない」と自分に言う
3. 準備を完璧にする（自信につながる）
4. 小規模な場で経験を積む
5. 100回プレイすれば慣れる

緊張は正常、プロも緊張する
```

### Q2: MCは必要？

**A: ジャンルによる**

```
House/Techno:
- ほぼ不要
- 音楽が主役
- たまに「Thank you」程度

Hip Hop/Open Format:
- 積極的にMC
- 観客を盛り上げる
- 個性を出す

自分のスタイルに合わせる
```

### Q3: トラブルが起きたら？

**A: 冷静に対処、予備プラン**

```
準備:
- 予備USB必須
- 予備ケーブル
- 基本的なトラブル対処を覚える

本番:
- 深呼吸
- 笑顔を保つ
- 観客は詳細を知らない
- プロに見せる
```
