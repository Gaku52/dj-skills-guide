## なぜプロトコルが重要か

**マナー違反 vs プロの振る舞い:**
```
マナー違反:
- 遅刻
- 機材を雑に扱う
- 他DJに失礼
- スタッフに横柄
→ 二度と呼ばれない

プロの振る舞い:
- 時間厳守
- 機材を大切に
- 他DJと協力
- スタッフに敬意
→ リピートブッキング

プロトコル = プロの証
```

## 1. クラブプロトコルとは

### 定義

```
クラブプロトコル = クラブでの暗黙のルール

要素:
- 時間管理
- 機材の扱い
- 人間関係
- エチケット
- プロフェッショナリズム

目的: 円滑な運営、良好な関係
```

### なぜ重要か

**業界は狭い:**
```
DJ業界:
- 東京でも狭い
- 大阪でも狭い
- 噂はすぐ広まる

悪い噂:
「あのDJ、遅刻した」
「機材壊した」
「態度が悪い」
→ 業界から干される

良い評判:
「あのDJ、プロフェッショナル」
「時間厳守」
「協力的」
→ ブッキング増える

評判 = キャリア
```

## 2. 時間厳守の重要性

### 絶対に遅刻しない

**到着時間:**
```
ルール: 自分のセット時間の60分前到着

例:
あなたのセット: 22:00-23:00
到着時間: 21:00

理由:
- セットアップ時間（15分）
- サウンドチェック（10分）
- 前のDJ観察（20分）
- 余裕（15分）

遅刻 = 最悪のマナー違反
```

### 遅刻しそうな場合

**緊急時の対応:**
```
もし遅刻しそうなら:

1. 即座に連絡（30分前に気づいた時点で）
   - プロモーターに電話
   - 理由と到着時刻を伝える

2. 謝罪
   「申し訳ありません、電車遅延で20分遅れます」

3. 到着したら再度謝罪

4. 挽回
   - 完璧なセット
   - スタッフに感謝

理由なき遅刻 = アウト
```

## 3. DJブースでのマナー

### ブースは神聖な場所

**ルール:**
```
✓ DJと関係者のみ
✓ 飲み物は蓋付きのみ
✓ 機材に触れない（自分のターン以外）
✓ 大声で話さない
✓ 写真は許可を得てから

✗ 友達を連れ込む
✗ お酒をこぼす
✗ 勝手に機材を触る
✗ DJの集中を妨げる
```

### 前のDJへの敬意

**スムーズな交代:**
```
あなたのターンが来たら:

1. ブースに入る許可を得る
   - 「入ってもいいですか？」
   - 前のDJに確認

2. 静かにセットアップ
   - 前のDJがプレイ中
   - 邪魔しない
   - 静かに準備

3. 最後の曲まで待つ
   - 前のDJが終わるまで
   - 焦らない

4. 交代時に挨拶
   「お疲れ様でした」
   「良いセットでしたね」

敬意を払う
```

### 次のDJへの配慮

**スムーズな引き継ぎ:**
```
あなたのセットが終わったら:

1. 次のDJにスペースを譲る
   - 機材を片付ける
   - ブースを空ける

2. 引き継ぎ情報
   （もし聞かれたら）
   - 「観客はTech Houseで盛り上がってました」
   - 「エネルギーは高めです」

3. 挨拶
   「頑張ってください」

4. ブースから退出
   - すぐに退出
   - 長居しない

次のDJのため
```

## 4. 機材の扱い方

### 他人の機材は自分の物以上に大切に

**基本原則:**
```
クラブの機材:
- 数百万円
- 壊したら弁償
- 超高額

扱い方:
✓ 優しく
✓ 丁寧に
✓ 自分の物より大切に

✗ 乱暴に
✗ 雑に
✗ 無理に動かす
```

### やってはいけないこと

**絶対NG:**
```
✗ フェーダーを無理やり動かす
✗ ノブを限界まで回す
✗ ボタンを強く叩く
✗ ケーブルを引っ張る
✗ 機材に飲み物を置く
✗ 機材に寄りかかる
✗ ジャンプして振動を与える

壊したら:
→ 弁償（数十万円）
→ 二度と呼ばれない
→ 業界で悪評
```

### DDJ-FLX4持ち込みの場合

**自分の機材でも:**
```
✓ 丁寧にセットアップ
✓ ケーブルをテープで固定
✓ 終わったら速やかに撤収
✓ クラブのスペースに配慮

クラブの機材を使わせてもらっている
感謝の気持ち
```

## 5. クラブスタッフとの関係

### スタッフ全員に敬意を

**スタッフの種類:**
```
関わるスタッフ:
- プロモーター（ブッキング担当）
- 音響スタッフ（音響担当）
- バーテンダー
- セキュリティ
- 受付スタッフ

全員に敬意を払う
```

### 基本的な礼儀

**到着時:**
```
1. 受付で挨拶
   「こんばんは、今夜DJを担当します[名前]です」

2. プロモーターに挨拶
   「お世話になります」
   「今日はありがとうございます」

3. 音響スタッフに挨拶
   「よろしくお願いします」

4. バーテンダーに挨拶
   （後で飲み物をもらう可能性）

第一印象が大事
```

### 終了後の挨拶

**終わったら:**
```
必ず全員にお礼:

1. プロモーターに
   「ありがとうございました」
   「また呼んでいただけたら嬉しいです」

2. 音響スタッフに
   「音響ありがとうございました」

3. バーテンダーに
   「ありがとうございました」

4. セキュリティに
   「お疲れ様でした」

感謝の気持ち = 再ブッキング
```

## 6. 他DJとの協力

### DJコミュニティ

**横のつながり:**
```
同じイベントのDJ:
- ライバルではない
- 仲間
- コミュニティ

良い関係:
✓ 情報交換
✓ 助け合い
✓ ネットワーク拡大
✓ 将来のコラボ

悪い関係:
✗ 孤立
✗ 情報なし
✗ ブッキング減る
```

### ネットワーキング

**交流の仕方:**
```
他のDJに:

1. 挨拶
   「はじめまして、[名前]です」
   「今夜DJします」

2. 褒める
   「あなたのセット素晴らしかったです」
   「特に[曲名]が良かったです」

3. 連絡先交換
   「Instagram交換しましょう」
   「連絡先交換しませんか」

4. 後日フォロー
   InstagramでDM
   「先日はありがとうございました」

ネットワーク = キャリア
```

## 7. プロモーターとのコミュニケーション

### プロモーターとは

```
プロモーター = イベントを企画・運営する人

役割:
- DJをブッキング
- イベント告知
- 会場手配
- 予算管理

最も重要な人
```

### 良好な関係を築く

**やるべきこと:**
```
✓ 時間厳守
✓ プロフェッショナル
✓ 要求が合理的
✓ SNSでイベント告知協力
✓ 感謝の気持ち
✓ イベント後にお礼メッセージ

例（終了後のメッセージ）:
「今夜はありがとうございました。
素晴らしいイベントでした。
また呼んでいただけたら嬉しいです。」

プロモーターが満足 = 再ブッキング
```

### やってはいけないこと

**NG行動:**
```
✗ ギャラの不満を公に言う
✗ イベントの悪口
✗ 無理な要求
✗ プロモーターを無視
✗ 契約を破る

評判が悪くなる
```

## 8. 音響スタッフとの関係

### 音響スタッフの重要性

```
音響スタッフ = あなたの音を作る人

役割:
- Master音量調整
- EQ調整
- モニター調整
- トラブル対応

音響スタッフ次第で音質が変わる
```

### コミュニケーション

**やり方:**
```
サウンドチェック時:

1. 挨拶
   「よろしくお願いします」

2. 音量確認
   「この音量で大丈夫ですか？」
   （勝手に決めない）

3. 問題があれば丁寧に
   「モニターが少し聞こえにくいです」
   「もう少し上げていただけますか？」
   （命令口調NG）

4. お礼
   「ありがとうございます」

本番中:

問題があれば:
- 手を上げて合図
- ジェスチャーで説明
- 感謝を忘れずに

終了後:

必ずお礼:
「音響ありがとうございました」
「素晴らしい音でした」

良好な関係 = 良い音
```

## 10. トラブル対応

### 機材トラブル

**対処法:**
```
機材が故障した:

1. 音響スタッフに即座に報告
   - 手を上げる
   - 「機材トラブルです」

2. 冷静に対応
   - 焦らない
   - 笑顔を保つ

3. スタッフに任せる
   - 自分で無理に直さない
   - プロに任せる

4. 予備プラン
   - 予備USB
   - 別の機材

5. 謝罪（もし自分のミスなら）

プロは冷静
```

### 観客とのトラブル

**対処法:**
```
観客が迷惑行為:

1. 無視（最初は）
   - 反応しない
   - プレイに集中

2. 悪化したら
   - セキュリティに合図
   - 自分で対応しない

3. 絶対に喧嘩しない
   - プロフェッショナル
   - 大人の対応

セキュリティの仕事
DJは音楽に集中
```

## 11. よくある質問

### Q1: ギャラの交渉はどうすれば？

**A: 事前にメールで**

```
タイミング:
- ブッキング時
- 事前にメールで
- 当日は絶対NG

交渉例:
「今回のギャラはいくらですか？」
「最低ギャラは[金額]円です」

丁寧に:
✓ 「相談させてください」
✓ 「調整可能ですか？」

横柄に:
✗ 「安すぎる」
✗ 「もっとよこせ」

プロフェッショナルに
```

### Q2: 観客が少ない場合は？

**A: 全力でプレイ**

```
観客10人でも:
✓ 全力でプレイ
✓ プロフェッショナル
✓ 手を抜かない

理由:
- その10人が次に呼んでくれるかも
- プロモーターが見ている
- プロは観客数に関係なく全力

観客数 ≠ プレイの質
```

### Q3: 他DJが時間オーバーしたら？

**A: プロモーターに相談**

```
対処:
1. プロモーターに相談
   「次は私のターンですが...」

2. プロモーターが対応
   - DJに伝える
   - 時間調整

3. 自分は怒らない
   - 大人の対応
   - プロフェッショナル

直接文句を言わない
プロモーター経由
```

## まとめ

- **時間厳守**: 60分前到着、遅刻は最悪のマナー違反、連絡必須
- **DJブース**: 神聖な場所、前後のDJに敬意、静かにセットアップ
- **機材**: 優しく丁寧に、壊したら弁償、飲み物注意
- **スタッフ**: 全員に敬意、挨拶とお礼、音響スタッフは重要
- **他DJ**: 仲間、ネットワーキング、褒める、連絡先交換
- **プロモーター**: 最重要人物、感謝の気持ち、イベント後にお礼
- **音響スタッフ**: 音を作る人、丁寧なコミュニケーション、お礼必須
- **NG行動**: 遅刻、泥酔、機材破損、時間オーバー、悪口厳禁
- **トラブル**: 冷静に対応、スタッフに任せる、プロフェッショナル
- **ギャラ**: 事前にメールで交渉、当日はNG、丁寧に
