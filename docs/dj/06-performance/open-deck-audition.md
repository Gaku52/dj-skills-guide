# オープンデッキ・トライアウト

初心者がプロへの第一歩を踏み出すための完全ガイド。

## この章で学ぶこと

- トライアウトとは
- 準備するもの

- プレイ戦略（短時間で印象）
- 成功の秘訣

- 次のステップ

## なぜオープンデッキが重要か

**キャリアのスタート:**
```
問題:
初心者はどうやってギグを獲得？
→ 実績ない
→ 誰も雇わない
→ 経験積めない

解決:
オープンデッキ = チャンス
→ 実力を証明
→ プロモーターが見る
→ ブッキング獲得

オープンデッキ = キャリアの入り口
```

---

## 1. オープンデッキとは

### 定義

```
オープンデッキ = 誰でも参加できるDJセッション

特徴:
- 事前登録（通常）
- 短時間プレイ（15-30分）
- 審査なし（ほとんど）
- 経験不問
- 無料または少額

目的: 経験を積む、実力を見せる
```

### 開催場所

**よくある場所:**
```
- 小規模バー
- クラブの早い時間帯（18:00-20:00）
- DJ学校のイベント
- レコードショップ
- コミュニティスペース

頻度:
- 毎週または隔週
- 固定曜日（例: 毎週水曜）

東京、大阪等の都市部に多い
```

---

## 2. トライアウトとは

### 定義

```
トライアウト = オーディション形式のオープンデッキ

特徴:
- 審査あり
- 勝者がレジデンシー獲得
- 競争的
- ハイレベル
- プレッシャー大

例:
「Club Asia DJ Audition」
「Contact DJ Trial」

勝つ = 定期ブッキング
```

### オープンデッキ vs トライアウト

**違い:**
```
オープンデッキ:
✓ カジュアル
✓ 審査なし
✓ 全員プレイ可能
✓ プレッシャー小
✓ 初心者向け

トライアウト:
- 競争的
- 審査あり
- 勝者のみ報酬
- プレッシャー大
- 中級者以上

まずはオープンデッキから
```

---

## 3. 準備するもの

### 必須

**機材:**
```
□ DDJ-FLX4（持ち込みの場合）
□ ノートPC
□ USBメモリ x 2（メイン + 予備）
□ ヘッドフォン
□ ケーブル類

または:
- 会場のCDJ使用
- USBメモリのみ持参

事前に確認必須
```

### セットリスト

**完璧な30分セット:**
```
準備:
1. 30分ぴったりのセットリスト
   - 7-10曲
   - BPM昇順
   - エネルギーカーブ

2. Plan A, B, C
   - メインセット（Plan A）
   - 代替曲（Plan B）
   - 緊急曲（Plan C）

3. 全曲を100回聴く
   - 完全に覚える
   - ミックスポイント完璧
   - Hot Cue設定済み

準備 = 自信 = 成功
```

---

## 4. 審査ポイント

### プロモーターが見るポイント

**Top 5:**
```
1. 技術（30点）
   - ミックス精度
   - ビートマッチング
   - EQ使い方
   - トランジション

2. 選曲（30点）
   - 曲の質
   - 流れ
   - 独自性
   - 会場に合う

3. プレゼンス（20点）
   - ステージング
   - 自信
   - エネルギー
   - 観客とのコネクション

4. 時間管理（10点）
   - 時間厳守
   - 30分ぴったり
   - オーバーしない

5. プロフェッショナリズム（10点）
   - 機材の扱い
   - マナー
   - 態度

技術 + 選曲 = 60%
最も重要
```

---

## 5. プレイ戦略

### 30分で印象を残す

**戦略:**
```
時間が短い:
→ 最初の1分が勝負
→ 強い曲で始める
→ エネルギーを保つ
→ 最後も強く

30分の流れ:

0-5分: 強いスタート
- 印象的な曲
- エネルギー中〜高
- 技術を見せる

5-20分: メインパート
- 流れを作る
- 選曲センスを見せる
- 安定したミックス

20-25分: ピーク
- 最高のエネルギー
- 最高の曲
- 観客を盛り上げる

25-30分: 強い終わり
- 記憶に残る曲
- エネルギー高いまま
- 「もっと聴きたい」

最初と最後 = 最重要
```

### やってはいけないこと

**NG行動:**
```
✗ ウォームアップから始める
   理由: 時間が短い、退屈

✗ 実験的すぎる曲
   理由: リスク高い、観客困惑

✗ 技術を見せつけすぎ
   理由: スクラッチ連発等、やりすぎ

✗ エネルギー低い曲
   理由: 盛り上がらない

✗ 時間オーバー
   理由: 次のDJに迷惑

✗ 緊張で下を向く
   理由: プレゼンスゼロ

安全に、でも印象的に
```

## 6. 成功の秘訣

### 秘訣1: 完璧な準備

**準備が全て:**
```
やること:
✓ セットリストを100回練習
✓ 全てのミックスポイント暗記
✓ Hot Cue完璧
✓ 予備プラン準備
✓ 機材動作確認

当日:
- 緊張しない
- 自信を持つ
- 準備したことをやるだけ

準備 = 自信
```

### 秘訣2: 独自性

**他と違うことをする:**
```
差別化:
✓ 他のDJが使わない曲
✓ 独自のミックススタイル
✓ 個性的な選曲
✓ 自分らしさ

避ける:
✗ Beatport Top 10だけ
✗ 他DJの完全コピー
✗ 無個性

記憶に残る = ブッキング
```

### 秘訣3: 観客を見る

**プレゼンス:**
```
技術だけじゃダメ:
✓ 顔を上げる
✓ 観客とアイコンタクト
✓ 笑顔
✓ エネルギーを出す

下を向きっぱなし:
✗ 印象に残らない
✗ 自信なさそう

プレゼンス = プロっぽい
```

## 7. 失敗例と対処

### 失敗例1: 緊張でミス連発

**問題:**
```
緊張:
→ 手が震える
→ ビートマッチミス
→ 焦る
→ さらにミス
→ パニック
```

**対処:**
```
ミスしたら:
1. 深呼吸
2. 次の曲に集中
3. 笑顔を作る（強制的に）
4. 「大丈夫」と自分に言う

1つのミス ≠ 失敗
全体で判断される

落ち着く
```

### 失敗例2: 時間オーバー

**問題:**
```
30分のはずが35分:
→ 次のDJに迷惑
→ プロモーター激怒
→ 印象最悪
```

**対処:**
```
予防:
✓ タイマーセット（25分でアラーム）
✓ 最後の曲は短め（3分）
✓ 余裕を持つ

もしオーバーしそうなら:
- 1曲飛ばす
- 早めに終わる

時間厳守 = プロの証
```

### 失敗例3: 観客ゼロ

**問題:**
```
オープンデッキ:
→ 観客が少ない（5人）
→ モチベーション下がる
→ 手を抜く
```

**対処:**
```
✓ 観客数に関係なく全力
✓ プロモーターが見ている
✓ 練習の機会と考える
✓ 5人でも全力

観客5人 = 5人のファン獲得チャンス
```

## 9. プロモーターへのアプローチ

### オープンデッキ後

**成功したら:**
```
終了後:

1. プロモーターに挨拶
   「今日はありがとうございました」
   「楽しかったです」

2. 連絡先交換
   「連絡先交換させてください」
   Instagram、メールアドレス

3. 後日フォローアップ
   （3日後にメール）:

   「先日はありがとうございました。
   またの機会があれば嬉しいです。

   私のミックス: [SoundCloudリンク]

   よろしくお願いします。」

4. 定期的に連絡
   - 月1回程度
   - しつこくない
   - 新しいミックスを送る

関係構築 = 次のブッキング
```

### プロモーターの視点

**何を求めているか:**
```
プロモーターの視点:

✓ 技術的に安定
✓ 観客を盛り上げる
✓ プロフェッショナル
✓ 時間厳守
✓ トラブルなし
✓ 協力的

求めている:
- 安心して任せられるDJ
- 問題を起こさない
- 観客が満足

プロモーターの立場で考える
```

## 11. よくある質問

### Q1: 全く経験なしでも参加できる？

**A: オープンデッキはOK、トライアウトは難しい**

```
オープンデッキ:
✓ 初心者歓迎
✓ 経験不問
✓ 練習の場

ただし:
- 最低限の技術は必要
- ビートマッチングできる
- 30分プレイできる

推奨:
- 自宅で6ヶ月練習
- 友達の前で10回
- その後オープンデッキ

準備してから参加
```

### Q2: 何回参加すればギグ獲得？

**A: 個人差大、平均5-10回**

```
目安:
- 5-10回参加
- プロモーターに顔を覚えてもらう
- 技術向上
- ネットワーク拡大

早い人: 1回で獲得
遅い人: 20回以上

継続 = 成功
```

### Q3: 年齢制限は？

**A: 会場による、18歳以上が多い**

```
一般的:
- 18歳以上（バー、クラブ）
- 20歳以上（アルコール提供）

例外:
- オールエイジズイベント
- DJ学校のイベント

事前確認必須
```

## まとめ

- **オープンデッキ**: 誰でも参加可能、15-30分プレイ、経験を積む場
- **トライアウト**: オーディション形式、審査あり、勝者がレジデンシー
- **準備**: 完璧な30分セット、Plan A/B/C、100回練習
- **審査**: 技術30点、選曲30点、プレゼンス20点、時間管理10点、マナー10点
- **戦略**: 強い始まり、メイン、ピーク、強い終わり、最初と最後が重要
- **成功の秘訣**: 完璧な準備、独自性、観客を見る、プレゼンス
- **失敗対処**: ミスしても落ち着く、時間厳守、観客少なくても全力
- **見つけ方**: Google検索、SNS、クラブに直接問い合わせ
- **アプローチ**: 終了後挨拶、連絡先交換、後日フォロー、定期連絡
- **次のステップ**: 5-10回参加、ミックスアップロード、小規模ギグ、定期ギグ
