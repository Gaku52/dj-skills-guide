# パフォーマンス準備

本番で最高のパフォーマンスを発揮するための完璧な準備方法を学びます。

## この章で学ぶこと

- 完璧な準備の重要性
- 持ち物チェックリスト
- セットリスト最終確認
- メンタル準備
- 本番前ルーティン
- 食事と睡眠管理
- 服装選び
- 当日のスケジュール
- 緊急時の対応

## なぜ準備が重要か

**準備不足 vs 完璧な準備:**
```
準備不足:
- 忘れ物
- セットリスト未完成
- 睡眠不足
- 緊張しすぎ
→ 失敗

完璧な準備:
- 全て揃っている
- セットリスト完璧
- 体調万全
- 自信
→ 成功

準備 = 成功の80%
```

## 1. 準備の重要性

### プロの準備

```
プロDJ:
- 1週間前から準備開始
- 当日は完璧な状態
- 何も心配なし
- パフォーマンスに集中

アマチュア:
- 前日に慌てて準備
- 当日忘れ物
- 不安だらけ
- パフォーマンスに集中できない

準備時間 = 結果
```

## 2. 持ち物チェックリスト

### 必須アイテム（絶対忘れない）

**機材関連:**
```
□ DDJ-FLX4
□ ノートPC（充電済み）
□ 電源ケーブル（PC用）
□ USBケーブル（DDJ-FLX4接続用）
□ ヘッドフォン
□ 予備USBメモリ（同じ内容）

これを忘れたら終わり
前日夜にバッグに入れる
```

**ケーブル類（予備）:**
```
□ 予備USBケーブル
□ 予備RCAケーブル（DDJ-FLX4用）
□ 変換アダプター（6.3mm ↔ 3.5mm）
□ 電源タップ（任意）

ケーブルは消耗品
予備必須
```

**デジタル準備:**
```
□ Rekordboxに全曲ロード済み
□ Hot Cue全て設定済み
□ プレイリスト完成
□ 全曲解析済み
□ PCのバッテリー70%以上
□ Rekordboxライセンス有効
```

### 推奨アイテム

**パフォーマンス向上:**
```
□ 水（500ml）- 水分補給
□ タオル（汗拭き用）
□ ガムまたはミント（口臭対策）
□ スマートフォン（充電済み、Shazam用）
□ モバイルバッテリー
□ 名刺（ネットワーキング用）
□ ペン（セットリスト調整用）
```

**緊急時対応:**
```
□ 胃腸薬（緊張で腹痛の場合）
□ 痛み止め（頭痛の場合）
□ 目薬（長時間プレイ）
□ リップクリーム（乾燥対策）
□ マスク（感染症対策）
```

## 3. セットリスト最終確認

### 1週間前: セットリスト作成

**完璧なセットリスト:**
```
作成:
1. プレイ時間確認（60分？90分？）
2. ジャンル確認（何を期待されているか？）
3. 会場の雰囲気を考慮
4. 20-30曲選曲（60分の場合）

Rekordboxでプレイリスト作成:
- プレイリスト名: [日付]_[会場名]
- 例: 2025-01-15_Club_Asia

順番に並べる:
1. ウォームアップ（0-15分）
2. エネルギー構築（15-35分）
3. ピーク（35-50分）
4. クールダウン（50-60分）
```

### 3日前: 全曲確認

**全て聴き直す:**
```
作業:
1. セットリストの全曲を聴く
2. 各曲のポイント確認:
   - Hot Cueの位置
   - ミックスポイント
   - エネルギーレベル
3. 流れを確認
4. 問題があれば曲を変更

時間: 2時間

完璧になるまで調整
```

### 前日: 最終調整

**Plan A, B, Cを準備:**
```
Plan A（メインセットリスト）:
- 完璧に準備した曲順
- 理想的な流れ

Plan B（代替曲）:
- 観客の反応が悪い場合
- エネルギー調整用
- 10曲追加

Plan C（緊急用）:
- 絶対に盛り上がる曲
- セーフティネット
- 5曲

全てRekordboxの別プレイリストに
```

## 4. メンタル準備

### 1週間前: ポジティブな自己暗示

**自己肯定:**
```
毎日5分:

自分に言い聞かせる:
「準備は完璧」
「技術は十分」
「楽しむだけ」
「失敗しても死なない」
「プロDJも緊張する」

ポジティブな言葉:
✓ 「できる」
✓ 「準備した」
✓ 「自信がある」

ネガティブな言葉は禁止:
✗ 「失敗するかも」
✗ 「不安」
✗ 「無理」

言葉 = 現実
```

### 3日前: ビジュアライゼーション

**成功をイメージ:**
```
毎晩寝る前、10分:

イメージする:
1. 会場に入る
2. セットアップ完璧
3. 最初の曲をプレイ
4. 観客が盛り上がる
5. 完璧なミックス
6. 拍手喝采
7. 満足して終わる

脳は現実とイメージを区別しない
→ 成功体験として記憶
→ 本番で自信

毎日繰り返す
```

## 6. 食事と睡眠管理

### 前日の食事

**何を食べるか:**
```
昼食（前日）:
✓ 通常通り
✓ 消化の良いもの
✓ 食べ慣れたもの

夕食（前日）:
✓ 早めに（19:00までに）
✓ 軽め
✓ 炭水化物中心
✓ アルコール控えめ

避けるべき:
✗ 辛いもの（腹痛リスク）
✗ 揚げ物（胃もたれ）
✗ 生もの（食中毒リスク）
✗ 大量のアルコール
```

### 当日の食事

**本番までの食事:**
```
朝食（当日）:
✓ 7:30頃
✓ 軽め
✓ パン、果物、ヨーグルト
✓ カフェイン少なめ

昼食（当日）:
✓ 12:00-13:00
✓ 通常量
✓ エネルギー補給
✓ 水分補給

軽食（本番3時間前）:
✓ おにぎり、バナナ
✓ 消化が良いもの
✓ エネルギー補給

本番直前（1時間前）:
✗ 何も食べない
✗ 水分補給のみ

理由: 満腹だと集中できない
```

### 睡眠管理

**前日の睡眠:**
```
就寝時間:
- 23:00までに寝る
- 7-8時間確保
- 0:00以降は避ける

睡眠の質を上げる:
✓ 寝る1時間前にスマホを見ない
✓ 暗い部屋
✓ 快適な温度
✓ リラックス

睡眠不足 = 集中力低下
絶対に十分な睡眠を
```

## 7. 服装選び

### ジャンル別の服装

**House / Techno:**
```
スタイル:
- カジュアル
- 黒が基本
- シンプル

推奨:
✓ 黒のTシャツ
✓ 黒のパンツ
✓ スニーカー
✓ シンプルなアクセサリー

避ける:
✗ スーツ（硬すぎ）
✗ 派手すぎ
✗ サンダル
```

**Hip Hop / Open Format:**
```
スタイル:
- ストリート
- カジュアル
- 個性的

推奨:
✓ グラフィックTシャツ
✓ ジーンズ
✓ スニーカー（ナイキ、アディダス等）
✓ キャップ
✓ アクセサリー

個性を出す
```

### 実用的な服装

**動きやすさ重視:**
```
✓ 動きやすい服
✓ 汗をかいても大丈夫
✓ 長時間立っても疲れない靴
✓ ポケット付き（USBメモリ収納）

避ける:
✗ サンダル（危険）
✗ ヒール（疲れる）
✗ 厚手の服（暑い）
✗ タイトすぎる服（動きにくい）
```

## 9. 緊急時の対応

### 忘れ物をした場合

**DDJ-FLX4を忘れた:**
```
対処法:
1. 家に戻る（時間があれば）
2. プロモーターに連絡
3. 借りられるか確認
4. CDJでプレイ（可能なら）

予防:
✓ 前日夜にバッグに入れる
✓ 出発前に必ずチェック
```

**USBメモリを忘れた:**
```
対処法:
1. 予備USBを使う
2. Dropbox/Google Driveから再ダウンロード（事前にアップロード必須）
3. ストリーミング（Beatport Link等）

予防:
✓ 予備USBメモリ必須
✓ クラウドバックアップ
```

### 体調不良の場合

**当日体調が悪い:**
```
軽い体調不良:
- 痛み止め、胃腸薬
- 水分補給
- なんとか乗り切る

重い体調不良:
- プロモーターに連絡
- 代わりのDJ手配
- キャンセル

無理は禁物
```

## 10. よくある質問

### Q1: 本番前に練習すべき？

**A: 軽く確認程度**

```
おすすめ:
- 当日朝に30分
- セットリストの最初の3曲
- ミックスポイント確認

やりすぎ注意:
✗ 何時間も練習
✗ 疲れる
✗ 飽きる

新鮮な気持ちで本番へ
```

### Q2: お酒は飲んでいい？

**A: 本番後まで我慢**

```
本番前:
✗ アルコール禁止
理由: 判断力低下、ミス増加

本番中:
✗ 1杯も禁止
理由: 集中力低下

本番後:
✓ 好きなだけ
✓ 打ち上げで祝杯

プロは本番前に飲まない
```

### Q3: 緊張で眠れない場合は？

**A: リラックス法を試す**

```
対処法:
1. 深呼吸（4-7-8法）
2. ホットミルク
3. リラックス音楽
4. 軽いストレッチ
5. 読書

それでも眠れない:
- 6時間でもOK
- 横になるだけでも休まる
- 緊張は正常

無理に寝ようとしない
```

## まとめ

- **持ち物**: DDJ-FLX4、PC、ケーブル類、予備USB、ヘッドフォン、水
- **セットリスト**: 1週間前に作成、3日前に全曲確認、前日にPlan A/B/C準備
- **メンタル**: ポジティブな自己暗示、ビジュアライゼーション、成功イメージ
- **ルーティン**: 朝のルーティン確立、会場1時間前到着、セットアップ15分
- **食事**: 前日は早めに軽め、当日朝は軽め、本番3時間前に軽食、直前は水のみ
- **睡眠**: 7-8時間確保、23:00までに就寝、質の高い睡眠
- **服装**: ジャンルに合わせる、動きやすさ重視、黒が基本
- **スケジュール**: 余裕を持った時間配分、早めの到着、最終確認
- **緊急対応**: 予備機材、クラウドバックアップ、体調管理
