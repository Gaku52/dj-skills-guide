DJセットを録音し、自己分析やプロモーションに活用する完全ガイド。

- Mixcloud、SoundCloudアップロード

## なぜレコーディングが重要か

**録音しない vs 録音する:**
```
録音しない:
- 記録が残らない
- 成長が見えない
- プロモーションできない
- 後悔

録音する:
- 成長記録
- 自己分析可能
- プロモーション素材
- ポートフォリオ

レコーディング = DJ成長の必須ツール
```

### 3つの目的

**目的1: 自己分析**

録音を聴き直す:

聴くだけではわからない
録音で客観的に分析

**目的2: プロモーション**
```
使い道:
- SoundCloudにアップロード
- プロモーターに送る
- SNSでシェア

音源 = 実力の証明
```

**目的3: 成長記録**
```
6ヶ月前のミックス vs 今:
- 明らかな成長
- 技術向上
- 選曲センス向上
- モチベーション

記録 = 成長の可視化
```

## 2. Rekordboxでの録音方法

### 基本設定（Rekordbox 7.2.7）

**録音機能の有効化:**
```
Rekordbox 7.2.7の録音機能:

1. Preferences（設定）を開く
   - macOS: Rekordbox > Preferences
   - Windows: File > Preferences

2. Recording（録音）タブを選択

3. 設定:
   ✓ Enable Recording: ON
   ✓ Recording Format: WAV（高音質）
   ✓ Recording Quality: 24bit / 44.1kHz（推奨）
   ✓ Recording Folder: 保存先を選択
     例: ~/Music/DJ Recordings/

4. OKをクリック

これで録音準備完了
```

### 録音手順

**Step 1: 録音開始**
```
Rekordboxの録音ボタン:

1. Rekordbox下部に録音ボタン（●）
   - 赤い丸ボタン

2. 録音開始:
   - 録音ボタン（●）をクリック
   - ボタンが赤く点灯
   - 「Recording...」表示

3. プレイ開始:
   - 通常通りDJプレイ
   - 全て録音される

注意:
- 録音開始してからプレイ
- 録音ボタンを押し忘れない
```

**Step 2: プレイ中**
```
録音中:
- 赤い●が点滅
- 録音時間表示
- 通常通りプレイ

確認:
- 時々録音時間を確認
- 録音が止まっていないか
```

**Step 3: 録音停止**
```
プレイ終了後:

1. 録音停止ボタン（■）をクリック
   - 録音が停止
   - 自動で保存

2. ファイル確認:
   - 設定した保存先フォルダ
   - ファイル名: 日付_時刻.wav
   - 例: 2025-01-15_203045.wav

3. バックアップ:
   - 外付けHDDにコピー
   - クラウドにアップロード
```

## 3. 音質設定

### フォーマット選択

**WAV vs MP3:**
```
WAV（推奨）:
✓ 無圧縮
✓ 最高音質
✓ 編集に最適
✓ ファイルサイズ大（60分 = 600MB）

MP3:
- 圧縮
- 音質劣化
- ファイルサイズ小（60分 = 100MB）
- アップロード用

録音はWAV、アップロード時にMP3変換
```

### ビットレートとサンプルレート

**最適な設定:**
```
推奨設定（Rekordbox）:

Format: WAV
Bit Depth: 24bit（または16bit）
Sample Rate: 44.1kHz

理由:
- CD品質
- 編集に十分
- ファイルサイズ適度

上級者（余裕があれば）:
- 24bit / 48kHz
- さらに高音質
- ファイルサイズ大
```

## 4. 録音のTips

### Tip 1: ゲインステージングに注意

**録音レベル:**
```
重要: クリッピングを避ける

Master Level:
- 赤に入らない
- 緑〜黄色
- ピークで-3dB以下

クリッピング = 音が歪む
→ 修復不可能
→ 録音失敗

ゲイン控えめ = 安全
```

### Tip 2: 録音テスト

**本番前に必ずテスト:**
```
テスト手順:

1. 5分間テスト録音
2. 再生して確認:
   - 音が録れているか
   - 音質は良いか
   - ノイズはないか
3. 問題なければ本番

本番で録音失敗 = 最悪
必ずテスト
```

### Tip 3: バックアップ録音

**2系統で録音:**
```
安全策:

方法1: Rekordbox
方法2: スマートフォン（音声レコーダー）

両方録音:
- Rekordboxが失敗しても
- スマホの録音が残る
- 保険

プロは必ずバックアップ
```

## 5. 編集とマスタリング

### 基本的な編集

**Audacity（無料ソフト）:**
```
Audacity = 無料の音声編集ソフト

ダウンロード:
https://www.audacityteam.org/

基本編集:

1. ファイルを開く
   - File > Open > 録音ファイル

2. 不要部分をカット
   - 最初の無音部分
   - 最後の無音部分
   - 範囲選択 → Delete

3. フェードイン/アウト
   - 最初の5秒: Effect > Fade In
   - 最後の10秒: Effect > Fade Out

4. ノーマライズ（音量最適化）
   - Effect > Normalize
   - Peak amplitude: -1.0 dB
   - Apply

5. エクスポート
   - File > Export > Export as MP3
   - Quality: 320kbps（最高音質）
```

### マスタリング（上級）

**音質向上:**
```
プロのマスタリング:

1. EQ調整
   - Effect > Equalization
   - 低音強化（80-100Hz）
   - 高音クリア（10kHz以上）

2. コンプレッサー
   - Effect > Compressor
   - 音圧を上げる
   - 音量を均一化

3. リミッター
   - Effect > Limiter
   - クリッピング防止

初心者は不要
Normalizeだけで十分
```

## 6. Mixcloud、SoundCloudアップロード

### Mixcloud（ミックス専用）

**Mixcloudとは:**
```
Mixcloud = DJミックス専用プラットフォーム

特徴:
✓ 著作権クリア（使用曲を報告）
✓ DJミックスに特化
✓ プロDJも使用
✓ 無料

URL: https://www.mixcloud.com/
```

**アップロード手順:**
```
1. Mixcloudアカウント作成
   - 無料登録

2. Upload（アップロード）
   - Upload > Select File
   - MP3ファイル選択（320kbps推奨）

3. 情報入力:
   - Title: [日付] Tech House Mix
   - Description: セット内容説明
   - Tags: Tech House, Deep House等
   - Picture: カバー画像（1400x1400px）

4. Tracklist（重要）:
   - 使用曲リスト
   - アーティスト名 - 曲名
   - 全て入力（著作権のため）

5. Publish（公開）
   - 公開ボタン
   - 完了

共有してSNSでプロモーション
```

### SoundCloud（音楽全般）

**SoundCloudとは:**
```
SoundCloud = 音楽共有プラットフォーム

特徴:
✓ 幅広いジャンル
✓ アーティスト多数
✓ 無料プラン（3時間まで）
✓ 有料プラン（無制限）

URL: https://soundcloud.com/
```

**アップロード手順:**
```
1. SoundCloudアカウント作成

2. Upload
   - Upload > Choose files
   - MP3選択（320kbps）

3. 情報入力:
   - Title: Tech House Mix - Jan 2025
   - Genre: House
   - Tags: Tech House, DJ Mix等
   - Description: セット内容
   - Cover Art: 画像（800x800px以上）

4. Privacy:
   - Public（公開）
   - Private（非公開）- プロモーター用

5. Save（保存）

共有リンクをコピー
```

## 8. 自己分析方法

### 録音を聴き直す

**分析の手順:**
```
録音から1週間後:

1. 全て聴き直す（集中して）

2. チェックポイント:
   □ ミックスポイント適切か
   □ EQ調整は良いか
   □ エネルギーカーブは滑らかか
   □ 選曲の流れは良いか
   □ ミスはあったか

3. メモを取る:
   - 良かった点
   - 改善点
   - 次回に活かす

4. 改善リストを作る

録音 = 最高の先生
```

### 成長の記録

**定期的に聴き比べる:**
```
6ヶ月前のミックス vs 今:

比較:
- ミックス精度
- 選曲センス
- エネルギー管理
- 全体の質

成長が見える:
→ モチベーション向上
→ さらに練習

記録は宝
```

## 10. 著作権の注意点

### 使用曲の報告

**Mixcloud:**
```
Mixcloudの利点:
- 著作権団体と契約済み
- 使用曲を報告すればOK
- アーティストにロイヤリティ支払われる

必須:
- Tracklist（全曲リスト）
- アーティスト名 - 曲名
- 正確に入力

Tracklist入力 = 合法
```

**SoundCloud:**
```
注意:
- 著作権管理が厳しい
- 曲によってはブロック
- アカウント停止リスク

対処:
- オリジナル曲のみ
- 許可を得た曲のみ
- またはPrivate設定

リスクあり
```

### YouTube

**著作権が最も厳しい:**
```
YouTube:
✗ 商用音楽使用は基本NG
✗ Content ID で自動検出
✗ 削除、収益化停止

対処:
- オリジナル曲のみ
- ロイヤリティフリー音楽
- または覚悟の上で

リスク高い
Mixcloud推奨
```

## 11. よくある質問

### Q1: 練習ミックスも録音すべき？

**A: 全て録音推奨**

```
理由:
✓ 成長記録
✓ 練習でも良いミックスができる
✓ 後で聴き返せる

ストレージ:
- 外付けHDD（安い）
- 1TB = 数百時間分

全て録音、後で選別
```

### Q2: 毎回アップロードすべき？

**A: 厳選してアップロード**

```
基準:
- 自信のあるミックス
- ミスが少ない
- エネルギーカーブが良い

頻度:
- 月1-2回
- 質を重視

全部アップロードは逆効果
```

### Q3: カバーアートはどう作る？

**A: Canvaで簡単作成**

```
Canva（無料ツール）:
https://www.canva.com/

1. テンプレート選択
   - 「Album Cover」
   - 1400x1400px

2. 編集:
   - DJ名
   - Mix名
   - 画像

3. ダウンロード
   - PNG形式

10分で作成可能
```

## まとめ

- **重要性**: 自己分析、プロモーション、成長記録の3つの目的
- **Rekordbox録音**: Preferences > Recording > WAV / 24bit / 44.1kHz設定
- **録音手順**: 録音ボタン（●）> プレイ > 停止（■）> 自動保存
- **音質**: WAV推奨、録音時はクリッピング注意、ゲイン控えめ
- **編集**: Audacityで無音カット、フェードイン/アウト、ノーマライズ
- **Mixcloud**: DJミックス専用、Tracklist必須、著作権クリア
- **SoundCloud**: 幅広いジャンル、無料3時間、320kbps MP3
- **自己分析**: 1週間後に聴き直し、改善点メモ、6ヶ月ごとに成長確認
- **プロモーション**: SNSシェア、プロモーターに送付、定期配信
- **著作権**: Mixcloud推奨（Tracklist入力必須）、YouTubeは厳しい
