```

プロのセットアップ:
- 10分かけて完璧に準備
- 全ての音量最適化
- モニター完璧

準備 = 成功の80%
```

## 3. DDJ-FLX4セットアップ手順

### Step 1: 機材の配置

**ブースでの配置:**
```
1. DJブースに入る
   - 前のDJが終わるのを待つ
   - スペース確認

2. DDJ-FLX4を設置
   場所:
   - ブースの中央
   - 安定した場所
   - 手が届きやすい高さ

3. ノートPCを配置
   - DDJ-FLX4の後ろ
   - 画面が見える位置
   - 安定した場所

注意:
✗ 飲み物をコントローラーの近くに置かない
✗ 不安定な場所
✗ ケーブルが邪魔な位置
```

### Step 2: ケーブル接続

**接続順序（重要）:**
```
正しい順序:

1. ノートPC電源接続（バッテリー保護）
2. DDJ-FLX4とPCをUSBケーブル接続
   - DDJ-FLX4背面のUSB端子
   - PCのUSB端子
   - しっかり差し込む

3. ヘッドフォン接続
   - DDJ-FLX4前面のヘッドフォン端子
   - 6.3mm標準ジャックまたは3.5mm

4. DDJ-FLX4の電源を入れる
   - 背面の電源スイッチON
   - LEDが点灯

5. クラブのミキサーまたはスピーカーに接続
   （DDJ-FLX4の場合、クラブのシステムに接続する場合）
   - Master Out（背面）からミキサーへ
   - RCAケーブル（赤/白）

USB給電の場合:
- AC電源不要
- PCから給電
- ケーブル1本で完結
```

### Step 3: Rekordbox起動と設定

**Rekordbox 7.2.7:**
```
1. Rekordboxを起動
   - アプリケーション起動
   - 数秒待つ

2. DDJ-FLX4が認識されているか確認
   - 左上に「DDJ-FLX4」表示
   - もし表示されない:
     → Preferences > Audio
     → Audio Device: DDJ-FLX4選択

3. オーディオ設定確認
   Preferences > Audio:

   ✓ Audio Device: DDJ-FLX4
   ✓ Sample Rate: 44100 Hz（推奨）
   ✓ Buffer Size: 256 samples（推奨）

   Buffer Size:
   - 小さい（128）: レスポンス速い、CPU負担大
   - 大きい（512）: レスポンス遅い、CPU負担小
   - 256が最適

4. キーボードショートカット無効化（重要）
   Preferences > Controller:
   ✓ Keyboard Shortcuts: OFF

   理由: 誤操作防止
```

## 4. サウンドチェック

### サウンドチェックとは

```
サウンドチェック = 音量レベルの最適化

目的:
- Master音量の確認
- 各チャンネルのゲイン調整
- モニターバランス確認
- 音質チェック

時間: 5-10分
絶対に省略しない
```

### Step 1: Master音量確認

**基本手順:**
```
1. テスト曲を1曲ロード
   - Deck Aに
   - よく知っている曲
   - 音圧が標準的な曲

2. Deck Aを再生
   - Playボタン押す
   - 音が出ない（まだMasterが0）

3. Channel Fader（縦フェーダー）を上げる
   - Deck Aのフェーダーを上（MAX）

4. Trimノブ（ゲイン）を調整
   - まず12時（中央）
   - レベルメーター確認

5. レベルメーター確認
   DDJ-FLX4のレベルメーター:
   - 緑: 良い（-18dB ~ -6dB）
   - 黄色: 注意（-6dB ~ -3dB）
   - 赤: 大きすぎ（0dB以上）

   目標: 緑〜黄色の境界（-6dB付近）

6. Master Level（Master音量）を上げる
   - 徐々に上げる
   - クラブスタッフに確認
   「この音量で大丈夫ですか？」

7. 適切な音量に調整
   - 大きすぎず
   - 小さすぎず
   - スタッフのOKをもらう
```

### Step 2: 両デッキの音量を揃える

**ゲインマッチング:**
```
1. Deck Aでテスト曲再生中

2. Deck Bに別の曲をロード

3. Deck Bのヘッドフォンキューボタン押す
   - ヘッドフォンでDeck Bを聴く
   - まだスピーカーからは出ない

4. Deck BのTrim（ゲイン）を調整
   - レベルメーターを見る
   - Deck Aと同じレベルに
   - 緑〜黄色の境界

5. 両デッキを比較
   - Deck A再生
   - Deck Bに切り替え
   - 音量が同じか確認

6. 全ての曲で繰り返す
   （本番中も常にゲイン調整）

ゲインが揃っている = プロ
```

## 5. モニターバランス調整

### モニターとは

```
モニター = ヘッドフォンから聞こえる音

2つの音源:
1. Master（スピーカーから出ている音）
2. Cue（次にプレイする曲）

ミックス比を調整可能
```

### DDJ-FLX4でのモニターバランス

**Mixing（ミキシング）ノブ:**
```
DDJ-FLX4のヘッドフォンセクション:

Mixingノブ（中央上部）:
- 左（Master）: スピーカーの音のみ
- 中央: Master + Cue 50/50
- 右（Cue）: 次の曲のみ

おすすめ設定:

初心者:
→ 右（Cue）寄り
理由: 次の曲をしっかり聞く

中級者:
→ 中央
理由: 両方バランスよく

上級者:
→ 状況に応じて調整

自分に合った設定を見つける
```

### ヘッドフォン音量調整

**Headphone Levelノブ:**
```
DDJ-FLX4のヘッドフォン音量:

Level（音量）ノブ:
- 左: 小さい
- 右: 大きい

適切な音量:
✓ 曲のビートがはっきり聞こえる
✓ 会場の音も少し聞こえる
✓ 耳が痛くならない

大きすぎ:
✗ 耳を痛める
✗ 会場の音が聞こえない
✗ 観客の反応がわからない

小さすぎ:
✗ ビートマッチできない
✗ 音質確認できない

中間〜やや大きめが最適
```

## 6. クラブでの標準的なセットアップ

### CDJセットアップとの違い

**DDJ-FLX4の利点:**
```
CDJセットアップ（標準的なクラブ）:
- CDJ x 2台
- DJミキサー x 1台
- USBメモリ必須
- 複雑な接続

DDJ-FLX4セットアップ:
- DDJ-FLX4のみ
- ノートPC
- USBケーブル1本
- シンプル

セットアップ時間:
CDJ: 15-20分
DDJ-FLX4: 5-10分

DDJ-FLX4の方が簡単
```

### クラブのサウンドシステムへの接続

**2つの方法:**
```
方法1: DDJ-FLX4から直接出力（推奨）

接続:
DDJ-FLX4背面 Master Out（RCA）
→ クラブのミキサーまたはアンプ

利点:
✓ シンプル
✓ トラブル少ない
✓ 初心者向け

方法2: クラブのミキサー経由

接続:
DDJ-FLX4 → クラブのミキサー → スピーカー

利点:
✓ クラブのEQ使える
✓ 他DJと共有しやすい

初心者は方法1推奨
```

## 7. トラブル回避

### よくあるトラブルと対処

**トラブル1: 音が出ない**
```
原因:
- ケーブル接続ミス
- Master Level が0
- Channel Fader が0
- オーディオ設定ミス

対処:
1. 全てのケーブル確認
2. Master Level を上げる
3. Channel Fader を上げる
4. Rekordbox Audio設定確認
5. DDJ-FLX4再起動
```

**トラブル2: 音が歪む**
```
原因:
- Trim（ゲイン）が大きすぎ
- Master Level が大きすぎ
- クリッピング

対処:
1. レベルメーター確認（赤点灯？）
2. Trimを下げる
3. Master Level を下げる
4. EQを使いすぎていないか確認
```

**トラブル3: モニターが聞こえない**
```
原因:
- ヘッドフォン接続ミス
- Headphone Level が0
- Cueボタン押していない

対処:
1. ヘッドフォンしっかり差し込む
2. Headphone Level を上げる
3. Cueボタンが点灯しているか確認
```

**トラブル4: 片方のチャンネルから音が出ない**
```
原因:
- RCAケーブルの片方が抜けている
- ケーブル不良

対処:
1. 両方のRCAケーブル確認（赤/白）
2. 予備ケーブルに交換
3. 別の出力端子を使う
```

## 8. セットアップチェックリスト

### 機材設置

```
□ DDJ-FLX4をブース中央に設置
□ ノートPCを安定した場所に設置
□ USBケーブル接続（PC ↔ DDJ-FLX4）
□ ヘッドフォン接続
□ 電源接続（AC電源の場合）
□ Master Out接続（クラブシステムへ）
□ DDJ-FLX4電源ON
□ 全てのケーブルをテープで固定（任意）
```

### Rekordbox設定

```
□ Rekordbox起動
□ DDJ-FLX4認識確認
□ Audio Device設定確認
□ Sample Rate: 44100 Hz
□ Buffer Size: 256 samples
□ Keyboard Shortcuts: OFF
□ 全ての曲がライブラリにあるか確認
□ セットリストプレイリスト表示
```

### サウンドチェック

```
□ テスト曲をDeck Aにロード
□ 再生開始
□ Trim（ゲイン）調整
□ レベルメーター確認（緑〜黄色）
□ Master Level調整
□ 音量をスタッフに確認
□ Deck Bにテスト曲ロード
□ Deck Bのゲイン調整
□ 両デッキの音量を揃える
□ モニターバランス調整（Mixingノブ）
□ ヘッドフォン音量調整
```

### 最終確認

```
□ セットリスト準備完了
□ 予備USBメモリ準備
□ 予備ケーブル準備（任意）
□ スマートフォン充電済み（Shazam用）
□ 水分補給用の水準備
□ 深呼吸
□ プレイ開始準備完了
```

## 9. プロのTips

### Tip 1: 前のDJを観察

**学びのチャンス:**
```
前のDJがプレイ中:
✓ セットアップ方法を見る
✓ 音量レベルを聞く
✓ 観客の反応を見る
✓ 使っている機材を確認

学べること:
- どれくらいの音量が適切か
- どのジャンルが受けているか
- エネルギーレベル

自分のセットに活かす
```

### Tip 2: 余裕を持った到着

**時間配分:**
```
最低30分前到着:
- セットアップ: 15分
- サウンドチェック: 10分
- 余裕: 5分

理想は60分前:
- 前のDJ観察: 30分
- セットアップ: 15分
- サウンドチェック: 10分
- 余裕: 5分

遅刻 = 最悪
プロは早く着く
```

### Tip 3: ケーブルをテープで固定

**トラブル防止:**
```
ケーブルが抜けるリスク:
- 踊って動く
- 手が当たる
- 誰かが引っ掛ける

対処:
✓ ガムテープで床に固定
✓ ケーブルをまとめる
✓ 邪魔にならない配置

プロは必ずやる
```

### Tip 4: 予備機材

**安心のための準備:**
```
必須の予備:
✓ 予備USBメモリ（同じ内容）
✓ 予備USBケーブル
✓ 予備ヘッドフォン

任意の予備:
- 予備RCAケーブル
- 予備電源ケーブル
- 予備ノートPC（上級者）

トラブル時に即対応
```

## 10. よくある質問

### Q1: サウンドチェック中に前のDJがプレイしている場合は？

**A: 音を出さずに確認**

```
方法:
1. 機材セットアップは可能
2. Rekordbox起動
3. ヘッドフォンで全て確認
4. 音は出さない

前のDJが終わったら:
1. 30秒でサウンドチェック
2. すぐにプレイ開始

準備は全て済んでいる状態
```

### Q2: クラブの音響スタッフがいる場合は？

**A: 必ずコミュニケーション**

```
やること:
1. 挨拶
2. 「サウンドチェックしていいですか？」
3. 音量を徐々に上げる
4. 「この音量で大丈夫ですか？」
5. スタッフの指示に従う

スタッフとの良好な関係 = 成功
```

### Q3: セットアップ時間がない場合は？

**A: 最低限の確認**

```
最低限（3分）:
1. 接続確認
2. Rekordbox起動確認
3. Master音量確認

理想的ではないが緊急時の対処

普段は余裕を持つこと
```

## まとめ

- **到着**: 60分前が理想、最低30分前
- **接続**: PC電源 → USB → ヘッドフォン → DDJ-FLX4電源ON
- **Rekordbox**: Audio設定確認（44100Hz, 256samples）
- **サウンドチェック**: Master音量確認、ゲイン調整、両デッキ揃える
- **モニター**: Mixingノブで調整、Cue寄り〜中央推奨
- **トラブル**: 音が出ない → 接続確認、レベルメーター確認
- **チェックリスト**: 機材設置 → Rekordbox設定 → サウンドチェック → 最終確認
- **プロのTips**: 前のDJ観察、余裕持った到着、ケーブル固定、予備機材
- **時間配分**: セットアップ15分、サウンドチェック10分、余裕5分
