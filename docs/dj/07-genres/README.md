# ジャンル別ガイド

各ジャンルの特徴、歴史、DJ技法を深く理解し、ジャンルに応じたプレイをマスターします。

## このセクションで学ぶこと

- House、Techno、Hip Hop等の主要ジャンル
- 各ジャンルの歴史と文化
- ジャンル別のDJ技法の違い
- 代表的アーティストと必聴曲
- ジャンルごとのセット構築
- クロスジャンルミキシング
- オープンフォーマットDJ
- ジャンルの進化とトレンド

## なぜジャンル別の理解が重要か

**ジャンル理解なし vs ジャンル理解あり:**
```
ジャンル理解なし:
- 表面的な知識
- 文脈がわからない
- 選曲が浅い
- クラシックを知らない
→ アマチュア

ジャンル理解あり:
- 深い知識
- 歴史と文化を理解
- 選曲に深み
- クラシックから最新まで
→ プロフェッショナル

ジャンル理解 = DJ の深み
```

**技術だけでは不十分:**
```
技術完璧でも:
- House の歴史を知らない
- Techno の文化を知らない
- Hip Hop のルーツを知らない
→ 浅いDJ

技術 + ジャンル知識:
- 歴史を理解
- 文化を尊重
- ルーツを知る
→ リスペクトされるDJ

知識 = 尊敬
```

## このセクションの構成

### Phase 1: 電子音楽の基礎（House / Techno）

**House:**
- 1980年代シカゴ発祥
- 4つ打ち、グルーヴ重視
- Deep House、Tech House、Progressive
- DJ技法: スムーズなミックス、ロングトランジション

**Techno:**
- 1980年代デトロイト発祥
- ダーク、工業的、ミニマル
- Minimal Techno、Industrial、Acid
- DJ技法: ハードカット、ループ、エフェクト多用

### Phase 2: ストリート文化（Hip Hop）

**Hip Hop:**
- 1970年代ニューヨーク発祥
- ターンテーブリズム、スクラッチ
- Boom Bap、Trap、Drill
- DJ技法: スクラッチ、ジャグリング、カットイン

### Phase 3: ベースミュージック（Dubstep / D&B）

**Dubstep:**
- 2000年代ロンドン発祥
- ハーフタイム、重低音、ドロップ
- Brostep、Riddim、Deep Dubstep
- DJ技法: ドロップタイミング、ベース管理

**Drum & Bass:**
- 1990年代UK発祥
- 高速BPM（160-180）、ブレイクビート
- Liquid、Neurofunk、Jump Up
- DJ技法: 高速ミックス、リワインド

### Phase 4: トランスとオープン

**Trance:**
- 1990年代ドイツ発祥
- ビルドアップ、エモーショナル
- Progressive Trance、Uplifting、Psytrance
- DJ技法: ロングビルドアップ、エモーショナルミックス

**Open Format:**
- ジャンル制限なし
- マルチジャンルミキシング
- Pop、R&B、EDM、Hip Hop混在
- DJ技法: 即興、幅広い知識、読み

## 学習の順序

### Week 1-2: House

```
目標: House の理解

学習:
1. 歴史（シカゴ、Frankie Knuckles、Larry Levan）
2. サブジャンル（Deep、Tech、Progressive）
3. 代表的アーティスト50名
4. 必聴クラシック100曲

実践:
- House 60分セット作成
- Deep House のみ30分
- Tech House のみ30分
```

### Week 3-4: Techno

```
目標: Techno の理解

学習:
1. 歴史（デトロイト、Juan Atkins、Derrick May）
2. サブジャンル（Minimal、Industrial、Acid）
3. 代表的アーティスト50名
4. 必聴クラシック100曲

実践:
- Techno 60分セット作成
- Minimal のみ30分
- 速いTechno（130+ BPM）30分
```

### Week 5-6: Hip Hop

```
目標: Hip Hop の理解

学習:
1. 歴史（ブロンクス、Kool Herc、Grandmaster Flash）
2. サブジャンル（Boom Bap、Trap、Drill）
3. ターンテーブリズム
4. スクラッチ技術

実践:
- Hip Hop 60分セット作成
- スクラッチ練習（Baby Scratch）
- カットインミックス
```

### Week 7-8: その他ジャンル

```
Dubstep、D&B、Trance、Open Format:
- 各ジャンル1週間
- 歴史と文化理解
- セット作成

Open Formatは最後:
- 全てのジャンル理解後
- マルチジャンルミキシング
```

## DDJ-FLX4でのジャンル別アプローチ

### House / Techno

**特徴:**
```
- ロングミックス（1-2分）
- EQ重視
- スムーズなトランジション

DDJ-FLX4設定:

- Beat FX: Echo、Reverb
- Trim: 慎重に（音圧高い）
```

### Hip Hop

**特徴:**
```
- 短いミックス（5-15秒）
- カットイン
- スクラッチ

DDJ-FLX4設定:
- クロスフェーダー使用
- Performance Pads: Hot Cue
- Slip Mode: ON（スクラッチ時）
```

### Dubstep / D&B

**特徴:**
```
- ドロップタイミング重要
- ベース管理
- エネルギー高い

DDJ-FLX4設定:
- Beat FX: Filter、Flanger
- EQ: ベースコントロール
- ゲインステージング重要
```

## ジャンルの選び方

### 自分の好きなジャンルから

**推奨順序:**
```
1. 好きなジャンルを1つ選ぶ
   - House / Techno / Hip Hop

2. そのジャンルを徹底的に学ぶ
   - 3ヶ月集中
   - 歴史、文化、技法

3. 得意ジャンル確立
   - 「House DJ」として認知される
   - 専門性

4. 徐々に拡大
   - 関連ジャンル（HouseならTechno）
   - 6ヶ月後

最初から全部は無理
1つずつマスター
```

### ジャンルと会場

**需要:**
```
クラブ:
- House、Techno（最も需要大）
- Dubstep、D&B（UK、若い世代）
- Trance（フェスティバル）

バー、ラウンジ:
- Open Format（最も需要大）
- Deep House、Lofi Hip Hop
- Chill系

パーティー、結婚式:
- Open Format必須
- Pop、R&B、Hip Hop、EDM

需要を考慮して選ぶ
```

## よくある質問

### Q1: 最初にどのジャンルを学ぶべき？

**A: 好きなジャンル、または House**

```
推奨: House
理由:
✓ 世界で最も人気
✓ ギグの需要が最も多い
✓ 技術的に基礎的
✓ 他ジャンルへの応用が効く

でも:
好きなジャンルが一番
モチベーションが続く
```

### Q2: 複数ジャンルを同時に学べる？

**A: 推奨しない、1つずつ**

```
理由:
✗ 中途半端になる
✗ 専門性がない
✗ 記憶が混乱

推奨:
✓ 1ジャンル3ヶ月集中
✓ マスターしてから次
✓ 専門性確立

焦らない
```

### Q3: ジャンルミックスはあり？

**A: 上級者向け、慎重に**

```
例: House → Techno
- 可能（BPM近い）
- スムーズに移行

例: House → Hip Hop
- 難しい（BPM大きく違う）
- Open Format DJのスキル必要

まずは1ジャンルを極める
その後クロスジャンル
```

## まとめ

- **ジャンル理解**: 歴史、文化、技法の深い理解が必須
- **House**: 世界で最も人気、ギグ需要大、基礎的技術
- **Techno**: アンダーグラウンド、ベルリン、ハードカット
- **Hip Hop**: DJ文化の原点、スクラッチ、ターンテーブリズム
- **Dubstep/D&B**: ベースミュージック、ドロップ、高速
- **Trance**: エモーショナル、ビルドアップ、フェスティバル
- **Open Format**: マルチジャンル、バー需要大、即興力
- **学習順序**: 1ジャンル3ヶ月集中、専門性確立、徐々に拡大
