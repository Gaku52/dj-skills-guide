## 1. D&B の歴史

### 起源（1990年代初期 UK）

**1992-1994: Jungle の誕生**
```
場所: ロンドン、UK

起源:
Breakbeat Hardcore + Reggae/Dub + Hip Hop
= Jungle

特徴:
- 高速ブレイクビート（160-180 BPM）
- レゲエベース
- MCing
- サンプリング

キーパーソン:
- Goldie
- Grooverider
- Fabio
- LTJ Bukem

1992年: 「Jungle Sound」確立
アンダーグラウンドレイヴシーン
```

### 進化（1995-2000）

**Jungle → Drum & Bass:**
```
1995年: 名前変更
"Jungle" → "Drum & Bass"
理由: より洗練、メインストリーム

1995-1997: ゴールデンエイジ
- Goldie「Timeless」(1995) - 名盤
- Roni Size「New Forms」(1997) - Mercury Prize
- LTJ Bukem「Logical Progression」

特徴:
- よりメロディアス
- Jazz の影響
- プロダクション向上

1998-2000: Tech Step登場
- より暗い
- ミニマル
- テクニカル
```

### 現代（2000-現在）

**サブジャンル爆発:**
```
2000-2005: Liquid Funk
- メロディアス
- ソウルフル
- ボーカル

2005-2010: Neurofunk
- 暗い
- 複雑なベースライン
- テクニカル

2010-2015: Jump Up
- パーティー向け
- キャッチー
- エネルギッシュ

2015-現在: 多様化
- 全サブジャンル共存
- グローバル化
- フェスティバル人気

D&B は進化し続ける
```

## 2. D&B の特徴

### 音楽的特徴

**基本構造:**
```
BPM: 160-180（標準は 174）

リズム:
- ブレイクビート（複雑）
- Amen Break（最も有名）
- Think Break
- シンコペーション

ベースライン:
- Reese Bass（うねる）
- Sub Bass（低音）
- Neurofunk（複雑）

構造:
1. イントロ（16-32バー）
2. ビルドアップ（16バー）
3. ドロップ（32-64バー）
4. ブレイクダウン（16-32バー）
5. セカンドドロップ（32-64バー）
6. アウトロ（16バー）

高速 = 集中力必要
```

### Amen Break とは

**最も有名なブレイク:**
```
起源:
「Amen, Brother」by The Winstons (1969)
- 6秒のドラムソロ
- 世界で最もサンプリングされた

D&B での使用:
- ほぼ全ての Jungle/D&B 曲
- スピードアップ（160+ BPM）
- チョップ、リアレンジ

Amen Break = D&B の DNA
```

## 5. DJ技法とミキシング

### 高速ミキシング

**特徴:**
```
D&B = 最速ミキシング

理由:
- BPM 174（速い）
- 曲が短い（4-5分）
- エネルギー維持

ミックス頻度:
- 2-3分に1曲
- 1時間で20-30曲
- House の2倍

集中力必要
```

### ダブルドロップ

**D&B特有の技術:**
```
ダブルドロップ = 2曲のドロップを同時に

手順:
1. 曲Aのドロップ再生中
2. 曲Bをキュー（ドロップ開始位置）
3. ビートマッチング
4. 曲Aのドロップと曲Bのドロップを同時に
5. エネルギー爆発

効果:
- 2倍のエネルギー
- 観客大興奮
- D&B の醍醐味

難易度: 高
練習必須
```

### リワインド（巻き戻し）

**MCカルチャー:**
```
リワインド = 曲を巻き戻して再度プレイ

タイミング:
- 曲が盛り上がった
- MCが煽る「Rewind!」
- DJがレコードを逆回転
- 最初から再度プレイ

効果:
- 観客大興奮
- エネルギー最高潮
- UK D&B 文化

DDJ-FLX4:
- Slip Modeで巻き戻しエフェクト
- またはHot Cueでジャンプバック

D&B 文化の象徴
```

## 7. セット構築

### 60分 D&B セット例

**フェーズ1: ウォームアップ（0-15分）**
```
BPM: 170-172
エネルギー: 4-5 / 10
スタイル: Liquid

選曲:
- メロディアス
- ボーカル
- ポジティブ

例:
1. High Contrast「If We Ever」170 BPM
2. Netsky「Come Alive」171 BPM
3. Fred V & Grafix「Major Happy」172 BPM
4. Logistics「Together」172 BPM

約8曲（2分ごと）
```

**フェーズ2: エネルギー構築（15-35分）**
```
BPM: 172-174
エネルギー: 6-7 / 10
スタイル: Liquid → Neurofunk移行

選曲:
- よりエネルギッシュ
- ベース強調
- テンポ上げる

例:
5. Sub Focus「Rock It」172 BPM
6. Wilkinson「Afterglow」173 BPM
7. Dimension「Whip Slap」174 BPM
8. Phace「Cold Champagne」174 BPM

約10曲
```

**フェーズ3: ピーク（35-50分）**
```
BPM: 174-176
エネルギー: 8-10 / 10
スタイル: Neurofunk / Jump Up

選曲:
- 最もハード
- ダブルドロップ
- リワインド

例:
9. Noisia「Stigma」174 BPM
10. Andy C「Heartbeat Loud」175 BPM - ダブルドロップ
11. Macky Gee「Tour」175 BPM
12. DJ Hazard「Mr. Happy」176 BPM - リワインド！

約8曲
```

**フェーズ4: クールダウン（50-60分）**
```
BPM: 174-172
エネルギー: 6-7 / 10
スタイル: Liquid回帰

選曲:
- メロディアス
- クラシック
- 記憶に残る

例:
13. Calibre「Even If」173 BPM
14. Etherwood「Begin by Letting Go」172 BPM
15. Goldie「Inner City Life」170 BPM - クラシックで締め
```

## 8. DDJ-FLX4での実践

### 高速ミキシング対応

**設定:**
```
Sync機能:
- D&B は速いのでSync推奨
- ビートマッチングは手動でも可能
- 両方練習

Hot Cue:
- 必須
- イントロ、ドロップ設定
- 高速アクセス

Performance Pads:
- Hot Cue Mode
- 即座にドロップへ
```

### リワインド エフェクト

**DDJ-FLX4:**
```
方法1: Slip Mode + 巻き戻し
1. Slip Mode ON
2. Jogを逆回転
3. エフェクト的

方法2: Hot Cue
1. Hot Cue 1に曲の開始設定
2. ドロップ中にHot Cue 1押す
3. 即座に最初から

方法2が簡単
```

**Week 1-2: 高速ミキシング**
```
練習内容:
1. D&B 20曲選ぶ

3. Sync使用
4. 60分セット作成

目標:
- スムーズな移行

```
