## なぜ Hip Hop が重要か

**Hip Hop = DJ文化の原点:**
```
事実:
- DJ という概念の発明（1973年）
- ターンテーブルを楽器に
- スクラッチの誕生
- Hip Hop 4大要素の1つ

Hip Hop を学ぶ = DJ の原点を理解
```

**Hip Hop 4大要素:**
```
1. MCing (ラップ)
2. DJing（ターンテーブリズム）
3. Breakdancing (ブレイクダンス)
4. Graffiti (グラフィティ)

DJing = Hip Hop の心臓部
```

## 1. Hip Hop の歴史

### 起源（1973年 ブロンクス）

**1973年8月11日: DJ文化誕生**
```
場所: ニューヨーク、ブロンクス
イベント: DJ Kool Herc のパーティー
住所: 1520 Sedgwick Avenue

革命:
DJ Kool Herc の発明:
- 2台のターンテーブル使用
- 同じレコードの「ブレイク」をループ
- "Merry-Go-Round" 技法
- ブレイクダンサーが踊る

これがDJingの始まり
```

### ゴールデンエイジ（1975-1985）

**技術革新:**
```
1975-1978: スクラッチの発明
Grand Wizzard Theodore:
- 偶然レコードを擦る
- スクラッチ誕生
- 新しい表現方法

1979年: 商業的ブレイク
「Rapper's Delight」by Sugarhill Gang
- 最初のヒップホップヒット曲
- 全世界に拡散

1981年: ターンテーブリズム確立
Grandmaster Flash:
- クイックミックス理論
- パンチフレージング
- スピンバック
- 技術の体系化

1983年: 映画「Wild Style」
- Hip Hop 文化を記録
- グローバル影響

1984-1985: DMC開始
- DJ ミックス コンペティション
- 技術の標準化
```

### 進化（1985-2000年代）

**ジャンルの拡大:**
```
1985-1995: ゴールデンエイジ
- DJ Jazzy Jeff & The Fresh Prince
- DJ Premier（Gang Starr）
- Pete Rock
- J Dilla

1995-2005: ターンテーブリスト時代
- DJ Q-Bert（最高技術）
- Mix Master Mike
- DJ Craze
- DJ Babu

2005-現在: デジタル時代
- Serato、Traktor登場
- DVS（Digital Vinyl System）
- コントローラーDJ増加
- 伝統 vs 革新
```

## 2. Hip Hop の特徴

### 音楽的特徴

**基本構造:**
```
BPM: 80-100（Boom Bap）
      135-160（Trap）

リズム:
- Boom Bap（古典的）
  キック・スネア・キック・スネア
- Trap（現代）
  ハイハット3連符

楽器:
- ドラムマシン（Akai MPC最重要）
- サンプラー
- ターンテーブル
- ボーカル（ラップ）

サンプリング:
- Funk、Soul、Jazz から
- ループ
- チョップ
- クリエイティブ

雰囲気:
- グルーヴ重視
- ライム（韻）
- ストーリーテリング
```

## 3. サブジャンル

### Boom Bap（クラシック）

**特徴:**
```
BPM: 85-95
年代: 1980-2000年代
雰囲気: オールドスクール、グルーヴィー

音楽的特徴:
- シンプルなドラム
- サンプリング重視
- ジャズ、Soulのループ
- スクラッチ

代表的アーティスト:
- DJ Premier（Gang Starr）
- Pete Rock
- J Dilla
- 9th Wonder
- DJ Jazzy Jeff

必聴曲:
- Gang Starr「Mass Appeal」
- Pete Rock & CL Smooth「They Reminisce Over You (T.R.O.Y.)」
- J Dilla「Donuts」(Album)

DJ技法: スクラッチ、ジャグリング
```

### Trap（現代）

**特徴:**
```
BPM: 135-160
年代: 2010年代-現在
雰囲気: ダーク、重低音、エネルギッシュ

音楽的特徴:
- 808ベース（重い）
- ハイハット3連符（トリプレット）
- スネアロール
- ドロップ

代表的アーティスト:
- RL Grime
- Baauer
- Flosstradamus
- TroyBoi
- Ekali

必聴曲:
- Baauer「Harlem Shake」
- RL Grime「Core」
- Flosstradamus「Mosh Pit」

DJ技法: ドロップタイミング、ベース管理
```

### Lofi Hip Hop

**特徴:**
```
BPM: 70-90
年代: 2010年代-現在
雰囲気: チル、勉強用、リラックス

音楽的特徴:
- レコードノイズ
- ジャズサンプル
- メロウ
- インストゥルメンタル

代表的アーティスト:
- Nujabes（日本）
- J Dilla（影響大）
- Tomppabeats
- joji
- eevee

必聴曲:
- Nujabes「Feather」
- joji「Sanctuary」

DJ技法: スムーズなミックス、雰囲気重視
```

### Drill

**特徴:**
```
BPM: 60-70（Half Time）
年代: 2010年代-現在
雰囲気: ダーク、攻撃的

音楽的特徴:
- ハーフタイム
- ダークなメロディ
- 808ベース
- 攻撃的なボーカル

代表的アーティスト:
- Chief Keef（シカゴ）
- Pop Smoke（UK Drill）
- Headie One（UK）

必聴曲:
- Chief Keef「I Don't Like」
- Pop Smoke「Dior」

DJ技法: エネルギー管理、ベース重視
```

## 4. ターンテーブリズム

### 定義

```
ターンテーブリズム = ターンテーブルを楽器として演奏

技術:
- スクラッチ
- ジャグリング
- ビートジャグリング
- トランスフォーム

目的: 音楽を作る、リズムを作る
```

### 基本技術

**Baby Scratch:**
```
最も基本:
1. レコードを前後に動かす
2. クロスフェーダーは開けたまま
3. リズムに合わせて

練習: 30分/日 x 2週間
```

**Forward Scratch:**
```
1. レコードを前に押す
2. 音が出る
3. 戻す

基礎中の基礎
```

**Transformer:**
```
クロスフェーダー使用:
1. レコード動かす
2. クロスフェーダーを素早く開閉
3. 音が切れる（Transformer効果）

中級技術
```

**Crab Scratch:**
```
上級:
1. 指を順番にクロスフェーダーに
2. 高速で音を切る
3. カニの足のように

プロ技術
詳細は [スクラッチ](../04-advanced-techniques/scratching.md)
```

## 6. DJ技法とミキシング

### Hip Hop ミキシング原則

**短いミックス:**
```
特徴:
- 5-15秒
- カットイン多用
- スクラッチ
- ビートジャグリング

手順:
1. 次の曲をキュー（イントロ or ドロップ）
2. ビートマッチング
3. カットイン（クロスフェーダーまたはボリューム）
4. 5-15秒で完全切り替え

Hip Hop = 短く、インパクト
```

### クロスフェーダー使用

**重要:**
```
House/Techno:
- クロスフェーダーあまり使わない
- ボリュームフェーダー中心

Hip Hop:
- クロスフェーダー多用
- スクラッチ必須
- カットイン

DDJ-FLX4:
- クロスフェーダー標準装備
- カーブ調整可能
```

### スクラッチ実践

**セット中の使い方:**
```
イントロ/アウトロ:
- スクラッチでリズム追加
- エネルギー増加

ブレイク:
- 2-4ビートスクラッチ
- 観客を盛り上げる

ドロップ前:
- スクラッチでテンション
- ビルドアップ

適度に使う（やりすぎ注意）
```

## 8. セット構築

### 60分 Hip Hop セット例（パーティー）

**フェーズ1: ウォームアップ（0-15分）**
```
BPM: 85-90
エネルギー: 4-5 / 10

選曲:
- クラシック
- グルーヴィー
- 誰でも知っている

例:
1. DJ Jazzy Jeff「Summertime」88 BPM
2. A Tribe Called Quest「Can I Kick It?」90 BPM
3. Gang Starr「Mass Appeal」90 BPM
```

**フェーズ2: エネルギー構築（15-35分）**
```
BPM: 90-95
エネルギー: 6-7 / 10

選曲:
- 2000年代ヒット
- 観客が歌える
- エネルギー高い

例:
4. 50 Cent「In Da Club」90 BPM - 観客大喜び
5. Eminem「Lose Yourself」92 BPM
6. Kanye West「Gold Digger」93 BPM
7. Jay-Z「99 Problems」94 BPM
```

**フェーズ3: ピーク（35-50分）**
```
BPM: 140-160（Trap）
エネルギー: 8-9 / 10

選曲:
- 最新Trap
- ドロップ強い
- エネルギー最高

例:
8. RL Grime「Core」150 BPM
9. Baauer「Harlem Shake」140 BPM - ダンスタイム
10. Travis Scott「SICKO MODE」155 BPM
11. Migos「Bad and Boujee」145 BPM
```

**フェーズ4: クールダウン（50-60分）**
```
BPM: 90-85
エネルギー: 5-6 / 10

選曲:
- クラシック回帰
- 感動系
- 記憶に残る

例:
12. Tupac「California Love」92 BPM
13. Notorious B.I.G.「Juicy」90 BPM
14. Dr. Dre「Still D.R.E.」93 BPM - 締め

クラシックで締める = リスペクト
```

## 9. DDJ-FLX4での実践

### クロスフェーダー設定

**DDJ-FLX4:**
```
クロスフェーダーカーブ:
- Sharp（急）: スクラッチ向き
- Smooth（滑らか）: ミックス向き

推奨: Sharp設定
理由: Hip Hop はカットイン多用

調整: DDJ-FLX4 Utility設定
```

### Performance Pads活用

**Hot Cue:**
```
Hip Hop に最適:

設定例:
Pad 1: イントロ開始
Pad 2: ドロップ
Pad 3: ボーカル開始
Pad 4: アウトロ
Pad 5-8: スクラッチポイント

即座にジャンプ可能
```

### 練習方法

**Week 1-2: カットイン**
```
練習内容:
1. Hip Hop 10曲選ぶ
2. 5-15秒でカットイン
3. クロスフェーダー使用
4. 50回繰り返し

目標:
- タイミング完璧
- スムーズな切り替え
```

**Week 3-4: スクラッチ追加**
```
練習内容:
1. Baby Scratch練習
2. セット中に2-4ビート追加
3. 録音して分析

目標:
- 自然なスクラッチ
- リズムに合う
```

## まとめ

- **歴史**: 1973年ブロンクス、DJ Kool Herc、DJ文化の発明
- **特徴**: BPM 80-100（Boom Bap）/135-160（Trap）、サンプリング、グルーヴ
- **サブジャンル**: Boom Bap、Trap、Lofi、Drill（それぞれ時代と雰囲気異なる）
- **ターンテーブリズム**: スクラッチ、ジャグリング、楽器としてのターンテーブル
- **DJ技法**: 短いミックス（5-15秒）、カットイン、クロスフェーダー多用
- **代表的DJ**: Grandmaster Flash、DJ Jazzy Jeff、DJ Premier、Q-Bert等
- **必聴曲**: 1970年代起源から2020年代Trapまで100曲以上
- **セット構築**: クラシック→2000年代→Trap→クラシック回帰
- **DDJ-FLX4**: クロスフェーダーSharp設定、Hot Cue多用、スクラッチ練習
