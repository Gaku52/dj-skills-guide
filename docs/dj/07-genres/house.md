## なぜ House が重要か

**House = 電子音楽の基礎:**
```
事実:
- 世界で最も人気のジャンル
- クラブギグの70%が House
- DJ人口の50%が House DJ
- 学びやすく、奥が深い

House をマスター = DJ の基礎
```

## 1. House の歴史

### 起源（1980年代シカゴ）

**1977-1984: 誕生**
```
場所: シカゴ

キーパーソン:
- Frankie Knuckles（"The Godfather of House"）
- Ron Hardy
- Larry Levan（ニューヨーク）

The Warehouse（クラブ）:
- Frankie Knuckles が DJ
- Disco、Soul、Funk をミックス
- ドラムマシン（Roland TR-808）追加
- "House Music" の名前の由来

1984年: 最初の House 曲
「On and On」by Jesse Saunders
```

### ゴールデンエイジ（1985-1995）

**シカゴからヨーロッパへ:**
```
1985-1987: シカゴ House 確立
- Trax Records、DJ International
- Marshall Jefferson「Move Your Body」
- Fingers Inc.「Can You Feel It」

1987-1990: UK、ヨーロッパへ
- Acid House ブーム
- Summer of Love（1988）
- イビサ島が中心地に

1990-1995: グローバル化
- Deep House、Tech House登場
- ニューヨーク、デトロイト、ベルリン
- 世界中に拡散
```

### 現代（1995-現在）

**進化と多様化:**
```
1995-2005: サブジャンル爆発
- Progressive House
- Tribal House
- Electro House

2005-2015: EDM ブーム
- David Guetta、Calvin Harris
- メインストリーム化
- フェスティバル中心

2015-現在: アンダーグラウンド回帰
- Tech House 人気（Carl Cox、Jamie Jones）
- Deep House（Black Coffee、Kerri Chandler）
- Minimal House

House は常に進化
```

## 2. House の特徴

### 音楽的特徴

**基本構造:**
```
BPM: 120-130（標準は 125）

リズム:
- 4つ打ち（Four on the floor）
- キックが4分音符で連打
- ドン・ドン・ドン・ドン

楽器:
- ドラムマシン（Roland TR-808、909）
- ベースライン（深い、グルーヴィー）
- ハイハット（オープン/クローズ）
- クラップ/スネア

ボーカル:
- ソウルフル（Deep House）
- サンプリング（Tech House）
- アカペラ

雰囲気:
- グルーヴ重視
- ダンサブル
- ポジティブ
```

**特徴:**
```

音楽的特徴:

- John Digweed

必聴曲:

会場: フェスティバル、大規模クラブ
```

### Soulful House

**特徴:**
```
BPM: 120-125
雰囲気: ソウルフル、エモーショナル

音楽的特徴:
- ゴスペル、ソウルの影響
- 生ボーカル
- ピアノ、オルガン
- 温かい

代表的アーティスト:
- Frankie Knuckles
- Louie Vega
- Masters at Work
- Tony Humphries

必聴曲:

- Masters at Work「To Be in Love」

会場: アンダーグラウンドクラブ、老舗クラブ

## 5. DJ技法とミキシング

### House のミキシング原則

**ロングミックス:**
```
特徴:
- 1-2分のミックス
- スムーズな移行
- グルーヴを壊さない

手順:
1. 次の曲をキュー（イントロから）
2. ビートマッチング
3. ゆっくりフェードイン（32-64ビート）
4. EQ調整（低音から）
5. 両曲が混ざる（32ビート）
6. ゆっくりフェードアウト

合計: 1-2分

House = ロングミックスの芸術
```

### EQ の使い方

**3バンド EQ:**
```
LOW（ベース）:
- 最も重要
- 両曲のベースは被らない
- 片方を完全にカット
- 徐々に切り替え

MID（ボーカル、メロディ）:
- 少し下げる程度
- 完全カットしない
- バランス調整

HIGH（ハイハット、シンバル）:
- 微調整
- 明るさを調整

EQ = House ミックスの鍵
```

### トランジション技法

**Echo Out:**
```
使用タイミング:
- ピークから次の曲へ
- エネルギーを保つ

手順:
1. 現在の曲にEcho（DDJ-FLX4のBeat FX）
2. Echo時間を伸ばす
3. 次の曲をミックスイン
4. Echoを切る

効果: スムーズで音楽的
```

**Filter Sweep:**
```
使用タイミング:
- ビルドアップからドロップ
- エネルギー変化

手順:
1. 現在の曲にLow Pass Filter
2. 徐々にフィルター閉じる
3. 次の曲をミックスイン
4. フィルター開く

効果: ダイナミック
```

## 7. セット構築

### 60分 Tech House セット例

**フェーズ1: ウォームアップ（0-15分）**
```
BPM: 122-125
エネルギー: 3-4 / 10

選曲:
- グルーヴィーだが落ち着いている
- ボーカル少なめ
- ベースライン重視

例:
1. Patrick Topping「Voicemail」122 BPM
2. Solardo「Tribesmen」124 BPM
3. CamelPhat「Breathe」125 BPM
```

**フェーズ2: エネルギー構築（15-35分）**
```
BPM: 125-128
エネルギー: 5-7 / 10

選曲:
- より Energy 高い
- パーカッシブ
- ボーカルサンプル

例:
4. Fisher「Crowd Control」126 BPM
5. Chris Lake「Turn Off the Lights」127 BPM
6. Jamie Jones「Hungry for the Power」128 BPM
7. Green Velvet「La La Land」128 BPM
```

**フェーズ3: ピーク（35-50分）**
```
BPM: 128-130
エネルギー: 8-9 / 10

選曲:
- 最もエネルギッシュ
- ドロップあり
- 観客が知っている曲

例:
8. Fisher「Losing It」128 BPM - 観客大喜び
9. Chris Lake「Operator」129 BPM
10. Carl Cox「I Want You」130 BPM
```

**フェーズ4: クールダウン（50-60分）**
```
BPM: 128-126
エネルギー: 6-7 / 10

選曲:
- 少しエネルギー下げる
- グルーヴ重視
- 記憶に残る曲

例:
11. Solardo「XTC」127 BPM
12. Patrick Topping「Be Sharp」126 BPM
13. CamelPhat & Elderbrook「Cola」126 BPM

最後は印象的な曲で締める
```

## 8. DDJ-FLX4での実践

### 推奨設定

**Smart CFX:**
```
Low Cut推奨:
- House のミックスに最適
- ベース被りを防ぐ
- スムーズな移行

使用タイミング:
- トランジション時
- ベース切り替え時
```

**Beat FX:**
```
Echo:
- トランジションに
- フレーズ終わりに

Reverb:
- アトモスフェリック
- ビルドアップに

Filter:
- ダイナミックな変化
- ドロップ前に

House = Echo と Filter が基本
```

### ミックス練習

**Week 1-2:**
```
練習内容:
1. Tech House 10曲選ぶ
2. 全曲のビートマッチング練習
3. EQミックス（ベース切り替え）
4. 2曲ミックス x 50回

目標:
- 完璧なビートマッチ
- スムーズなベース切り替え
```

**Week 3-4:**
```
練習内容:
1. 30分セット作成
2. エネルギーカーブ意識
3. トランジション技法追加
4. 録音して分析

目標:
- 30分ノーミス
- 自然な流れ
```

## 9. House DJ になるための道筋

```

```
実践:

- レジデンシー目指す
- 毎週配信（Twitch）
- ファンベース構築

目標:
- 月1回以上のギグ
- House DJ として認知される
```

## まとめ

- **歴史**: 1980年代シカゴ、Frankie Knuckles、4つ打ちの誕生
- **特徴**: BPM 120-130、グルーヴ重視、ダンサブル、ポジティブ
- **サブジャンル**: Deep、Tech、Progressive、Soulful（それぞれ特徴異なる）
- **DJ技法**: ロングミックス（1-2分）、EQ重視、グルーヴ維持
- **代表的アーティスト**: Carl Cox、Jamie Jones、Fisher、Black Coffee等
- **必聴曲**: 1980年代クラシックから2020年代最新まで100曲以上
- **セット構築**: ウォームアップ→構築→ピーク→クールダウン
- **DDJ-FLX4**: Smart CFX Low Cut、Echo/Filter使用、ベース管理重要
- **道筋**: 3ヶ月基礎、6ヶ月スキル向上、1年でプロ
