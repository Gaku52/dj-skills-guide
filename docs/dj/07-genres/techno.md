## 1. Techno の歴史

### 起源（1980年代デトロイト）

**1981-1985: 誕生**
```
場所: デトロイト、ミシガン州

The Belleville Three（ベルヴィル三人衆）:
- Juan Atkins（"The Originator"）
- Derrick May
- Kevin Saunderson

影響:
- Kraftwerk（ドイツ）
- Giorgio Moroder（イタリア）
- Parliament-Funkadelic
- Chicago House

1985年: 最初の Techno 曲
Juan Atkins (as Model 500)「No UFO's」

特徴:
- 未来的
- 機械的
- ミニマル
- インダストリアル
```

### ヨーロッパ展開（1988-1995）

**ベルリンが中心地に:**
```
1989年: ベルリンの壁崩壊
→ 東ベルリンの廃墟がクラブに
→ Techno の聖地誕生

重要なクラブ:
- Tresor（1991-）廃墟の銀行
- Berghain（2004-）発電所跡
- E-Werk（1993-1997）

UK:
- Warp Records（Sheffield）
- 808 State
- The Prodigy（初期）

ドイツ:
- Sven Väth
- Richie Hawtin（カナダ→ドイツ）
- Adam Beyer（スウェーデン）

1990年代: Techno グローバル化
```

### 現代（1995-現在）

**進化と多様化:**
```
1995-2005: ミニマルブーム
- Richie Hawtin (Plastikman)
- Ricardo Villalobos
- Luomo

2005-2015: ハードテクノ人気
- Adam Beyer、Drumcode
- Charlotte de Witte
- I Hate Models

2015-現在: Industrial リバイバル
- 暗い、速い（135+ BPM）
- Paula Temple
- Kobosil
- Dax J

Techno は常に進化
最もアンダーグラウンド
```

## 2. Techno の特徴

### 音楽的特徴

**基本構造:**
```
BPM: 125-150（標準は 130-135）

リズム:
- 4つ打ち（House と同じ）
- より機械的、正確
- ミニマル

楽器:
- ドラムマシン（Roland TR-909最重要）
- シンセサイザー（暗い、攻撃的）
- サンプラー
- エフェクト多用

ボーカル:
- ほぼなし
- あってもサンプル
- 音楽が主役

雰囲気:
- ダーク
- 工業的
- 催眠的
- トランス状態誘発

House との違い:
- House: 明るい、グルーヴィー
- Techno: 暗い、ミニマル
```

## 3. サブジャンル

### Minimal Techno

**特徴:**
```
BPM: 125-130
雰囲気: ミニマル、催眠的、深い

音楽的特徴:
- 最小限の要素
- リピート、変化少ない
- ディープ
- 長い曲（8-10分）

代表的アーティスト:
- Richie Hawtin (Plastikman)
- Ricardo Villalobos
- Robert Hood
- Jeff Mills（初期）
- Luciano

必聴曲:
- Plastikman「Spastik」
- Robert Hood「Minus」
- Ricardo Villalobos「Dexter」

会場: アンダーグラウンドクラブ、ベルリン
```

### Detroit Techno（オリジナル）

**特徴:**
```
BPM: 120-135
雰囲気: ソウルフル、未来的

音楽的特徴:
- House の影響（ソウルフル）
- メロディアス
- エモーショナル
- クラシック

代表的アーティスト:
- Juan Atkins (Model 500)
- Derrick May
- Kevin Saunderson (Inner City)
- Carl Craig
- Jeff Mills

必聴曲:
- Derrick May「Strings of Life」
- Inner City「Good Life」
- Jeff Mills「The Bells」

会場: クラシック Techno イベント
```

### Industrial Techno / Hard Techno

**特徴:**
```
BPM: 135-150+
雰囲気: ダーク、攻撃的、激しい

音楽的特徴:
- 歪んだキック
- ノイズ
- 速い
- 暗い

代表的アーティスト:
- Charlotte de Witte
- I Hate Models
- Kobosil
- Dax J
- Paula Temple
- Amelie Lens

必聴曲:
- Charlotte de Witte「Selected」
- I Hate Models「Warehouse Memories」
- Dax J「Escape the System」

会場: Berghain、大規模フェスティバル
```

### Acid Techno

**特徴:**
```
BPM: 130-140
雰囲気: サイケデリック、トリッピー

音楽的特徴:
- Roland TB-303（ベースシンセ）
- 独特な「酸っぱい」サウンド
- 催眠的
- グルーヴィー

代表的アーティスト:
- Plastikman
- Phuture（House 寄り）
- Chris Liberator
- D.A.V.E. The Drummer

必聴曲:
- Plastikman「Acid House」
- Phuture「Acid Tracks」

会場: レイヴ、アンダーグラウンド
```

## 5. DJ技法とミキシング

### Techno のミキシング原則

**ハードカット vs ロングミックス:**
```
ハードカット（多用）:
- 8-16ビートで切り替え
- クロスフェーダーまたはボリューム
- ダイナミック
- エネルギー維持

ロングミックス:
- 32-64ビート（House より短い）
- ミニマル曲に
- レイヤー重視

Techno = 両方使い分け
```

### EQ の使い方（Techno特有）

**アグレッシブなEQ:**
```
LOW（ベース）:
- 完全カット多用
- ドラマチックな変化
- ビルドアップに

MID:
- ブーストもあり
- エネルギー増加
- House より積極的

HIGH:
- カット/ブースト大胆に
- エフェクト的に使用

Techno = EQ を楽器として使う
```

### ループとエフェクト

**ループ多用:**
```
4-8ビートループ:
- トランジション時
- ビルドアップ作成
- エネルギー調整

DDJ-FLX4:
- Performance Pads でループ
- Loop In/Out ボタン
- Slip Mode使用

Techno DJはループマスター
```

**エフェクト:**
```
Reverb:
- スペース感
- トランジション

Delay/Echo:
- リズミック
- ビルドアップ

Filter:
- 最重要
- Low Pass多用
- ビルドアップ→ドロップ

Techno = エフェクト重要
```

## 7. セット構築

### 90分 Techno セット例（ベルリンスタイル）

**フェーズ1: ウォームアップ（0-20分）**
```
BPM: 125-128
エネルギー: 2-3 / 10

選曲:
- Minimal、Deep
- ゆっくり始める
- グルーヴ構築

例:
1. Petre Inspirescu「Zwer」125 BPM
2. Raresh「Mana」126 BPM
3. Ricardo Villalobos「Fizpatrick」127 BPM
4. Robert Hood「Minus」128 BPM

ベルリンは長いウォームアップ
```

**フェーズ2: エネルギー構築（20-50分）**
```
BPM: 128-132
エネルギー: 4-6 / 10

選曲:
- よりパーカッシブ
- ループ多用
- 徐々に暗く

例:
5. Ben Klock「Subzero」129 BPM
6. Marcel Dettmann「Korrupt」130 BPM
7. Len Faki「Death by Silence」131 BPM
8. Adam Beyer「Your Mind」132 BPM
9. Joseph Capriati「C'est La Vie」132 BPM
```

**フェーズ3: ピーク（50-75分）**
```
BPM: 132-138
エネルギー: 7-9 / 10

選曲:
- Hard、Industrial
- 速い
- ダーク

例:
10. Charlotte de Witte「Selected」135 BPM
11. I Hate Models「Warehouse Memories」136 BPM
12. Dax J「Escape the System」137 BPM
13. Jeff Mills「The Bells」138 BPM - クラシック
14. Amelie Lens「In My Mind」138 BPM
```

**フェーズ4: クールダウン（75-90分）**
```
BPM: 135-130
エネルギー: 6-5 / 10

選曲:
- 少し落ち着く
- メロディアス
- クラシック

例:
15. Derrick May「Strings of Life」133 BPM - 感動
16. Carl Craig「At Les」130 BPM
17. Robert Hood「One Touch」128 BPM

デトロイトクラシックで締める = リスペクト
```

## 10. Techno 文化とマナー

### Berghain文化

**世界で最も有名なTechnoクラブ:**
```
ルール:
- 写真撮影禁止
- 音楽に集中
- ドレスコード（黒）
- 入場審査厳しい

精神:
- 音楽が主役
- DJ崇拝しない
- 踊ることが目的
- リスペクト

Techno = 音楽への献身
```

### DJとしての姿勢

**アンダーグラウンド精神:**
```
✓ 商業主義を避ける
✓ アートとして音楽を
✓ コミュニティ重視
✓ リスペクト

✗ 金儲け優先
✗ 派手なパフォーマンス
✗ 自己中心的

Techno = 謙虚、音楽中心
```

## まとめ

- **歴史**: 1980年代デトロイト、The Belleville Three、未来的サウンド
- **特徴**: BPM 125-150、ダーク、工業的、ミニマル、4つ打ち
- **サブジャンル**: Minimal、Detroit、Industrial/Hard、Acid（それぞれ個性強い）
- **DJ技法**: ハードカット多用、ループ重要、エフェクト（Filter）、EQアグレッシブ
- **代表的アーティスト**: Jeff Mills、Richie Hawtin、Charlotte de Witte、Adam Beyer等
- **必聴曲**: デトロイトクラシックから最新Industrialまで100曲以上
- **セット構築**: 長いウォームアップ、徐々にハード、クラシックで締める
- **DDJ-FLX4**: High Pass Filter、Beat FX Filter多用、ループマスター
- **文化**: ベルリン精神、音楽中心、アンダーグラウンド、リスペクト
