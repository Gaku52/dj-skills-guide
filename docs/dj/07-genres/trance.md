# Trance

感情的な高揚感と壮大なビルドアップが特徴的なトランス。クラブからフェスまで、観客を感動的な音楽の旅へと導くジャンルです。

## 1. Trance の歴史

### 起源（1990年代初期 ドイツ/ゴア）

**1992-1994: Trance の誕生**

```
場所: フランクフルト、ベルリン（ドイツ）

起源:
Techno + Acid House + New Age
= Trance

特徴:
- 反復的なシンセメロディ
- エモーショナルなビルドアップ
- 130-140 BPM
- トライバルなリズム
```

**キーパーソン:**
- Sven Väth（Frankfurt）
- Paul van Dyk（Berlin）
- Jam & Spoon
- Cosmic Baby

**初期の名曲:**
- Age of Love - The Age of Love (1990)
- Energy 52 - Café Del Mar (1993)
- Jam & Spoon - Stella (1992)
- Paul van Dyk - For An Angel (1994)

### 1995-1999: ゴアトランスとプログレッシブ

**ゴアトランス（Goa Trance）:**

```
場所: ゴア、インド

特徴:
- サイケデリックなサウンド
- 複雑なシンセライン
- スピリチュアルな雰囲気
- 138-145 BPM

代表的アーティスト:
- Astral Projection
- Juno Reactor
- Infected Mushroom (初期)
- Hallucinogen
```

**プログレッシブトランス台頭:**
- より洗練されたサウンド
- ハウス的な要素
- 長いミックス展開
- Sasha、John Digweed の影響

### 2000-2004: 黄金時代


## 2. 音楽的特徴

### BPM とリズム

**典型的なBPM:**
- Progressive Trance: 128-132 BPM
- Uplifting Trance: 136-140 BPM
- Psytrance: 138-145 BPM
- Tech Trance: 135-138 BPM

**リズムパターン:**

```
キックドラム:
4/4 ビート（4つ打ち）
|ドン|　　|ドン|　　|ドン|　　|ドン|　　|
 1    2    3    4

特徴:
- Deep でパワフルなキック
- オープンハイハット（16分音符）
- ドライブ感のあるベースライン
```

### メロディとハーモニー

**トランス特有のメロディ:**

```
典型的な展開:

1. Intro（イントロ）
   └─ ミニマルなビート
      時間: 0:00-1:30

2. Build-up（ビルドアップ）
   └─ シンセが徐々に追加
      時間: 1:30-3:00

3. Breakdown（ブレイクダウン）
   └─ 感情的な展開
      ボーカルやメロディ
      時間: 3:00-4:30

4. Climax（クライマックス）
   └─ 全要素が爆発
      最高潮の瞬間
      時間: 4:30-6:00

5. Outro（アウトロ）
   └─ 徐々に減衰
      次の曲へ
      時間: 6:00-8:00
```

**コードプログレッション:**
- マイナーキーが多い
- エモーショナルな展開
- アルペジオシンセ
- パッドサウンド

### サウンドデザイン

**典型的な音色:**

```
1. シンセリード:
   - TB-303風アシッドライン
   - Supersaw（壮大なサウンド）
   - アルペジエーター

2. パッド:
   - アンビエントな雰囲気
   - リバーブたっぷり
   - ストリングス的

3. プラック:
   - 短いスタッカート音
   - リズミカルな要素
   - 16分音符パターン

4. エフェクト:
   - ホワイトノイズスウィープ
   - フィルタースウィープ
   - リバースシンバル
```

## 6. DJ技法とミキシング

### 基本ミックス技法

**ロングミックス（2-4分）:**

```
トランスの基本:

理由:
- 長いイントロ/アウトロ構造
- ビルドアップを活かす
- ジャーニー重視
- 感情的な展開

推奨ミックス時間:
- Progressive: 3-4分
- Uplifting: 2-3分
- Psytrance: 1-2分
- Tech Trance: 2-3分
```

**EQミックス:**

```
DDJ-FLX4 での設定:

Track A（プレイ中）:
├─ HI: 12時（0）
├─ MID: 12時（0）
└─ LOW: 12時（0）

Track B（次の曲、イントロ）:
├─ HI: 9時（-∞）
├─ MID: 9時（-∞）
└─ LOW: 12時（0） ← ベースは残す

徐々に:
1. B の HI を上げる（30秒）
2. B の MID を上げる（30秒）
3. A の HI を下げる（30秒）
4. A の MID を下げる（30秒）
5. A の LOW を下げる（30秒）
6. スムーズにトランジション完了
```

### ブレイクダウンミックス

**感情的なトランジション:**

```
タイミング:

Track A: ブレイクダウン開始
       ↓
      (キックが抜ける)
       ↓
Track B: イントロ開始
       ↓
      (ベースラインのみ)
       ↓
Track A: フェードアウト
       ↓
Track B: ビルドアップ開始
       ↓
      (クライマックスへ)

時間: 約2-3分
```

**具体例:**

```
曲: Above & Beyond - Sun & Moon

0:00-1:30: イントロ
          → 前の曲とミックス開始

3:00-4:30: ブレイクダウン
          → 次の曲のイントロ開始
          → 感情的な瞬間

4:30-6:00: クライマックス
          → 前の曲完全フェードアウト
```

### キーミキシング（ハーモニック）

**トランスでの重要性:**

```
理由:
- メロディが豊富
- ハーモニーが重要
- 感情的な展開

推奨トランジション:
Perfect 5th: +7 semitones
Perfect 4th: +5 semitones
Minor 3rd: +3 semitones

例:
Track A: 8A (A minor)
       ↓
Track B: 8B (C major) ← Perfect!
       または
Track B: 9A (E minor) ← Perfect!
```

**DDJ-FLX4 での確認:**

```
Rekordbox:
1. [Collection] で曲を選択
2. [Key] カラムで確認
   例: 8A, 5B, 12A

3. ソート機能:
   - [Key] をクリックしてソート
   - 同じキーの曲が集まる

4. 次の曲選択:
   - ±1 の番号（例: 8A → 9A）
   - 同じ番号で A↔B（例: 8A → 8B）
```

### エフェクト活用

**DDJ-FLX4 エフェクト:**

**1. Echo（エコー）:**

```
使用タイミング:
- ブレイクダウン前
- ビルドアップ中
- トランジション

設定:
[BEAT] ボタン: 1/2（2分音符）
[LEVEL/DEPTH] ノブ: 12-1時

手順:
1. ブレイクダウン8小節前
2. Echo ON
3. LEVEL を徐々に上げる
4. ブレイクダウン突入でOFF
```

**2. Reverb（リバーブ）:**

```
使用タイミング:
- ボーカルパート
- ブレイクダウン
- 感情的な瞬間

設定:
[BEAT] ボタン: 2/1（2小節）
[LEVEL/DEPTH] ノブ: 10-12時

効果:
- 空間的な広がり
- 感情的な深み
```

**3. Filter（フィルター）:**

```
High Pass Filter:
- イントロ開始時
- 徐々に曲を導入

Low Pass Filter:
- ブレイクダウン前
- トランジション
- 雰囲気作り

Smart CFX (DDJ-FLX4):
[SMART CFX] ノブを回す:
- 左: Low Pass Filter
- 右: High Pass Filter
```

### エナジーマネジメント

**90分セットの構築:**

```
0-15分: Warm Up（導入）
├─ BPM: 128-132
├─ スタイル: Progressive
└─ 目的: 雰囲気作り

15-30分: Build Up（上昇）
├─ BPM: 132-136
├─ スタイル: Progressive → Uplifting
└─ 目的: エナジー上昇

30-60分: Peak Time（ピーク）
├─ BPM: 136-140
├─ スタイル: Uplifting/Anthem
└─ 目的: クライマックス、感動

60-75分: Mid Energy（中盤）
├─ BPM: 136-138
├─ スタイル: Tech Trance/Vocal
└─ 目的: 一息、でもエナジー維持

75-90分: Closing（クロージング）
├─ BPM: 138-140
├─ スタイル: Anthem/Classic
└─ 目的: 最高の瞬間、記憶に残る
```

## 8. DDJ-FLX4 での実践

### 基本セットアップ

**Rekordbox 設定:**

```
1. トラックプレップ:
   □ すべての曲に [Key] 分析
   □ [BPM] 確認
   □ [Beatgrid] 調整
   □ [Memory Cue] 設定（イントロ、ブレイクダウン、クライマックス）

2. プレイリスト作成:
   □ Key でソート
   □ BPM でソート
   □ ジャンル別フォルダ

3. Hot Cue 設定:
   A: イントロ終わり（ミックスイン）
   B: ブレイクダウン開始
   C: クライマックス
   D: アウトロ開始（ミックスアウト）
```

**ハードウェア設定:**

```
DDJ-FLX4:

[MASTER LEVEL] ノブ:
- クラブ: 12時
- 自宅: 10時

[HEADPHONES MIX] ノブ:
- Master 50% / Cue 50%（12時）

[TRIM] ノブ:
- すべて12時でスタート
- 曲ごとに微調整（レベルメーター確認）

[TEMPO] フェーダー:
- センター（Sync 使用推奨）
```

### トランス特有の設定

**Smart CFX の活用:**

```
トランスに最適なエフェクト:

[SMART CFX] ノブ:

左回転（Low Pass Filter）:
- ブレイクダウン前
- トランジション
- 雰囲気作り

使用例:
Track A（プレイ中）: ブレイクダウン前
└─ SMART CFX を左に回す
   → 徐々にHI がカット
   → ブレイクダウンが際立つ

右回転（High Pass Filter）:
- イントロ開始
- 新しい曲の導入
- トランジション

使用例:
Track B（次の曲）: イントロ開始
└─ SMART CFX を右に回す
   → 徐々にLOW がカット
   → ベースだけ残る
   → 綺麗にミックス
```

**Merge FX の活用:**

```
[MERGE FX] ボタン:

機能:
2つのトラックを自動でミックス

トランスでの使用:
- Progressive → Uplifting トランジション
- キーが合っている曲同士
- 実験的なミックス

注意:
- 完全自動なのでコントロール不可
- クリエイティブな瞬間用
- 通常は手動ミックス推奨
```

### トラブル対応

**よくある問題:**

**問題1: キーが合わない**

```
症状:
- 不協和音
- 気持ち悪いハーモニー

解決:
1. すぐに次の曲の LOW をカット
2. HI/MID も下げる
3. 別の曲を選択
4. キーが合う曲を探す（Rekordbox Key カラム）
5. やり直し

予防:
- 事前にキー分析
- プレイリストをキーでソート
- リハーサル
```

**問題2: ブレイクダウンのタイミングずれ**

```
症状:
- 2曲のブレイクダウンが重なる
- エナジーが下がりすぎ

解決:
1. どちらか1曲のブレイクダウンをスキップ
   - Hot Cue C（クライマックス）にジャンプ
2. または、片方を早めにフェードアウト

予防:
- 曲構造を把握
- ブレイクダウン位置に Memory Cue
- リハーサルで確認
```
