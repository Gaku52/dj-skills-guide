# 聴力保護 ⚠️ 最重要

DJの命は耳。一度失った聴力は二度と戻らない。長期キャリアを守るための聴力保護完全ガイドです。

## ⚠️ 緊急警告

```
聴力損失は不可逆（元に戻らない）

一度ダメージを受けた聴覚細胞:
→ 再生しない
→ 永久に失われる
→ 治療法なし

プロDJ引退の最大理由:
難聴・耳鳴り（ティニタス）

予防が唯一の対策
```

## 聴力損失のメカニズム

### 耳の構造

**音が聞こえる仕組み:**

```
外耳 → 中耳 → 内耳 → 脳

内耳（蝸牛）:
├─ 有毛細胞（約16,000個）
│  - 音を電気信号に変換
│  - 非常に繊細
│  - 再生不可能 ← 重要！
│
└─ 大音量で破壊される
   → 永久に失われる
   → 難聴
```

### 音圧レベル（dB）と危険性

**デシベル（dB）スケール:**

```
0 dB: 聞こえる最小音
10 dB: 呼吸音
20 dB: 木の葉のそよぎ
30 dB: 図書館
40 dB: 静かなオフィス
50 dB: 通常の会話
60 dB: レストラン
70 dB: 掃除機

───── 安全ライン ─────

85 dB: 繁華街 ← 8時間以上でリスク
90 dB: 芝刈り機 ← 2時間以上でリスク
95 dB: オートバイ ← 47分以上でリスク
100 dB: ナイトクラブ ← 15分以上でリスク
105 dB: DJ ブース ← 5分以上でリスク
110 dB: ロックコンサート ← 2分以上でリスク
115 dB: 雷 ← 30秒以上でリスク
120 dB: ジェットエンジン ← 即座にダメージ
130 dB: 痛みを感じる ← 即座に有害

警告:
100dB で2時間のDJセット
= 確実に聴覚ダメージ
```

### 露出時間とダメージ

**85dB ルール:**

```
国際基準（NIOSH）:

85 dB: 8時間まで安全
88 dB: 4時間まで
91 dB: 2時間まで
94 dB: 1時間まで
97 dB: 30分まで
100 dB: 15分まで
103 dB: 7.5分まで
106 dB: 3.75分まで

3dB 上がるごとに:
安全時間が半分に

DJブース（105dB）で2時間:
→ 安全時間の24倍超過
→ 深刻なダメージ
```

## 難聴の症状と種類

### 初期症状（要注意）

**チェックリスト:**

```
□ DJ後に耳が詰まった感じ
□ セット後に音がこもる
□ 翌日まで聞こえづらい
□ 高音が聞こえにくい
□ テレビの音量を上げがち
□ 会話が聞き取りづらい
□ 「え？」と聞き返すことが多い
□ 騒がしい場所で会話困難

1つでも該当:
→ 要注意
→ すぐに保護開始

3つ以上該当:
→ 既にダメージあり
→ 医師に相談
→ 徹底的な保護必要
```

### 難聴の種類

**1. 一時的難聴（TTS）**

```
Temporary Threshold Shift

症状:
- ライブ後に聞こえづらい
- 数時間〜数日で回復
- 耳が詰まった感じ

原因:
- 大音量への短時間露出
- 有毛細胞の一時的疲労

危険:
- 「回復するから大丈夫」× 間違い
- 繰り返すと永久ダメージに
- 警告サイン
```

**2. 永久的難聴（PTS）**

```
Permanent Threshold Shift

症状:
- 聴力が戻らない
- 特定の周波数が聞こえない
- 通常4000Hz（高音）から

原因:
- 長期的な大音量露出
- TTS の繰り返し
- 有毛細胞の永久破壊

特徴:
- 不可逆
- 治療法なし
- 進行性
```

**3. 耳鳴り（ティニタス）**

```
Tinnitus

症状:
- 常に「キーン」音
- 静かな場所で特に気になる
- 集中力低下
- 睡眠障害
- 精神的苦痛

原因:
- 聴覚神経のダメージ
- 脳が「音がない」を勘違い
- 幻の音を作り出す

現実:
- 治療法なし
- 一生続く
- QOL の大幅低下

DJキャリア終了の最大要因
```

（12時の75%程度）

ルール:
「隣の部屋で会話できる音量」
```

**本番時（クラブ）:**

```
問題:
- クラブは100-110dB

### 3. 休息時間の確保

**耳の休息ルール:**

```
85dB ルール:

85dB で8時間露出後:
→ 16時間の静音が必要

100dB で1時間露出後:
→ 16時間の静音が必要

実践:

金曜夜 DJ（2時間、100dB）:
→ 土曜は静かに過ごす
→ ヘッドフォン使用避ける
→ 大音量避ける

連続ギグは危険:
金曜・土曜連続:
→ 回復時間なし
→ ダメージ蓄積

推奨:
週2-3回まで
間に休息日
```

**日常の聴力休息:**

```
毎日の習慣:

□ 朝: 静かな時間（30分）
  - 瞑想
  - 読書
  - 無音

  - イヤホンなしで過ごす
  - 静かなカフェ

□ 夜: リラックス
  - 小音量の音楽
  - または無音

週1回:
完全な静音日
- 音楽聞かない
- DJ しない
- 耳を完全休息
```

## 定期的な聴力検査

### 聴力検査の重要性

**早期発見が鍵:**

```
理由:

1. 初期ダメージは自覚なし
   - 4000Hz から始まる
   - 日常会話は 500-3000Hz
   - 気づいたときは手遅れ

2. 進行を止められる
   - 早期発見 → 保護強化
   - ダメージ拡大防止

3. ベースライン確立
   - 今の聴力を記録
   - 将来と比較

推奨:
年1回の聴力検査
```

### 検査の種類

**1. 純音聴力検査（PTA）**

```
内容:
- 様々な周波数の音
- 聞こえる最小音量を測定
- 125Hz - 8000Hz

結果:
オージオグラム（聴力図）

正常: 0-25dB で聞こえる
軽度難聴: 26-40dB
中等度難聴: 41-70dB
重度難聴: 71dB以上

DJ注目:
4000Hz のノッチ
→ 騒音性難聴の典型
```

**2. 高周波数聴力検査**

```
内容:
- 8000Hz 以上
- 9000Hz, 10000Hz, 12000Hz...
- 初期ダメージ発見

DJには重要:
通常の検査では見つからない
早期ダメージを発見
```

### 検査を受ける場所

**選択肢:**

```
1. 耳鼻咽喉科
   - 健康保険適用
   - 詳細な検査
   - 医師の診断

2. 補聴器店
   - 無料検査も
   - 簡易的

3. 聴覚検査センター
   - 専門的
   - 産業医学

推奨:
まず耳鼻咽喉科
→ ベースライン確立
→ 以降は年1回チェック
```

## 耳鳴り（ティニタス）対策

### 耳鳴りの兆候

**初期症状:**

```
□ DJ後に「キーン」音
□ 静かな場所で音が聞こえる
□ 夜、寝る前に気になる
□ 一時的だが頻繁に

警告:
すでにダメージあり
即座に保護強化
```

### 対処法

**既に耳鳴りがある場合:**

```
医療:
1. すぐに耳鼻咽喉科受診
2. ステロイド治療（初期のみ有効）
3. 早ければ早いほど良い

日常対策:
□ 大音量完全避ける
□ 常に耳栓使用
□ カフェイン・アルコール控える
□ ストレス管理
□ 十分な睡眠

サウンドセラピー:
- ホワイトノイズ
- 自然音
- 耳鳴りをマスキング

心理的対応:
- 受容
- CBT（認知行動療法）
- サポートグループ

現実:
完治はほぼ不可能
共存を学ぶ
予防が最重要
```
