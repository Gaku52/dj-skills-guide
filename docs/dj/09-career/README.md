# DJキャリア構築

技術と選曲をマスターしても、それだけではプロにはなれない。ビジネススキルを身につけ、キャリアを築く完全ガイドです。

---

## このセクションについて

DJとして活動するには、音楽的なスキルだけでなく、**ビジネススキル**も必要です。

```
プロDJ = 技術 × 選曲 × ビジネススキル

技術だけでは:
収入にならない

選曲だけでは:
機会が得られない

ビジネススキルがあれば:
持続可能なキャリアを築ける
```

このセクションでは、DJ活動を**仕事**にするための実践的なビジネス知識を学びます。

---

## なぜビジネススキルが重要なのか

### プロとアマチュアの違い

**技術レベル:**

```
アマチュア:
技術スキル 8/10
ビジネススキル 2/10
→ 趣味で終わる

プロ:
技術スキル 8/10
ビジネススキル 8/10
→ 仕事として成立

差は技術ではなく:
ビジネススキル
```

### 現実

**厳しい事実:**

```
DJの95%:
収入だけでは生活できない

DJの80%:
本業を持ちながら活動

DJの5%:
専業DJとして生活

専業になるには:
ビジネススキルが必須
```

でも諦める必要はありません。正しい知識と行動で、誰でもプロを目指せます。

### ステップバイステップ


## よくある質問

### Q1: DJだけで生活できる？

**A:** 可能だが、時間がかかる

```
現実:
専業DJ になるまで:
- 3-5年かかる人が多い
- 副業から始めるのが安全

収入源を多様化:
- クラブDJ
- ウェディングDJ
- イベントDJ
- DJ レッスン
- 音楽制作
- ライブ配信

複数の収入源:
安定したキャリアの鍵
```

### Q2: DJ名はどう決める？

**A:** 覚えやすく、検索しやすく

```
良い例:
- 短い（2-3文字）
- 発音しやすい
- 検索で出てくる
- SNSで空いている

悪い例:
- 長すぎる（5文字以上）
- 既に有名DJと同じ
- 特殊文字が多い

時間をかけて:
一生使う名前だから
```

### Q3: ソーシャルメディアは必須？

**A:** 2025年現在、ほぼ必須

```
理由:
1. プロモーターがチェック
   - フォロワー数
   - エンゲージメント
   - コンテンツ質

2. ファン獲得
   - 直接つながる
   - セット情報発信

3. ブランディング
   - 自分を知ってもらう
   - 差別化

最低限:
Instagram + SoundCloud
```

### Q4: 初ギグはいくらもらえる？

**A:** 初めてなら無料〜¥10,000

```
相場:

オープンデッキ:
¥0（無料）

小さいバー/カフェ:
¥5,000-10,000

プライベートパーティ:
¥10,000-20,000

クラブ（平日）:
¥10,000-30,000

最初は:
お金より経験
```

### Q5: 契約書は必要？

**A:** 必須。口約束は危険

```
トラブル例:
「当日ギャラが半額に」
「急にキャンセルされた」
「音源使用で訴えられた」

契約書があれば:
お互いに安心
トラブル防止

最低限の内容:
- 日時、場所
- 演奏時間
- ギャラ
- キャンセルポリシー
```
