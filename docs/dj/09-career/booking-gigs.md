# ブッキングとギグ獲得

待っているだけでは仕事は来ない。積極的に動いてギグを獲得する実践的な戦略と、初ギグまでの完全ロードマップです。

- ブッキングエージェント
- 初ギグの準備
- リピートを増やす方法

## なぜ積極的なアプローチが必要か

**プロモーターは忙しい:**

```
現実:

プロモーターの元に:
毎週数十人のDJから連絡

受動的に待つ:
埋もれる、見つけてもらえない

積極的にアプローチ:
チャンスが生まれる

「仕事をください」ではなく:
「価値を提供します」の姿勢

行動 = 成功への近道
```

## ギグの種類

### 7つのギグタイプ

**それぞれの特徴:**

```
1. オープンデッキ

内容: 誰でも参加できる DJ イベント
時間: 15-30分
ギャラ: ¥0（無料）
難易度: ★☆☆☆☆

メリット:
- 経験を積める
- プロモーターに見てもらえる
- DJ仲間と交流
- 実践練習

デメリット:
- 無料
- 短時間

おすすめ:
初心者は必ず参加

─────────────────

2. バー/カフェ

内容: 背景音楽DJ
時間: 2-4時間
ギャラ: ¥5,000-15,000
難易度: ★★☆☆☆

メリット:
- 初ギグに最適
- プレッシャー少ない
- 長時間経験

デメリット:
- ギャラ低め
- 観客の反応薄い

選曲:
落ち着いた雰囲気
会話できる音量

─────────────────

3. クラブ（サポート/ウォームアップ）

内容: メインDJ前の時間帯
時間: 1-2時間
ギャラ: ¥10,000-30,000
難易度: ★★★☆☆

メリット:
- クラブ経験
- まだ人が少ない = 失敗しても
- メインDJ の技を学べる

デメリット:
- 観客少ない時間
- 盛り上げすぎNG

選曲:
徐々にエネルギー上げる

─────────────────

4. クラブ（メインタイム）

内容: ピークタイム DJ
時間: 2-3時間
ギャラ: ¥30,000-100,000
難易度: ★★★★☆

メリット:
- 満員のフロア
- 高ギャラ
- 実績になる

デメリット:
- プレッシャー大
- 経験必要

要求:
フロアを盛り上げる力

─────────────────

5. プライベートパーティ

内容: 誕生日、企業イベント等
時間: 2-4時間
ギャラ: ¥20,000-50,000
難易度: ★★☆☆☆

メリット:
- 高ギャラ
- 比較的気楽
- 平日昼間も

デメリット:
- リクエスト対応必須
- 幅広いジャンル必要

準備:
ポップ、定番曲必須

─────────────────

6. ウェディング

内容: 結婚式・披露宴 DJ
時間: 3-6時間
ギャラ: ¥50,000-150,000
難易度: ★★★★☆

メリット:
- 高ギャラ
- 定期的に依頼
- 昼間の仕事

デメリット:
- 専門知識必要
- ミス許されない
- 幅広いジャンル

準備:
ウェディング専用トレーニング

─────────────────

7. フェスティバル

内容: 野外/大型イベント
時間: 1-2時間
ギャラ: ¥50,000-数百万（レベルによる）
難易度: ★★★★★

メリット:
- 大観衆
- 知名度UP
- キャリアの節目

デメリット:
- ブッキング困難
- 実績必須

目標:
中級〜上級者
```

## ギグの見つけ方

### オンライン

**インターネットで探す:**

```
1. イベント情報サイト

日本:
- Resident Advisor (RA)
  → クラブイベント情報
- EventNavi
- こくちーず
- Facebook Events

検索:
「DJ募集」「オープンデッキ」
「[あなたの都市] DJ」

2. SNS

Instagram:
- ハッシュタグ検索
  #dj募集 #オープンデッキ
  #tokyodj #djevent

- ローカルクラブをフォロー
  → 投稿チェック

3. DJ コミュニティ

Reddit:
- r/DJs
- 地域別サブレディット

Facebook グループ:
- 「Tokyo DJs」
- 「DJ ネットワーク」

Discord:
- DJ サーバー多数

4. 求人サイト

Indeed, タウンワーク:
「DJ」で検索
→ バー/クラブ求人

5. クラウドソーシング

coconala, ランサーズ:
「イベントDJ」依頼
→ 初心者でも応募可能
```

### オフライン

**足で稼ぐ:**

```
1. 直接訪問

ターゲット:
- ローカルバー
- カフェ
- 小さいクラブ

アプローチ:
「DJ 募集していますか？」
→ 名刺、デモミックス渡す

タイミング:
昼間、忙しくない時間

2. イベント参加

毎週末:
ローカルイベントに行く

やること:
- 観察
- DJ/プロモーターと話す
- 名刺交換
- 次回も来る

顔を覚えてもらう:
チャンス増える

3. オープンデッキ

最重要:
毎回参加

効果:
- 実践経験
- プロモーターに見てもらえる
- 他DJと交流
- ギグにつながる可能性大

4. DJ スクール/コミュニティ

イベント:
DJ スクール主催のイベント
→ 生徒が出演機会

ネットワーク:
先生、先輩からの紹介
```

## プロモーターへのアプローチ

### リサーチ

**知ってからアプローチ:**

```
Step 1: ターゲット選定

あなたのジャンルに合う:
□ Techno DJ なら Techno イベント
□ House DJ なら House ナイト

リスト作成:
10-20のイベント/会場

Step 2: 徹底リサーチ

各イベントについて:
□ 主催者は誰？
□ どんな雰囲気？
□ どんなDJ が出ている？
□ どんな音楽？
□ 観客層は？

実際に行く:
最低1回は参加

Step 3: 接点を見つける

SNS:
□ フォロー
□ 投稿にいいね、コメント
□ ストーリーズに反応

イベント:
□ 参加
□ 挨拶
□ 名刺交換

関係構築:
いきなり営業× NG
```

### アプローチメール

**テンプレート:**

```
件名:
「DJ出演のご相談 - DJ GAKU（Techno）」

本文:

「
[イベント名/主催者名] 様

はじめまして。
東京を拠点に Techno DJ として活動している
DJ GAKU と申します。

先日、[会場名] で開催された
[イベント名] に参加させていただき、
素晴らしい雰囲気と音楽に感動いたしました。

特に [メインDJ名] のセットが印象的で、
私も同じような方向性の Techno を
プレイしております。

もし今後、ウォームアップや
サポートDJ の機会がございましたら、
ぜひご検討いただけないでしょうか。

▼ Demo Mix (Techno, 60min)
https://soundcloud.com/djgaku/xxx

▼ Instagram
https://instagram.com/djgaku

▼ EPK (Electronic Press Kit)
https://drive.google.com/xxx

ご多忙中恐れ入りますが、
ご検討のほど、よろしくお願いいたします。

DJ GAKU
Email: dj@example.com
Tel: 090-XXXX-XXXX
」

ポイント:
1. 具体的（イベント名、DJ名）
   → ちゃんと来てる証明

2. 謙虚
   → 「チャンスください」

3. 簡潔（300文字程度）
   → 忙しい人向け

4. リンク明記
   → すぐ確認できる

5. 連絡先
   → 返信しやすく
```

### 対面アプローチ

**イベント後に:**

```
タイミング:
イベント終了後
DJ/プロモーターが片付け中

アプローチ:

「お疲れ様です。
今日のイベント、素晴らしかったです。

私も DJ をやっていて、
いつかこの会場でプレイできたらと
思っています。

もしよければ、デモを聴いて
いただけませんか？」

→ 名刺、USBメモリ渡す

注意:
- 忙しそうなら避ける
- 簡潔に（1-2分）
- しつこくしない
- 笑顔で

印象:
良ければ覚えてもらえる
```

## デモミックス戦略

### 複数パターン用意

**ターゲット別に:**

```
Pattern 1: メインデモ（60分）

目的: あなたの真骨頂
内容: ベストミックス
使用: メインタイム応募

Pattern 2: ウォームアップ（30分）

目的: 徐々に盛り上げる
内容: BPM低め → 徐々に↑
使用: サポートDJ応募

Pattern 3: ピークタイム（30分）

目的: フロアを沸かせる
内容: ハイエナジー
使用: メインタイム応募

Pattern 4: ラウンジ/バー（30分）

目的: 落ち着いた雰囲気
内容: Deep, Melodic
使用: バー/カフェ応募

最低2本:
メイン + ターゲット別1本
```

### デモの送り方

**プロフェッショナルに:**

```
形式:

SoundCloud リンク（推奨）:
- ストリーミング可能
- トラックリスト記載
- すぐ聴ける

Google Drive（オプション）:
- MP3 (320kbps)
- ファイル名: DJ_GAKU_Techno_Mix_60min.mp3
- 共有設定: 誰でも閲覧可

USBメモリ（対面時）:
- MP3
- ファイル名明記
- ラベル貼る

NG:
- 大容量メール添付
- WeTransfer（期限切れる）
- WAV（重すぎる）
```

## オープンデッキ攻略

### 準備

**完璧な準備:**

```
1週間前:

□ セットリスト作成（30分）
  - 15-20曲
  - 順番決定
  - キー分析

□ Hot Cue 設定
  - 全曲イントロ/アウトロ
  - ドロップポイント

□ リハーサル（3回以上）
  - 完璧に通せるまで

3日前:

□ USBメモリ準備
  - Rekordbox でエクスポート
  - バックアップUSB も

□ 服装決定
  - ブランドイメージに合う
  - 動きやすい

前日:

□ 十分な睡眠
□ USBメモリ確認
□ 会場の場所・時間確認
□ イメージトレーニング

当日:

□ 早めに到着（30分前）
□ 受付で登録
□ 会場の雰囲気確認
□ 他DJのプレイ観察
□ 深呼吸、リラックス
```

### プレイ中

**全力で楽しむ:**

```
最初の1分:

強い曲で始める:
観客の注目を集める

アイコンタクト:
フロアを見る、笑顔

10-20分:

エネルギーカーブ:
徐々に盛り上げる

フロアリーディング:
反応を見ながら調整

ミスしても:
止まらない、続ける

最後の5分:

ピークを作る:
印象に残るフィニッシュ

次のDJへ:
スムーズに引き継ぐ

終了後:

挨拶:
プロモーター、他DJ に感謝

片付け:
ケーブル、機材を整理

交流:
名刺配布、SNSフォロー

印象:
プレイ + 人柄が大事
```

## リピートを増やす方法

### 初ギグ後

**次につなげる:**

```
当日:

□ 全力でプレイ
□ 時間厳守
□ トラブル対応も冷静に
□ プロモーターに感謝

翌日:

□ お礼メール送信

「昨日はありがとうございました。
素晴らしい経験をさせていただきました。
またの機会がございましたら、
ぜひよろしくお願いいたします。」

□ SNS で投稿
  「@[会場名] でプレイさせていただきました！
   素晴らしい夜をありがとうございました」
  → タグ付け

1週間後:

□ 録音があれば SoundCloud にアップ
  → タグ付けして共有

1ヶ月後:

□ 軽くフォローアップ
  「その後いかがでしょうか。
   また機会があれば嬉しいです」

3ヶ月後:

□ 新作デモ送付
  「新しいミックスを作りました。
   もしよろしければ聴いてください」

リピート率を上げる:
- プロフェッショナル
- 時間厳守
- トラブルなし
- 良い印象
```

### プロとしての振る舞い

**信頼を築く:**

```
時間厳守:

□ セットアップ: 30分前到着
□ プレイ開始: 時間通り
□ プレイ終了: 時間通り
□ 片付け: 速やかに

遅刻:
二度と呼ばれない

コミュニケーション:

□ メール: 24時間以内返信
□ 確認事項: 事前に質問
□ 変更: すぐ連絡

柔軟性:

□ リクエスト: 可能なら対応
□ 時間変更: 柔軟に
□ トラブル: 冷静に

態度:

□ 謙虚
□ プロフェッショナル
□ 協力的
□ ポジティブ

評判:
業界は狭い
良い評判が次の仕事を呼ぶ
```

## ブッキングエージェント

### エージェントとは

**仕事を取ってくれる人:**

```
役割:
- ギグ探し
- 交渉
- スケジュール管理
- 契約書作成

手数料:
ギャラの 10-20%

メリット:
- 営業不要
- プロのネットワーク
- 良い条件交渉
- 時間節約

デメリット:
- 手数料
- 初心者は契約困難

タイミング:
月2-4本以上ギグがあるなら検討
```

### エージェント探し

**信頼できる相手:**

```
探し方:

1. リサーチ
   - 同ジャンルDJ が所属
   - 実績ある
   - 評判良い

2. アプローチ
   - デモ、EPK 送付
   - 実績を示す
   - 熱意を伝える

3. 面談
   - 条件確認
   - 手数料
   - 契約内容

注意:
- 契約書を熟読
- 専属か非専属か
- 解約条件
- 手数料率

最初は:
自分で営業
→ 実績作ってからエージェント
```

## よくある質問

### Q1: 初ギグまで何ヶ月？

**A:** 3-6ヶ月が平均

```
タイムライン:

積極的に動けば:
- オープンデッキ: 1ヶ月
- バー/カフェ: 3ヶ月
- クラブ: 6ヶ月-1年

受動的なら:
数年かかる可能性

行動量:
結果に直結
```

### Q2: 断られたらどうする？

**A:** 気にせず次へ

```
現実:
断られるのが普通
10件送って1件返信あれば良い

対処:
1. 落ち込まない
2. フィードバックあれば活用
3. 次のターゲットへ
4. 改善して再挑戦

諦めない:
継続が成功の鍵
```

### Q3: 機材は何を持参？

**A:** 基本は USB のみ

```
通常:

会場の機材使用:
- CDJ + DJM（クラブ）
- コントローラー（バー）

持参するもの:
□ USBメモリ（2本、バックアップ）
□ ヘッドフォン
□ ケーブル（念のため）

DDJ-FLX4:
小さい会場なら持参可能
事前に確認
```
