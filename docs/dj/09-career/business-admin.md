# ビジネス管理

音楽だけでは食べていけない。確定申告、経費管理、保険など、DJ の地味だが重要なビジネス知識の完全ガイドです。

- 保険・年金

## なぜビジネス管理が必要か

**法律で決まっている:**

```
現実:

収入があれば:
確定申告が必要（条件あり）

申告しないと:
- 脱税
- 追徴課税
- 罰金
- 最悪、刑事罰

知識があれば:
- 合法的に節税
- トラブル回避
- 安心して活動

面倒だけど:
避けて通れない
```

## 確定申告の基礎

### 確定申告とは

**1年間の収入・支出を報告:**

```
確定申告:
前年1月1日〜12月31日の
収入と支出を税務署に報告

目的:
所得税を計算
納付 or 還付

期間:
毎年2月16日〜3月15日

方法:
- e-Tax（オンライン）推奨
- 税務署に直接
- 郵送

罰則:
申告しないと追徴課税
```

### 誰が申告必要？

**条件を確認:**

```
会社員で副業DJ:

副業収入 - 経費 > ¥200,000
→ 確定申告必要

例:
DJ収入 ¥300,000
経費 ¥80,000
所得 ¥220,000
→ 申告必要

例:
DJ収入 ¥250,000
経費 ¥80,000
所得 ¥170,000
→ 申告不要

専業DJ（フリーランス）:

所得 > ¥480,000
→ 確定申告必要

実質:
ほぼ全員必要

学生:

アルバイト + DJ:
合計所得 > ¥480,000
→ 申告必要

親の扶養:
年収 ¥103万円以下
→ 扶養控除受けられる
超えると親の税金↑

まとめ:
少しでも収入あれば
申告を検討
```

## 経費計上

### 経費とは

**事業のために使ったお金:**

```
経費にできる:
事業に必要な支出
→ 所得を減らせる
→ 税金↓

所得 = 収入 - 経費

例:
収入 ¥500,000
経費 ¥0
所得 ¥500,000
税金 約¥50,000（10%）

収入 ¥500,000
経費 ¥200,000
所得 ¥300,000
税金 約¥30,000（10%）

差額 ¥20,000 節税！

経費が多いほど:
税金が安くなる
```

### DJ の経費

**何が経費になる？**

```
✓ 間違いなく経費:

□ 機材購入
  DDJ-FLX4: ¥25,000
  ヘッドフォン: ¥15,000
  ケーブル: ¥3,000

□ 音楽購入
  Beatport: 月¥10,000
  iTunes: 月¥5,000

□ ソフトウェア
  Rekordbox: ¥0（無料）
  サブスクリプション: 月¥2,000

□ 通信費（按分）
  インターネット: 月¥5,000 × 30% = ¥1,500
  スマホ: 月¥8,000 × 30% = ¥2,400

□ 交通費
  ギグへの交通費全額

□ 会場費
  スタジオ練習: 1回¥2,000

□ 宣伝費
  名刺: ¥3,000
  フライヤー: ¥10,000

□ 勉強費
  DJ スクール: ¥50,000
  本: ¥3,000

△ グレーゾーン（按分必要）:

□ 家賃
  自宅を事務所/スタジオとして使用
  → 面積比で按分（10-30%）

□ 光熱費
  練習に使う分
  → 時間で按分（10-20%）

□ 食事
  打ち合わせ時のみ
  → 交際費として

□ 服
  ステージ衣装のみ
  → 私服は NG

□ イベント参加費
  リサーチ目的
  → 研修費として

✗ 経費にならない:

□ プライベート飲食
□ 普段着
□ 趣味の旅行
□ DJ と関係ない支出

ポイント:
「DJ 事業に必要」と説明できるか
```

### 按分（あんぶん）

**事業用とプライベートを分ける:**

```
按分とは:
事業用の割合だけ経費にする

例1: 自宅家賃

家賃 ¥100,000/月
部屋全体 40㎡
DJ スペース 8㎡

按分:
8㎡ ÷ 40㎡ = 20%

経費:
¥100,000 × 20% = ¥20,000/月
年間 ¥240,000

例2: スマホ

月額 ¥10,000
DJ 使用 30%（推定）

経費:
¥10,000 × 30% = ¥3,000/月
年間 ¥36,000

例3: インターネット

月額 ¥6,000
DJ 使用 50%

経費:
¥6,000 × 50% = ¥3,000/月
年間 ¥36,000

按分比率:
合理的に説明できればOK
税務調査で聞かれる可能性
```

## 領収書・レシート管理

### 保管義務

**7年間保管:**

```
法律:
領収書・レシートは7年間保管義務

理由:
税務調査で確認される

保管方法:

1. 紙で保管
   □ 月別に封筒に入れる
   □ ファイルに貼る
   □ スキャンして PDF + 原本保管

2. デジタル化（2022年〜 OK）
   □ スマホで撮影
   □ 会計ソフトにアップロード
   □ クラウド保存
   □ 原本は廃棄可能

推奨:
デジタル化
紛失リスク↓
```

### 領収書のもらい方

**正しい形式:**

```
必要事項:

□ 日付
□ 宛名（あなたの名前 or 屋号）
□ 金額
□ 但し書き（何を買ったか）
□ 発行者名・住所

レシート:
上記があれば OK

宛名なし:
¥30,000 未満なら OK

クレジットカード:
利用明細が領収書代わり

オンライン購入:
画面スクショ、メールを保存
```

## 開業届と青色申告

### 開業届

**個人事業主として届出:**

```
開業届:
「事業を始めました」という届出

提出先:
税務署

期限:
開業から1ヶ月以内

費用:
無料

メリット:
- 屋号が使える
- 事業用口座開設
- 青色申告できる
- 社会的信用

デメリット:
- 失業保険もらえない
- 扶養から外れる可能性

いつ出す？:

副業DJ:
年収 ¥200,000 超えたら検討

専業DJ:
即提出

出さなくても:
白色申告は可能
```

### 青色申告 vs 白色申告

**どちらで申告する？**

```
白色申告:

特徴:
- 簡単
- 帳簿が簡易
- 特典なし

控除:
¥0

青色申告:

特徴:
- 複雑
- 複式簿記が必要
- 特典あり

控除:
¥100,000（簡易簿記）
¥650,000（複式簿記 + e-Tax）

例:

所得 ¥1,000,000

白色:
課税所得 ¥1,000,000
税金 約¥100,000

青色（¥650,000控除）:
課税所得 ¥350,000
税金 約¥35,000

差額 ¥65,000 節税！

青色申告のメリット:

□ 最大¥650,000控除
□ 赤字を3年間繰越
□ 家族への給料を経費に
□ 30万円未満の機材即経費

デメリット:

□ 帳簿が複雑
□ 事前申請必要

結論:
収入 > ¥500,000 なら青色推奨
```

### 青色申告承認申請

**事前申請が必要:**

```
提出先:
税務署

期限:
開業から2ヶ月以内
or
青色申告したい年の3月15日まで

費用:
無料

手続き:
1. 開業届提出
2. 青色申告承認申請書提出

両方同時に出せる
```

## 節税戦略

### 合法的な節税

**知識で差がつく:**

```
1. 経費を漏らさない
   → 全ての経費を計上

2. 青色申告
   → ¥650,000控除

3. iDeCo
   → 全額所得控除
   月¥23,000 → 年¥276,000控除

4. ふるさと納税
   → 実質¥2,000で返礼品

5. 小規模企業共済
   → 退職金積立 + 全額控除

6. 医療費控除
   → 年¥100,000超なら控除

7. 寄付金控除
   → 寄付で控除

8. 家族への給料（青色専従者給与）
   → 事業手伝ってもらう

年間数十万円:
節税可能
```

## よくある質問

### Q1: バレたらどうなる？

**A:** 追徴課税、罰金

```
申告しないと:

1. 無申告加算税
   15-20%上乗せ

2. 延滞税
   年7.3-14.6%

3. 重加算税（悪質な場合）
   35-40%

例:
本来の税金 ¥100,000
無申告発覚
→ ¥120,000 + 延滞税

正直に:
申告するのが得
```

### Q2: 赤字でも申告必要？

**A:** した方が良い

```
赤字の場合:

税金:
¥0

申告:
任意だが推奨

理由:
- 青色申告なら赤字繰越
- 証明になる
- 還付がある場合も

赤字でも:
申告推奨
```

### Q3: 税理士は必要？

**A:** 収入次第

```
不要:
年収 < ¥500万円
→ 会計ソフトで十分

検討:
年収 > ¥500万円
→ 複雑になる

費用:
年間 ¥100,000-300,000

メリット:
- 時間節約
- 正確
- 節税アドバイス
- 税務調査対応

最初は:
自分で挑戦
→ 必要になったら依頼
```
