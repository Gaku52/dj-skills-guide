## なぜ著作権知識が必要か

**知らないとトラブルに:**

```
起こりうること:

□ 訴訟
□ 損害賠償請求
□ アカウント停止
□ キャリア終了

実例:
「SoundCloud にミックスアップ → 削除」
「YouTube で著作権侵害 → アカウントBAN」
「無許可配信 → 訴えられた」

知識があれば:
合法的に活動できる
安心してキャリア構築
```

## 著作権の基礎

### 著作権とは

**創作物を守る権利:**

```
著作権:
音楽、絵画、文章など
創作物を作った人の権利

音楽の場合:

1. 著作権（楽曲）
   作詞家、作曲家の権利
   例: 歌詞、メロディ

2. 著作隣接権（音源）
   実演家、レコード会社の権利
   例: ボーカル、演奏、録音

2つの権利が存在
両方をクリアする必要
```

### 保護期間

**いつまで保護される？**

```
著作権:
作者の死後70年
（2018年12月30日以降）

著作隣接権:
発行後70年

パブリックドメイン:
保護期間が切れた作品
→ 自由に使える

例:
ベートーヴェン、モーツァルト
→ 楽曲は自由

でも:
現代の録音（演奏）
→ まだ保護されている

注意必要
```

## DJプレイと著作権

### クラブ・バーでのプレイ

**会場が使用料を払っている:**

```
通常:

クラブ/バー/イベント会場:
JASRAC、NexTone と包括契約
→ 使用料を年間/月間で支払い

DJ は:
自由にプレイ OK

ただし:

小さいバー、カフェ:
契約していない場合も

確認:
「著作権の手続きは？」
→ プロモーター/オーナーに

DJ の責任:
通常なし
（会場が契約してれば）

でも:
録音・配信は別
```

### プライベートパーティ

**注意が必要:**

```
公の場:
著作権使用料必要

例:
- 企業パーティ
- ウェディング（披露宴）
- 大規模な誕生日パーティ

手続き:
主催者が JASRAC に申請
→ 使用料支払い

プライベート（真の意味で）:
- 自宅パーティ（友人のみ）
- 家族の集まり

→ 使用料不要

DJ として:
「著作権の手続きは？」
→ 事前確認必須
```

## JASRAC と NexTone

### 2つの団体

**音楽著作権を管理:**

```
JASRAC（ジャスラック）:
日本音楽著作権協会
設立: 1939年
管理楽曲: 約200万曲以上
シェア: 約95%

NexTone（ネクストーン）:
設立: 2016年
管理楽曲: 約10万曲
シェア: 約5%

役割:
- 著作権者の代理
- 使用料徴収
- 著作権者に分配

DJ/会場:
どちらかと契約
→ 両方の楽曲使用可能
```

### 使用料の仕組み

**会場が支払う:**

```
クラブ・ライブハウス:

包括契約:
年間 or 月間定額

金額:
会場の広さ、収容人数による

例:
小規模クラブ（50人）:
月額 ¥10,000-30,000

中規模クラブ（200人）:
月額 ¥50,000-100,000

大規模クラブ（500人+）:
月額 ¥100,000-300,000

イベント単位:

都度申請:
1回のイベントごと

金額:
入場料 × 観客数 × 数%

例:
入場料 ¥3,000 × 100人 × 5%
= ¥15,000

DJ:
通常支払い不要
会場/主催者が負担
```

## 録音・配信の著作権

### 録音の注意点

**個人使用 vs 公開:**

```
個人使用:
自分で聴くだけ
→ OK、問題なし

公開:
SoundCloud、YouTube 等
→ 著作権処理必要

問題:
DJ ミックスは他人の楽曲使用
→ 許可が必要

現実:
完全に合法的な配信は困難
```

### SoundCloud

**グレーゾーン:**

```
SoundCloud のスタンス:

公式:
著作権侵害は禁止

現実:
DJ ミックス多数存在

Content ID:
著作権者が申し立て可能
→ 削除される場合も

リスク:

低リスク:
- マイナー楽曲
- アンダーグラウンド
- Remix、Edit

高リスク:
- メジャーレーベル
- トップ40
- 有名曲

削除されたら:
- アカウント警告
- 繰り返すと BAN

対策:

1. 説明欄に記載
   「プロモーション目的」
   「著作権者に権利帰属」

2. トラックリスト明記
   全曲リスト記載

3. 商用利用しない
   無料公開のみ

4. 申し立てあれば即削除
   争わない

完全合法:
無理
グレーゾーンで活動
```

### YouTube

**Content ID が厳しい:**

```
Content ID:
自動で著作権検出

結果:

1. ブロック
   動画が見れない

2. 収益化無効
   広告収入なし

3. 収益が著作権者に
   著作権者が広告収入獲得

4. ミュート
   音声が消される

DJ ミックス:
ほぼ確実に検出される

対策:

1. 著作権フリー音楽のみ
   - Epidemic Sound
   - Artlist
   - 自作曲

2. ライセンス取得
   難しい、コスト高

3. 収益化しない
   Content ID は発動するが
   ブロックされにくい

4. Twitch、Mixcloud 検討
   DJ 向けプラットフォーム

完全合法:
著作権フリー音楽のみ
現実的には難しい
```

### Mixcloud

**DJ に最適:**

```
Mixcloud:
DJ ミックス専門プラットフォーム

特徴:

著作権処理済み:
Mixcloud が使用料支払い
→ 合法的にアップロード可能

制約:

- ダウンロード不可
- スキップ制限
- 同じアーティスト連続再生制限

DJ として:

利点:
- 合法
- 著作権の心配なし
- トラックリスト自動認識

欠点:
- SoundCloud より知名度低い
- 無料版は制限あり
- Pro版は有料（月額$15）

推奨:
メインは Mixcloud
サブで SoundCloud

合法的に活動したいなら:
Mixcloud 一択
```

## ライブ配信

### Twitch

**ゲーム配信だけじゃない:**

```
Twitch:
DJ ライブ配信可能

著作権:

Content ID:
音楽検出あり

対応:

1. Soundtrack by Twitch
   著作権フリー音楽
   → 安全

2. 商用楽曲使用
   リスクあり
   ミュート、BAN の可能性

3. アーカイブ
   ミュートされる場合多い

推奨:

ライブ:
リスク低め

アーカイブ:
削除推奨
```

### Instagram Live / YouTube Live

**厳しい:**

```
Instagram Live:
音楽検出あり
配信停止のリスク

YouTube Live:
Content ID 発動
ブロック、ミュート

対策:
著作権フリー音楽
または
短時間（30分以内）
```

## トラブル事例

### よくある問題:**

```
事例1: SoundCloud アカウント削除

原因:
メジャー楽曲ミックス繰り返しアップロード

結果:
アカウント永久BAN

教訓:
マイナー楽曲、バランス

─────────────────

事例2: YouTube 収益化停止

原因:
著作権侵害の申し立て

結果:
チャンネル収益化無効

教訓:
YouTube は避ける or 著作権フリーのみ

─────────────────

事例3: 損害賠償請求

原因:
無許可で商用配信
CD販売

結果:
訴訟、数百万円の賠償

教訓:
商用利用は絶対NG

─────────────────

事例4: イベント中止

原因:
会場が JASRAC 未契約
発覚して中止

結果:
損害、評判悪化

教訓:
事前確認必須
```

## 合法的な活動方法

### ベストプラクティス

**安全に活動:**

```
DJ プレイ:

□ 会場の著作権契約確認
□ プライベートパーティは主催者に確認
□ 自分で確認できないなら契約書に記載

録音:

□ 個人使用のみ: OK
□ SoundCloud: グレーゾーン、注意して
□ Mixcloud: 合法、推奨
□ YouTube: 避ける
□ ダウンロード販売: 許可必須

ライブ配信:

□ Twitch: 注意して
□ Instagram/YouTube: 著作権フリーのみ推奨
□ Mixcloud Live: 合法

プロモーション:

□ 30秒以内のクリップ: 比較的安全
□ フェアユース主張（米国）
□ トラックリスト明記
□ 商用利用しない

自作曲・Remix:

□ オリジナル: 自由に使える
□ Remix: 元曲の許可必要
□ Bootleg: グレーゾーン、注意

完璧に合法:
難しいが、リスク最小化は可能
```

## よくある質問

### Q1: 全ての DJ が違法？

**A:** グレーゾーンが多い

```
クラブプレイ:
合法（会場が契約してれば）

録音アップロード:
グレーゾーン
Mixcloud なら合法

現実:
多くのDJ がグレーゾーンで活動

リスク:
削除、BAN
訴訟は稀

推奨:
リスクを理解して
最小化する努力
```

### Q2: 許可を取れば OK？

**A:** 理論上は可能、現実は困難

```
必要な許可:

1. 楽曲著作権（JASRAC/NexTone）
2. 原盤権（レコード会社）
3. 実演家の権利

手続き:
複雑、時間がかかる
コスト高い

現実:
個人DJ には非現実的

例外:
メジャーレーベルと契約
→ サポートあり
```

### Q3: 著作権フリー音楽だけで DJ できる？

**A:** 可能だが限定的

```
著作権フリー音楽:

プラットフォーム:
- Epidemic Sound
- Artlist
- Bensound

特徴:
- 安全
- 合法

でも:
- 選択肢少ない
- 有名曲なし
- ジャンル限定

用途:
- YouTube 投稿
- 企業イベント
- 安全第一の場合

クラブDJ:
現実的ではない
```

## まとめ

### 著作権 5つのポイント

```
1. 知識を持つ
   └─ 無知は言い訳にならない

2. 会場の契約確認
   └─ プレイ前に必ず

3. Mixcloud 活用
   └─ 合法的な配信

4. リスクを理解
   └─ グレーゾーンを認識

5. 商用利用は慎重に
   └─ 許可なしは絶対NG
```

### 今日からできること

```
□ Mixcloud アカウント作成
□ 次のギグで会場に著作権確認
□ SoundCloud 説明欄に著作権表記追加
□ YouTube アップロードは控える
□ JASRAC、NexTone のサイトを読む
```

### 最後に

```
著作権:
複雑、完璧な理解は困難

でも:
基本を知る
リスクを最小化

合法的に:
できる限り努力

音楽への敬意:
著作権を守ることも含む
```
