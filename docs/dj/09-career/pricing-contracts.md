お金の話を避けてはいけない。適正な料金設定と、自分を守る契約書の知識で、プロとして活動する完全ガイドです。

## なぜ料金設定が重要か

**プロとしての価値:**

```
適正な料金:
- 自分の価値を示す
- 持続可能な活動
- プロ意識

安すぎる:
- 疲弊する
- 軽く見られる
- 業界全体の相場を下げる

高すぎる:
- ギグが取れない
- 期待値が上がりすぎる

バランスが重要
```

## 料金設定の方法

### 考慮すべき要素

**単純に時間だけではない:**

```
1. 演奏時間

基本:
1時間 ¥10,000-30,000（中級者）

でも:
30分 = 半額 ではない
→ 準備時間は同じ

2. 準備時間

含まれる:
- リハーサル
- プレイリスト作成
- Hot Cue 設定
- 移動時間

例:
2時間ギグ = 実質6-8時間

3. 経費

計算:
□ 交通費
□ 機材（減価償却）
□ 音楽購入費
□ 通信費（SoundCloud 等）

4. 経験・スキル

スキル高い:
料金も高く設定できる

5. 需要

人気DJ:
高く設定できる

6. 会場の予算

大きい会場/イベント:
予算も大きい

7. 集客力

あなたのファン:
何人来るか
→ 会場の収入に影響

計算式:

基本料金:
時給 ¥10,000（中級者）
× 演奏時間

+ 準備時間:
¥5,000/時間 × 準備時間

+ 経費:
交通費、機材費等

+ プレミアム:
スキル、集客力

= 合計金額
```

## 値上げのタイミング

### いつ上げるか

**成長に合わせて:**

```
値上げのサイン:

□ ギグ依頼が断りきれない
  → 需要 > 供給

□ スキルが大幅に向上
  → 価値↑

□ 実績が増えた
  → 信頼性↑

□ フォロワーが増えた
  → 集客力↑

□ 現在の料金で満足できない
  → モチベーション↓

タイミング:

6ヶ月-1年ごと:
10-20%値上げ

例:
初心者: ¥10,000
6ヶ月後: ¥12,000
1年後: ¥15,000
1.5年後: ¥18,000
2年後: ¥20,000

値上げ方法:

既存クライアント:
「来月から料金改定します」
→ 1ヶ月前に通知

新規クライアント:
新料金を提示

レギュラー:
据え置きも検討
→ 感謝の気持ち
```

## 交渉術

### 準備

**交渉前に:**

```
リサーチ:

□ 会場の規模
□ 通常のギャラ相場
□ イベントの予算感
□ 他DJのギャラ（推測）

最低ライン:

これ以下では受けない:
¥XX,000

理由:
- 経費
- 時間
- 自分の価値

希望額:

理想的な金額:
¥YY,000

妥協点:

落とし所:
¥ZZ,000

例:
最低: ¥15,000
妥協: ¥20,000
希望: ¥25,000
```

### 交渉の流れ

**ステップバイステップ:**

```
Step 1: 相手の提示を聞く

プロモーター:
「ギャラはいくら？」

あなた:
「まず、イベントの詳細を
教えていただけますか？」

情報収集:
- 日時
- 時間
- 会場
- 観客数
- 他の出演DJ

Step 2: 条件を確認

□ 演奏時間
□ サウンドチェック時間
□ 機材（会場 or 持参）
□ 交通費
□ 食事・ドリンク
□ 宿泊（遠方の場合）

Step 3: 提示

あなた:
「通常、この条件でしたら
¥XX,000 でお受けしています」

→ 希望額の10-20%高めから

Step 4: 相手の反応

高すぎる:
「予算が...」

→ 交渉開始

OK:
「わかりました」

→ 即決定

Step 5: 妥協案

あなた:
「では、¥YY,000 ではいかがでしょうか」

または

「時間を短くして¥ZZ,000」
「次回は¥XX,000 で」

Win-Win:
お互いが納得できる点

Step 6: 確定

口頭合意後:
「では、契約書を送ります」

→ 必ず書面で
```

### 交渉のコツ

**プロとして:**

```
DO:

✓ 自信を持つ
  → 自分の価値を知る

✓ 柔軟性
  → 全てにNOではない

✓ Win-Win を探す
  → お互いにメリット

✓ 礼儀正しく
  → プロフェッショナル

✓ 代替案を提示
  → 時間短縮、次回割引等

DON'T:

✗ 最初から最低額
  → 交渉の余地なし

✗ 感情的
  → 冷静に

✗ 悪態をつく
  → 評判が落ちる

✗ すぐに妥協
  → 安く見られる

✗ 曖昧にする
  → 後でトラブル
```

## 契約書の基礎

### なぜ契約書が必要か

**自分を守るため:**

```
トラブル例:

口約束だけ:
「¥30,000 って言ったのに
当日¥15,000 しかもらえなかった」

「急にキャンセルされた。
準備したのに」

「音源を勝手に使われた」

契約書があれば:
法的に守られる
トラブル回避
お互いに安心
```

### 契約書に含めるべき内容

**最低限の項目:**

```
■ 基本情報

□ 契約日
□ 双方の名前・連絡先
□ イベント名

■ イベント詳細

□ 日時
  例: 2025年6月15日（土）
      22:00-24:00（2時間）

□ 会場
  例: Club XYZ
      東京都渋谷区...

□ 演奏時間
  例: 2時間

□ サウンドチェック
  例: 21:30-21:50

■ ギャラ

□ 金額
  例: ¥50,000（税込）

□ 支払い方法
  例: 当日現金払い
      or 3日以内振込

□ 支払いタイミング
  例: 演奏終了後即座
      or 翌月末

□ 交通費
  例: 実費別途支給
      or ギャラに含む

■ 機材

□ 会場提供機材
  例: CDJ-3000 × 2, DJM-900NXS2

□ DJ 持参機材
  例: USB × 2, ヘッドフォン

□ トラブル時の責任
  例: 会場機材故障は会場負担

■ キャンセルポリシー

□ 主催者都合
  7日前まで: 無料
  3-6日前: ギャラの50%支払い
  2日前以降: ギャラの100%支払い

□ DJ 都合
  7日前まで: 無料
  3-6日前: 違約金¥XX,000
  2日前以降: 違約金¥YY,000

□ 不可抗力（天災等）
  双方協議の上決定

■ 録音・録画

□ イベント録音の可否
  例: 可 / 不可

□ 録音物の使用権
  例: DJ が SoundCloud 投稿可
      主催者がプロモーション使用可

□ 商用利用
  例: 要別途協議

■ その他

□ 肖像権
  例: イベント写真の使用許可

□ 損害賠償
  例: 故意・重過失の場合のみ

□ 準拠法
  例: 日本国法律に準拠

□ 管轄裁判所
  例: 東京地方裁判所

■ 署名

主催者:
署名: ________________
日付: ________________

DJ:
署名: ________________
日付: ________________
```

## トラブル回避

### よくあるトラブル

**事前に防ぐ:**

```
トラブル1: ギャラの未払い

原因:
口約束のみ

予防:
- 契約書必須
- 前払い or 当日現金
- 領収書発行

トラブル2: 急なキャンセル

原因:
キャンセルポリシーなし

予防:
- 契約書に明記
- 前金50%

トラブル3: 時間超過要求

原因:
「もう1時間やって」

予防:
- 契約書に時間明記
- 延長料金事前決定

トラブル4: 機材トラブル

原因:
「CDJ 壊れた、弁償して」

予防:
- 責任範囲を明記
- 保険加入（プロは検討）

トラブル5: 録音物の無断使用

原因:
権利が曖昧

予防:
- 録音の可否を明記
- 使用範囲を規定
```

## 支払い条件

### タイミング

**いつもらうか:**

```
パターン1: 当日現金（推奨）

メリット:
- 確実
- すぐもらえる

デメリット:
- 大金を持ち歩く

おすすめ:
小〜中規模ギグ

パターン2: 前払い50% + 当日50%

メリット:
- キャンセルリスク↓
- 双方安心

デメリット:
- 手間

おすすめ:
大きいギグ、不安な場合

パターン3: 後日振込

メリット:
- 楽

デメリット:
- 未払いリスク
- 遅延の可能性

おすすめ:
信頼できる相手のみ

パターン4: 全額前払い

メリット:
- 確実

デメリット:
- 相手が不安

おすすめ:
遠方、初取引

領収書:
必ず発行・保管
確定申告に必要
```

## よくある質問

### Q1: 最初から高く設定すべき？

**A:** 経験とのバランス

```
初心者:
低めから（¥0-10,000）
→ 経験優先

でも:
安すぎない
→ 自分の価値を下げる

目安:
バイト時給 × 2-3倍
```

### Q2: 友人のイベントは無料？

**A:** ケースバイケース

```
無料でOK:
- 小規模な友人パーティ
- お互い様の関係
- 経験のため

有料にすべき:
- 商業イベント
- 主催者が儲かる
- 自分の時間・労力が大きい

友人価格:
通常の50-70%
お互いWin-Win
```

### Q3: 値切られたらどうする？

**A:** 最低ラインを守る

```
相手:
「¥30,000 は高い、¥15,000 で」

対応:

1. 価値を説明
   「準備時間、経験を考えると...」

2. 代替案
   「では時間を1時間に短縮で¥20,000」
   「次回は¥25,000 で」

3. 断る勇気
   最低ライン以下なら
   「申し訳ありませんが...」

安売りしない:
自分の価値を守る
```
