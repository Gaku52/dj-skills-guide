## EPK（Electronic Press Kit）作成

### EPKとは

**プロモーター向けの資料:**

```
EPK = あなたの履歴書

含める内容:
1. プロフィール
2. DJ写真
3. バイオ
4. デモミックス
5. SNS リンク
6. 過去の実績（あれば）
7. 連絡先

形式:
- PDF（1-2ページ）
- ウェブページ
- Google Drive フォルダ

目的:
プロモーターが判断しやすく
```

### EPK作成ステップ

**Canvaで簡単作成:**

```
Step 1: Canva で「EPK」検索
テンプレート選択
→ DJ/音楽系

Step 2: カスタマイズ

ページ1: 基本情報
┌─────────────────────────┐
│  [DJ写真]               │
│                         │
│  DJ GAKU                │
│  Techno / House DJ      │
│                         │
│  Tokyo, Japan           │
│                         │
│  Instagram: @djgaku     │
│  SoundCloud: djgaku     │
│  Email: dj@example.com  │
└─────────────────────────┘

ページ2: バイオ + デモ
┌─────────────────────────┐
│  Biography              │
│  ───────────────────    │
│  [100-200文字の経歴]    │
│                         │
│  Demo Mixes             │
│  ───────────────────    │
│  🎵 Summer Mix 2025     │
│     soundcloud.com/... │
│                         │
│  🎵 Deep House Mix      │
│     soundcloud.com/... │
└─────────────────────────┘

Step 3: PDF でダウンロード

Step 4: Google Drive にアップロード
共有リンク作成
→ 誰でも閲覧可能に設定

所要時間: 1時間
```

## デモミックスの作り方

### デモミックスの重要性

**あなたのスキルを証明:**

```
プロモーターは:
デモを聴いて判断

良いデモ:
ギグにつながる

悪いデモ:
機会を失う

デモ = 最重要ツール
```

### デモミックスの基準

**プロモーターが求めること:**

```
1. 長さ
   推奨: 30-60分
   理由: 聴きやすい長さ

2. クオリティ
   - ミスが少ない
   - スムーズなミックス
   - 音質が良い

3. 選曲
   - 一貫性がある
   - エネルギーカーブ
   - 個性が出ている

4. ターゲットに合う
   - クラブ向け → エネルギー高め
   - ラウンジ向け → 落ち着き
   - フェス向け → ピーク多め

完璧である必要はない:
でも最善を尽くす
```

### Rekordbox でデモ録音

**DDJ-FLX4 で完璧な録音:**

```
準備（前日）:

1. プレイリスト作成
   □ 30-60分分の曲（15-30曲）
   □ 順番を決める
   □ Hot Cue 設定
   □ キー分析完了

2. リハーサル
   □ 1回通しで練習
   □ ミスした箇所を確認
   □ 改善

録音当日:

1. Rekordbox 設定
   [Preferences] > [Controller] > [Rec]

   保存先: /DJ/Demo_Mixes/
   フォーマット: WAV (44.1kHz, 16-bit)

2. 環境準備
   □ 静かな時間帯
   □ 集中できる環境
   □ 電話OFF
   □ 通知OFF

3. 録音開始
   DDJ-FLX4:
   [REC] ボタン 押す

   または

   Rekordbox:
   R キーを押す

4. プレイ
   - リラックス
   - 楽しむ
   - ミスしても続ける
   - 完璧を求めすぎない

5. 録音終了
   [REC] ボタン もう一度押す

録音後:

1. 聴き直し
   □ 大きなミスはないか
   □ 音質は OK か
   □ 満足できるか

2. 必要なら再録音
   納得いくまで（最大3-5回）

3. MP3 に変換
   Rekordbox で:
   WAV → MP3 (320kbps)

4. タイトルをつける
   例: DJ_GAKU_Techno_Mix_2025.mp3
```

## SoundCloud 戦略

### プロフィール最適化

**第一印象を整える:**

```
プロフィール写真:
- 高解像度
- 顔がはっきり（or ロゴ）
- プロフェッショナル

カバー画像:
- 横長（2480 x 520px）
- ブランドカラー使用
- DJ名表示

バイオ:
- 100文字程度
- コンセプト明記
- SNS リンク
- 連絡先

例:
「Tokyo を拠点とする Techno DJ。
深夜の旅に誘う、ダークでメロディアスなセット。

📧 Booking: dj@example.com
📸 Instagram: @djgaku」
```

### アップロード戦略

**定期的に質の高いコンテンツ:**

```
頻度:
月2-4本
→ 多すぎず、少なすぎず

ミックスの種類:

1. ジャンル別ミックス
   - Techno Mix
   - House Mix
   - Deep House Mix

2. 季節/イベント別
   - Summer Mix 2025
   - Halloween Mix
   - New Year Mix

3. テーマ別
   - Peak Time Mix
   - Warm Up Mix
   - After Hours Mix

タイトル付け:

良い例:
「DJ GAKU - Melodic Techno Mix | June 2025」
→ 誰が、何を、いつ が明確

悪い例:
「mix1」
→ 情報なし

説明文:

テンプレート:
「[ジャンル] mix recorded live on DDJ-FLX4.

Tracklist:
1. Artist - Track [Label]
2. Artist - Track [Label]
...

Follow me:
Instagram: @djgaku
Booking: dj@example.com」

タグ:
最大15個
関連ワード:
- ジャンル名
- アーティスト名
- 地域名

例:
#techno #melodictechno #housemusic
#tokyodj #djmix #technomusic
```

### 再生数を増やす

**実践的テクニック:**

```
1. シェア
   録音直後:
   □ Instagram ストーリーズ
   □ Facebook
   □ Twitter
   □ DJ コミュニティ

2. プレイリスト参加
   SoundCloud グループ:
   - Techno DJs
   - House Music
   - Underground DJs

   リポスト/いいね で交流

3. Repost 交換
   他のDJと:
   「お互いリポストしませんか？」
   → Win-Win

4. エンゲージメント
   コメントには返信
   他のDJのミックスにコメント
   → コミュニティ参加

5. SEO
   タイトル、説明に:
   検索されるキーワード含める

継続が鍵:
数ヶ月で効果
```

## よくある質問

### Q1: どれくらいで結果が出る？

**A:** 最低3-6ヶ月

```
現実:
すぐには結果出ない

初ギグまで:
- 早い人: 3ヶ月
- 平均: 6ヶ月
- 遅い人: 1年以上

焦らず:
継続が鍵
```

### Q2: デモは何本必要？

**A:** 最低2-3本

```
種類:

1. メインジャンル（60分）
   - あなたの真骨頂
   - 最高の出来

2. ピークタイム（30分）
   - エネルギー高め
   - クラブ向け

3. ウォームアップ（30分、オプション）
   - 落ち着き
   - バー/ラウンジ向け

使い分け:
ターゲットに応じて
```

### Q3: 無料で出演すべき？

**A:** 初期は経験優先

```
初ギグ:
¥0 でも OK
→ 経験が財産

2-3回目以降:
少額でも交渉
→ 価値を示す

10回以上:
しっかりギャラ交渉
→ プロとして

無料 = 価値なし
と思われないように
```
