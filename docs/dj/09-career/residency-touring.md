## レジデンシーとは

### 定義

**定期的な出演契約:**

```
レジデンシー:
特定の会場で定期的に DJ プレイ

頻度:
- 毎週
- 隔週
- 毎月

期間:
数ヶ月〜数年

例:
「毎週金曜、Club XYZ でプレイ」
「毎月第1土曜、Bar ABC でプレイ」

特徴:
安定した仕事
会場との長期関係
```

---

## レジデンシーのメリット

### DJ側のメリット

**安定と成長:**

```
1. 安定収入
   毎月の収入が予測できる

   例:
   月4回 × ¥30,000 = ¥120,000/月

2. スキル向上
   定期的にプレイ
   → 急速に成長

3. ファン獲得
   毎週来る客
   → あなたのファンになる

4. 実験の場
   新しい曲、技術を試せる
   → リスク低い

5. 知名度向上
   「〇〇といえば△△」
   → ブランディング

6. 交渉力
   信頼関係
   → ギャラ交渉しやすい

7. 他ギグにつながる
   レジデンシーが実績
   → 他会場からオファー

8. ネットワーク
   ゲストDJ と交流
   → 人脈拡大
```

### 会場側のメリット

**Win-Win:**

```
1. 安定したクオリティ
   毎回同じDJ
   → 質が保証される

2. 集客
   DJ のファンが来る
   → 売上↑

3. コスト削減
   毎回探す手間なし

4. ブランディング
   「〇〇の日は△△DJ」
   → 会場の特色

お互いにメリット:
長期関係が築ける
```

## レジデンシー獲得方法

### ステップ1: ターゲット選定

**どの会場を狙う？**

```
条件:

□ あなたのジャンルに合う
□ 通える距離
□ 雰囲気が好き
□ 客層が合う
□ 定期イベントがある/ありそう

リサーチ:

□ 実際に通う（3回以上）
□ 現在レジデントDJ がいるか
□ 客の入り具合
□ プロモーターの人柄

候補:
3-5会場リストアップ
```

### ステップ2: 関係構築

**すぐには獲得できない:**

```
Month 1-2: 常連になる

□ 週1回通う
□ 音楽を楽しむ
□ 軽く挨拶
□ SNS でフォロー

Month 3-4: 交流深める

□ プロモーターと会話
□ 他の常連と仲良く
□ イベント手伝い申し出
□ フィードバック提供

Month 5-6: 信頼構築

□ オープンデッキ参加
□ 良いプレイを見せる
□ 継続的に通う
□ コミュニティの一員に

焦らない:
時間をかけて信頼を
```

### ステップ3: 提案

**タイミングが重要:**

```
良いタイミング:

□ レジデントDJ が辞めた
□ 新しい曜日/イベント開始
□ あなたのプレイが好評
□ プロモーターから相談された

アプローチ:

「もし定期的にプレイする機会があれば、
ぜひ検討していただけませんか。

毎週/隔週/毎月で、
[曜日] に [時間帯] のプレイを
させていただけたら嬉しいです。

デモミックスもありますので、
ぜひ聴いていただけないでしょうか。」

謙虚に、でも自信を持って

提案内容:

□ 頻度（週1/隔週/月1）
□ 曜日
□ 時間帯
□ ギャラ
□ 開始日

柔軟性:
相手の要望も聞く
```

### ステップ4: 契約

**条件を明確に:**

```
契約内容:

□ 頻度（毎週金曜等）
□ 演奏時間（22:00-24:00等）
□ ギャラ（¥XX,000/回）
□ 期間（3ヶ月、6ヶ月、1年等）
□ 更新条件
□ キャンセルポリシー
□ 独占条項（他会場 OK/NG）

書面で:
口約束はトラブルの元

試用期間:
最初3ヶ月は試用
→ お互い様子見
→ 問題なければ更新
```

## レジデンシー成功の秘訣

### 継続のコツ

**長く続けるために:**

```
1. 一貫性
   毎回同じクオリティ
   → 信頼される

2. 柔軟性
   フロアの反応に合わせる
   → マンネリ化しない

3. 新曲導入
   毎週新しい曲
   → 常連が飽きない

4. コミュニケーション
   客と交流
   → ファンが増える

5. プロ意識
   遅刻しない
   準備万端
   → プロモーターが安心

6. 進化
   技術向上
   選曲進化
   → 成長を見せる

長期レジデンシー:
これらが鍵
```

## ツアーとは

### 定義

**複数都市での連続公演:**

```
ツアー:
短期間に複数の会場でプレイ

例:
Week 1: 東京
Week 2: 大阪
Week 3: 名古屋
Week 4: 福岡

または:

Fri: 東京
Sat: 大阪
Sun: 名古屋

規模:
- 国内ツアー
- アジアツアー
- 世界ツアー

初心者:
国内ツアーから
```

## ツアー計画

### ステップ1: 目的設定

**なぜツアーをする？**

```
目的:

□ 知名度拡大
□ 新しい市場開拓
□ ファン獲得
□ 収入増加
□ 経験値稼ぎ
□ ネットワーク拡大

明確な目的:
計画がブレない
```

### ステップ2: ルート設計

**効率的に:**

```
考慮事項:

□ 地理的な近さ
□ 移動時間
□ 移動コスト
□ ギャラ
□ 会場の質

例: 国内ツアー（1週間）

Day 1 (金): 東京 → 名古屋（新幹線2時間）
Day 2 (土): 名古屋 → 大阪（新幹線1時間）
Day 3 (日): 大阪 → 神戸（電車30分）
Day 4 (月): 休息
Day 5 (火): 神戸 → 京都（電車1時間）
Day 6 (水): 京都 → 東京（新幹線2.5時間）

効率的:
無駄な移動を減らす
```

### ステップ3: ブッキング

**会場を確保:**

```
方法:

1. 自分で営業
   各都市のクラブにメール
   EPK、デモ送付

2. ブッキングエージェント
   複数都市を一括で
   手数料10-20%

3. ローカルプロモーターと提携
   その都市の専門家に依頼

4. 既存のつながり
   他都市のDJ仲間に紹介依頼

タイムライン:

3-6ヶ月前:
ブッキング開始

2ヶ月前:
確定

1ヶ月前:
プロモーション開始
```

### ステップ4: 予算計画

**収支を計算:**

```
収入:

ギャラ:
東京: ¥50,000
名古屋: ¥30,000
大阪: ¥40,000
神戸: ¥25,000
京都: ¥30,000

合計: ¥175,000

支出:

交通費:
新幹線: ¥40,000
電車: ¥5,000

宿泊費:
4泊 × ¥8,000 = ¥32,000

食費:
6日 × ¥3,000 = ¥18,000

その他:
¥10,000

合計: ¥105,000

利益:
¥175,000 - ¥105,000 = ¥70,000

最初は:
利益少ない
経験が目的
```

**DDJ-FLX4 の強み:**

```
DDJ-FLX4:

重さ: 2.1kg
サイズ: コンパクト

持ち運び:
✓ スーツケースに入る
✓ 機内持ち込み可能
✓ 電車移動も楽

他の機材:

必須:
□ DDJ-FLX4
□ USBケーブル
□ 電源ケーブル
□ USBメモリ × 2（バックアップ）
□ ヘッドフォン

- 精密機器シール

→ 持ち運び不要

### 宿泊

**コスト削減:**

```
選択肢:

1. ホテル
   快適、高い
   1泊 ¥8,000-15,000

2. ビジネスホテル
   普通、手頃
   1泊 ¥5,000-8,000

3. カプセルホテル
   狭い、安い
   1泊 ¥3,000-5,000

4. ゲストハウス/ホステル
   共同、安い
   1泊 ¥2,000-4,000

5. DJ 仲間の家
   無料、交流
   お礼必須

6. 会場が用意
   交渉次第
   ギャラ減額と引き換え

推奨:
ビジネスホテル
コスパ良い
```

**効率的に:**

```
国内:

- 速い

- 遅い

- 安い

- 自由

海外:

飛行機:
必須

機材:
- 機内持ち込み

ビザ:
事前確認必須
