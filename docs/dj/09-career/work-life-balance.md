## なぜバランスが重要か

**燃え尽きたら終わり:**

```
DJ だけに集中:

短期的:
- 急成長
- 多くのギグ
- 知名度↑

長期的:
- 疲弊
- バーンアウト
- 人間関係崩壊
- 健康悪化
→ 続けられない

バランスがあれば:
- 持続可能
- 安定収入
- 幸福度↑
- 長期キャリア

マラソン:
スプリントではない
```

## 時間管理術

### タイムブロッキング

**時間を区切る:**

```
方法:

1週間を事前計画:

月曜 18:00-20:00: DJ練習
火曜 19:00-21:00: SNS管理
水曜 18:00-20:00: DJ練習
木曜 20:00-22:00: 新曲探し
金曜 19:00-23:00: ギグ準備
土曜 20:00-05:00: ギグ
日曜 終日: 休息

ブロックしたら:
その時間は DJ に集中

メリット:
- 無駄が減る
- 集中力↑
- 罪悪感↓

ツール:
- Google カレンダー
- Notion
- 紙の手帳
```

### ポモドーロテクニック

**集中と休憩:**

```
方法:

25分: 集中して作業
5分: 休憩
→ 4セット繰り返し
15-30分: 長い休憩

DJ 練習:

Set 1 (25分): ビートマッチング
休憩 (5分)
Set 2 (25分): 選曲
休憩 (5分)
Set 3 (25分): トランジション
休憩 (5分)
Set 4 (25分): セット録音
長い休憩 (30分)

合計: 2.5時間

効果:
集中力維持
疲れにくい
```

### デッドタイムの活用

**隙間時間を使う:**

```
通勤時間（往復1時間）:

□ 新曲チェック
□ DJ ポッドキャスト
□ ミックス聴く
□ SNS チェック/投稿

昼休み（30分）:

□ SNS 投稿
□ メール返信
□ イベント情報収集

待ち時間:

□ アイデアメモ
□ トラックリスト作成

積み重ねれば:
週5-10時間確保
```

## 優先順位

### 重要度×緊急度マトリックス

**何を優先する？**

```
┌─────────────┬─────────────┐
│ 緊急 重要   │ 緊急 重要でない│
│             │             │
│ ・明日のギグ│ ・SNS 通知  │
│ ・機材故障  │ ・雑談      │
│             │             │
│ → すぐやる  │ → 後で/断る │
├─────────────┼─────────────┤
│ 重要 緊急でない│重要でない 緊急でない│
│             │             │
│ ・練習      │ ・暇つぶし  │
│ ・スキル向上│ ・ダラダラSNS│
│ ・人脈作り  │             │
│             │             │
│ → 計画的に  │ → やめる    │
└─────────────┴─────────────┘

多くの人:
「緊急」に追われる

成功する人:
「重要だが緊急でない」に時間を使う

DJ で言えば:
練習、スキル向上、人脈作り
→ ここに時間を
```

### 「NO」と言う勇気

**全てを受けない:**

```
断るべきギグ:

□ ギャラが最低ライン以下
□ 移動時間 > 演奏時間
□ 自分のジャンルと合わない
□ 怪しいプロモーター
□ 体調が悪い
□ 本業に支障が出る
□ 家族との大事な予定

「NO」の言い方:

「ありがとうございます。
残念ながら、その日は
別の予定が入っております。

また次回の機会に
ぜひよろしくお願いいたします。」

丁寧に、でもはっきりと

最初は:
全て受けたくなる

でも:
選択することが成長
```

**サポートが必要:**

```
家族への説明:

1. 情熱を伝える
   「音楽が本当に好き」
   「人を楽しませたい」

2. 計画を示す
   「まず1年は副業で」
   「将来はプロを目指す」

3. 配慮を示す
   「家族の時間も大事にする」
   「経済的には本業で安定」

4. 実績を見せる
   ギグの動画
   観客の反応
   → 理解が深まる

反対されたら:

焦らず:
小さな実績から
徐々に理解を得る
```

## バーンアウト予防

### サインを見逃さない

**チェックリスト:**

```
□ DJ が楽しくない
□ 練習が義務に感じる
□ ギグが憂鬱
□ 慢性的な疲労
□ 睡眠障害
□ イライラが増えた
□ 集中力低下
□ 音楽を聴きたくない
□ 社交的になれない

3つ以上:
バーンアウトの危険

対処:
すぐに休息
```

### 予防策

**燃え尽きる前に:**

```
1. 週1日は完全OFF
   DJ のことを一切しない
   機材にも触らない

2. 睡眠優先
   最低7時間
   ギグ後も早めに寝る

3. 運動
   週2-3回、30分
   ストレス発散

4. 趣味（DJ以外）
   音楽以外の趣味を持つ
   → リフレッシュ

5. 人との交流
   DJ仲間以外の友人と
   → 視野が広がる

6. 定期的な休暇
   3ヶ月に1回、1週間休み
   → リセット

予防:
治療より重要
```

## よくある質問

### Q1: 会社にバレたらクビ？

**A:** 会社による

```
確認:

就業規則:
副業禁止か確認

許可制:
申請すれば OK の場合も

バレる原因:
- SNS（本名使用）
- 住民税（副業収入で↑）
- 同僚に話す

対策:
- 芸名使用
- SNS で本名出さない
- 住民税を自分で納付
- 同僚に言わない

最近:
副業 OK の会社増加中
```

### Q2: 睡眠時間が足りない

**A:** 優先順位の見直し

```
対策:

1. ギグ頻度を減らす
   月4回 → 月2回

2. 練習時間を調整
   毎日2時間 → 週3日1.5時間

3. 睡眠を最優先
   練習 < 睡眠

4. 効率化
   ダラダラ練習をやめる

健康第一:
睡眠を削るのは NG
```

### Q3: 家族が反対する

**A:** 対話と実績

```
アプローチ:

1. 話を聞く
   なぜ反対か理解

2. 懸念に答える
   「本業は辞めない」
   「家族の時間も大事にする」

3. 期限を設ける
   「1年やって結果出なければ考え直す」

4. 実績を見せる
   ギグ動画、収入

時間をかけて:
理解してもらう
```
