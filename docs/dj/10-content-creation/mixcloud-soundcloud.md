## Mixcloud vs SoundCloud

### 比較表

**どちらを使う？**

```
┌─────────────┬──────────────┬──────────────┐
│             │  Mixcloud    │  SoundCloud  │
├─────────────┼──────────────┼──────────────┤
│ 著作権      │ 合法         │ グレーゾーン │
│ 使用料      │ 運営が支払い │ 個人責任     │
│ 削除リスク  │ 低い         │ 高い         │
│ ダウンロード│ 不可（有料版のみ）│ 可（設定次第）│
│ スキップ    │ 制限あり     │ 自由         │
│ 収益化      │ 可能         │ 可能         │
│ ユーザー数  │ 少なめ       │ 多い         │
│ DJ特化      │ はい         │ いいえ       │
│ リーチ      │ 狭い         │ 広い         │
│ SEO         │ 弱い         │ 強い         │
└─────────────┴──────────────┴──────────────┘

推奨:
Mixcloud をメイン（合法的）
SoundCloud をサブ（リーチ拡大）
```

Step 2: プロフィール設定

### アップロード手順

**ミックスを投稿:**

```
Step 1: Upload 開始

トップページ:
[Upload] ボタンクリック

Step 2: ファイル選択

ドラッグ&ドロップ:
または [Select file]

対応形式:
MP3, AAC, M4A, FLAC
(WAVは非対応 → MP3に変換)

最大サイズ:
4GB

最大長さ:
6時間

アップロード開始:
進行状況表示

Step 3: 基本情報入力

Title:（必須）
「DJ GAKU - Melodic Techno Mix | December 2025」

フォーマット:
[DJ名] - [ジャンル] Mix | [月 年]

Description:
ミックスの説明（任意）

例:
「A deep journey through melodic techno.
Recorded live on DDJ-FLX4 with Rekordbox.

Perfect for late-night drives or deep focus.

📧 Booking: dj@example.com
📸 Instagram: @djgaku
🎵 More mixes: mixcloud.com/djgaku」

Tags:（最大5個）
techno, melodictechno, djmix, technomusic, dj

Step 4: カバーアート

Upload Picture:
クリックして画像選択

推奨サイズ:
1400×1400px（正方形）

Canvaで作成:
- ブランドカラー
- DJ名
- ミックス名
- 日付

Step 5: トラックリスト

Tracklist:（重要！）
[Add tracks]

各曲を追加:
Artist - Track Name

例:
1. Amelie Lens - Feel It
2. Charlotte de Witte - Sgadi Li Mi
3. Adam Beyer - Your Mind

理由:
- 著作権者に報告
- リスナーが曲を知れる
- プロフェッショナル

自動認識:
Mixcloud が曲を自動認識
→ アーティストに使用料分配

Step 6: カテゴリ設定

Category:
Music

Subcategory:
Techno（あなたのジャンル）

Show Type:
DJ Mix

Step 7: 公開設定

Visibility:
- Public（公開）推奨
- Private（非公開）
- Unlisted（リンク知ってる人のみ）

Publish Date:
- Now（即座）
- Schedule（予約投稿）

Step 8: 公開

[Publish] クリック:
アップロード完了

処理中:
数分〜数十分

完了:
URL が発行される

mixcloud.com/djgaku/melodic-techno-mix-dec-2025

SNSでシェア:
すぐに拡散
```

## 再生数を増やすテクニック

### SEO最適化

**検索されやすく:**

```
タイトル:

良い例:
「DJ GAKU - Melodic Techno Mix | December 2025 | Dark & Deep」

要素:
- DJ名
- ジャンル
- 月・年
- 特徴（Dark, Deep等）

悪い例:
「Mix 1」
→ 検索されない

説明文:

キーワード:
ジャンル、アーティスト名を含める

トラックリスト:
全曲記載
→ アーティスト名で検索される

タグ:

関連ワード全て:
- ジャンル
- サブジャンル
- 気分（dark, deep, melodic）
- 機材（djmix, pioneerdj）
```

## よくある質問

### Q1: どちらを優先すべき？

**A:** Mixcloud 優先、SoundCloud も

```
理由:

Mixcloud:
- 合法的
- 削除リスク低い
- DJ向け

SoundCloud:
- リーチ広い
- SEO強い
- 発見されやすい

両方投稿:
手間だが効果的
クロスリンク（お互いリンク）
```

### Q2: 削除されたらどうする？

**A:** 諦めて次へ

```
SoundCloud:

削除通知:
著作権者からの申し立て

対処:
- 争わない
- 削除を受け入れる
- 次のミックスを投稿

繰り返すと:
アカウント停止リスク

予防:
- マイナー曲多め
- トラックリスト明記
- 著作権表記
```

### Q3: 再生数が伸びない

**A:** 時間と継続

```
現実:
初投稿: 10-50再生
3ヶ月後: 100-500再生
6ヶ月後: 500-2000再生
1年後: 2000-10000再生

対策:
- 継続投稿（月2-4本）
- SNS拡散
- コミュニティ参加
- 質を上げる

焦らない:
誰もが通る道
```
