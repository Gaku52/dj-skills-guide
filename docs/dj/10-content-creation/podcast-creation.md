# ポッドキャスト作成

定期配信で固定ファンを獲得する。DJ ポッドキャストの始め方から配信戦略までの完全ガイドです。

## この章で学ぶこと

- ポッドキャストとは
- Anchor.fm で簡単配信
- Spotify Podcasts 登録
- Apple Podcasts 登録
- エピソード構成
- 定期配信のコツ
- リスナー獲得戦略

---

## ポッドキャストとは

**ラジオ番組のオンデマンド版:**

```
ポッドキャスト:
音声コンテンツの定期配信

特徴:
- エピソード形式
- RSS フィード配信
- 複数プラットフォームで聴ける
- ダウンロード可能
- 無料

DJ ポッドキャスト:
- ミックス
- トーク
- 新曲紹介
- ゲストDJ

メリット:
- 固定ファン獲得
- 定期的なエンゲージメント
- ブランディング
- 収益化の可能性
```

## Anchor.fm で始める

### なぜ Anchor.fm？

**最も簡単:**

```
Anchor.fm の利点:

1. 完全無料
   費用ゼロ

2. 簡単
   アプリで完結

3. 自動配信
   Spotify, Apple Podcasts, Google Podcasts
   → 一度で全プラットフォームに

4. 収益化
   Spotify Ads
   サポーター機能

5. 分析
   再生数、リスナー属性

デメリット:
ほぼなし

初心者:
Anchor 一択
```

### アカウント作成

**5分で完了:**

```
Step 1: アクセス

ブラウザ:
https://anchor.fm

または:
Anchor アプリ（iOS/Android）

Step 2: サインアップ

[Get started] クリック

方法選択:
- Email
- Google
- Spotify

Email 推奨:
email@example.com
パスワード設定

Step 3: ポッドキャスト情報

Podcast Name:（必須）
「GAKU's Techno Journey」

Description:（必須）
番組説明（300文字程度）

例:
「Monthly techno mix podcast from Tokyo.
Deep, dark & melodic sounds for the underground.

Hosted by DJ GAKU.

New episode every first Friday of the month.

📸 Instagram: @djgaku
📧 Booking: dj@example.com」

Language:
Japanese or English

Category:
Music

Subcategory:
Electronic

Step 4: カバーアート

Upload artwork:
クリック

推奨サイズ:
3000×3000px（正方形）

最小サイズ:
1400×1400px

Canva で作成:
- ポッドキャスト名
- DJ名
- ビジュアル

Step 5: 完了

[Create podcast] クリック

ポッドキャスト作成完了！

URL発行:
anchor.fm/djgaku
```

## エピソード作成

### 構成

**3パート構成:**

```
Part 1: イントロ（2-5分）

内容:
- 挨拶
- 自己紹介
- エピソード概要
- 今月のトピック

例:
「こんにちは、DJ GAKU です。
GAKU's Techno Journey Episode 5 へようこそ。

今月は Melodic Techno を特集します。
Amelie Lens、Charlotte de Witte の新作や、
アンダーグラウンドの注目トラックを
60分のミックスでお届けします。

それでは、音楽をお楽しみください」

録音:
スマホのボイスメモで OK

Part 2: ミックス（60分）

内容:
- メインのDJミックス
- Rekordbox で録音済み

Part 3: アウトロ（2-3分）

内容:
- トラックリスト
- 次回予告
- SNS案内
- 感謝

例:
「ご視聴ありがとうございました。

本日のトラックリストは
説明欄をご覧ください。

次回は来月第1金曜日、
Techno の新作を特集します。

Instagram @djgaku でも
日々の活動を発信していますので、
ぜひフォローしてください。

それでは、また次回お会いしましょう」

合計:
65-70分
```

### Anchor でエピソードアップロード

**ステップバイステップ:**

```
Step 1: 新規エピソード

Anchor ダッシュボード:
[New episode] クリック

Step 2: 音声アップロード

3つのファイル:
1. イントロ（MP3）
2. ミックス（MP3）
3. アウトロ（MP3）

[Upload] でそれぞれ追加

または:

1つのファイル:
Audacity で結合済み
→ アップロード

Step 3: エピソード情報

Episode Title:（必須）
「Episode 5 - Melodic Techno | December 2025」

フォーマット:
Episode [番号] - [テーマ] | [月 年]

Description:（必須）

テンプレート:

「Episode 5 of GAKU's Techno Journey

🎵 This month: Melodic Techno

A 60-minute journey through deep and melodic techno sounds.

📋 Tracklist:
00:00 DJ GAKU - Intro
02:30 Amelie Lens - Feel It
07:45 Charlotte de Witte - Sgadi Li Mi
...
62:30 DJ GAKU - Outro

📸 Follow: instagram.com/djgaku
📧 Booking: dj@example.com

Next episode: Jan 3, 2026

#techno #melodictechno #djmix #podcast」

Season & Episode:（任意）
Season 1, Episode 5

Step 4: 設定

Type:
Full Episode

Explicit:
Clean（通常）

Step 5: 公開

Publish:
- Now（即座）
- Schedule（予約）

[Publish episode] クリック

完了:
数分で全プラットフォームに配信
```

## プラットフォーム配信

**Anchor から自動:**

```
Anchor.fm から配信先:

✓ Spotify
✓ Apple Podcasts
✓ Google Podcasts
✓ Pocket Casts
✓ Overcast
✓ その他多数

設定:

確認:

Anchor ダッシュボード
```

## 定期配信のコツ

### 頻度

**無理のない計画:**

```
推奨頻度:

月1回:（おすすめ）
- 第1金曜日
- 負担少ない
- 継続しやすい

隔週:
- 毎月1日, 15日
- やや負担

週1回:
- 毎週金曜日
- 負担大
- プロレベル

初心者:
月1回から
→ 慣れたら増やす

一貫性:
継続が最重要
```

### 制作スケジュール

**月1回の場合:**

```
Week 1（配信日の3週間前）:

□ テーマ決定
□ 曲選び開始
□ プレイリスト作成

Week 2（2週間前）:

□ プレイリスト完成
□ Hot Cue 設定
□ リハーサル

Week 3（1週間前）:

□ ミックス録音
□ イントロ/アウトロ録音
□ 編集（必要なら）
□ カバーアート作成

Week 4（配信週）:

月曜:
□ 最終チェック
□ エピソード情報作成

水曜:
□ Anchor にアップロード
□ 予約投稿設定

金曜（配信日）:
□ 公開確認
□ SNS シェア
□ コメント対応

余裕を持って:
締め切りに追われない
```

## 収益化

### Spotify Ads

**Anchor の機能:**

```
条件:
- Anchor 利用
- Spotify 配信
- 一定の再生数

設定:
Anchor ダッシュボード
[Monetization] > [Sponsorships]

収益:
再生回数に応じて

目安:
1000再生 = $15-25

ハードル:
高くない、試す価値あり
```

### サポーター機能

**ファンから支援:**

```
Anchor Supporter:

リスナーが:
月額サポート

金額:
任意（$1-$50等）

リターン:
- 特典エピソード
- 早期アクセス
- シャウトアウト

設定:
Anchor ダッシュボード
[Monetization] > [Supporter]

難しいが、成長すれば可能
```

## よくある質問

### Q1: トークが苦手

**A:** 短く、または音楽のみ

```
対策:

最小構成:
イントロ（30秒）
「GAKU's Techno Journey Episode 5
今月は Melodic Techno です。お楽しみください」

ミックス（60分）

アウトロ（30秒）
「ありがとうございました。次回もお楽しみに」

合計61分:
トーク少なめ

または:

音楽のみ:
ミックスだけでもOK
説明は文章で
```

### Q2: どれくらいで成長する？

**A:** 6ヶ月-1年

```
現実:

Episode 1: 10-50再生
Episode 5: 50-200再生
Episode 10: 200-500再生
Episode 20: 500-2000再生

成長は:
緩やか

継続が:
唯一の道
```

### Q3: 著作権は大丈夫？

**A:** グレーゾーン

```
現実:
DJ ミックスは著作権的にグレー

Spotify:
削除されるリスクあり

対策:
- 説明に著作権表記
- アンダーグラウンド曲多め
- 削除されたら諦める

Mixcloud との違い:
Mixcloud は合法
ポッドキャストはグレー

リスク承知で:
多くのDJが配信中
```
