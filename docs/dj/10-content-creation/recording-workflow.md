# 録音ワークフロー

完璧な録音は、すべてのコンテンツの基礎。Rekordbox と DDJ-FLX4 で最高品質のミックスを録音する完全ガイドです。

## この章で学ぶこと

- Rekordbox 録音設定
- DDJ-FLX4 での録音手順
- フォーマット選択（WAV vs MP3）
- ゲイン調整とレベル管理
- 録音中のベストプラクティス
- ファイル管理とバックアップ
- トラブルシューティング

---

## なぜ録音が重要なのか

**すべてのコンテンツの出発点:**

```
録音したミックス:

→ SoundCloud/Mixcloud にアップロード
→ ポッドキャストに使用
→ YouTube動画の音源
→ プロモーター用デモ
→ 自己分析・改善

質の高い録音:
全てのクオリティを左右する

録音が悪いと:
全てが台無し
```

## Rekordbox 録音設定

### 初期設定

**一度設定すれば OK:**

```
Step 1: 環境設定を開く

Rekordbox メニュー:
[Rekordbox] > [Preferences]（Mac）
[File] > [Preferences]（Windows）

または:
[Cmd + ,]（Mac）
[Ctrl + ,]（Windows）

Step 2: 録音タブを選択

左メニュー:
[Controller] > [Rec]

設定画面が開く

Step 3: 保存先設定

Rec Folder:
[Browse] をクリック

推奨フォルダ構造:
/Users/あなたの名前/DJ/
  ├── Recordings/
  │   ├── Practice/（練習用）
  │   ├── Demos/（デモ用）
  │   └── Live/（ライブ録音）
  └── Music/

作成して選択:
[DJ] > [Recordings] > [Practice]

Step 4: ファイル形式設定

Rec Format:
WAV or MP3 を選択

WAV 設定:
- Sample Rate: 44100 Hz
- Bit Depth: 16 bit
- 非圧縮、最高音質
- ファイルサイズ大（60分 約600MB）

MP3 設定:
- Bit Rate: 320 kbps（最高品質）
- ファイルサイズ小（60分 約140MB）
- 圧縮、音質劣化わずか

推奨:
最初は WAV で録音
→ 後で MP3 に変換
（音質の選択肢を残す）

Step 5: ファイル名設定

Auto Naming:
チェックを入れる

Format:
%Y-%m-%d_%H%M_Recording

例:
2025-12-07_2200_Recording.wav

カスタマイズも可能:
%Y: 年（4桁）
%m: 月（2桁）
%d: 日（2桁）
%H: 時（24時間）
%M: 分（2桁）

Step 6: その他設定

Split Rec:
OFF（通常は不要）

Rec Level:
0 dB（デフォルト）

完了:
[OK] をクリック

設定完了！
```

## DDJ-FLX4 での録音手順

### 準備

**録音前のチェック:**

```
30分前:

□ Rekordbox 起動
□ DDJ-FLX4 接続確認
□ プレイリスト準備
□ Hot Cue 設定完了
□ ヘッドフォン動作確認

15分前:

□ リハーサル（軽く通す）
□ ミスしそうな箇所確認
□ トイレ、水分補給
□ 部屋を静かに（通知OFF）

5分前:

□ 深呼吸
□ 最初の2曲を確認
□ リラックス
```

### 録音開始

**ステップバイステップ:**

```
Step 1: 録音ボタン準備

DDJ-FLX4:
右上の [REC] ボタン確認

または

Rekordbox PC画面:
上部メニューの [●] ボタン

Step 2: レベル確認

Master Output:
メーター確認

理想的なレベル:
-6 dB〜 0 dB（黄色ゾーン）

赤ゾーン（クリップ）:
絶対に避ける
→ 歪む、修復不可能

TRIM/GAIN:
適切に調整済みか確認

Step 3: 録音開始

[REC] ボタンを押す:
LED が赤く点灯
→ 録音中

Rekordbox画面:
「Recording...」表示

または

ショートカットキー:
[R] キー（PC）

タイミング:
最初の曲のビート1から
または
数秒の無音 → 曲開始

Step 4: プレイ開始

通常通りDJプレイ:
□ リラックス
□ ミスを恐れない
□ ミスしても続ける
□ 楽しむ

録音中の注意:
□ マイク音声も録音される
□ PC通知音も録音される
□ 部屋の雑音も録音される

→ 静かな環境推奨

レベル監視:
定期的にメーターチェック
常に黄色ゾーン維持

Step 5: 録音終了

最後の曲が終わったら:
[REC] ボタンをもう一度押す

LED 消灯:
録音停止

または

[R] キー（PC）

自動保存:
設定フォルダに保存される

確認:
Rekordbox ブラウザで
Recordings フォルダ確認
```

## 録音中のベストプラクティス

### レベル管理

**歪みを防ぐ:**

```
ゲインステージング:

各チャンネル:
TRIM を適切に
→ チャンネルメーター -6〜0 dB

Master:
MASTER LEVEL で微調整
→ マスターメーター -6〜0 dB

常にモニター:
録音中は目を離さない

赤点灯したら:
すぐに MASTER LEVEL を下げる

理想:
終始 -6〜-3 dB
→ 余裕を持つ
→ 後処理で調整可能
```

### タイムスタンプメモ

**後で振り返る:**

```
録音中:

良い瞬間:
「25:30 - 完璧なトランジション」

ミス:
「42:15 - ビートズレ」

メモ方法:

1. 紙に書く（アナログ）
   時間: 内容

2. スマホメモ（デジタル）
   タイマー見ながら記録

3. 音声メモ（録音後確認時）

後で:
改善点の分析に役立つ
次回に活かす
```

## ファイル管理

### 整理術

**見つけやすく:**

```
フォルダ構造:

/DJ/Recordings/
  ├── 2025/
  │   ├── 01-January/
  │   │   ├── 2025-01-05_Practice_Techno.wav
  │   │   └── 2025-01-12_Demo_House.wav
  │   ├── 02-February/
  │   └── 03-March/
  └── Archive/
      └── 2024/

命名規則:

YYYY-MM-DD_Type_Genre.wav

例:
2025-12-07_Practice_Techno.wav
2025-12-10_Demo_House.wav
2025-12-15_Live_Festival.wav

Type:
- Practice: 練習
- Demo: デモ
- Live: ライブ録音
- Podcast: ポッドキャスト用

Genre:
- Techno
- House
- HipHop
- Mix（複数ジャンル）

一目で分かる:
効率的
```

### バックアップ戦略

**大切な録音を守る:**

```
3-2-1 ルール:

3: 3箇所に保存
2: 2種類のメディア
1: 1箇所は外部（クラウド等）

実践例:

コピー1: PC内蔵ストレージ
/Users/あなた/DJ/Recordings/

コピー2: 外付けHDD/SSD
/Volumes/Backup/DJ/Recordings/

コピー3: クラウド
Google Drive / Dropbox / iCloud

頻度:
録音後すぐにバックアップ

自動化:
Time Machine（Mac）
File History（Windows）
→ 設定しておく

失うと:
取り返しがつかない
必ずバックアップ
```

## WAV から MP3 への変換

### いつ変換する？

**用途別に:**

```
WAV のまま:
□ マスタリング前
□ 編集予定
□ 最高音質保存

MP3 に変換:
□ SoundCloud アップロード
□ デモとして送付
□ スマホで聴く
□ ファイルサイズ削減

両方保存:
推奨（ストレージに余裕あれば）
```

### Rekordbox で変換

**簡単3ステップ:**

```
Step 1: ファイル選択

Rekordbox ブラウザ:
Recordings フォルダ

WAV ファイル:
右クリック

Step 2: 変換実行

メニュー:
[Convert to MP3]

設定:
Bit Rate: 320 kbps
Quality: High

保存先:
元のフォルダ

[Convert] クリック

Step 3: 確認

同じフォルダに:
.mp3 ファイルが生成

例:
2025-12-07_2200_Recording.wav
2025-12-07_2200_Recording.mp3

両方保存されている:
用途別に使い分け
```

## トラブルシューティング

### 録音されない

**チェックリスト:**

```
原因1: REC ボタンが押されていない
解決: [REC] ボタン確認、LED点灯確認

原因2: 保存先フォルダが存在しない
解決: Preferences で保存先確認、フォルダ作成

原因3: ディスク容量不足
解決: ストレージ確認、不要ファイル削除

原因4: Rekordbox が録音モードでない
解決: Performance モードか確認
```

### 音が歪む

**対処法:**

```
原因: ゲインが高すぎる

解決:

録音中:
1. MASTER LEVEL を下げる
2. 各チャンネルの TRIM を下げる
3. EQ を控えめに

次回から:
録音前にレベル確認
リハーサルで調整
常に -6〜-3 dB を目標
```

### ファイルが見つからない

**探し方:**

```
Step 1: 保存先確認

Preferences:
[Controller] > [Rec]
Rec Folder を確認

Step 2: Finder/エクスプローラーで検索

検索キーワード:
「Recording」「.wav」

日付で絞り込み:
今日、昨日

Step 3: Rekordbox ブラウザ

左メニュー:
[Recordings]

表示されない場合:
フォルダを右クリック
[Reload]
```

## よくある質問

### Q1: WAV と MP3 どちらで録音？

**A:** WAV 推奨、後で変換

```
理由:
- 最高音質で保存
- 後で選択肢が広がる
- 編集・マスタリングに有利

ストレージ節約:
古い録音は MP3 に変換
WAV は削除

最初は:
WAV で録音する習慣を
```

### Q2: 録音中にミスしたら？

**A:** 止めずに続ける

```
ミスの種類:

小さいミス:
そのまま続ける
→ 誰も気づかない

大きいミス:
続けて最後まで録音
→ 後で編集 or 再録音

録音を止めるのは:
- 機材トラブル
- 大きな外部ノイズ
のみ

完璧主義:
禁物、続けることが重要
```

### Q3: どれくらいの長さを録音？

**A:** 用途による

```
練習:
30-60分

デモミックス:
30-60分（60分推奨）

ポッドキャスト:
60-90分

ライブセット:
実際の長さ（2-4時間）

初心者:
30分から始める
→ 徐々に伸ばす
```
