## なぜ教えるのか

**教えることで学ぶ:**

```
Teaching is Learning Twice

教える側のメリット:

理解が深まる:
説明するために再学習
→ 自分の理解が深まる

整理される:
知識が体系化
→ より上手くなる

記憶定着:
アウトプットで覚える

評判向上:
「教えてくれる人」として認知
→ 人脈拡大

自信:
「教えられる」という自信

コミュニティ貢献:
恩返し

統計:

教える経験のあるDJ:
技術向上スピードが1.5倍速い

理由:
知識の再構築
新しい視点
質問から学ぶ

あなたも:
教わったはず
→ 次は返す番
```

## 初心者DJの指導

### いつから教えられる？

**意外と早い:**

```
必要な経験:

最低:
DJ歴6ヶ月

推奨:
DJ歴1年

条件:
基本技術ができる
ビートマッチ、EQ、ミキシング

完璧でなくてOK:
プロである必要なし

教えられること:

DJ歴6ヶ月:
完全初心者向け
機材の使い方
Rekordboxの基本

DJ歴1年:
初級者向け
ビートマッチ
選曲の基本

DJ歴2年+:
中級者向け
トランジション
ハーモニックミキシング

重要:
「自分より初心者」に教えられる
```

### 指導の基本

**良い教師になる:**

```
心構え:

謙虚:
「俺も学んでいる途中」

忍耐:
何度でも説明

ポジティブ:
褒めて伸ばす

共感:
初心者の気持ちを思い出す

教え方:

1. デモンストレーション
   「まずやってみせる」

2. 説明
   「なぜそうするのか」

3. 実践
   「やってみて」

4. フィードバック
   「良い！ただし〇〇を改善」

5. 反復
   「もう一度」

例（ビートマッチ）:

1. デモ:
   自分がビートマッチする

2. 説明:
   「ジョグホイールで微調整」
   「耳でズレを聴く」

3. 実践:
   生徒にやらせる

4. フィードバック:
   「タイミング良い！
   もう少しジョグを軽く」

5. 反復:
   10回練習

NG:

批判:
「下手だね」

比較:
「〇〇さんはすぐできた」

焦らせる:
「早くやって」

放置:
「勝手にやって」
```

### 1対1セッション

**効果的な指導:**

```
準備:

機材:
DDJ-FLX4（あなたの）
または生徒の機材

場所:
自宅 or スタジオ

時間:
1-2時間

曲:
練習用プレイリスト

流れ（初回）:

1. ヒアリング（15分）
   「目標は？」
   「どんな音楽が好き？」
   「何に困ってる？」

2. 現状確認（15分）
   「今できることを見せて」

3. 課題設定（5分）
   「今日は〇〇を習得しよう」

4. 指導（60分）
   デモ → 実践 → FB

5. 振り返り（10分）
   「今日の要点」
   「次回までの宿題」

宿題:

具体的:
「毎日30分ビートマッチ練習」

記録:
「10回成功したら動画送って」

次回:

確認:
宿題の進捗

次のステップ:
段階的に難易度UP

頻度:

週1回:
理想的

隔週:
現実的

月1回:
最低限
```
