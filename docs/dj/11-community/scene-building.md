## シーンとは

**文化としてのコミュニティ:**

```
シーンの定義:

特定の音楽ジャンル:
Techno、House等

地域:
東京、大阪、福岡

定期的なイベント:
毎週、毎月

コミュニティ:
DJs、観客、オーガナイザー

文化:
共有された価値観

有名なシーン:

ベルリン Techno:
Berghain、Tresor

発祥の地

Vent、Contact

大阪 Techno:
Circus、Noon

あなたの街:

シーンがない?:
作れる

50人から

継続:
数年かけて育てる

結果:
文化として根付く
```

## イベントシリーズ立ち上げ

### 企画段階

**明確なコンセプト:**

```
Step 1: コンセプト決定

ジャンル:
Techno（例）

サブジャンル:
Melodic Techno

BPM範囲:
125-135

雰囲気:
Dark & Melodic

ターゲット:
25-35歳

差別化:
既存イベントと何が違う？

例:
「DEEP - Melodic Techno Night
毎月第1金曜日、深夜まで
Melodic & Progressive Techno
125-135 BPM」

Step 2: 名前とブランド

イベント名:
短く、覚えやすく

例:
DEEP
UNITY
PULSE
RESONANCE

ロゴ:
Canva で作成

カラー:
ブランドカラー決定

Step 3: 頻度とスケジュール

月1回:
最も現実的

隔週:
やや大変

週1回:
プロレベル

固定日:
第1金曜日等

理由:
覚えやすい
習慣化

Step 4: 収支計画

初期コスト:
会場レンタル: ¥30,000
フライヤー: ¥5,000
SNS広告: ¥10,000
合計: ¥45,000

収入:
エントランス ¥2,000 × 30人
= ¥60,000

損益:
¥60,000 - ¥45,000 = +¥15,000

現実:
最初は赤字覚悟
```

### 会場探し

**適切な箱を見つける:**

```
会場の種類:

小規模バー:
30-50人
雰囲気良い
音響微妙

小規模クラブ:
50-100人
音響良い
レンタル高い

レンタルスペース:
フレキシブル
自分で機材持ち込み

屋外:
公園、ビーチ
許可必要

探し方:

1. Google マップ
   「クラブ [あなたの街]」

2. 既存イベント参加
   会場を下見

3. オーナーに直接
   営業時間外に訪問

4. SNS
   「レンタルスペース DJ」

交渉:

アプローチ:
オーナー/マネージャーに

提案:
「毎月第1金曜日に
Techno イベントを開催したい」

条件:
レンタル料
音量制限
時間

契約:
最初は1回ごと
うまくいけば長期契約

レンタル料相場:

小規模（50人）:
¥20,000-40,000/晩

中規模（100人）:
¥50,000-100,000/晩

交渉:
ドリンク売上の一部還元
→ レンタル料減額

Win-Win:
クラブも客が来る
= お互いにメリット
```

## よくある質問

### Q1: お金がないけど始められる？

**A:** 最小限から

```
最小構成:

会場:
友人のバー（無料）

DJ:
自分+友人（無料）

プロモーション:
SNSのみ（無料）

エントランス:
¥500-1,000

規模:
20-30人

徐々に:
利益で拡大
```

### Q2: 失敗したらどうする？

**A:** 学びに変える

```
失敗の定義:

客が来ない:
5人以下

大赤字:
-¥50,000以上

対処:

原因分析:
何が悪かった？

改善:
次回に活かす

継続:
3回は続ける

撤退基準:

6回やって:
毎回10人以下
→ コンセプト見直し

12回やって:
毎回赤字
→ 撤退検討

でも:

情熱があれば:
続ける

諦めない:
ベルリンのクラブも最初は小さかった
```

### Q3: 許可は必要？

**A:** 会場次第

```
クラブ/バー:

会場が許可持ち:
あなたは不要

レンタルスペース:

営業許可:
必要な場合あり

確認:
オーナーに

屋外:

公園:
許可必要（市役所）

ビーチ:
同上

音量:
近隣住民に配慮

JASRAC:

音楽著作権:
会場が包括契約してるか確認

していない場合:
自分で手続き

弁護士相談:
心配なら
```
