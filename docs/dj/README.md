# DJ スキルガイド

Rekordbox 7.2.7 を使った DJ スキルの習得ガイドです。基礎から応用、キャリア構築まで網羅します。

## このセクションについて

DJ として必要なすべてのスキルを、体系的に学べるように構成しています。

**対象:**
- これから DJ を始める初心者
- 基礎を固めたい中級者
- さらに技術を磨きたい上級者

**使用機材:**
- Rekordbox 7.2.7
- DJコントローラー / CDJ
- オーディオインターフェイス

---

## 学習の進め方

### 初心者の方

1. **[00-fundamentals](../00-fundamentals/)** で音楽の基礎を学ぶ
2. **[01-basics](#01-basics---dj基礎知識)** で DJ とは何かを理解
3. **[02-equipment](#02-equipment---機材ガイド)** で機材の使い方を習得
4. **[03-basic-techniques](#03-basic-techniques---基本テクニック)** で基礎技術を練習

1. 基礎の復習（必要に応じて）
2. **[04-advanced-techniques](#04-advanced-techniques---応用テクニック)** で技術向上
3. **[05-track-selection](#05-track-selection---選曲とセットリスト)** でセンスを磨く
4. **[06-performance](#06-performance---パフォーマンス)** で実践力を身につける

### 上級者の方

1. **[07-genres](#07-genres---ジャンル別ガイド)** で専門性を深める

## Rekordbox 7.2.7 の活用

このガイドでは、最新の Rekordbox 7.2.7 の機能をフル活用します。

### 主要機能

- **キー解析**: ハーモニックミキシング

- **Related Tracks**: AIによる曲推薦
- **エフェクト**: Echo、Reverb、Filter など
- **Cloudライブラリ**: どこでも同期
- **Lighting**: DMX対応照明制御

## 推奨学習時間

**初心者（0-3ヶ月）:**
- 毎日1時間の練習
- 基礎技術の習得に集中
- 目標: 30分のシンプルなセットを組める

**中級者（3-12ヶ月）:**
- 週5日、各2時間の練習
- 応用テクニックの習得
- 目標: 1時間のセットをスムーズに

**上級者（12ヶ月以降）:**
- 継続的な練習と実践
- ジャンル専門性の深化
- 目標: プロフェッショナルレベルのパフォーマンス
