# Ableton で DJ

**Ableton LiveをDJソフトウェアとして使用する方法を完全マスター**

Ableton LiveはDAWですが、DJソフトウェアとしても強力です。Session Viewでのビートマッチング、クロスフェーダー、エフェクトを駆使したDJセットを実現できます。

---

## Session ViewでのDJセットアップ

### トラック構成

```
Track 1-2: Deck A（左側）
Track 3-4: Deck B（右側）
Track 5-8: エフェクト
Return Tracks: Reverb, Delay, Filter
Master: クロスフェーダー
```

### クリップ配置

```
各トラックに楽曲をクリップとして配置:
  - 1 Clip = 1曲
  - Warp: On
  - BPM: 自動同期

Scene:
  - 1 Scene = 1セクション
```

---

## クリップでのビートマッチング

### 自動同期

```
Warp On:
  → Project TempoにBPMが自動同期
  → ビートマッチング不要

利点:
  - 完璧な同期
  - BPM差が大きくてもOK（100 vs 140でも可能）
```

### マニュアル調整

```
Clip View → Segment BPM:
  - 微調整可能
  - グリッドずれ修正
```

---

## クロスフェーダーとトランジション

### クロスフェーダー設定

```
Mixer Section:
  - Track 1-2: A側
  - Track 3-4: B側
  - Crossfader: A ←→ B

操作:
  - MIDIコントローラーのフェーダー
  - またはマウス
```

### トランジション

```
1. Deck Aで曲再生
2. Deck Bで次の曲を準備（Cue）
3. Crossfaderを徐々にB側へ
4. Deck Aフェードアウト、Deck Bフェードイン
```

---

## エフェクトラックの活用

### Send エフェクト

```
Return A: Echo
Return B: Reverb
Return C: Auto Filter

各トラックのSend:
  - リアルタイムで調整
  - トランジション時にSend上げる
```

### Insert エフェクト

```
各トラックに直接挿入:
  - EQ Eight: Hi/Mid/Lowカット
  - Filter: Filter Sweep
  - Beat Repeat: Stutter効果
```

## Rekordbox vs Ableton

### Rekordbox（DJ専用）

```
利点:
  - CDJ互換
  - Waveform表示
  - Hot Cue、Loop機能豊富
  - クラブ標準

欠点:
  - 制作機能なし
  - エフェクト限定的
```

### Ableton（DAW+DJ）

```
利点:
  - 制作とDJを同じソフトで
  - 無限のエフェクト
  - MIDI楽器演奏可能
  - Session Viewの自由度

欠点:
  - Waveform表示なし（クリップのみ）
  - CDJ非互換
  - 学習曲線高い
```

## ハイブリッドセットアップ

### Rekordbox + Ableton

```
Rekordboxで他のアーティストの曲をDJ
↓
Abletonに切り替え
↓
オリジナルトラックをライブ制作
↓
再びRekordboxへ

→ 両方の利点を活用
```
