**DJ視点を取り入れた楽曲制作を完全マスター**

制作者がDJの視点を理解することで、DJにプレイされやすい、ダンスフロアで効く楽曲を作成できます。Intro/Outroの長さ、ミックスポイント、キー設定など、実践的なテクニックを学びます。

## DJフレンドリーな曲構成

### 32小節 Intro/Outro

**なぜ32小節か**:
```
DJのミックス時間:
  - 標準: 32小節（4フレーズ）
  - 短い: 16小節（ミックスが難しい）
  - 長い: 64小節（余裕あり）

32小節 = 約1分（128 BPMの場合）
  → 十分なミックス時間
```

### 要素の追加/削除順序

**Intro（徐々に追加）**:
```
Bar 1-8: Kick
Bar 9-16: + Hi-Hat
Bar 17-24: + Bass
Bar 25-32: + Pad/Synth
Bar 33〜: Full Drop
```

**Outro（逆順で削除）**:
```
Bar 1-8: Full
Bar 9-16: - Lead
Bar 17-24: - Bass
Bar 25-32: Kick のみ
```

## ミックスポイントの設計

### ループ可能なセクション

**Intro（Bar 9-16）をループ化**:
```
- Bar 9と17が完全に繋がる
- DJが好きなタイミングで次の要素を追加可能

Ableton設定:
  - Loop: On
  - Loop Length: 8小節
```

### フレーズの切れ目

```
8小節ごとにフレーズが完結:
  - DJがトランジション開始しやすい
  - 4小節単位だと短すぎる
```

## キー設定の重要性

### ハーモニックミキシング対応

**明確なキー**:
```
1つのスケール内で完結:
  - C Major または A Minor
  - スケール外の音を避ける

Mixed In Key分析:
  → キーが正確に検出される
```

### Camelot Wheel

```
隣接キー:
  - 8A（Am）→ 9A（Em）
  - 8A（Am）→ 8B（C Major）

→ DJがスムーズにミックスできる
```

## BPMの選択

### ジャンル別標準BPM

```
Techno: 128-132 BPM
House: 122-126 BPM
Progressive House: 126-128 BPM
Trance: 136-140 BPM
Dubstep: 140 BPM
```

### BPMの一貫性

```
曲全体で一定BPM:
  - テンポ変化は避ける
  - DJのSync機能が正しく動作
```
