# Ableton Live 基礎

DJから楽曲制作へ。世界中のプロデューサーが使用するAbleton Liveの基礎を完全マスターするセクションです。

## このセクションで学ぶこと

- Ableton Liveのインターフェイス完全理解
- Session ViewとArrangement Viewの使い分け
- プロジェクト設定と環境設定
- Audio/MIDI設定の最適化
- ファイル管理とバックアップ戦略
- 効率的なワークフロー構築

---

## なぜAbleton Liveなのか

**DJから制作への最短ルート:**

```
Ableton Liveの強み:

DJフレンドリー:
Session View = ライブパフォーマンス向き
DJセットのように即興的に

直感的:
他DAWより学習曲線が緩やか
初心者に優しい

業界標準:
Techno、House制作の定番
多くのプロが使用

統合:
DJ + 制作 を1つのソフトで
Ableton Link でDJ機材と連携

実績:
Deadmau5、Richie Hawtin、
Carl Cox、Nina Kraviz
など多数のアーティストが使用

他DAWとの比較:

FL Studio:
Hip Hop、EDM向き
ステップシーケンサー強い

Logic Pro:
Macのみ
総合的だが重い

Cubase:
老舗、プロ仕様
初心者には複雑

Ableton Live:
電子音楽に最適
DJからの移行が楽

結論:
Techno/House DJなら
Ableton Live一択
```

## DJから制作への移行

**既にあるスキルを活かす:**

```
DJスキル → 制作スキル:

1. BPM理解:
   DJ: テンポ合わせ
   制作: プロジェクトテンポ設定

2. フレーズ構造:
   DJ: 32小節で展開
   制作: 曲構成に応用

3. EQ:
   DJ: 周波数カット
   制作: ミキシング

4. エフェクト:
   DJ: Echo、Filter
   制作: プラグイン活用

5. ハーモニックミキシング:
   DJ: キー合わせ
   制作: コード理論

6. 選曲センス:
   DJ: 曲の良し悪し
   制作: 何が良い音か分かる

7. フロアリーディング:
   DJ: 観客の反応
   制作: ダンサブルかどうか

アドバンテージ:

すでに耳が育っている:
何が良い曲か分かる

ジャンル知識:
Technoの構造理解済み

機材慣れ:
MIDI、オーディオ概念

ネットワーク:
DJ仲間にフィードバックもらえる

モチベーション:
自分の曲をDJセットで使える！

時間:
週10時間の練習
→ 3ヶ月で最初のトラック完成
→ 6ヶ月でリリースレベル
```

## よくある質問

### Q1: DJと制作、両立できる？

**A:** 可能、むしろ相乗効果

```
時間配分:

週10時間の場合:
DJ: 6時間（60%）
制作: 4時間（40%）

週20時間の場合:
DJ: 12時間
制作: 8時間

相乗効果:

制作 → DJ:
自作曲をプレイ
= 差別化

DJ → 制作:
フロアでの反応を制作に活かす

両立のコツ:
完全に分ける日を作る
月・水・金: DJ練習
火・木・土: 制作
日: 休み
```

### Q2: 音楽理論は必要？

**A:** 基礎だけでOK、実践で学ぶ

```
最低限:

スケール:
CメジャーとAマイナー

コード:
3和音（トライアド）

リズム:
4/4拍子

それ以上:
作りながら学ぶ

Techno/Houseの場合:
理論より感覚
ダンサブルかどうか

後で学ぶ:
00-fundamentals/ 参照
```

### Q3: Macじゃないとダメ？

**A:** WindowsでもOK

```
Mac:
Logic ProユーザーならMac
安定性高い

Windows:
Ableton LiveならOS関係なし
同じ機能

選び方:
今持ってるPCで十分
新しく買うなら好きな方

重要なのは:
RAM 16GB以上
SSD搭載
```
