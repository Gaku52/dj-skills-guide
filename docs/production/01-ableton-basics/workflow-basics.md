# ワークフロー基礎

効率的な作業の流れを確立する。ショートカットとテンプレートで制作スピードを10倍にします。

## この章で学ぶこと

- 効率的なワークフロー確立
- 必須ショートカット50選
- テンプレート活用術
- セッション管理
- 時短テクニック集
- 制作ルーティン

---

## なぜワークフローが重要なのか

**速さは正義:**

```
初心者の制作時間:

8小節ループ: 2時間
完成まで: 100時間

プロの制作時間:

8小節ループ: 30分
完成まで: 20時間

差の理由:

ワークフロー:
無駄な動作ゼロ
ショートカット使いこなし
テンプレート活用

結果:
5倍速い

あなたも:
1ヶ月で3倍速に
3ヶ月で5倍速に
```

## 基本ワークフロー

**制作の流れ:**

### アイデア → 完成までの8ステップ

```
Step 1: プロジェクト作成 (2分)
テンプレート選択
BPM設定
保存

Step 2: ドラムパターン (30分)
Kick配置
Hi-Hat追加
Snare/Clap追加
8小節ループ完成

Step 3: ベースライン (30分)
Bass音色選択
MIDI入力
グルーヴ調整

Step 4: メロディ/コード (1時間)
Synth音色
コード進行
メロディライン

Step 5: アレンジ (2時間)
Intro作成
Build-up
Drop
Break
Outro

Step 6: ミキシング (3時間)
音量バランス
EQ
Compressor

Step 7: マスタリング (1時間)
Master にLimiter
最終調整

Step 8: 書き出し (10分)
WAV/MP3
ID3タグ

合計: 約8時間
```

### セッション管理

```
セッションの長さ:

短期集中:
25分作業 + 5分休憩 (ポモドーロ)

標準:
90分作業 + 15分休憩

長時間:
3時間 (限界)

推奨:
90分セッション × 3回 / 日
= 約5時間

休憩の重要性:

耳の疲労:
90分で判断力低下
→ 休憩必須

集中力:
長時間は非効率

新鮮な耳:
休憩後に聴き直す
→ 改善点が見える
```

## テンプレート活用術

**毎回の手間を省く:**

### 基本テンプレート

```
Techno Template:

トラック構成:
1. Kick (+ Compressor)
2. Bass (+ Filter)
3. Hi-Hat
4. Percussion
5. Synth 1
6. Synth 2
7. FX
8. Vocal (空)

Return:
A. Reverb (Valhalla Reverb)
B. Delay (Echo)

Master:
Limiter

設定:
BPM: 128
Key: Am
Quantize: 1/16

保存:
File > Save Live Set as Default Set
→ "Techno Template"
```

### 使い方

```
新規プロジェクト:

File > New Live Set from Template
→ "Techno Template" 選択

効果:
トラック作成不要
すぐ制作開始

時短:
5分 → 30秒
```

### 複数テンプレート

```
用途別:

Techno Fast (128 BPM)
Techno Slow (125 BPM)
House (124 BPM)
Minimal (125 BPM)
Demo (録音用)
DJ Set (Arrangement用)

切り替え:
プロジェクトに応じて選択
```

## 時短テクニック集

**プロの技:**

### Browser高速検索

```
検索窓:

Cmd+F (Browser選択)
→ "kick" 入力
→ Enter

タグ:
Browser > Tags
→ Dark, Warm, Analog 等でフィルター

お気に入り:
右クリック > Add to Favorites
→ すぐアクセス

Recent:
Browser > Places > Recent
→ 最近使ったサンプル
```

### 複製テクニック

```
Cmd+D:
選択クリップ複製

Cmd+D × 3:
4回連続押し
→ 4個複製
→ 32小節が一瞬

Cmd+D + Cmd+E:
複製 + 編集
→ バリエーション作成
```

### グループ活用

```
複数トラック選択:
Shift+クリック

Cmd+G:
グループ化

効果:
まとめて音量調整
まとめてエフェクト
Mixer がすっきり

用途:
Drums グループ
Synths グループ
```

### Freeze Track

```
重いトラック:
右クリック > Freeze Track

効果:
CPU負荷激減
オーディオ化

解除:
Unfreeze
→ 再編集可能

いつ使う:
プロジェクトが重くなったら
10トラック以上ある場合
```

### 朝の制作 (2時間)

```
08:00-08:10: 準備
Ableton起動
プロジェクト開く
コーヒー淹れる

08:10-09:40: 制作 (90分)
新しいアイデア
集中タイム
休憩なし

09:40-09:50: 休憩
外の空気
ストレッチ

09:50-10:00: 保存・整理
Cmd+S
バックアップ確認
次回のTODO記録
```

### 夜の制作 (3時間)

```
20:00-21:30: Session 1 (90分)
ミキシング
細かい調整

21:30-21:45: 休憩

21:45-23:15: Session 2 (90分)
アレンジ
完成させる

23:15-23:30: 保存・聴き直し
Cmd+S
通し再生
満足度チェック
```

```
10:00-11:30: Morning Session

11:30-12:00: 休憩+ランチ

制作継続

13:30-14:00: 休憩

14:00-15:30: Afternoon Session 2
完成させる

15:30-16:00: レビュー
週の振り返り

## よくある質問

### Q1: ショートカットを覚えられない

**A:** 少しずつ

```
Week 1: 10個
Week 2: +5個
Week 3: +10個
Week 4: 全て

コツ:
使いながら覚える
壁紙にする
毎日触る

3ヶ月後:
無意識に使える
```

### Q2: どのテンプレートを使うべき？

**A:** 自分で作る

```
既存テンプレート:
他人のワークフロー
自分に合わない

自作テンプレート:
自分のワークフロー
完璧にフィット

方法:
普通に曲作る
→ 良かった構成を保存
→ テンプレート化
```

### Q3: 制作が遅い

**A:** ワークフロー見直し

```
チェック:

ショートカット:
使ってる？

活用してる？

完璧主義:
80%で次に進む

改善:

タイマー:
各ステップに時限設定

2時間で完成させる
