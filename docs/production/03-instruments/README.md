```

20-30%
特殊な音色のみ

## 音源の種類と使い分け

**シンセサイザーの3大方式:**

### 1. 減算式（Subtractive）

```
原理:

豊富な倍音 → フィルターで削る = 音色

代表:
Analog

特徴:
シンプル
温かい
アナログ的

用途:
パッド
ビンテージサウンド
```

### 2. FM式（Frequency Modulation）

```
原理:

オシレーター同士で変調 = 複雑な音色

代表:
Operator

特徴:
金属的
明るい
複雑

用途:
ベル
ブラス
リード
```

### 3. ウェーブテーブル式

```
原理:

波形テーブルをスキャン = 変化する音色

代表:
Wavetable

特徴:
モダン
多彩
Techno/House向き

用途:
ベース
リード
パッド
全て
```

### 使い分けチャート

```
作りたい音:

ベースライン:
→ Wavetable (第一選択)
→ Operator (金属的なベース)
→ Analog (温かみ)

リード:
→ Wavetable (モダン)
→ Operator (明るい)

パッド:
→ Wavetable (厚み)
→ Analog (温かみ)

ドラム:
→ Drum Rack (必須)

サンプル加工:
→ Sampler

ワンショット:
→ Simpler
```

## 重要な概念

### シンセサイザーの基本構造

```
信号の流れ:

Oscillator (オシレーター):
音の元
波形生成

↓

Filter (フィルター):
音色調整
明るさ・暗さ

↓

Amplifier (アンプ):
音量調整

↓

Effects (エフェクト):
Reverb, Delay等

制御:

Envelope (エンベロープ):
時間変化
ADSR

LFO (Low Frequency Oscillator):
周期的変化
ビブラート、トレモロ

Modulation (モジュレーション):
パラメーター間の接続
```

### 音作りの3ステップ

```
Step 1: 音色選択 (1分)

プリセットから探す:
近い音色

または:
ゼロから作る

Step 2: 調整 (5-10分)

Filter Cutoff:
明るさ調整

Envelope:
Attack, Decay調整

LFO:
動き追加

Step 3: エフェクト (5分)

Reverb:
空間

Delay:
奥行き

EQ:
最終調整

合計: 10-15分/音色
```

## よくある質問

### Q1: Wavetable vs Operator、どっち使う？

**A:** 迷ったらWavetable

```
Wavetable優先:

用途:
ベース、リード、パッド
全般的に使える

理由:
直感的
モダンな音
Techno/Houseに最適

Operator使用:

用途:
ベル、ブラス
金属的な音

理由:
Wavetableで作れない音色
```

### Q2: プリセット使うのは悪い？

**A:** 全く問題なし

```
プロの実態:

プリセット使用率:
50-70%

理由:
時間節約
高品質
十分使える

カスタマイズ:

プリセット選択:
70%完成

微調整:
Filter, Envelope
20-30%変更

完成:
オリジナル音色

推奨:

初心者:
プリセット100%でOK

中級者:
プリセット + 調整

上級者:
ゼロから作成 + プリセット併用
```

### Q3: 外部音源必要？

**A:** 不要

```
Ableton付属で十分:

Wavetable:
非常に高品質

Drum Rack:
無限の可能性

理由:

プロ制作:
付属音源70-80%

外部音源:
20-30%
特殊な音色のみ

推奨:

最初の6ヶ月:
付属音源のみ

理由:
使いこなすのに時間かかる

6ヶ月後:

物足りなければ:
Serum, Massive X等検討
