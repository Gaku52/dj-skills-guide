## なぜPresets活用が重要なのか

**時間の効率化:**

```
制作時間配分:

音色作成:
ゼロから → 30-60分/音色

プリセット使用:
選択 → 5分
調整 → 5-10分
合計 → 10-15分

差:
3-4倍速い

プロの実態:

プリセット使用率:
50-80%

ゼロから作成:
20-50%
特殊音色のみ

理由:

効率:
時間節約

品質:
プロが作った音色
高品質

創造性:
音作りより
楽曲構成に集中

誤解:

「プリセット = 悪」:
間違い

「ゼロから = プロ」:
間違い

真実:
プロほどプリセット活用
時間を効率的に使う
```

## プリセット改変

**70%から100%へ:**

### 基本アプローチ

```
プリセット選択:
70%完成

改変:
Filter Cutoff調整
Envelope調整
LFO追加

結果:
100%オリジナル

推奨パラメーター:

1. Filter Cutoff:
   明るさ調整
   最も効果的

2. Envelope:
   Attack, Release
   時間変化

3. Unison/Voices:
   厚み調整

4. Effects:
   Reverb, Delay

5. Modulation:
   LFO追加

避けるべき:

Oscillator変更:
音色根本変わる

Algorithm変更:
別の音色に

推奨:
微調整のみ
10-30%変更
```

### 改変例: Techno Bass

```
プリセット選択:

Wavetable > Bass > "Sub Bass"

改変:

1. OSC 2追加:
   Fat Saw
   Detune: +7 cent

2. Filter:
   Cutoff: 800 Hz → 1200 Hz
   明るく

3. Filter Envelope:
   Amount: +50%
   Decay: 600 ms

4. Unison:
   Voices: 2
   Detune: 15%

結果:
太く、明るいTechno Bass

時間:
10分

保存:
"My Techno Bass"
```

## サウンドデザイン基礎

**ゼロから作る:**

### 3ステップアプローチ

```
Step 1: 基本音色 (30%)

Oscillator選択:
Saw / Square / Sine

例:
Wavetable > Basic > Saw

Step 2: フィルター (40%)

Cutoff調整:
明るさ決定

Envelope:
時間変化

例:
Cutoff: 1000 Hz
Envelope: Decay 500ms

Step 3: 仕上げ (30%)

Unison:
厚み

Effects:
空間

例:
Unison: 2 Voices
Reverb: 20%

合計時間:
30-45分/音色

推奨:

初心者:
プリセットから

中級者:
プリセット + 改変

上級者:
ゼロから + プリセット併用
```

### 音色の3要素

```
要素1: 音程成分

Oscillator:
何の波形？

Pitch:
高い？低い？

要素2: 時間変化

Envelope:
Attack速い？遅い？

Release:
短い？長い？

要素3: 音色

Filter:
明るい？暗い？

Resonance:
鋭い？滑らか？

全て重要:
バランス
```

## Macro Knob カスタマイズ

**1つで複数制御:**

### Instrument Rack

```
作成:

1. Wavetable等をRackに:
   右クリック → Group

2. Instrument Rack化

3. Macro Knob表示:
   Show Macros

Map:

1. Macro 1選択

2. Map Mode: On

3. パラメータークリック:
   Filter Cutoff

4. Map

5. Range調整:
   Min/Max

例:

Macro 1:
- Filter Cutoff (0-100%)
- Reverb Wet (0-50%)

Macro 2:
- Envelope Decay (100-2000ms)
- LFO Rate (1/16 - 1Bar)

効果:
1つまみで複数制御

用途:

ライブ:
簡単操作

制作:
Automation簡単
```
