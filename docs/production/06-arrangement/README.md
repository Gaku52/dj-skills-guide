# Arrangement（アレンジメント）

楽曲の設計図を作ります。Intro・Buildup・Drop・Outroの完璧な配置でプロ楽曲を完成させます。

## この章で学ぶこと

- 曲構造の基本（8セクション）
- エネルギーカーブ管理（10段階）
- トランジション技法（4小節ルール）
- ビルドアップとドロップ（8小節定番）
- イントロとアウトロ（DJ対応32小節）
- アレンジメントワークフロー（完全な手順）
- ジャンル別アレンジメント
- タイムライン管理

## Arrangementの基本構造

**8つのセクション:**

### 主要セクション

```
1. Intro (イントロ):

長さ: 32小節
目的: DJ対応、導入
エネルギー: 2-3/10

要素:
最小限
Kick・Hi-Hat・Pad
ゆっくり追加

2. Verse (ヴァース):

長さ: 16-32小節
目的: メインメロディー前
エネルギー: 4-5/10

要素:
Bass追加
Lead控えめ
準備段階

3. Buildup (ビルドアップ):

長さ: 8小節 (定番)
目的: 緊張感・期待感
エネルギー: 5 → 9/10

要素:
Filter上昇
White Noise
Snare Roll
Kick消える (最後)

4. Drop (ドロップ):

長さ: 16-32小節
目的: 最大の盛り上がり
エネルギー: 10/10

要素:
全要素
最大音量
Kick復帰
Bass全開

5. Breakdown (ブレイクダウン):

長さ: 16-32小節
目的: 休息・変化
エネルギー: 3-4/10

要素:
Kick・Bass消える
Vocal・Padのみ
静寂

6. Buildup 2:

長さ: 8小節
目的: 2回目の盛り上がりへ
エネルギー: 4 → 9/10

7. Drop 2:

長さ: 16-32小節
目的: 最終盛り上がり
エネルギー: 10/10

要素:
Drop 1 + 追加要素
より豪華

8. Outro (アウトロ):

長さ: 32小節
目的: DJ対応、終了
エネルギー: 10 → 2/10

要素:
徐々に減る
Kick・Hi-Hat残る
次の曲へ
```

## 8小節ルール

**基本原則:**

### なぜ8小節？

```
理由1: DJミックス

DJ:
8小節・16小節単位
ミックスしやすい

結果:
クラブで使われる

理由2: 人間の認知

聴き手:
8小節 = 自然な区切り
記憶しやすい

理由3: 業界標準

全プロ:
8小節・16小節・32小節
統一

推奨:

全セクション:
8の倍数

Intro: 32小節
Verse: 16 or 32小節
Buildup: 8小節
Drop: 16 or 32小節
Breakdown: 16 or 32小節
Outro: 32小節

例外:

実験的:
7小節・9小節
可能だが難しい
```

## トランジション技法

**セクション間:**

### 4小節ルール

```
最後の4小節:

変化を予告

方法:

1. Filter上昇:
   Bar 5-8: Cutoff上昇

2. Fill (フィル):
   Bar 7-8: Snare Roll

3. 要素削除:
   Bar 7-8: Bass消える

4. FX追加:
   Bar 7-8: Riser

例 (Verse → Buildup):

Bar 61-64 (Verse最後):
Filter徐々に上昇
White Noise出現

Bar 65 (Buildup開始):
明確な変化
全要素変化

効果:
自然な流れ
予測可能
聴きやすい
```

## よくある失敗

### 1. セクション長すぎ

```
問題:
Intro 64小節
飽きる

解決:

Intro: 32小節最大
Verse: 16-32小節
Drop: 32小節最大

理由:
聴き手の集中力
```

### 2. エネルギー変化なし

```
問題:
ずっとエネルギー10/10
疲れる

解決:

Breakdown:
必ず挿入
エネルギー3-4/10

頻度:
2-3分ごと休息
```

### 3. DJ対応なし

```
問題:
Intro 8小節のみ
DJが使えない

解決:

Intro: 32小節
Outro: 32小節
Kick・Hi-Hat・Bassのみ

結果:
クラブで使われる
```

### 4. 8小節ルール無視

```
問題:
Buildup 7小節
DJミックス困難

解決:

全セクション:
8の倍数

例外:
ほぼなし
```

## チェックリスト

### 構成

```
□ 合計時間 4-6分
□ 全セクション 8の倍数
□ Intro 32小節
□ Outro 32小節
□ Buildup 8小節
□ Drop 16-32小節
□ Breakdown あり
□ マーカー配置済み
```

### エネルギー

```
□ エネルギーカーブ描いた
□ 2回のピークあり
□ 休息 (Breakdown) あり
□ 緩やかな上昇
□ 緩やかな下降
□ 単調でない
```

### DJ対応

```
□ Intro Kick・Hi-Hat・Bass のみ (最初)
□ Intro 32小節
□ Outro Kick・Hi-Hat・Bass のみ (最後)
□ Outro 32小節
□ Intro・Outro 同じ要素
□ ループ可能
```

```
□ 4小節ルール適用
□ Filter上昇 (Buildup前)
□ Fill追加 (セクション境界)
```
