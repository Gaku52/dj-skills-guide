## Aセクション・Bセクション

**構造の基本:**

### A-B-A構造

```
定義:

A: メインセクション
B: 対比セクション

パターン:
A → B → A

例:

Verse (A)
→ Chorus (B)
→ Verse (A)

メリット:

対比:
A vs B
明確

統一感:
Aで始まり
Aで戻る

円環構造:
美しい

実践 (Pop):

Intro (A)
→ Verse 1 (A)
→ Chorus 1 (B)
→ Verse 2 (A)
→ Chorus 2 (B)
→ Bridge (C)
→ Chorus 3 (B)
→ Outro (A)

構造:
A-A-B-A-B-C-B-A

多様:
飽きない
```

### Techno でのA-B

```
A: Drop
B: Breakdown

パターン:

Intro (導入)
→ Verse (準備)
→ Buildup (移行)
→ Drop (A)
→ Breakdown (B)
→ Buildup 2 (移行)
→ Drop 2 (A)
→ Outro (終了)

構造:
導入-準備-A-B-A-終了

A特徴:
エネルギー高
複雑
全要素

B特徴:
エネルギー低
シンプル
休息

対比:
A vs B
劇的
```
