# Song Structure（曲構造）

8つのセクションで完璧な楽曲を設計します。Intro・Verse・Buildup・Drop・Breakdownの配置を完全マスターします。

## この章で学ぶこと

- 8つのセクション詳細
- セクション別役割
- 長さの決め方
- Techno/House/Hip Hop構成
- タイムライン設計
- セクション配置戦略
- よくあるパターン

---

## なぜSong Structureが重要なのか

**楽曲の骨格:**

```
構造なし:

結果:
ランダム
方向性不明
聴きにくい

構造あり:

結果:
計画的
ストーリー
聴きやすい

プロとアマの差:

アマ:
思いつき配置
展開なし
単調

プロ:
計算された構造
明確な展開
2回のピーク

真実:

「名曲」=
完璧な構造

例:

有名曲:
全て同じ構造
8小節ルール
DJ対応

使用頻度:

100%
全楽曲必須
```

## 8つのセクション

**基本構成:**

### 1. Intro（イントロ）

```
目的:

楽曲への導入
DJ対応
雰囲気作り

長さ:

Techno/House: 32小節 (必須)
Hip Hop: 8小節
Pop: 4-8小節

理由:
DJミックス 16-32小節必要

エネルギー:

2-3/10
低い
徐々に上昇

要素:

最小限:
Bar 1-8: Kick・Hi-Hat
Bar 9-16: Bass追加
Bar 17-24: Percussion追加
Bar 25-32: Pad追加

メロディー:
なし (Intro)

Vocal:
なし

重要ルール:

Intro = Outro
同じ要素推奨

理由:
DJがループ可能
ミックスしやすい

Techno Intro例:

Bar 1-8:
Kick (4つ打ち)
Hi-Hat (16分音符)

Bar 9-16:
Bass追加
シンプルパターン

Bar 17-24:
Shaker追加
Percussion追加

Bar 25-32:
Pad追加
雰囲気作り
→ Verse へ
```

### 2. Verse（ヴァース）

```
目的:

メインセクション前
ストーリー展開
Vocal (ある場合)

長さ:

Techno/House: 16-32小節
Hip Hop: 16小節 (典型的)
Pop: 8-16小節

エネルギー:

4-5/10
中程度
準備段階

要素:

ほぼ全て:
Kick・Bass・Drums
Vocal・Lead (控えめ)
Pad・FX

まだ控えめ:
Leadフル開放なし
Filter少し暗め

Hip Hop Verse:

特徴:
ラップ
16小節 (8+8)
シンプルなBeat

Techno Verse:

特徴:
メロディー控えめ
グルーヴ重視
32小節

役割:

Buildup への準備
聴き手を温める
エネルギー徐々に上昇
```

### 3. Buildup（ビルドアップ）

```
目的:

緊張感
期待感
Drop への準備

長さ:

8小節 (最も一般的)
16小節 (長め)
4小節 (短め、まれ)

推奨: 8小節

エネルギー:

5 → 9/10
急上昇
最大の期待感

要素:

Filter上昇:
Cutoff 300 Hz → 8000 Hz
Resonance 20% → 70%

White Noise:
Riser
徐々に大きく

Snare Roll:
Bar 7-8
16分音符・32分音符

Kick:
Bar 8.4: 消える (瞬間停止)
または維持

Automation:

Send上昇:
Reverb 25% → 60%

Volume上昇:
FX徐々に大きく

8小節Buildup構成:

Bar 1-4 (Phase 1):
Filter 300 → 1000 Hz
White Noise出現

Bar 5-6 (Phase 2):
Filter 1000 → 5000 Hz
Snare細かく

Bar 7-8 (Phase 3):
Filter 5000 → 8000 Hz
Snare Roll全開
Bar 8.4: Kick消える

→ Drop!

重要:

最後の瞬間:
全て停止 (0.25拍)
または
Lowpass Filter 0 Hz

効果:
最大のインパクト
```

### 4. Drop（ドロップ）

```
目的:

最大の盛り上がり
解放感
楽曲のハイライト

長さ:

16-32小節
通常: 16小節 (1st Drop)
24-32小節 (2nd Drop)

エネルギー:

10/10
最大

要素:

全て:
Kick (復帰・全開)
Bass (最大音量)
Lead (全開)
Drums (全て)
Vocal (ある場合)
FX

Filter:

Bypass
または
Cutoff全開 (8000 Hz+)

1st Drop vs 2nd Drop:

1st Drop (Bar 73-88):
16小節
標準

2nd Drop (Bar 129-160):
32小節
より長い
FX追加
より豪華

Drop構成例:

Bar 1-8:
全要素
最大音量

Bar 9-16:
継続
または
要素少し変化

Bar 17-24 (2nd Dropのみ):
FX追加
Vocal追加

Bar 25-32 (2nd Dropのみ):
徐々に Outro へ移行

重要:

Drop = 楽曲の顔
最も記憶に残る
最高の音質必須
```

### 5. Breakdown（ブレイクダウン）

```
目的:

休息
対比
感動・空気感

長さ:

16-32小節
通常: 16小節

エネルギー:

3-4/10
低い
休息

要素:

最小限:
Vocal (ある場合)
Pad
FX・Ambience

削除:
Kick (完全に消える)
Bass (消える)
Drums (ほぼ消える)

効果:

対比:
Drop → Breakdown
10/10 → 3/10
劇的変化

感情:
静寂
感動的
空間

次への準備:
Buildup 2 へ
再びエネルギー上昇

Breakdown構成例:

Bar 1-8:
Kick・Bass消える
Vocal・Padのみ
静寂

Bar 9-16:
徐々に要素追加
Hi-Hat復帰
Percussion復帰
→ Buildup 2 へ

ジャンル別:

Techno:
32小節Breakdown
ゆっくり

House:
16小節Breakdown
Vocal中心

Hip Hop:
Breakdownなし
または Bridge
```

### 6. Bridge（ブリッジ）

```
目的:

変化
新しいセクション
Chorus への移行

使用:

Hip Hop・Pop: よく使う
Techno・House: まれ

長さ:

8小節
16小節

エネルギー:

5-6/10
中程度

要素:

異なるコード進行
異なるメロディー
変化

Hip Hop Bridge例:

Bar 1-8:
コード変化
メロディー変化
Vocal変化
→ Chorus へ

Techno:

Breakdownが代わり
Bridge まれ
```

### 7. Chorus（コーラス）

```
目的:

サビ
最も記憶に残る
繰り返し

使用:

Pop・Hip Hop: 必須
House: よく使う (Vocal)
Techno: まれ (Drop が代わり)

長さ:

8小節
16小節

エネルギー:

8-10/10
高い

要素:

全て
Vocal (キャッチー)
メインメロディー

繰り返し:

通常 2-3回
曲中で繰り返す

例:

Intro → Verse → Chorus
→ Verse → Chorus
→ Bridge → Chorus
→ Outro
```

### 8. Outro（アウトロ）

```
目的:

楽曲の終了
DJ対応
次の曲へ

長さ:

Techno/House: 32小節 (必須)
Hip Hop: 8小節
Pop: 4-8小節

エネルギー:

10 → 2/10
徐々に下降

要素:

徐々に削除:
Bar 1-8: Lead・Vocal消える
Bar 9-16: Pad・FX消える
Bar 17-24: Percussion消える
Bar 25-32: Kick・Hi-Hat・Bass のみ

最後:
Kick・Hi-Hat・Bass
次の曲とミックス可能

Intro との関係:

Outro = Intro (逆)
同じ要素

理由:
DJがループ可能
Outro → 別曲 Intro
シームレス

Techno Outro例:

Bar 145-152:
Lead消える
Vocal消える

Bar 153-160:
Pad消える
FX消える

Bar 161-168:
Percussion消える

Bar 169-176:
Kick・Hi-Hat・Bass のみ
ループ可能
→ 次の曲へ
```

## セクション配置戦略

**原則:**

### 1. 対比の法則

```
高 → 低:
Drop → Breakdown
劇的

低 → 高:
Breakdown → Buildup → Drop
感動的

繰り返し:
同じエネルギー続かない

推奨パターン:

低 (Intro)
→ 中 (Verse)
→ 高 (Drop)
→ 低 (Breakdown)
→ 高 (Drop 2)
→ 低 (Outro)

波型:
理想的
```

### 2. 8小節ルール

```
全セクション:
8の倍数

理由:
DJミックス
人間の認知
業界標準

例外:

Pop:
4小節あり

Hip Hop:
16小節Verse標準

Techno/House:
8小節厳守
```

### 3. 2回ピークルール

```
1回のみ:
物足りない

2回:
理想的
Drop 1・Drop 2

3回:
可能だが長くなる
7-8分

推奨:
2回のDrop
```

## まとめ

### Song Structure

```
□ 8つのセクション理解
□ Intro 32小節 (Techno/House)
□ Buildup 8小節
□ Drop 16-32小節
□ Breakdown 16-32小節
□ Outro 32小節 (DJ対応)
```

### セクション長さ

```
Techno: 6-7分 (192-224小節)
House: 4-5分 (152-176小節)
Hip Hop: 3-4分 (88-105小節)
```

### 重要原則

```
□ 8の倍数厳守
□ 対比の法則
□ 2回のピーク
□ DJ対応 Intro/Outro
□ Reference分析10曲
```
