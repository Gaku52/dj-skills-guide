**なぜ起こるか:**

### 原因1: 完璧主義

```
症状:

「完璧な曲」を目指す
100点以外ダメ
他人と比較
「プロレベルじゃないと」

結果:

何も作れない
始められない
途中で止まる

解決:

80%ルール
比較禁止
自分のペース
```

### 原因2: 選択肢過多

```
症状:

Plugin 100個
Sample 10万個
どれ使う？

結果:

決められない
探すだけで疲れる
制作できない

解決:

制限を設ける
1 Plugin
1 Sample Pack
シンプルに
```

```
症状:

毎日8時間制作
休みなし
睡眠不足

結果:

脳が疲れる
創造性ゼロ
ブロック

解決:

休養必須
週1-2日休み
睡眠7-8時間
```

### 原因4: インスピレーション不足

```
症状:

外出しない

結果:

新しいアイデア湧かない
マンネリ

解決:

新しい音楽
ライブ行く
散歩
```

### 原因5: プレッシャー

```
症状:

締め切り
「ヒット作らないと」

結果:

緊張
創造性低下
ブロック

解決:

プレッシャー解放
楽しむ
遊び心
```

### 原因6: 批判への恐怖

```
症状:

「批判されたら」

「恥ずかしい」
```
