# Production Workflow（制作ワークフロー）

アイデアから完成までの7つのフェーズを完全マスター。効率的な制作フローで完成率80%を実現します。

## この章で学ぶこと

- 7つのフェーズ
- 各フェーズの時間配分
- フェーズ移行基準
- 完全なワークフロー
- Ableton Liveでの実践
- プロの制作習慣
- 効率化テクニック

## 7つのフェーズ概要

**完全ワークフロー:**

### フェーズ構成

```
Phase 1: Idea（アイデア）
時間: 1-2時間
目的: コンセプト・BPM・Key決定
完成物: 8-16小節のスケッチ

Phase 2: Drums & Bass（ドラムとベース）
時間: 2-3時間
目的: リズムとベースライン
完成物: グルーヴ確立

Phase 3: Harmony（ハーモニー）
時間: 2-3時間
目的: コード・Pad
完成物: ハーモニー完成

Phase 4: Melody（メロディー）
時間: 2-3時間
目的: Lead・Vocal
完成物: 全要素完成

Phase 5: Arrangement（アレンジメント）
時間: 3-5時間
目的: 曲構造・展開
完成物: 完全な楽曲

Phase 6: Mixing（ミキシング）
時間: 3-5時間
目的: バランス・空間
完成物: ミックス完成

Phase 7: Mastering（マスタリング）
時間: 1-2時間
目的: 最終調整・ラウドネス
完成物: リリース可能

合計: 14-23時間

目標: 20時間以内
```

## プロの制作習慣

### 効率化テクニック

```
1. Template活用

Mixing Template:
全TrackにEQ・Comp・Send配置
各曲で使い回し

Drum Rack Template:
完璧なドラムセット保存
即座に使用

Mastering Chain:
EQ・Comp・Limiter保存

効果:
制作時間 30%短縮

各Phase:
タイマー設定
終了したら次へ

完璧主義防止:
80%で次のPhase
100%は存在しない

3. Reference常時比較

各Phase:
Referenceと比較
方向性確認

Final Check:
必ず比較
同等レベル確認

4. 休憩必須

50分作業:
10分休憩

2時間作業:
30分休憩

効果:
集中力維持
品質向上

5. Session分け

Day 1: Phase 1-4（制作）
Day 2: Phase 5（アレンジ）
Day 3: Phase 6-7（Mix・Master）

耳休め:
翌日聴くと客観的

効果:
判断力向上
```

## よくある失敗

### 1. Phase飛ばし

```
NG:
Phase 1から
いきなりArrangement

問題:
基礎がダメ
全部作り直し

解決:
順番厳守
各Phase完成後に次へ
```

### 2. 時間かけすぎ

```
NG:
Phase 2に10時間

問題:
完璧主義
完成しない

解決:
各Phase時間制限
3時間経過したら次へ
```

### 3. Reference無視

```
NG:
自分の感覚だけで制作

問題:
方向性ずれる
プロレベル届かない

解決:
Reference常時比較
各Phase確認
```

### 4. Mixing早すぎ

```
NG:
Phase 2でMixing開始

問題:
時間の無駄
後で調整必要

解決:
Mixingは Phase 6のみ
それまで大まかなバランスのみ
```
