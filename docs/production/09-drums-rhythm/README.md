# Drums & Rhythm（ドラム・リズム）

楽曲の土台となるドラムとリズムを完全マスターします。キック・スネア・ハイハットの設計、グルーヴとスウィング、ジャンル別リズムパターンを習得し、踊れるビートを作ります。

## この章で学ぶこと

- ドラムプログラミングの基礎（MIDI打ち込み）
- キックデザイン（Sub、Punch、Click）
- スネア・クラップ設計（Body、Snap、Tone）
- ハイハット・パーカッション（Open、Close、Ghost Note）
- グルーヴとスウィング（Quantize、Groove Pool）
- ジャンル別リズムパターン（Techno、House、Hip Hop、D&B）
- ドラム処理とミキシング（EQ、コンプ、パラレル）
- Drum Rackの完全活用

## なぜDrums & Rhythmが重要なのか

**リズムの差:**

```
リズムなし:
楽曲の印象:
メロディのみ
フワフワ
踊れない

リスナー反応:
退屈
すぐスキップ
記憶に残らない

リズムあり:
楽曲の印象:
グルーヴ感
パワフル
踊れる

リスナー反応:
体が動く
リピート再生
クラブで盛り上がる

プロとアマの差:

アマ:
ドラム: サンプルそのまま 90%
グルーヴ: なし
リズム: 単調

プロ:
ドラム: レイヤリング + 加工 80%
グルーヴ: Swing、ベロシティ調整
リズム: 複雑、変化

結果の違い:
アマ: 平坦、退屈
プロ: グルーヴィー、踊れる、クラブで使える
```

## 学習の順序

### ステップ1: ドラムプログラミング基礎（Week 1-2）

**目標**: 基本的な4つ打ちビートを作れる

```
学習内容:
1. Drum Rackの使い方
   - Pad配置（C1=Kick）
   - サンプル読み込み
   - Chain設定

2. MIDI打ち込み
   - キック: 1拍目ごと（4つ打ち）
   - スネア: 2拍目、4拍目
   - ハイハット: 8分音符

3. ベロシティ設定
   - 100-127（強い）
   - 80-100（普通）
   - 60-80（弱い）

4. Grid設定
   - 1/4: キック
   - 1/8: ハイハット
   - 1/16: ハイハットバリエーション

実践:
- drum-programming.md を完全理解
- 4つ打ちビート10パターン作成
- 各BPM（120、125、128 BPM）
```

### ステップ2: キックデザイン（Week 3-4）

**目標**: パワフルなキックを作る

```
学習内容:
1. キックの3要素
   - Sub（50-100 Hz）: 腹に響く
   - Punch（100-200 Hz）: パンチ
   - Click（2-8 kHz）: アタック

2. キック作成方法
   - Wavetableでシンセキック作成
   - サンプルレイヤリング
   - Pitch Envelopeでドゥーン音

3. キック加工
   - EQ Eight: HP 30 Hz
   - Saturator: ボディ追加
   - Glue Compressor: 凝縮

4. ジャンル別キック
   - Techno: 硬い、アタック強い
   - Deep House: 柔らかい、Sub強い
   - Hip Hop: 808キック、ローエンド

実践:
- kick-design.md で4種類作成
- Techno、House、Hip Hop、Dubstep
- スペクトラム確認
```

### ステップ3: スネア・クラップ設計（Week 5-6）

**目標**: パンチのあるスネア・クラップを作る

```
学習内容:
1. スネアの3要素
   - Body（150-300 Hz）: 胴鳴り
   - Snap（1-3 kHz）: パリッ
   - Tone（200-500 Hz）: 音程感

2. スネア作成方法
   - ノイズ + Tone
   - サンプルレイヤリング
   - Transient処理

3. クラップ設計
   - 複数サンプル重ね
   - Reverb + HP Filter
   - ステレオ化

4. ジャンル別スネア
   - Techno: 硬い、短い
   - House: 温かい、リバーブ
   - Hip Hop: 太い、長い

実践:
- snare-clap.md で4種類作成
- ジャンル別特性理解
- EQ、Reverbで調整
```

### ステップ4: ハイハット・パーカッション（Week 7-8）

**目標**: グルーヴを作るハイハットとパーカッション

```
学習内容:
1. ハイハット種類
   - Closed: 8分音符、16分音符
   - Open: アクセント
   - Ghost Note: ベロシティ 40-60

2. パーカッション
   - シェイカー: 16分音符
   - コンガ: アクセント
   - クラベス: リズム補強

3. 配置とタイミング
   - Quantize: 1/16
   - Swing: 10-30%
   - マイクロタイミング調整

4. ステレオ配置
   - Closed HH: Center
   - Open HH: L/R 20%
   - Percussion: L/R 40-80%

実践:
- hihat-percussion.md で4種類作成
- Swingとベロシティ調整

```

**目標**: 踊れるグルーヴを作る


### サンプル vs シンセドラム

**サンプル（使用頻度: 70%）**

```
メリット:
- 即座に使える
- リアルな音
- 多様性

デメリット:
- 個性が少ない
- 調整範囲が限定的

推奨ソース:
- Splice（サブスク）
- Loopmasters
- r/Drumkits（Reddit）
- 無料: 99Sounds、Bedroom Producers Blog

ジャンル別:
Techno: Techno/Industrial Packs
House: Classic House Drums
Hip Hop: 808、909、SP-1200
```

**シンセドラム（使用頻度: 30%）**

```
メリット:
- 完全コントロール
- オリジナル
- ピッチ調整可能

デメリット:
- 時間がかかる
- 知識必要

作成方法:
キック: Wavetable（Sine + Pitch Envelope）
スネア: Wavetable（Noise + Tone）
ハイハット: Operator（FM、高周波）

推奨:
初心者: サンプル 90%、シンセ 10%
中級者: サンプル 70%、シンセ 30%
上級者: サンプル 50%、シンセ 50%
```

## よくある質問（FAQ）

**Q1: サンプルとシンセドラム、どちらを使うべきですか？**

```
A: 両方を組み合わせる

推奨比率:
初心者: サンプル 90%、シンセ 10%
中級者: サンプル 70%、シンセ 30%
上級者: サンプル 50%、シンセ 50%

使い分け:
キック: シンセ（Sub）+ サンプル（Punch）
スネア: サンプル 80%
ハイハット: サンプル 100%
パーカッション: サンプル 100%

理由:
サンプル: リアル、多様性
シンセ: コントロール性、オリジナル
```

**Q2: Swingはどれくらいかけるべきですか？**

```
A: ジャンルで決める

推奨値:
Techno: 0%（ストレート）
House: 10-20%
Hip Hop: 30-50%
Drum & Bass: 0-10%

調整方法:
1. 0%でスタート
2. 10%ずつ上げる
3. 「これだ！」で止める
4. A/Bテスト

注意:
30%以上: 強いスウィング（Hip Hopのみ）
66%: Triplet（シャッフル）
過度なSwingは危険
```

**Q3: ドラムが他のトラックに埋もれます。どうすれば？**

```
A: EQとコンプレッションで調整

手順:
1. ドラム以外をMute
2. ドラムのみでミックス
3. 他のトラックを1つずつ追加

EQ処理:
キック: 5 kHz +3 dB（アタック）
スネア: 200 Hz -3 dB（濁り削除）
ベース: 100-200 Hz -2 dB（キックに譲る）

コンプレッション:
Glue Compressor（ドラムバス）
Ratio: 2:1
Threshold: -10 dB

サイドチェイン:
ベース → キック（サイドチェイン）
Ratio: 4:1
```

**Q4: プロのようなパンチのあるドラムを作るには？**

```
A: トランジェント処理 + パラレルコンプ

トランジェント処理:
1. Transient Shaper（Max for Live）
2. Attack: +5〜+10 dB
3. Sustain: -3〜-5 dB

パラレルコンプレッション:
1. Return Track作成
2. Compressor:
   - Ratio: 10:1
   - Threshold: -20 dB
   - Attack: 1 ms
   - Release: 50 ms
3. Send量: 30%

サチュレーション:
Saturator: Drive 3-6 dB
Curve: A Bit Warmer
```

**Q5: ジャンル別のBPMがわかりません**

```
A: 標準BPM一覧

ジャンル別:
Techno: 128-135 BPM
House: 120-128 BPM
  - Deep House: 120-125
  - Tech House: 125-128
Hip Hop: 80-100 BPM
  - Boom Bap: 85-95
  - Trap: 140-160（ハーフタイム感覚で70-80）
Drum & Bass: 170-180 BPM
Dubstep: 140 BPM（ハーフタイム感覚で70）
Trance: 130-140 BPM
Ambient: 60-90 BPM

決め方:
1. 参考曲のBPM確認（Beatportに記載）
2. Abletonでタップテンポ
3. ±3 BPMで微調整
```

**Q6: ドラムパターンがマンネリ化します**

```
A: 変化の7つのテクニック

1. Ghost Note追加
   - ベロシティ 40-60
   - 32分音符

2. フィルイン
   - 4小節、8小節ごと
   - スネアロール
   - タムフィル

3. パーカッション追加
   - シェイカー、コンガ、クラベス
   - ステレオ配置

4. ハイハットバリエーション
   - Open/Close切り替え
   - Triplet

5. Kick削除
   - 1小節丸ごと削除
   - ビルドアップ前

6. Reverse
   - Crash Reverseでビルドアップ

7. Automation
   - HP Filter 200→8000 Hz
   - 8小節かけて開く

実践:
16小節を4小節ごとに変化
A → A' → B → A''
```

## まとめ

ドラムとリズムは、楽曲の土台であり、最も重要な要素です。

**重要ポイント:**

1. **4つ打ち基本**: Techno、Houseの基礎
2. **レイヤリング**: キック（Sub + Punch + Click）
3. **Swing**: House 10-20%、Hip Hop 30-50%
4. **ベロシティ**: アクセント120+、通常90-110、Ghost 40-70
5. **EQ処理**: キック HP 30 Hz、スネア HP 100 Hz
6. **パラレルコンプ**: Send 30%、Ratio 10:1
7. **ジャンル理解**: BPM、Swing、音色を合わせる
