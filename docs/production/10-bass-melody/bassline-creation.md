# ベースライン作成

**グルーヴの源を完全マスター**

ベースラインは楽曲のグルーヴを決定づける最重要要素の一つです。ドラムがリズムの骨組みなら、ベースはその肉付けであり、楽曲に動きとエネルギーを与えます。このガイドでは、シンプルな4つ打ちベースから複雑なシンコペーションまで、あらゆるベースラインの作成方法を完全マスターします。

---

## この章で学ぶこと

- ✅ ルート音の選び方とスケールの関係
- ✅ リズムパターン（4つ打ち、Offbeat、シンコペーション）
- ✅ 音程の使い分け（Root、5th、Octave、経過音）
- ✅ ドラムとの連動によるグルーヴ作り
- ✅ ジャンル別ベースライン（Techno、House、Dubstep、Hip Hop）
- ✅ Ableton Live でのMIDI打ち込みテクニック

**学習時間**: 3-5時間
**難易度**: ★★☆☆☆ 初級〜中級

---

## なぜベースラインが重要なのか

### DJの視点から

**DJとして**:
- ベースラインが**フロアを揺らす**瞬間を体感している
- キック + ベースの組み合わせで**エネルギーが爆発**する
- ベースラインの変化で**セクションが切り替わる**のを知っている

**プロデューサーとして**:
- その「グルーヴ」を**自分で作り出す**ことができる
- キックとベースの**周波数関係**を完全にコントロール
- ベースラインで**楽曲の個性**を確立できる

### プロの意見

> "ベースラインはドラムの次に重要。これが弱いと楽曲全体が弱くなる。"
> — **Deadmau5**（Techno/Progressive House）

> "Houseではベースラインがグルーヴの90%を決める。ファンキーなベースがなければHouseじゃない。"
> — **MK**（House Legend）

> "Dubstepはベースが全て。サブベース + Wobbleベースで空間を支配する。"
> — **Skrillex**

### 数字で見る重要性

| ジャンル | ベースライン重要度 | 制作時間配分 |
|---------|----------------|-----------|
| **Techno** | ★★★★☆ (80%) | 2-3時間 / 20時間 |
| **House** | ★★★★★ (100%) | 3-4時間 / 20時間 |
| **Dubstep** | ★★★★★ (100%) | 4-5時間 / 20時間 |
| **Hip Hop** | ★★★★★ (100%) | 2-3時間 / 20時間 |
| **Trance** | ★★★☆☆ (60%) | 1-2時間 / 20時間 |

## 1. ベースラインの基礎理論

### 1.1 ルート音（Root Note）

**ルート音とは**:
- コード進行の**基準となる音**
- 最も安定した音
- ベースラインの**土台**

**例**: C Majorコードの場合
```
C Major: C - E - G
ルート音: C ← これがベースの基本
```

### 1.2 スケールとの関係

**C Major Scale**:
```
C - D - E - F - G - A - B - C
1   2   3   4   5   6   7   8
```

**ベースラインで使用頻度の高い音程**:
```
Root (1st):   C    - 最も安定、使用頻度80%
5th:          G    - 2番目に安定、使用頻度15%
Octave (8th): C(高)- Rootの1オクターブ上、使用頻度5%
3rd:          E    - メロディック、使用頻度<5%
```

**重要原則**:
- **Root音が最も重要** (80%以上使用)
- 5thで変化をつける
- 3rd、7thは高度なテクニック（後述）

### 1.3 音域の選択

**ベース最適音域**:
```
Sub Bass:  C1 - C2  (33 Hz - 65 Hz)  ← サブウーファー専用
Mid Bass:  C2 - C3  (65 Hz - 130 Hz) ← メインのベース音域
High Bass: C3 - C4  (130 Hz - 260 Hz) ← メロディックベース
```

**ジャンル別推奨音域**:
```
Techno:   C2 (65 Hz)  - シンプル、力強い
House:    E2 (82 Hz)  - ファンキー、温かい
Dubstep:  C1 (33 Hz)  - 超低域、サブウーファー破壊級
Hip Hop:  F1 (44 Hz)  - 808スタイル
D&B:      G2 (98 Hz)  - 高速、明瞭
```

## 2. リズムパターン

### 2.1 4つ打ち（Four on the Floor）

**最もシンプルで力強いパターン**

```
Kick:  |X---|X---|X---|X---|
Bass:  |C---|C---|C---|C---|
       1   2   3   4   1

音符: 4分音符
用途: Techno、House基本
```

**特徴**:
- キックと完全に同期
- シンプルで力強い
- ミキシングが最も簡単

**Ableton Live での打ち込み**:
```
1. C2のMIDI Noteを配置
2. Grid: 1/4 (4分音符)
3. Velocity: 100 (一定)
4. Length: 1/4 (レガート)
```

### 2.2 Offbeat（裏拍）

**グルーヴを生み出すパターン**

```
Kick:  |X---|X---|X---|X---|
Bass:  |-C--|-C--|-C--|-C--|
       1 & 2 & 3 & 4 &

音符: 8分音符（裏）
用途: House、Disco
```

**特徴**:
- キックの**直後**に鳴る
- ファンキーなグルーヴ
- Houseの定番パターン

**Ableton Live での打ち込み**:
```
1. C2を配置（1小節に4個）
2. Grid: 1/8
3. 位置: 1.1.2, 1.2.2, 1.3.2, 1.4.2
4. Velocity: 100
```

### 2.3 シンコペーション（Syncopation）

**複雑なグルーヴ**

```
Kick:  |X---|X---|X---|X---|
Bass:  |C-C-|--C-|C---|--C-|
       1 & 2 & 3 & 4 &

音符: 16分音符の組み合わせ
用途: Funky House、UK Garage
```

**特徴**:
- リズムが予測不可能
- ダンサブル、グルーヴィー
- 上級者向け

**Ableton Live での打ち込み**:
```
1. Grid: 1/16
2. 不規則な配置（上記パターン参照）
3. Velocity: 80-120（バリエーション）
4. Swing: 10-20%
```

### 2.4 ドット音符（Dotted Notes）

**トライバルなグルーヴ**

```
Kick:  |X---|X---|X---|X---|
Bass:  |C-----|C-----|C-----|
       1     .     2     .

音符: 8分音符ドット（3連系）
用途: Minimal Techno、Tribal
```

**特徴**:
- 3連符的なグルーヴ
- トライバル、催眠的
- Minimal Technoで頻繁に使用

## 3. 音程の使い分け

### 3.1 Root Only（ルート音のみ）

**最もシンプル、最も力強い**

```
C Major進行: C - F - G - C

Bass:  |C---|C---|C---|C---|
       |F---|F---|F---|F---|
       |G---|G---|G---|G---|
       |C---|C---|C---|C---|

各小節でコードのRoot音だけを演奏
```

**使用例**:
- Techno 80%
- Minimal Techno 95%
- Deep House 60%

**特徴**:
- 周波数が安定
- ミキシングが簡単
- シンプルで力強い

### 3.2 Root + 5th

**変化をつける基本テクニック**

```
C Major進行: C - F - G - C

Bass:  |C-G-|C-G-|C-G-|C-G-|
       |F-C-|F-C-|F-C-|F-C-|
       |G-D-|G-D-|G-D-|G-D-|
       |C-G-|C-G-|C-G-|C-G-|

Root → 5th を交互に
```

**使用例**:
- House 70%
- Tech House 80%
- Progressive House 60%

**特徴**:
- グルーヴが生まれる
- Rootより5th高い音程
- 動きがあるが安定

### 3.3 Root + Octave

**ダイナミックな変化**

```
C Major進行: C - F - G - C

Bass:  |C2--|C3--|C2--|C3--|
       低   高   低   高

オクターブの移動で緊張と弛緩
```

**使用例**:
- Progressive House 50%
- Trance 40%
- Dubstep（Drop前）

**特徴**:
- エネルギーの変化
- ビルドアップに最適
- 音域が広がる

### 3.4 Walking Bass（ウォーキングベース）

**メロディックなベースライン**

```
C Major進行: C - F - G - C

Bass:  |C-D-|E-F-|G-A-|B-C-|
        順番に音階を上がる

スケール内を歩くように移動
```

**使用例**:
- Funky House 60%
- Nu-Disco 70%
- Jazz House 80%

**特徴**:
- メロディック
- 複雑なハーモニー
- ジャズ的

## 4. ドラムとの連動

### 4.1 キックと完全同期（Techno）

**最も基本的な関係**

```
Kick:  |X---|X---|X---|X---|
Bass:  |C---|C---|C---|C---|

キックとベースが完全に一致
→ 最も力強い
```

**ミキシングのポイント**:
```
Kick: 60 Hz ブースト +3 dB
Bass: 60 Hz カット -2 dB

サイドチェイン:
Bass → Compressor (Sidechain: Kick)
→ Kickが鳴る瞬間だけBassが下がる
```

### 4.2 キック直後（House）

**グルーヴを生む関係**

```
Kick:  |X---|X---|X---|X---|
Bass:  |-C--|-C--|-C--|-C--|
        ↑ キックの直後

ファンキーなグルーヴ
```

**ミキシングのポイント**:
- サイドチェイン不要（被らない）
- キックとベースが補完関係

### 4.3 スネアと連動

**複雑なグルーヴ**

```
Kick:  |X---|X---|X---|X---|
Snare: |----|X---|----|X---|
Bass:  |C-C-|--C-|C-C-|--C-|
           ↑ スネアと同期

ベースがスネアを強調
```

**使用例**:
- Funky House
- UK Garage
- Drum and Bass

### 4.4 ハイハットと連動（D&B）

**高速グルーヴ**

```
Kick:      |X-------|X-------|
Snare:     |----X---|----X---|
Hi-Hat:    |X-X-X-X-|X-X-X-X-| (16分)
Bass:      |C---C-C-|----C-C-|
             ↑   ↑       ↑ ↑
           ハイハットと部分的に同期

Drum and Bass 特有
```

## 5. ジャンル別ベースライン

### 5.1 Techno

**特徴**: シンプル、力強い、ハイプノティック

**基本パターン**:
```
BPM: 125-135
音域: C2 (65 Hz)
リズム: 4つ打ち
音程: Root Only 80%

Kick:  |X---|X---|X---|X---|
Bass:  |C---|C---|C---|C---|
```

**応用パターン（Minimal Techno）**:
```
Bass:  |C----|--C-|-----|C---|
        少ない音数で緊張感
```

**推奨音色**:
- Wavetable: Basic Shapes → Saw
- Filter: Low Pass 200 Hz
- Envelope: Attack 0 ms、Decay 300 ms

### 5.2 House

**特徴**: ファンキー、グルーヴィー、ダンサブル

**基本パターン（Deep House）**:
```
BPM: 120-125
音域: E2 (82 Hz)
リズム: Offbeat
音程: Root + 5th

Kick:  |X---|X---|X---|X---|
Bass:  |-E--|-B--|-E--|-B--|
        Offbeat + Root/5th交互
```

**応用パターン（Funky House）**:
```
Bass:  |E-E-B---|E-F#-G-A-|
        Walking Bass + シンコペーション
```

**推奨音色**:
- Wavetable: Basic Shapes → Saw
- Filter: Low Pass 500 Hz
- Resonance: 30%
- Slight Detune: 温かみ

### 5.3 Dubstep

**特徴**: 超低域、Wobble、ハーフタイム

**基本パターン**:
```
BPM: 140 (ハーフタイム感覚70)
音域: C1 (33 Hz) ← サブベース
リズム: ハーフタイム

Kick:  |X-------|--------|X-------|--------|
Snare: |--------|X-------|--------|X-------|
Bass:  |C-------|--------|C-------|--------|

2拍ごとにキックとベース
```

**Wobble Bass（Drop）**:
```
Bass:  |C-C-C-C-C-C-C-C-| (8分音符)
Filter: LFO 1/16でワブルワブル

Wobble = フィルターのLFOモジュレーション
```

**推奨音色**:
- Sub Bass: Wavetable Sine (C1)
- Wobble Bass: Wavetable Saw
- Auto Filter: LFO Rate 1/16
- Resonance: 70%

### 5.4 Hip Hop

**特徴**: 808ベース、シンプル、重低音

**基本パターン（Trap）**:
```
BPM: 140-160
音域: F1 (44 Hz) ← 808スタイル
リズム: シンコペーション

Kick:  |X-------|X-------|X-------|X-------|
808:   |F-------|--F-----|F-F-----|--------|
       長いリリース + シンコペーション
```

**808ベースの特徴**:
```
長いリリース: 500-1000 ms
ピッチベンド: +12 semitones
Glide/Portamento: 50-100 ms
```

**推奨音色**:
- Operator: Algorithm 1
- Operator A: Sine (Coarse -24)
- Operator B: Sine (Coarse 0)
- Long Release: 800 ms
- またはSampler: 808サンプル

### 5.5 Drum and Bass

**特徴**: 高速、複雑、Reese Bass

**基本パターン**:
```
BPM: 170-180
音域: G2 (98 Hz)
リズム: 複雑なシンコペーション

Kick:  |X-------|X-------|
Snare: |----X---|----X---|
Bass:  |G-G-|--G-|G---|--G-|
       16分音符の組み合わせ
```

**Reese Bass（D&B定番）**:
```
2つのDetuneしたSawを重ねる:
Osc 1: Saw (Detune 0)
Osc 2: Saw (Detune -7 cents)
→ 独特のうねり
```

**推奨音色**:
- Wavetable: 2 Oscillators
- Osc 1/2: Saw、Detune 7 cents
- Filter: Low Pass 1000 Hz
- Unison: 4 voices

## 6. Ableton Live 実践

### 6.1 MIDI打ち込み基礎

**新規MIDIトラック作成**:
```
1. Cmd+Shift+T (新規MIDIトラック)
2. Wavetable をドラッグ&ドロップ
3. ダブルクリックで空のMIDI Clipを作成
```

**Grid設定**:
```
4つ打ち: Grid 1/4
Offbeat: Grid 1/8
シンコペーション: Grid 1/16
```

**音符の配置**:
```
1. ピアノロールでC2をクリック
2. ドラッグして長さ調整
3. Cmd+D で複製
4. Option+ドラッグで移動コピー
```

### 6.2 ベロシティ調整

**グルーヴのためのベロシティ**:

```
強調したい音: Velocity 110-120
通常の音: Velocity 100
弱い音: Velocity 80-90

例（4つ打ち）:
|100-|100-|100-|100-| ← 均一（退屈）
|110-|95--|105-|95--| ← バリエーション（グルーヴ）
```

**Ableton でのベロシティ編集**:
```
1. MIDI Note選択
2. ピアノロール下部のVelocityバー
3. ドラッグで調整
4. Option+ドラッグで複数同時調整
```

### 6.3 Length（音符の長さ）

**レガート vs スタッカート**:

```
レガート（長い）:
|C---------|C---------|
滑らか、繋がる

スタッカート（短い）:
|C-|  |C-|  |C-|  |C-|
歯切れ良い、グルーヴィー
```

**ジャンル別推奨**:
```
Techno: レガート 90% (1/4音符長)
House: 中間 70% (1/8音符長)
Funky House: スタッカート 50% (1/16音符長)
```

**設定方法**:
```
1. MIDI Note選択
2. 右端をドラッグして長さ調整
3. または数値入力: 1.1.1 - 1.1.4 (1/4長)
```

### 6.4 Quantize（クォンタイズ）

**グリッドに整列**:

```
手弾きMIDI:
|C--C---|C-C----|（ズレている）

Quantize後:
|C---|C---|C---|C---|（完璧に整列）
```

**設定**:
```
1. MIDI Clip選択
2. Cmd+U (Quantize)
3. Quantize To: 1/16
4. Amount: 100% (完全整列)
```

**部分的Quantize（グルーヴ維持）**:
```
Amount: 50-80%
→ 人間的なグルーヴを残す
```

### 6.5 Swing

**グルーヴを生む**:

```
Swing 0%:
|C-C-C-C-| (均等)

Swing 33%:
|C--C-C--C| (跳ねる)

Swing 66%:
|C---CC---C| (大きく跳ねる)
```

**設定方法**:
```
1. Clip選択
2. Clip View → Notes → Groove
3. Swing: 10-20% (Techno/House)
4. Swing: 30-40% (Hip Hop)
```

## 7. 高度なテクニック

### 7.1 Ghost Notes（ゴーストノート）

**かすかに聞こえる装飾音**

```
Main Bass: |C-------|C-------|
Ghost:     |--c--|--c--|
           Velocity 40-60

メインの間に小さい音
```

**使用例**:
- Funky House
- Hip Hop
- Drum and Bass

**設定**:
```
Velocity: 40-60 (小さく)
Length: 1/32 (短く)
位置: オフグリッド（わずかにずらす）
```

### 7.2 Slide/Glide（スライド）

**音程が滑らかに移動**

```
C → E へスライド:
ポルタメント使用

|C~~~~~E|
  滑らか
```

**Ableton設定**:
```
Wavetable:
→ Glide: 100-200 ms
→ 次の音へ滑らかに移動
```

**使用例**:
- Hip Hop 808
- Acid Techno
- UK Garage

### 7.3 Pitch Bend（ピッチベンド）

**一時的な音程変化**

```
|C---→C#+1|
   ベンドアップ

MIDI Pitch Bend:
+12 semitones (1オクターブ上)
```

**使用例**:
- Hip Hop (808スライド)
- Dubstep (Wobble前)
- Techno (エフェクト)

**設定**:
```
1. Clip View → Envelopes
2. Pitch Bend
3. 描画: 0 → +12 → 0
```

### 7.4 レイヤリング

**複数のベースを重ねる**

```
Track 1: Sub Bass (20-60 Hz)
Track 2: Mid Bass (60-250 Hz)
Track 3: High Bass (250-500 Hz)

3つを同時に鳴らす
```

**周波数分離**:
```
Sub: Low Pass 80 Hz
Mid: Band Pass 100-300 Hz
High: High Pass 300 Hz
```

**詳細**: [08-sound-design/layering.md](../08-sound-design/layering.md)

## 8. ミキシングのポイント

### 8.1 EQ処理

**ベースラインの基本EQ**:

```
High Pass: 30 Hz (ランブル除去)
Low Shelf: 60 Hz +2 dB (サブベース)
Low Cut (Kick用): 80 Hz -2 dB (キックとの分離)
High Cut: 300 Hz (中域以上カット)
```

**Techno ベースEQ例**:
```
EQ Eight:
1. High Pass 30 Hz (24 dB/oct)
2. Bell 65 Hz +3 dB Q=1.0 (力強さ)
3. Low Pass 200 Hz (12 dB/oct) (明瞭さ)
```

### 8.2 コンプレッション

**ダイナミクスの均一化**:

```
Compressor:
Ratio: 3:1 - 4:1
Threshold: -15 dB
Attack: 30 ms (Techno)、5 ms (Funky)
Release: 100 ms (Auto推奨)
Makeup Gain: +2〜3 dB
```

**目的**:
- ベロシティのばらつきを減らす
- 安定した音量
- パンチを維持

### 8.3 サイドチェインコンプレッション

**キックとの共存**:

```
Bass Track:
→ Compressor
→ Sidechain: 1-Kick

設定:
Ratio: 4:1 - 8:1 (強め)
Attack: 0 ms
Release: 100-150 ms
Threshold: -20 dB
```

**効果**:
- キックが鳴る瞬間だけベースが下がる
- 周波数の衝突を回避
- パンチの強いキック

**詳細**: [04-effects/eq-compression.md](../04-effects/eq-compression.md)

### 8.4 Saturation

**倍音を追加して前に出す**:

```
Saturator:
Drive: 3-6 dB
Curve: Warm (温かい倍音)
Dry/Wet: 50-70%
```

**効果**:
- 小さいスピーカーでも聞こえる
- 倍音で存在感
- アナログ的な温かみ

**目標**: シンプルなベースラインを作成できる

**Day 3-4: Offbeat ベース**
1. E2でOffbeatベースライン

**Day 5-7: Root + 5th**
1. C Major進行でベースライン

## 10. よくある失敗と対処法

### 失敗1: ベースラインが聞こえない

**原因**:
- 音域が低すぎる（C1以下）
- ヘッドフォンで確認している
- キックとマスキングしている

**対処法**:
```
1. Mid Bass (C2-C3) も追加
2. クラブシステムで確認
3. EQ: 80-150 Hz をブースト +2 dB
4. Saturation で倍音追加
```

---

### 失敗2: キックとベースが喧嘩する

**原因**:
- 同じ周波数帯域（60-80 Hz）
- 同時に鳴っている

**対処法**:
```
サイドチェインコンプレッション:
Bass → Compressor (Sidechain: Kick)
Ratio 6:1、Attack 0 ms、Release 100 ms
```

---

### 失敗3: ベースラインが単調

**原因**:
- Root音だけ
- ベロシティが均一
- リズムが単純

**対処法**:
```
1. Root + 5th を使う
2. Velocity 80-120 でバリエーション
3. シンコペーション追加
4. Ghost Notes で装飾
```

---

### 失敗4: グルーヴがない

**原因**:
- Quantize 100%で完璧すぎる
- ドラムと関係ない

**対処法**:
```
1. Quantize Amount 50-80%
2. Swing 10-20%
3. キック/スネアと連動
4. ベロシティバリエーション
```
