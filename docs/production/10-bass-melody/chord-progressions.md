# コード進行

**楽曲の感情とハーモニーを決定する**

コード進行は楽曲の「感情の設計図」です。同じメロディでも、背後のコード進行次第で明るくも暗くも聞こえます。このガイドでは、音楽理論の基礎から定番コード進行10選、実践的な作成テクニックまで、プロレベルのハーモニー構築を完全マスターします。

## なぜコード進行が重要なのか

### DJの視点から

**DJとして**:
- コード進行の変化で**楽曲のセクションが切り替わる**
- ハーモニックミキシングで**キー（調）の重要性**を理解
- マイナーキーとメジャーキーで**雰囲気が変わる**ことを体感

**プロデューサーとして**:
- その「感情」を**自分で設計**できる
- コード進行だけで**楽曲の80%が決まる**
- ハーモニーで**独自性**を確立できる

### プロの意見

> "コード進行は楽曲の感情。I-V-vi-IVは永遠に機能する。"
> — **Max Martin**（ヒット量産プロデューサー）

> "Progressive Houseでは、コード進行がすべて。メロディは後から付ければいい。"
> — **Deadmau5**

> "同じ4つのコードで1000曲作れる。重要なのはボイシングとリズム。"
> — **Calvin Harris**

## 1. コード理論の基礎

### 1.1 トライアド（3和音）

**メジャーコード（明るい）**:
```
C Major:
  C - E - G
  Root - Major 3rd - Perfect 5th
  半音数: 0 - 4 - 7

特徴: 明るい、楽しい、安定
```

**マイナーコード（暗い）**:
```
A Minor:
  A - C - E
  Root - Minor 3rd - Perfect 5th
  半音数: 0 - 3 - 7

特徴: 暗い、悲しい、メランコリック
```

**違いは3rdだけ**:
```
C Major: C - E (Major 3rd, 4半音)
C Minor: C - Eb (Minor 3rd, 3半音)

→ たった1半音の違いで雰囲気が変わる
```

### 1.2 ダイアトニックコード

**C Major Scaleから作られる7つのコード**:

```
Scale: C - D - E - F - G - A - B - C

Chord I:   C - E - G  = C Major (CM)
Chord ii:  D - F - A  = D minor (Dm)
Chord iii: E - G - B  = E minor (Em)
Chord IV:  F - A - C  = F Major (FM)
Chord V:   G - B - D  = G Major (GM)
Chord vi:  A - C - E  = A minor (Am)
Chord vii: B - D - F  = B diminished (Bdim)
```

**パターン**:
```
Major Scale:
  I   ii  iii IV  V   vi  vii°
  Maj min min Maj Maj min dim

覚え方: 大小小大大小減
```

**機能**:
```
Tonic (安定):     I, iii, vi
Subdominant (準備): II, IV
Dominant (緊張):   V, vii°

進行の原則:
  Tonic → Subdominant → Dominant → Tonic
  (安定 → 準備 → 緊張 → 解決)
```

### 1.3 7thコード

**トライアド + 7th**:

```
C Major 7 (CMaj7):
  C - E - G - B
  Root - M3 - P5 - Major 7th

特徴: ジャジー、洗練、Progressive House
```

**その他の7thコード**:
```
C Dominant 7 (C7):    C - E - G - Bb (ブルース的)
C Minor 7 (Cm7):      C - Eb - G - Bb (ジャズ的)
C Minor Major 7 (CmM7): C - Eb - G - B (ドラマティック)
```

**使用頻度**:
```
EDM: Triadが90%、7thは10%
Jazz/Soul House: 7thが60%
```

## 2. 定番コード進行10選

### 2.1 I-V-vi-IV（最強の進行）

**C Majorの場合**:
```
I:  C Major  (C - E - G)
V:  G Major  (G - B - D)
vi: A minor  (A - C - E)
IV: F Major  (F - A - C)

進行: C → G → Am → F
```

**特徴**:
- ポップミュージック史上最も使用される
- 明るく、キャッチー
- ループしやすい

**使用例**:
- "Let It Be" - The Beatles
- "Don't Stop Believin'" - Journey
- "Someone Like You" - Adele
- "Wake Me Up" - Avicii

**EDMでの使用**:
- Progressive House 70%
- Pop EDM 80%
- Future Bass 60%

### 2.2 vi-IV-I-V（悲しげ）

**C Majorの場合**:
```
vi: A minor
IV: F Major
I:  C Major
V:  G Major

進行: Am → F → C → G
```

**特徴**:
- マイナーから始まる（暗い）
- 徐々に明るくなる（希望）
- エモーショナル

**使用例**:
- "All of Me" - John Legend
- "Faded" - Alan Walker

### 2.3 ii-V-I（ジャズの王道）

**C Majorの場合**:
```
ii: D minor  (D - F - A)
V:  G Major  (G - B - D)
I:  C Major  (C - E - G)

進行: Dm → G → C
```

**特徴**:
- ジャズで最も重要
- 強い解決感
- Sophisticatedな印象

**使用例**:
- Jazz House
- Nu-Disco
- Soulful House

### 2.4 I-IV-V（ロックの基本）

**C Majorの場合**:
```
I:  C Major
IV: F Major
V:  G Major

進行: C → F → G
```

**特徴**:
- シンプル、力強い
- ロック、ブルース
- 3コードだけ

**EDMでの使用**:
- Big Room House
- Electro House

### 2.5 i-VII-VI-V（マイナー進行）

**A Minorの場合**:
```
i:   A minor  (A - C - E)
VII: G Major  (G - B - D)
VI:  F Major  (F - A - C)
V:   E Major  (E - G# - B)

進行: Am → G → F → E
```

**特徴**:
- ダーク、ドラマティック
- ハードスタイル、メタル的
- Deep House、Techno

**使用例**:
- Deep House 50%
- Melodic Techno 60%

### 2.6 I-vi-IV-V（50's進行）

**C Majorの場合**:
```
I:  C Major
vi: A minor
IV: F Major
V:  G Major

進行: C → Am → F → G
```

**特徴**:
- 1950年代ポップ
- ノスタルジック
- 安定感

### 2.7 I-V-vi-iii-IV-I-IV-V（Canon進行）

**C Majorの場合**:
```
C → G → Am → Em → F → C → F → G

パッヘルベルのカノン
```

**特徴**:
- クラシック由来
- 非常に美しい
- Progressive House、Trance

### 2.8 i-VI-III-VII（Andalusian）

**A Minorの場合**:
```
i:   A minor
VI:  F Major
III: C Major
VII: G Major

進行: Am → F → C → G
```

**特徴**:
- スペイン、フラメンコ的
- エキゾチック
- Techno、Progressive House

### 2.9 I-bVII-IV（モーダル）

**C Majorの場合**:
```
I:    C Major
bVII: Bb Major（借用和音）
IV:   F Major

進行: C → Bb → F
```

**特徴**:
- ロック的
- パワフル
- Big Room House

### 2.10 i-iv-v（マイナー3コード）

**A Minorの場合**:
```
i:  A minor
iv: D minor
v:  E minor

進行: Am → Dm → Em
```

**特徴**:
- ダーク、ミニマル
- Techno、Deep House

## 3. Ableton Live 実践

### 3.1 MIDI打ち込み基礎

**新規トラックでコード作成**:
```
1. Cmd+Shift+T（新規MIDIトラック）
2. 音源: Wavetable, Analog, または Samplerでピアノ音色
3. 空のMIDI Clipを作成
```

**C Majorコードの打ち込み**:
```
1. ピアノロールで以下の3音を同時に配置:
   - C3
   - E3
   - G3

2. 長さ: 1小節（4拍）

3. Velocity: 100（全音同じ）
```

### 3.2 コード進行の打ち込み（I-V-vi-IV）

```
Bar 1: C Major
  - C3, E3, G3

Bar 2: G Major
  - G3, B3, D4

Bar 3: A minor
  - A3, C4, E4

Bar 4: F Major
  - F3, A3, C4

→ 4小節のループ
```

### 3.3 Scaleツール活用

**スケール設定**:
```
1. MIDI Clip選択
2. Clip View → Notes → Scale
3. Root: C
4. Scale: Major
5. Fold: On

→ C, D, E, F, G, A, Bだけ表示
```

**コード作成が簡単に**:
```
Fold Onの状態:
  - 縦に3音選択 → 自動的にコード
  - 間違った音が弾けない
```

### 3.4 Chord MIDIエフェクト

**コードを自動生成**:

```
1. MIDI Effect → Chord
2. Shift 1: +4 (Major 3rd)
3. Shift 2: +7 (Perfect 5th)

→ C を弾くと C-E-G (C Major) が鳴る
```

**設定例**:
```
Major Chord:
  Shift 1: +4
  Shift 2: +7

Minor Chord:
  Shift 1: +3 (Minor 3rd)
  Shift 2: +7

7th Chord:
  Shift 1: +4
  Shift 2: +7
  Shift 3: +11 (Major 7th)

→ 1音打ち込むだけでコード完成
```

## 4. コードボイシング

### 4.1 基本形vs転回形

**基本形（Root Position）**:
```
C Major: C3 - E3 - G3
         Root 3rd 5th

特徴: 安定、標準
```

**第1転回形（1st Inversion）**:
```
C Major: E3 - G3 - C4
         3rd 5th Root

ベース音: E
特徴: 軽やか、動きがある
```

**第2転回形（2nd Inversion）**:
```
C Major: G3 - C4 - E4
         5th Root 3rd

ベース音: G
特徴: 不安定、トランジションに使用
```

### 4.2 スムーズな声部進行

**近い音程で移動（Voice Leading）**:

**悪い例（ジャンプが大きい）**:
```
C Major: C3 - E3 - G3
F Major: F3 - A3 - C4  ← 全音が大きく移動
```

**良い例（最小限の移動）**:
```
C Major: C3 - E3 - G3
F Major: C3 - F3 - A3  ← C3は共通、E→F（1音）、G→A（2音）

→ スムーズ、自然な流れ
```

**原則**:
```
1. 共通音はそのまま維持
2. 他の音は最小限の移動
3. 2度（1-2音）以内の移動が理想
```

### 4.3 オープン vs クローズドボイシング

**クローズドボイシング（狭い）**:
```
C Major: C3 - E3 - G3
         1オクターブ以内

特徴: コンパクト、明瞭
用途: House、Pop
```

**オープンボイシング（広い）**:
```
C Major: C3 - G3 - E4
         1オクターブ以上

特徴: 広がり、エアリー
用途: Jazz、Progressive House
```

### 4.4 ダブリング

**ルート音を倍にする**:
```
C Major: C2 - C3 - E3 - G3 - C4
         ↑       ↑           ↑
         Root音を3回（低音、中音、高音）

特徴: 厚み、力強さ
用途: Big Room House、Anthemicなトラック
```

## 5. ジャンル別コード進行

### 5.1 Progressive House

**定番進行**:
```
Am - F - C - G (vi-IV-I-V)

特徴:
  - エモーショナル
  - マイナーから始まる
  - ビルドアップで盛り上がる

音色: Saw Pad, Pluck

BPM: 125-128

例: "Language" - Porter Robinson
```

### 5.2 Trance

**定番進行**:
```
C - Am - F - G (I-vi-IV-V)

または:

Am - F - C - G (vi-IV-I-V)

特徴:
  - エピック、感動的
  - 長いビルドアップ（16-32小節）
  - メジャー7thを追加

音色: Supersaw, Epic Pad

BPM: 138-140
```

### 5.3 Deep House

**定番進行**:
```
Am - Dm - G - C (i-iv-VII-III)

または:

Dm7 - G7 - CMaj7 - Am7 (ii7-V7-IMaj7-vi7)

特徴:
  - ジャジー
  - 7thコード必須
  - 温かみのある音色

音色: Rhodes, Warm Pad

BPM: 120-125
```

### 5.4 Techno

**定番進行**:
```
Am - Am - Am - Am (i-i-i-i)

または:

Am - F - Am - F (i-VI-i-VI)

特徴:
  - ミニマル
  - 変化が少ない
  - ダーク、催眠的

音色: Dark Pad, Acid

BPM: 125-135
```

### 5.5 Future Bass

**定番進行**:
```
C - Am - F - G (I-vi-IV-V)

または:

vi - IV - I - V（暗→明）

特徴:
  - 明るく、感動的
  - 7thコード、テンション
  - ボーカルとの相性◎

音色: Supersaw, Vocal Chops

BPM: 140-170
```

## 6. リズムとボイシング

### 6.1 4つ打ちコード（House）

```
Kick:  |X---|X---|X---|X---|
Chord: |C---|G---|Am--|F---|

各拍でコードチェンジ
リズム: 4分音符
```

### 6.2 Offbeat コード（Funky House）

```
Kick:  |X---|X---|X---|X---|
Chord: |-C--|-G--|-Am-|-F--|

Kickの直後にコード
リズム: 8分音符（裏拍）
```

### 6.3 シンコペーション

```
Chord: |C-C-|--G-|Am--|--F-|
       1 & 2 & 3 & 4 &

不規則なリズム
グルーヴィー
```

### 6.4 Pluck パターン（Progressive House）

```
C Majorコード（C-E-G）を分解:

|C-E-G-E-|C-E-G-C5-|

Arpeggioのように1音ずつ
音色: Pluck Synth
リズム: 16分音符
```

## 7. ミキシング

### 7.1 EQ処理

**コードの基本EQ**:
```
EQ Eight:

1. High Pass 100-200 Hz
   （ベースとの分離）

2. Low-Mid Cut 250-400 Hz -2 dB
   （マッディネス除去）

3. Presence Boost 1-2 kHz +1 dB
   （明瞭さ）

4. Air Boost 8-10 kHz +1 dB
   （輝き）
```

**ジャンル別**:
```
Deep House: 400 Hz周辺を温かくブースト
Trance: 4 kHz以上を明るくブースト
Techno: 2 kHz以下をダークにカット
```

### 7.2 サイドチェイン（必須）

**Kickとの共存**:
```
Chord Track:
→ Compressor
→ Audio From: 1-Kick

設定:
  - Ratio: 4:1
  - Threshold: -20 dB
  - Attack: 10 ms
  - Release: 150 ms

効果:
  - Kickが鳴る瞬間、コードが下がる
  - グルーヴ、ダンサビリティ
  - Progressive House必須
```

### 7.3 ステレオ幅

**コードの最適幅**:
```
Pad Chord: 100% (広く)
Pluck Chord: 60-80% (中央寄り)

Utility:
  - Width: 100% (Pad)
  - Width: 70% (Pluck)
```

### 7.4 Reverb/Chorus

**空間と厚み**:
```
Return Track A: Reverb
  - Hall 2.5s
  - Pre-Delay: 30 ms
  - Dry/Wet: 100%

Chord Send A: 30-40%

Return Track B: Chorus
  - Rate: 0.5 Hz
  - Depth: 30%

Chord Send B: 20% (温かみ)
```

## 9. よくある失敗と対処法

### 失敗1: コードが合わない音がする

**対処法**:
```
1. Scaleツールで In Key 確認
2. ダイアトニックコードのみ使用
3. 3rd（3度）の音を確認
   - Majorなら4半音（例: C→E）
   - Minorなら3半音（例: A→C）
```

---

### 失敗2: コード進行が単調

**対処法**:
```
1. リズムバリエーション（Offbeat、シンコペーション）
2. ボイシング変化（転回形）
3. 7thコード追加
4. 一部のコードをマイナーに（モーダル・インターチェンジ）
```

---

### 失敗3: ベースラインと合わない

**対処法**:
```
1. ベースはコードのRoot音を演奏
   C Major → ベースはC
2. または5th（G）を使用
3. 転回形の場合、ベース音を別トラックで指定
```

---

### 失敗4: コードが埋もれる

**対処法**:
```
1. EQ: 1-2 kHz +2 dB（明瞭さ）
2. サイドチェイン（Kickとの共存）
3. ステレオ幅100%（Pad）
4. Reverb/Chorus で空間を作る
```

## まとめ

### コード進行の核心

1. **ダイアトニックコード**: I, ii, iii, IV, V, vi, vii°
2. **定番進行**: I-V-vi-IV が最強
3. **ボイシング**: スムーズな声部進行
4. **リズム**: 4つ打ち、Offbeat、シンコペーション
5. **ミキシング**: サイドチェイン必須
