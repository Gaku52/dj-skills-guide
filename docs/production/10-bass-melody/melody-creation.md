# メロディ作成

**記憶に残る音楽の魂を作り出す**

メロディは楽曲の「顔」であり、聴き手が口ずさみ、記憶に残る最も重要な要素です。ドラムがリズム、ベースがグルーヴなら、メロディは楽曲の**感情**と**個性**を決定づけます。このガイドでは、音楽理論の基礎から実践的な作曲テクニックまで、プロレベルのメロディ作成を完全マスターします。

---

## この章で学ぶこと

- ✅ モチーフ（2-4小節の核）の作り方
- ✅ スケールとキーの実践的理解

- ✅ リズムとシンコペーション

- ✅ ベロシティとヒューマナイゼーション
- ✅ リファレンストラックからの学習法

**学習時間**: 4-6時間
**難易度**: ★★★☆☆ 中級

## なぜメロディ作成が重要なのか

### DJの視点から

**DJとして**:
- メロディで**観客の感情が動く**瞬間を体感
- **口ずさめる曲**がフロアを盛り上げる
- メロディの有無で**楽曲の記憶に残り方**が変わる

**プロデューサーとして**:
- その「感動」を**自分で作り出せる**
- 音楽理論を使って**無限のメロディ**を生成
- オリジナリティで**他と差別化**できる

### プロの意見

> "メロディがなければ、その曲は忘れられる。記憶に残るメロディこそが名曲を作る。"
> — **Avicii**

> "シンプルなメロディほど強い。C-D-E-Dの4音だけで名曲は作れる。"
> — **Calvin Harris**

> "メロディは感情。喜び、悲しみ、希望、全てメロディで表現できる。"
> — **Hans Zimmer**

### 数字で見る重要性

| ジャンル | メロディ重要度 | 制作時間配分 | 記憶に残る要因 |
|---------|-------------|-----------|-------------|
| **Trance** | ★★★★★ (100%) | 5-6時間 / 20時間 | メロディ80% |
| **Progressive House** | ★★★★★ (100%) | 4-5時間 / 20時間 | メロディ70% |
| **Techno** | ★★☆☆☆ (40%) | 1-2時間 / 20時間 | リズム60% |
| **Dubstep** | ★★★☆☆ (60%) | 2-3時間 / 20時間 | ドロップ50% |
| **House** | ★★★★☆ (80%) | 3-4時間 / 20時間 | メロディ60% |

## 1. メロディの基礎理論

### 1.1 メロディとは何か

**定義**:
```
メロディ = 単一の音が時間軸上で連続すること

要素:
  - 音程（Pitch）: C, D, E, F...
  - リズム（Rhythm）: いつ鳴るか
  - 長さ（Duration）: どれだけ伸びるか
```

**コードとの違い**:
```
コード: 複数の音を同時に鳴らす
メロディ: 1音ずつ順番に鳴らす

例:
  コード C Major: C + E + G (同時)
  メロディ: C → E → G → E (順番)
```

### 1.2 モチーフ（Motif）

**音楽の最小単位**:
```
モチーフ = 2-4小節の短いメロディ

例（「きらきら星」）:
  C C G G | A A G - |
  2小節のモチーフ

これを繰り返す:
  C C G G | A A G - | (モチーフA)
  F F E E | D D C - | (モチーフB)
```

**良いモチーフの条件**:
```
1. シンプル（3-7音程度）
2. 記憶に残る
3. 繰り返しやすい
4. 2-4小節に収まる
```

### 1.3 音域の選択

**最適音域**:
```
メロディ推奨音域:
  C3 (131 Hz) - C5 (523 Hz)

理由:
  - 人間の声の音域に近い
  - 楽器として最も表現力がある
  - ミキシングしやすい
```

**ジャンル別音域**:
```
Trance Lead:     C4 - C5 (高め、エピック)
House Vocal:     C3 - C4 (中音域、歌いやすい)
Techno Melody:   C3 - G3 (低め、ダーク)
Progressive:     G3 - E4 (中高音域、感動的)
```

**避けるべき音域**:
```
C2以下: ベース領域と被る
C5以上: 耳に刺さる、疲れる
```

## 2. スケールとキー

### 2.1 メジャースケール（明るい）

**構造**:
```
C Major Scale:
C - D - E - F - G - A - B - C
全  全 半  全  全  全 半

音程（度数）:
C: Root (1st)
D: 2nd
E: 3rd (Major 3rd)
F: 4th
G: 5th
A: 6th
B: 7th (Major 7th)
C: Octave (8th)
```

**特徴**:
- 明るい、楽しい、希望的
- ポップミュージック、Trance、Progressive Houseで頻繁に使用

**メロディ例**:
```
|C-E-G-E-|D-F-A-F-|C-E-G-C5-|
 明るく上昇するメロディ
```

### 2.2 マイナースケール（暗い）

**Natural Minor Scale**:
```
A Minor Scale:
A - B - C - D - E - F - G - A
全 半  全  全 半  全  全

相対キー:
  A Minor = C Majorと同じ音
```

**Harmonic Minor（よりドラマティック）**:
```
A Harmonic Minor:
A - B - C - D - E - F - G# - A
                    ↑ 半音上げる

特徴:
  - エキゾチック
  - Techno、Middle Eastern風
  - テンション高い
```

**特徴**:
- 暗い、悲しい、メランコリック
- Deep House、Techno、Tranceの感動的パートで使用

**メロディ例**:
```
|A-C-E-D-|C-E-G-F-|E-D-C-A-|
 暗く下降するメロディ
```

### 2.3 Ableton Scaleツール

**使い方**:
```
1. MIDI Clip選択
2. Clip View → Notes → Scale
3. Root Note: C
4. Scale: Major または Minor
5. Fold: On（スケール外の音を非表示）

効果:
  - ピアノロールにスケール内の音だけ表示
  - 間違った音を弾けない
  - 初心者でも安心
```

**推奨設定**:
```
初心者: Fold On（スケール外非表示）
中級者: Fold Off、In Key ハイライト
上級者: Scale Off（完全な自由）
```

## 3. モチーフの作り方

### 3.1 シンプルから始める

**2音のモチーフ**:
```
C - E (Root → 3rd)

繰り返し:
|C-E-|C-E-|C-E-|C-E-|

これだけで機能する
```

**3-4音のモチーフ**:
```
C - E - G - E (Root → 3rd → 5th → 3rd)

|C-E-G-E-|C-E-G-E-|
 安定感のあるメロディ
```

**5-7音のモチーフ**:
```
C - D - E - F - G - A - G (スケール上昇)

|C-D-E-F-|G-A-G-F-|
 動きのあるメロディ
```

### 3.2 音程の動き

**上昇（Ascending）**:
```
C → E → G → C5

感情: 希望、エネルギー増加、ビルドアップ
用途: イントロ → ドロップへの移行
```

**下降（Descending）**:
```
C5 → G → E → C

感情: 安定、リリース、解決
用途: ドロップ後の落ち着き
```

**アーチ型（Arch）**:
```
C → E → G → E → C (上昇 → 下降)

感情: 完結感、バランス
用途: フレーズの終わり
```

**ジグザグ（Zigzag）**:
```
C → E → D → F → E → G

感情: 動的、予測不可能
用途: 複雑なメロディ
```

### 3.3 リズムパターン

**均等リズム（Even）**:
```
|C-E-G-E-|C-E-G-E-|
 4分音符、均等

特徴: シンプル、安定
```

**シンコペーション（Syncopation）**:
```
|C--E-G--|--E-C---|
 オフビート

特徴: グルーヴィー、ダンサブル
```

**ドット音符（Dotted）**:
```
|C---E-|--G---|
 8分音符ドット

特徴: トライバル、催眠的
```

### 3.4 繰り返しと変化（70% + 30%）

**黄金比**:
```
繰り返し 70%: 安心感、記憶に残る
変化 30%: 飽きさせない、興味を維持

例:
  Bar 1-2: モチーフA (オリジナル)
  Bar 3-4: モチーフA (繰り返し) ← 70%
  Bar 5-6: モチーフA (繰り返し)
  Bar 7-8: モチーフA' (変化) ← 30%
```

**変化の種類**:
```
1. 音程変化:
   オリジナル: C - E - G - E
   変化:       C - E - A - G (1音変更)

2. リズム変化:
   オリジナル: |C-E-G-E-|
   変化:       |C--EG--E|

3. オクターブ変化:
   オリジナル: C4 - E4 - G4 - E4
   変化:       C5 - E5 - G5 - E5 (1オクターブ上)

4. 逆行:
   オリジナル: C → E → G → E
   変化:       E → G → E → C (逆順)
```

## 4. Ableton Live 実践

### 4.1 MIDI打ち込み基礎

**新規トラック作成**:
```
1. Cmd+Shift+T (新規MIDIトラック)
2. Wavetable または他の音源をロード
3. ダブルクリックで空のMIDI Clipを作成
```

**Scale設定**:
```
1. Clip View → Notes → Scale
2. Root: C
3. Scale: Major
4. Fold: On

→ C, D, E, F, G, A, Bだけが表示される
```

**音符配置**:
```
1. ピアノロールでC4をクリック
2. 長さ: 1/4音符
3. Cmd+D で複製
4. 次の音程（E4）に移動コピー
```

### 4.2 ベロシティ調整

**ベロシティ = 音の強さ（0-127）**

**均一ベロシティ（機械的）**:
```
|C-E-G-E-| 全て100
 ↓ 退屈、機械的
```

**バリエーション（人間的）**:
```
|C  -E -G  -E -|
 110 95 105 95

アクセント:
  - 強拍: 110-120
  - 弱拍: 80-95
  - 通常: 100
```

**調整方法**:
```
1. MIDI Note選択
2. ピアノロール下部のVelocityバー
3. ドラッグで調整
4. または直接数値入力
```

### 4.3 ヒューマナイゼーション

**タイミングのずらし**:

**完璧すぎる（機械的）**:
```
|C---|E---|G---|E---|
 グリッドに完全整列
```

**わずかにずらす（人間的）**:
```
|C-|  E-|  G|  -E-|
  ↑    ↑   ↑    ↑
 少し早い 少し遅い

→ グルーヴ、生命感
```

**設定方法**:
```
1. MIDI Note選択
2. Option+矢印キー（微調整）
3. または Velocity Randomize:
   - MIDI Effect → Velocity
   - Random: -10 to +10

→ わずかなバリエーション
```

### 4.4 Quantize（クォンタイズ）

**100% Quantize（完璧整列）**:
```
手弾き: |C--|E-|--G|E--|
Quantize後: |C-|E-|G-|E-|

→ グリッドに完璧整列
```

**部分的 Quantize（グルーヴ維持）**:
```
Quantize Settings:
  - Amount: 70%
  - Quantize To: 1/16

→ 70%だけ整列、30%は元のタイミング
→ 人間的なグルーヴを維持
```

## 5. メロディ作成テクニック

### 5.1 スケール内での自由な動き

**C Major Scale内**:
```
自由に選べる音: C, D, E, F, G, A, B

ランダムに選んでも「合う」:
  |C-F-D-G-|E-A-F-C-|

→ スケール内ならハーモニーが成立
```

**コード進行と合わせる**:
```
コード: C - F - G - C

メロディ（コードトーン中心）:
  |E-G-C-|A-C-F-|B-D-G-|E-G-C-|
   C Major  F Major  G Major  C Major
   のコードトーン
```

### 5.2 コールアンドレスポンス

**会話のようなメロディ**:
```
Call（問いかけ）:
  |C-D-E-F-|G------|
   上昇、未解決

Response（答え）:
  |G-F-E-D-|C------|
   下降、解決

→ 緊張と弛緩
```

**使用例**:
- Trance: ビルドアップ（Call）→ ドロップ（Response）
- House: Aメロ（Call）→ Bメロ（Response）

### 5.3 経過音（Passing Tone）

**スケール外の音を一瞬使う**:
```
C Major Scale: C, D, E, F, G, A, B

経過音（C#）を挿入:
  |C-C#-D-E-|
     ↑ 一瞬だけスケール外

効果:
  - テンション
  - ジャズ的
  - 複雑なハーモニー
```

**使い方**:
```
1. スケール内の2音の間に挿入
2. 短い音価（1/16音符）
3. 弱いベロシティ（80-90）

例:
  |C4 (100) - C#4 (80) - D4 (100)|
              ↑ 経過音
```

### 5.4 レイヤリング（複数のメロディ）

**主旋律（Lead）+ 対旋律（Counter Melody）**:

**Lead Melody**:
```
音域: C4 - C5 (高音域)
|C5-E5-G5-E5-|
```

**Counter Melody**:
```
音域: C3 - C4 (低音域)
|G3-C4-E4-C4-|

→ Leadを補完、ハーモニー
```

**詳細**: [counter-melody.md](./counter-melody.md)

## 6. ジャンル別メロディ

### 6.1 Trance

**特徴**: エピック、感動的、長いフレーズ

**音域**: C4 - C5 (高め)
**スケール**: Major / Harmonic Minor
**長さ**: 8-16小節

**例**:
```
C Major進行: C - Am - F - G

Melody:
Bar 1-2: |E5-D5-C5-D5-|E5-G5-C6---|
         上昇、エネルギー増加

Bar 3-4: |G5-F5-E5-D5-|C5---------|
         下降、解決

繰り返し: 70%
変化: Bar 7-8で1オクターブ上
```

**推奨音色**:
- Supersaw Lead（Wavetable）
- Unison 8 voices
- Reverb: Hall 3.0s
- Delay: 1/8 Dotted

### 6.2 Progressive House

**特徴**: 感情的、ビルドアップ重視

**音域**: G3 - E4 (中高音域)
**スケール**: Minor（メランコリック）
**長さ**: 4-8小節

**例**:
```
Am - F - C - G進行:

Melody:
Bar 1: |A4-C5-E5-C5-| (Am)
Bar 2: |F4-A4-C5-A4-| (F)
Bar 3: |G4-E4-C4-E4-| (C)
Bar 4: |G4-B4-D5-B4-| (G)

特徴:
  - コードトーンに沿う
  - ビルドアップで1オクターブ上
```

**推奨音色**:
- Pluck Synth
- ショートリリース（200 ms）
- Sidechain to Kick

### 6.3 Techno

**特徴**: ミニマル、ダーク、繰り返し

**音域**: C3 - G3 (低め)
**スケール**: Minor / Phrygian（ダーク）
**長さ**: 2-4小節

**例**:
```
A Phrygian Scale: A - Bb - C - D - E - F - G

Melody（ミニマル）:
|A3----|A3----|A3-C4-|A3----|
 シンプル、催眠的

変化:
Bar 8: |A3-D4-|A3-E4-|
       わずかな変化で大きなインパクト
```

**推奨音色**:
- Acid Lead（Operator FM）
- Filter Cutoff モジュレーション
- Minimal Reverb

### 6.4 Dubstep

**特徴**: ドロップ中心、音色変化が主役

**音域**: C3 - C4 (中音域)
**スケール**: Minor
**長さ**: 2-4小節（ドロップ前）

**例**:
```
Pre-Drop Melody:
|E4-G4-|A4-C5-|B4-D5-|C5----|
 上昇、緊張

Drop:
ベースのWobbleが主役
メロディは背景または無し
```

**推奨音色**:
- Vocal Chops
- Pad（背景）
- Post-Drop: メロディ復活

## 7. ミキシング

### 7.1 EQ処理

**メロディの基本EQ**:
```
EQ Eight:
1. High Pass 200-300 Hz
   （ベース領域を避ける）

2. Low-Mid Cut 400-800 Hz -2 dB
   （マッディネス除去）

3. Presence Boost 2-4 kHz +2 dB
   （明瞭さ）

4. Air Boost 10-12 kHz +1 dB
   （輝き）
```

**ジャンル別**:
```
Trance Lead: 4 kHz +3 dB（前に出す）
House Pluck: 2 kHz +2 dB（明瞭）
Techno Melody: 1 kHz -2 dB（ダーク）
```

### 7.2 ステレオ幅

**メロディの最適幅**:
```
Lead Melody: 60-80%
  - 中央寄り
  - ボーカルと同じ位置
  - 明瞭に聞こえる

Pad Melody: 100%
  - 広く
  - 背景として
```

**Utility設定**:
```
Utility:
  - Width: 70%

確認:
  - Mono互換性チェック
  - Correlation Meter: +0.5以上
```

### 7.3 Reverb/Delay

**空間の作り方**:

**Reverb（奥行き）**:
```
Return Track A: Reverb
  - Type: Hall
  - Decay Time: 2.0s（Trance 3.0s）
  - Pre-Delay: 20 ms
  - Dry/Wet: 100%

Lead Melody Send A: 20-30%
```

**Delay（リズミックエコー）**:
```
Return Track B: Delay
  - Time: 1/8 Dotted（3連系）
  - Feedback: 30%
  - Dry/Wet: 100%

Lead Melody Send B: 15-25%
```

**詳細**: [04-effects/reverb-delay.md](../04-effects/reverb-delay.md)

## 9. よくある失敗と対処法

### 失敗1: メロディが思いつかない

**対処法**:
```
1. Scaleツールを使う（間違った音を弾けない）
2. 2-3音から始める（C-E-G）
3. リファレンストラックをコピーして変更
4. ランダム生成 → 編集
```

---

### 失敗2: メロディが単調

**対処法**:
```
1. リズムバリエーション（シンコペーション）
2. ベロシティ 80-120
3. 繰り返しと変化 70% + 30%
4. オクターブ変化
```

---

### 失敗3: コード進行と合わない

**対処法**:
```
1. コードトーンを中心にする
   C Major: C, E, G
2. スケール内で動く
3. Ableton Scale: In Keyハイライト使用
```

---

### 失敗4: メロディが埋もれる

**対処法**:
```
1. EQ: 2-4 kHz +2 dB（明瞭さ）
2. ステレオ幅: 60-80%（中央）
3. 他の楽器: 2-4 kHz -2 dB（スペース作り）
4. コンプレッション（前に出す）
```

## まとめ

### メロディ作成の核心

1. **モチーフ**: 2-4小節の核を作る
2. **スケール**: C Major（明るい）/ A Minor（暗い）
3. **音域**: C3-C5（人間の声に近い）
4. **繰り返しと変化**: 70% + 30%
5. **ヒューマナイゼーション**: ベロシティ、タイミング
