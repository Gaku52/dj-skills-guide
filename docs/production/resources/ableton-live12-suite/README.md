## 🔍 Ableton Live内での検索方法

### ブラウザで検索

1. **ブラウザを開く**: Cmd/Ctrl + Option/Alt + B
2. **Places → User Library または Packs**
3. **検索ボックスにキーワード入力**:
   - `Bass` → ベース系
   - `Lead` → リード系
   - `Pad` → パッド系
   - `Analog` → Analogシンセ
   - `Wavetable` → Wavetableシンセ
   - `Techno` → Techno関連（少ない）

### プリセットの場所

```
~/Music/Ableton/Factory Packs/
├── Synth Essentials/
│   └── Instruments/
│       ├── Analog/
│       │   ├── Bass/
│       │   ├── Synth Lead/
│       │   └── Synth Pad/
│       ├── Wavetable/
│       ├── Operator/
│       ├── Tension/
│       └── Collision/
├── Electric Keyboards/
├── Drum Essentials/
└── ... (全31 Packs)
```

## 🎓 学習パス

### ステップ1: 基本のシンセを覚える（1-2週間）

1. **Analog**から始める
   - [シンセサイザーガイド - Analog](./synthesizers.md#3-analog)
   - 41個のプリセットで実験
   - シンプルで分かりやすい

2. **Wavetable**に進む
   - 100個のプリセット（最多）
   - モダンなサウンド
   - ビジュアル的なフィードバック

### ステップ2: Pack探索（2-4週間）

1. **Synth Essentials**を徹底的に
   - 1,674個のコンテンツ
   - 全シンセのプリセット網羅

2. **ジャンル特化Packを探索**
   - 自分のジャンルに合ったPackを選ぶ
   - [Pack詳細リスト](./packs-list.md)を参照

### ステップ3: 応用（継続）

1. **Operator**でFM合成を学ぶ
2. **エクスペリメンタルなPack**（Drift、Meld等）
3. **自分のプリセットライブラリ構築**
