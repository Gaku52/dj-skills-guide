# Resources - リソース

学習を助けるリファレンス情報と推奨リソースをまとめています。

## このセクションについて

DJ と楽曲制作の学習を効率化するための、用語集、学習ロードマップ、推奨ツールなどをまとめています。

**活用方法:**
- わからない用語を調べる
- 学習の道筋を確認する
- おすすめの曲やツールを探す
- コミュニティに参加する

---

## コンテンツ一覧

### [用語集](./glossary.md)

DJ と楽曲制作に関する用語を網羅的に解説。

**カテゴリー:**
- **一般音楽用語**: BPM、キー、スケール、コードなど
- **DJ用語**: ビートマッチング、クロスフェーダー、キューイングなど
- **制作用語**: MIDI、オーディオ、ミキシング、マスタリングなど
- **機材用語**: CDJ、コントローラー、インターフェイスなど
- **エフェクト用語**: リバーブ、ディレイ、コンプレッサーなど
- **ジャンル用語**: House、Techno、Dubstep など

**使い方:**
- アルファベット順で検索
- カテゴリー別で参照
- 関連用語へのリンク

### [DJ 学習ロードマップ](./learning-path-dj.md)

DJ スキルを段階的に習得するためのロードマップ。

**レベル別:**
- **初心者（0-3ヶ月）**: 基礎固め
- **初級（3-6ヶ月）**: 基本テクニック習得
- **中級（6-12ヶ月）**: 応用テクニック
- **上級（12ヶ月以降）**: プロフェッショナル

**各レベルで学ぶこと:**
- 必須スキル
- 練習方法
- 目標設定
- チェックリスト

### [楽曲制作 学習ロードマップ](./learning-path-production.md)

Ableton Live での楽曲制作を段階的に習得するためのロードマップ。

**レベル別:**
- **初心者（0-6ヶ月）**: DAWの基礎
- **初級（6-12ヶ月）**: 曲を完成させる
- **中級（12-24ヶ月）**: クオリティ向上
- **上級（24ヶ月以降）**: プロレベル

**各レベルで学ぶこと:**
- 必須知識
- 制作ワークフロー
- 完成曲の目標
- スキルチェック

### [推奨練習曲](./recommended-tracks.md)

DJ 練習に最適な楽曲リスト。

**カテゴリー別:**
- **初心者向け**: シンプルな構造、一定BPM
- **ビートマッチング練習**: 4つ打ち、明確なビート
- **ハーモニックミキシング**: キー別グループ
- **ジャンル別**: House、Techno、Hip Hop など
- **トランジション練習**: Intro/Outro が長い曲
- **エフェクト練習**: ブレイク、ビルドアップ

**情報:**
- 曲名、アーティスト
- BPM、キー
- 練習ポイント
- 購入リンク

### [サンプルパック](./sample-packs.md)

楽曲制作に使える推奨サンプルパック。

**カテゴリー:**
- **無料パック**: 初心者向け
- **有料パック**: プロクオリティ
- **ジャンル別**: House、Techno、Dubstep など
- **楽器別**: ドラム、ベース、シンセ
- **Ableton専用**: Live Pack

**情報:**
- パック名、提供元
- 価格
- 内容（サンプル数、種類）
- おすすめポイント

### [プラグイン・ツール一覧](./plugins-tools.md)

DJ と制作に役立つソフトウェアとツール。

**DJ ツール:**
- **DJ ソフトウェア**: Rekordbox、Serato、Traktor
- **解析ツール**: Mixed In Key、KeyFinder
- **レコーディング**: Audacity、OBS
- **ストリーミング**: Twitch、YouTube

**制作ツール:**
- **DAW**: Ableton Live、FL Studio、Logic Pro
- **VST シンセ**: Serum、Vital、Pigments
- **VST エフェクト**: FabFilter、Valhalla、iZotope
- **ユーティリティ**: SPAN、Youlean Loudness Meter
- **サンプル管理**: Loopcloud、Splice

**情報:**
- ツール名、開発元
- 価格（無料/有料）
- 機能概要
- 推奨用途

## 活用方法

### 日常的な使い方

**学習中:**
```
1. わからない用語が出たら用語集で確認
2. 次に学ぶべきことをロードマップで確認
3. 練習曲で実践
```

**制作中:**
```
1. サンプルパックで素材を探す
2. プラグインで音作り
3. インスピレーションで行き詰まり解消
```

**コミュニティ:**
```
1. 質問があれば掲示板やDiscordで
2. イベントに参加して交流
3. コースで体系的に学習
```
