# 楽曲制作 学習ロードマップ

Ableton Live での楽曲制作を段階的に習得するためのロードマップです。

---

## 初心者（0-6ヶ月）

### 目標
- DAWの基礎操作
- シンプルなループ作成

### 必須知識
1. Ableton Liveインターフェイス
2. Audio/MIDI基礎
3. ドラムプログラミング
4. ベースライン作成
5. 基本エフェクト（EQ, Reverb）

### 制作ワークフロー
- 8小節ループ作成
- Kick, Bass, Hi-Hat, Synth
- 月2-4ループ完成

### 完成曲の目標
- 8-16小節ループ
- 4トラック（Kick, Bass, Hi-Hat, Synth）
- シンプルな音色（プリセット使用OK）

---

## 初級（6-12ヶ月）

### 目標
- 曲を完成させる
- Intro → Drop → Outro

### 必須知識
1. アレンジメント基礎
2. ミキシング基礎
3. 音源（Wavetable, Analog）
4. エフェクトチェイン
5. マスタリング基礎（Limiter）

### 制作ワークフロー
- フルトラック（3-5分）
- 8トラック程度
- 月1曲完成

### 完成曲の目標
- 3-5分フルトラック
- Intro → Buildup → Drop → Outro
- -14 LUFS マスタリング

---

## 中級（12-24ヶ月）

### 目標
- クオリティ向上
- リリース可能レベル

### 必須知識
1. サウンドデザイン
2. ミキシング完全習得
3. アレンジメント応用
4. リファレンストラック分析
5. プロモーション基礎

### 制作ワークフロー
- ジャンル特化
- 月2曲完成
- SoundCloudリリース

### 完成曲の目標
- リリース可能クオリティ
- オリジナル音色
- プロとの差を縮める

---

## 上級（24ヶ月以降）

### 目標
- プロレベル
- レーベルリリース

### 必須知識
1. 完全なワークフロー
2. プロ級ミキシング・マスタリング
3. レーベルへのデモ送信
4. リミックス・コラボ
5. キャリア構築

### 制作ワークフロー
- 月3-4曲完成
- レーベルリリース
- ライブ演奏

### 完成曲の目標
- レーベルリリース
- DJにプレイされる
- ストリーミング配信
